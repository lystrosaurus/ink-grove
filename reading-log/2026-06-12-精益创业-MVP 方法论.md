# 阅读记录：精益创业

## 基本信息
- **日期**：2026-06-12
- **书籍**：精益创业
- **作者**：埃里克·莱斯

## 今日阅读概览

莱斯提出精益创业方法论：开发 - 测量 - 认知的反馈循环，MVP 最小可行产品，验证式学习，创新核算。核心是降低创业失败率，用科学方法管理创新。

## 核心框架

### 开发 - 测量 - 认知循环

```
创意 → 开发 (MVP) → 测量 (数据) → 认知 (学习) → 调整方向
  ↑                                                    │
  └────────────────────────────────────────────────────┘
```

### 关键概念

| 概念 | 说明 |
|------|------|
| MVP | 最小可行产品，快速验证假设 |
| 验证式学习 | 用实验验证商业假设 |
| 创新核算 | 衡量创新进展的指标 |
| 转型/坚持 | 根据数据调整方向 |

## 关键洞见

1. 创业是管理不确定性
2. 失败要快，学习要快
3. 数据驱动而非直觉
