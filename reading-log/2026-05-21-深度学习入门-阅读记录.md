# 2026-05-21 - 深度学习入门

日期：2026-05-21
书名：深度学习入门
作者：斋藤康毅

## 阅读进度
通读全书，重点关注感知机、神经网络基础、反向传播、CNN 和 RNN 的手动实现。

## 摘录与感悟
- 这本书最大的特点是不依赖 TensorFlow/PyTorch 等框架，而是用纯 Python + NumPy 从零实现所有算法
- 从最简单的感知机（与门、或门、异或门）开始，逐步搭建起完整的神经网络
- 反向传播一章的图解是全书最精彩的部分——用计算图的方式让复杂的链式法则变得直观
- CNN（卷积神经网络）一章详细讲解了卷积层、池化层的工作原理，以及为什么 CNN 适合图像处理
- 书中代码全部可运行，且注释详细，是理论+实践结合的最佳范例
- 难度梯度设计合理，即使数学基础一般也能跟上

## 疑问与待解
- 书中实现的是教学版本，与工业级框架（PyTorch）的实现差距在哪里
- 读完这本书后下一步应该看什么——是直接上手项目还是继续深入理论
