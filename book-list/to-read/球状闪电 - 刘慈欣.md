# 球状闪电

## 基本信息
- **书名**: 球状闪电
- **作者**: 刘慈欣
- **出版社**: 四川科学技术出版社
- **出版年份**: 2020
- **页数**: 320
- **分类**: 科学探索

## 选书理由
《三体》之前刘慈欣的长篇代表作，讲述主人公追寻球状闪电奥秘的一生。从物理学研究到武器开发，科学的好奇心与人类命运的纠缠。"我的一生都是为那一刻活着。"这部作品展现了刘慈欣早期对科学、战争、爱情的思考，是理解其创作脉络的重要文本。希望在科学探索的执着中，理解纯粹好奇心的力量。
