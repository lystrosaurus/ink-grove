# 书单管理

本目录用于管理阅读清单，按阅读状态分为三个子目录。

---

## 目录结构

```
book-list/
├── to-read/        # 待读书单 - 想读还没开始的书
├── reading/        # 在读书单 - 正在翻阅中的书
└── done/           # 已读书单 - 读完的书，可附简短评价
```

## 快速统计

```bash
# 统计各状态书籍数量
echo "待读：$(ls book-list/to-read/ | wc -l)"
echo "在读：$(ls book-list/reading/ | wc -l)"
echo "已读：$(ls book-list/done/ | wc -l)"
```

## 使用指南

### 添加新书

1. 从 `templates/book-entry.md` 复制模板
2. 在 `book-list/to-read/` 目录创建文件，命名：`书名 - 作者.md`
3. 填写书籍信息和"为什么想读"

### 开始阅读

1. 从 `to-read/` 移动文件到 `reading/`
2. 更新书单条目中的状态为"在读"
3. 填写开始日期

### 完成阅读

1. 移动文件到 `done/`
2. 更新状态为"已读"
3. 填写完成日期和"一句话评价"
4. 在 `knowledge-notes/` 创建知识笔记（可选）

## 阅读建议

### 初学者路径（从 L1 难度开始）

**第一本推荐**: 《苏菲的世界》- 西方哲学入门  
**或**: 《小狗钱钱》- 理财启蒙  
**或**: 《被讨厌的勇气》- 心理学入门

### 进阶者路径（L2-L3 难度）

**推荐**: 《思考，快与慢》- 认知科学经典  
**或**: 《理想国》- 西方哲学奠基  
**或**: 《活着》- 中国当代文学

### 高阶者路径（L3-L4 难度）

**推荐**: 《纯粹理性批判》- 康德哲学  
**或**: 《资治通鉴》- 中国史学经典  
**或**: 《存在与时间》- 海德格尔现象学
