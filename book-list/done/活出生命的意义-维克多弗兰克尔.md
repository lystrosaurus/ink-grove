# 活出生命的意义

状态：已读

作者：维克多弗兰克尔

一句话评价：心理学与文学的完美结合，弗兰克尔以奥斯维辛集中营的亲身经历为基础，提出"意义治疗"理论——人可以被剥夺一切，但无法被剥夺选择自己态度的自由。

核心收获：
- 理解了意义治疗的核心：人生的根本动力不是追求快乐（弗洛伊德）或权力（阿德勒），而是追求意义
- 弗兰克尔的洞见：痛苦本身没有意义，但我们面对痛苦的态度可以赋予它意义
- 集中营的观察揭示了极端环境下人性的两面——有人沦为野兽，有人成为圣人
