# 活出生命的意义

状态：已读

作者：维克多·弗兰克尔

一句话评价：维克多·弗兰克尔（Viktor Frankl）"心理学与文学的完美结合"——以"奥斯维辛集中营的'亲身经历'"为基础——提出"'意义治疗'理论"——"人可以被剥夺一切——但无法被剥夺'选择自己态度的自由'"。"人生的根本动力不是'追求快乐'"——"也不是'追求权力'"——是"'追求意义'"。"痛苦本身没有意义——但我们'面对痛苦的态度'可以'赋予它'意义"。

核心收获：
- **意义治疗的核心**: 弗兰克尔的"意义治疗"（logotherapy）——"人生的根本动力'不是'追求快乐（弗洛伊德）"——"也不是'追求权力（阿德勒）"——是"'追求意义'"。弗兰克尔说"人'不怕'痛苦"——"人怕的是'没有意义的'痛苦"。"如果痛苦'有意义'"——"人'就能'承受"——"甚至'从中成长'"。"如果痛苦'没有意义'"——"人就会'崩溃'"
- **集中营的经历**: 弗兰克尔在书中"前半部分"描述了"他在奥斯维辛集中营的经历"。他"失去了父母、兄弟、妻子"——"经历了'非人的'折磨"——"但他'活了下来'"。他在集中营中"观察到'一个现象'"——"活下来的人'不是'最强壮的'"——"也不是'最聪明的'"——是"'有'意义感'的人"——"那些'有'活下去的理由'的人"——"比如'想再见到'亲人的人"——"比如'想完成'某项工作的人"
- **选择态度的自由**: 弗兰克尔最著名的洞见——"人可以被剥夺一切——但无法被剥夺'选择自己态度的自由'"。"纳粹可以剥夺他的'衣服'"——"剥夺他的'名字'"——"剥夺他的'尊严'"——"剥夺他的'家人'"——"但'有一样东西'他们'剥夺不了'"——"他选择'如何面对'这一切的自由"。在"刺激和回应之间有一个空间"——"在这个空间中你有选择回应的自由"——"这个选择'定义了'你是谁"
- **痛苦的意义**: 弗兰克尔说"痛苦本身没有意义"——"但我们'面对痛苦的态度'可以'赋予它'意义"。"你得了癌症"——"癌症本身'没有意义'"——"但你'面对癌症的态度'"——"你'在癌症中学到的东西'"——"你'通过癌症'对生命的'理解'"——"这些'是有意义的'"。弗兰克尔说"不要问'生活能给我什么'"——"要问'生活'期望我'做什么'"——"在'每一个处境'中'找到'意义"
- **三种发现意义的途径**: 弗兰克尔提出了"三种发现意义的途径"：1）**创造**——"你做一件事"——"工作、艺术、养育孩子"——"通过'创造价值'来'发现意义'"；2）**体验**——"你体验一件事"——"爱一个人"——"欣赏美"——"体验自然"——"通过'体验'来'发现意义'"；3）**态度**——"你在'无法改变的苦难'中选择的态度"——"即使'什么都做不了'"——"你仍然可以'选择'以什么态度'面对'"
- **存在真空**: 弗兰克尔提出了"存在真空"（existential vacuum）的概念——"当人感觉'生活没有意义'时的状态"——表现为"无聊"、"冷漠"、"成瘾"、"从众"。弗兰克尔说"存在真空'是'现代人的'通病'"——"物质丰富了"——"但'精神空虚了'"——"人们'不知道'为什么活着"——"这就是'存在真空'"。解决办法"不是'娱乐'"——是"找到'意义'"
- **维克多·弗兰克尔的背景**: 维克多·弗兰克尔（Viktor Frankl, 1905-1997）——奥地利精神病学家——"意义治疗"创始人。他在"集中营中'失去了一切'"——但"他的'意义感'让他'活了下来'"。他"战后写了这本书"——"成为了'心理学史上最畅销的'著作之一"——"全球销量超过 1200 万册"。他还写了"《医生与灵魂》""《无意识的上帝》"——"但《活出生命的意义》是他'最著名'的作品"
