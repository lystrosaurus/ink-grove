# 全唐诗

状态：已读

作者：多人辑

一句话评价：中国诗歌的"数据库"——四万八千多首诗，装下了整个唐朝的兴衰、情感和审美。不是用来"读完"的，是用来"遇到"的。

核心收获：
- 全唐诗收录2200多位诗人的48900多首诗，是中国文学最大的单一朝代诗歌集合
- 按时间线读唐诗能看到整个唐朝的兴衰：初唐的朝气、盛唐的豪迈、中唐的沉郁、晚唐的感伤
- 诗人风格的多样性：李白飘逸、杜甫沉郁、王维空灵、白居易平实、李商隐隐晦
- 唐诗的"意境"美学：不是直白叙述，而是用画面传递情感。"大漠孤烟直，长河落日圆"十个字胜过千言万语
- 全唐诗不是用来"读完"的，是用来"翻到哪首是哪首"的。每一次偶遇都可能是一首好诗
