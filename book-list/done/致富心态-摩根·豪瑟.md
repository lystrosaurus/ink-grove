# 致富心态

状态：已读

作者：摩根·豪瑟

一句话评价：摩根·豪瑟（Morgan Housel）的"近年来最好的理财书之一'"——"'用 20 个短篇故事揭示人们与金钱的关系中的心理陷阱'"。"'理财成功不取决于你有多聪明，而取决于你的行为'"——"'知道什么时候该停下来，比知道怎么赚更多更重要'"。"'复利的力量被严重低估'"——"'巴菲特 99% 的财富是 65 岁以后获得的'"

核心收获：
- **行为比智商重要**: 豪瑟提出了"'理财成功不取决于你有多聪明，而取决于你的行为'"的核心观点'"。"'智商'和'理财'是'两回事'"——"'很多'高智商'的'人'理财'失败'"。"'为什么'"——"'因为'理财'不是'数学问题'"——"'是'行为问题'"。"'你'可以'懂'很多'理财'知识'"——"'但'你'控制'不了'情绪'"——"'你'还是'会'失败'"。"'成功的'投资者'"——"'不是'最'聪明'的'"——"'是'最'有'耐心'的'"——"'是'最'能'控制'情绪'的'"。"'如何'改善'行为'"——"'认识'自己的'心理'陷阱'"——"'制定'规则'"——"'坚持'执行'"
- **足够的概念**: 豪瑟强调了"'足够'（Enough）"的概念'"。"'知道'什么时候'该'停下来'"——"'比'知道'怎么'赚'更多'更'重要'"。"'很多'人'赚'了'很多'钱'"——"'但'还'想'赚'更多'"——"'结果'失去'了'一切'"。"'盖茨比'心态'"——"'永远'觉得'不够'"——"'永远'在'追逐'更多'"——"'永远'不'快乐'"。"'什么'是'足够'"——"'当'你'的'基本需求'满足'后'"——"'就'应该'停下来'"——"'享受'生活'"。"'贪婪'是'危险'的'"——"'当'你'拿'已经'拥有'的'东西'去'冒险'"——"'想'得到'你'不'需要'的'东西'"——"'你'可能'失去'一切'"
- **复利的力量**: 豪瑟指出"复利的力量被严重低估'"。"'巴菲特'99% 的'财富'是'65 岁'以后'获得'的'"。"'为什么'"——"'因为'复利'需要'时间'"。"'复利'的'公式'"——"'本金×(1+ 收益率)^时间'"——"'时间'是'指数'"——"'最'重要'"。"'早期'的'收益'看起来'很少'"——"'但'时间'越'长'"——"'增长'越'快'"——"'曲线'越'陡峭'"。"'如何'利用'复利'"：1）**早开始**——"'时间'越'长'越好'"；2）**持续**——"'不'中断'投资'"；3）**耐心**——"'等待'复利'发挥作用'"。"'巴菲特'的'秘诀'"——"'他'从'10 岁'开始'投资'"——"'他'活'到'90 多岁'"——"'时间'是'他'的'朋友'"
- **运气与风险**: 豪瑟"讨论了运气和风险的作用'"。"'成功'不'全是'能力'"——"'有'运气'成分'"。"'失败'不'全是'愚蠢'"——"'有'风险'成分'"。"'比尔·盖茨'的'成功'"——"'他'很'努力'"——"'但'他'也'很'幸运'"——"'他'上'了'有'电脑'的'高中'"——"'那'在'当时'很'罕见'"。"'所以'不要'盲目'崇拜'成功者'"——"'他'的'经验'可能'不'适用'你'"。"'也'不要'鄙视'失败者'"——"'他'可能'只是'运气'不好'"。"'应对'策略'"——"'尊重'运气'和'风险'"——"'做'最'坏'打算'"——"'留'安全边际'"
- **金钱的真正价值**: 豪瑟"探讨了金钱的真正价值'"。"'金钱'的'最大'价值'"——"'是'给你'控制'时间'的'能力'"。"'你'可以'选择'做'什么'"——"'你'可以'选择'不'做'什么'"——"'这是'金钱'的'真正'价值'"。"'很多'人'赚'了'钱'"——"'但'没有'时间'花'"——"'那'钱'有'什么'用'"。"'财务自由'"——"'不'是'买'豪车'——'豪宅'"——"'是'有'选择'的'自由'"。"'如何'定义'富有'"——"'不'是'有'多少'钱'"——"'是'有'多少'自由'"
- **历史与人性**: 豪瑟"分析了历史和人性的关系'"。"'历史'不'会'简单'重复'"——"'但'人性'不变'"。"'贪婪'——'恐惧'——'从众'——'过度自信'"——"'这些'人性'几百年来'没有'变化'"。"'所以'金融危机'会'重复'发生'"——"'因为'人性'不变'"。"'理解'人性'"——"'理解'自己'"——"'才能'在'投资'中'成功'"。"'不要'以为'这次'不一样'"——"'这次'总是'一样'"。"'投资'的'最大'挑战'"——"'不'是'市场'"——"'是'自己'"
- **豪瑟的背景**: 摩根·豪瑟（Morgan Housel）——美国财经作家、前专栏作家——"《致富心态》的作者'"。"他'在'《华尔街日报》'——'《动机》'等'媒体'写'专栏'"。"《致富心态》'2020 年'出版'"——"'迅速'成为'畅销书'"。"豪瑟'的'写作风格'"——"'用'故事'讲'道理'"——"'不'用'复杂'的'理论'"。"'他'的'观点'朴实'"——"'但'深刻'"——"'适合'普通人'"。"豪瑟'的'其他'作品'"——"'《金钱心理学》'——'他'的'博客'"——"'都'很'受欢迎'"
