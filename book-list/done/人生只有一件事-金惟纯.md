# 人生只有一件事

状态：已读

作者：金惟纯

一句话评价：把人生浓缩成"活好"两个字，用朴素的故事和真诚的自省告诉你：所有外在追求最终都回到如何和自己相处。

核心收获：
- "人生只有一件事，就是活好"——不是外在成就，而是内在状态
- 放下"应该"，回到"想要"。很多痛苦来自用别人的标准要求自己
-  relationships的核心是"我愿意"，不是"你应该"
- 面对生死、疾病、失去时，最能检验一个人是否"活好"
- 修行不在深山，在日常——每一次选择都是练习"活好"的机会
