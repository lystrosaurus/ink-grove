# 假面饭店

状态：已读

作者：东野圭吾

一句话评价：用豪华酒店做舞台的悬疑小说，把"服务行业的微笑"变成最大的悬疑道具——你永远不知道面具后面藏着什么。新田和山岸的搭档模式为续集《假面之夜》埋下伏笔。

核心收获：
- **酒店是天然悬疑舞台**: 连续杀人案的线索指向东京"科尔特西亚"酒店，刑警新田浩介伪装成前台卧底。酒店的特殊性在于——客人来了又走，员工知道所有人的秘密但必须保持沉默。这是悬疑小说最理想的封闭空间
- **新田与山岸的搭档**: 酒店员工山岸尚美是唯一知道新田真实身份的人。两人从互不信任到默契配合，代表了两种职业精神的碰撞——刑警追求真相，酒店人保护客人隐私。这两种价值观的冲突和融合是全书最好看的部分
- **"假面"的双重含义**: 客人的面具（隐藏犯罪意图）和酒店员工的面具（职业微笑）。服务人员必须微笑，但微笑下面是什么？东野圭吾用"假面"这个隐喻探讨了服务行业的本质——专业精神是否意味着"隐藏真实的自己"？
- **每个客人都有嫌疑**: 东野圭吾的叙事技巧是让每个客人都"看起来像凶手"——行为可疑的男人、带着秘密的女人、反复出现的陌生人。但真相往往是最意想不到的那个人
- **服务行业的尊严**: 这本书最打动人的不是悬疑，是对酒店员工的尊重。山岸尚美说："我们的工作不是'伺候人'，是创造让客人感到幸福的体验。"东野圭吾用一本悬疑小说写了一封给服务行业的情书
- **东野圭吾的节奏控制**: 悬疑感持续累积，但从来不"吓人"。他用细节堆积不安感——一个眼神、一个动作、一句看似平常的话。直到最后一章才揭开真相，读者回头一看才发现所有线索早已埋好
- **与《假面之夜》的关系**: 本书是"假面三部曲"的第一部，新田和山岸的故事在《假面之夜》中继续。两本书的叙事结构不同——《假面饭店》是酒店内多线并行，《假面之夜》是单夜审讯室中的心理战
