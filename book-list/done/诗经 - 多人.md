# 诗经

状态：已读

作者：多人（西周至春秋）

一句话评价：中国"第一部诗歌总集'"——"'诗三百'"——"'先秦社会生活的百科全书'"。"'关关雎鸠，在河之洲'开篇优美'"——"'昔我往矣，杨柳依依'意境深远'"。"'中国文学源头'"——"'现实主义传统的开端'"

核心收获：
- **风雅颂的分类**: 诗经"分为风、雅、颂三类'"。"'风'"：1）**十五国风**——"'各地'民歌'"；2）**内容**——"'爱情'——'劳动'——'生活'"；3）**特点**——"'生动'——'朴实'"。"'雅'"：1）**大雅**——"'朝廷'乐歌'"；2）**小雅**——"'贵族'宴饮'"；3）**内容**——"'政治'——'讽谏'"。"'颂'"：1）**周颂**——"'祭祀'周王'"；2）**鲁颂**——"'祭祀'鲁公'"；3）**商颂**——"'祭祀'商祖'"；4）**内容**——"'歌颂'祖先'功德'"。"'风'是'诗经'的'精华'"——"'最'有'生命力'"
- **赋比兴的手法**: 诗经"有赋、比、兴三种表现手法'"。"'赋'"：1）**直陈其事**——"'直接'描述'"；2）**例子**——"'七月流火，九月授衣'"。"'比'"：1）**比喻**——"'用'一个'东西'比喻'另一个'"；2）**例子**——"'手如柔荑，肤如凝脂'"。"'兴'"：1）**起兴**——"'先'说'别的'"——"'再'说'主题'"；2）**例子**——"'关关雎鸠，在河之洲。窈窕淑女，君子好逑'"——"'先'说'鸟'"——"'再'说'人'"。"'赋比兴'影响'了'后世'诗歌'"
- **爱情诗**: 诗经"有很多爱情诗'"。"'《关雎》**——"'关关雎鸠，在河之洲'"——"'男子'追求'女子'"。"'《蒹葭》**——"'蒹葭苍苍，白露为霜'"——"'追求'伊人'"——"'可望'不可'即'"。"'《静女》**——"'静女其姝，俟我于城隅'"——"'约会'"。"'《氓》**——"'氓之蚩蚩，抱布贸丝'"——"'弃妇'的'哀怨'"。"'这些'爱情诗'"——"'真挚'——'热烈'"——"'不'像'后世'那么'含蓄'"
- **劳动歌谣**: 诗经"记录了大量劳动场景'"。"'《七月》**——"'七月流火，九月授衣'"——"'农事'活动'"——"'按月'描述'农活'"。"'《芣苢》**——"'采采芣苢，薄言采之'"——"'采集'劳动'"。"'《伐檀》**——"'坎坎伐檀兮，置之河之干兮'"——"'伐木'劳动'"。"'这些'歌谣'"——"'反映'了'先民'的'劳动'生活'"——"'是'珍贵'的'历史'资料'"
- **战争徭役**: 诗经"也反映了战争和徭役'"。"'《采薇》**——"'昔我往矣，杨柳依依。今我来思，雨雪霏霏'"——"'征人'的'哀愁'"。"'《击鼓》**——"'击鼓其镗，踊跃用兵'"——"'战争'的'残酷'"。"'《东山》**——"'我徂东山，慆慆不归'"——"'征夫'思乡'"。"'这些'诗'"——"'不'歌颂'战争'"——"'表达'的'是'对'战争'的'厌倦'"——"'对'和平'的'渴望'"
- **祭祀颂歌**: 诗经"有祭祀颂歌'"。"'《生民》**——"'讲述'周族'始祖'后稷'的'传说'"。"'《大明》**——"'歌颂'周文王'——'周武王'的'功德'"。"'这些'颂歌'"——"'用于'祭祀'仪式'"——"'有'宗教'功能'"。"'但'文学'价值'不'高'"——"'太'正式'"——"'太'庄重'"
- **诗经的成书**: 诗经"是如何成书的'"。"'诗经'305 篇'"——"'创作'时间'跨度'500 多年'"——"'西周'初期'到'春秋'中期'"。"'谁'收集'的'"——"'相传'孔子'删定'"——"'但'学者'有'争议'"。"'诗经'如何'流传'"——"'口头'传唱'"——"'后来'被'记录'"。"'诗经'的'地位'"——"'儒家'五经'之一'"——"'中国'诗歌'源头'"——"'影响'了'两千'多年'"
