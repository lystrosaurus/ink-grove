# 设计心理学

状态：已读

作者：唐纳德诺曼

一句话评价：设计领域的开山之作，提出"以用户为中心"的设计理念，揭示了好设计与坏设计的区别在于是否符合人类的认知习惯。

核心收获：
- 理解了"示能"（Affordance）概念：物品的外观应该暗示它的用法——门把手应该让人知道是推还是拉
- 好设计不是好看的设计，而是好用的设计——美观但难用的产品是失败的
- 用户的"错误"往往是设计的失败——不要怪用户笨，要怪设计没有考虑到人的认知局限
