# 心智社会

状态：已读

作者：马文明斯基

一句话评价：人工智能先驱马文·明斯基的心理学理论巨著，提出"心智是由无数简单代理组成的社会"这一革命性观点。

核心收获：
- 理解了明斯基的"心智社会"理论——心智不是单一的实体而是由无数简单代理（agent）组成的复杂系统
- 情感、记忆、意识都可以用"代理"的交互来解释——这是一种还原论但又富有洞察力的视角
- 明斯基作为 AI 先驱的独特视角——他试图用计算机科学的思维来理解人类心智
