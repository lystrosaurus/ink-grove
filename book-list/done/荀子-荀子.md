# 荀子

状态：已读

作者：荀子

一句话评价：荀子的"先秦儒家集大成之作'"——"'提出'性恶论'和'化性起伪'的教育观'"——"'与孟子的'性善论'形成鲜明对比'"。"'天行有常，不为尧存，不为桀亡'"——"'自然界有自己的规律'"——"'是中国古代最重要的哲学著作之一'"

核心收获：
- **性恶论**: 荀子提出了"'性恶论'"的核心观点'"。"'人性本恶'"——"'人生来'就'有'欲望'"——"'好利'——'好色'——'好'安逸'"。"'如果'顺从'本性'"——"'就'会'争夺'"——"'会'混乱'"。"'这'与'孟子'的'性善论'相反'"——"'孟子'说'人'生来'有'善端'"——"'荀子'说'人'生来'有'恶端'"。"'但'荀子'不'悲观'"——"'他'说'人性'可以'改造'"——"'通过'教育'和'礼义'"。"'所以'性恶论'不'是'说'人'没救'了'"——"'是'说'需要'教育'"
- **化性起伪**: 荀子提出了"'化性起伪'"的教育观'"。"'化性'是'改变'本性'"——"'起伪'是'人为'的'努力'"。"'伪'不'是'虚伪'"——"'是'人为'"——"'与'自然'相对'"。"'如何'化性起伪'"：1）**学习**——"'学习'礼义'法度'"；2）**师法**——"'向'老师'学习'"；3）**环境**——"'在'好'的'环境'中'熏陶'"。"'荀子'特别'重视'教育'"——"'他'说'人之'性'恶'其'善'者'伪'也'"——"'人'的'善'是'后天'努力'的'结果'"
- **礼的作用**: 荀子"强调了礼的重要性'"。"'礼'是'什么'"——"'礼'是'社会'规范'"——"'是'道德'准则'"——"'是'行为'标准'"。"'礼'的'作用'"：1）**节制欲望**——"'让'人'的'欲望'有'节制'"；2）**区分等级**——"'让'社会'有'秩序'"；3）**教化人民**——"'让'人'变'善'"。"'荀子'说'礼'是'治国'的'根本'"——"'没有'礼'"——"'社会'就'会'混乱'"。"'礼'不'是'束缚'"——"'礼'是'保护'"——"'保护'社会'的'和谐'"
- **天行有常**: 荀子"提出了自然观'"。"'天行有常，不为尧存，不为桀亡'"——"'自然界'有'自己'的'规律'"——"'不'以'人'的'意志'为'转移'"。"'尧'是'圣君'"——"'桀'是'暴君'"——"'但'自然'规律'不'因'他们'而'改变'"。"'这种'观点'是'唯物'的'"——"'反对'迷信'"——"'反对'天命'"。"'荀子'说'制天命而用之'"——"'人'应该'认识'自然'规律'"——"'利用'自然'规律'"——"'不'是'顺从'天命'"
- **认识论**: 荀子"探讨了人如何认识世界'"。"'人'通过'感官'认识'世界'"——"'耳'——'目'——'口'——'鼻'——'身'"。"'但'感官'可能'出错'"——"'需要'心'来'判断'"。"'心'是'思维'器官'"——"'心'能'思考'"——"'能'判断'是非'"。"'如何'获得'真知'"：1）**虚**——"'不' preconceived'"；2）**一**——"'专注'"；3）**静**——"'平静'"。"'这种'认识论'强调'理性'"——"'强调'学习'"
- **荀子与孟子**: 荀子"与孟子的对比'"。"'相同点'"：1）**都是儒家**——"'都'重视'道德'"——"'都'主张'仁义'"；2）**都重视教育**——"'都'认为'教育'能'改变'人'"。"'不同点'"：1）**人性论**——"'孟子'性善'"——"'荀子'性恶'"；2）**方法**——"'孟子'主张'扩充'善端'"——"'荀子'主张'改造'本性'"；3）**影响**——"'孟子'被'尊'为'亚圣'"——"'荀子'长期'被'忽视'"。"'但'荀子'的'影响'很'大'"——"'他的'学生'韩非'——'李斯'"——"'是'法家'代表'"
- **荀子的背景**: 荀子（约公元前 313-前 238）——战国末期思想家、教育家——"儒家代表人物'"。"他'是'赵国人'"——"'曾'在'齐国'稷下学宫'讲学'"——"'三'为'祭酒'"（学宫'首领）。"他'的'学生'包括'韩非'——'李斯'"——"'后来'成为'法家'代表'"。"《荀子》'32 篇'"——"'大部分'是'荀子'自己'所写'"——"'这'在'先秦'诸子'中'很'罕见'"。"荀子'的'思想'"——"'对'汉代'儒学'影响'很大'"——"'但'宋代'后'被'边缘化'"——"'因为'他'的'性恶论'不'符合'理学'"
