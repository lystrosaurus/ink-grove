# 高效能人士的第八个习惯

状态：已读

作者：史蒂芬柯维

一句话评价：《高效能人士的七个习惯》的续作，在信息时代和知识经济背景下提出"第八个习惯"——找到你自己的声音，并启发他人找到他们的声音。

核心收获：
- 理解了"八个习惯"的框架：前七个习惯是关于个人效能，第八个习惯是关于找到"声音"（独特的个人天赋和热情）
- " Voice = 天赋 + 热情 + 良知 + 意义的交集"——这个公式帮助人们找到自己的人生方向
- 领导力不再是"管理"，而是"释放他人的潜能"
