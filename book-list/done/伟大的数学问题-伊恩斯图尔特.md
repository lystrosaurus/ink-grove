# 伟大的数学问题

状态：已读

作者：伊恩-斯图尔特

一句话评价：把数学史上最美的未解之谜和已解之谜用讲故事的方式呈现，让不懂数学的人也能感受到"数学之美"。

核心收获：
- 费马大定理：350年的智力马拉松，从费马的"空白太小写不下"到怀尔斯的证明，是人类毅力的极限考验
- 四色定理：任何地图只需要四种颜色就能让相邻区域颜色不同。证明靠的是计算机穷举——数学家第一次承认"我证不出来，但机器能"
- 庞加莱猜想：三维空间中任何封闭的曲面都等价于球面。佩雷尔曼证明了它，然后拒绝了菲尔兹奖和百万美元奖金
- 哥德尔不完备定理：任何足够复杂的数学系统里，都存在"既不能证明也不能证伪"的命题。数学不是万能的
- 数学问题的价值不在于"有没有用"，而在于"它是否揭示了深层的结构"
