# 周期

状态：已读

作者：霍华德·马克斯

一句话评价：霍华德·马克斯的投资哲学力作——用"周期"的视角理解市场波动：经济有周期、信贷有周期、情绪有周期、房地产有周期。投资者无法预测周期何时转折，但可以判断"我们现在在周期的什么位置"，并据此调整仓位。第二层次思维和逆向投资是穿越周期的核心能力。

核心收获：
- **周期的必然性**: 霍华德·马克斯的核心观点——"周期是不可避免的"。市场、经济、信贷、情绪、房地产——一切都有周期性波动。原因很简单：人是有情绪的，而情绪会导致"过度反应"。好的时候人们"过度乐观"（推高价格），坏的时候人们"过度悲观"（压低价格）。这种"钟摆效应"是周期的根源。马克斯说"树不会长到天上去"——任何"永远涨"的信念都是错的
- **判断位置 vs 预测时机**: 马克斯反复强调"不要试图预测市场何时转折"（timing the market），而是"判断我们现在在周期的什么位置"（positioning）。预测时机几乎不可能做到，但判断位置是可行的——通过观察估值水平、信贷宽松度、投资者情绪、市场参与度等指标，你可以知道"现在是便宜还是贵""是谨慎还是贪婪"
- **第二层次思维**: 马克斯最著名的概念。第一层次思维说"这是一家好公司，所以股票值得买"。第二层次思维说"这是一家好公司，但人人都知道它是好公司，所以股票已经太贵了，不值得买"。第二层次思维的核心是"逆向思考"——当大多数人一致看多时，你要警惕；当大多数人一致看空时，你要兴奋。"超额收益来自与共识不同的正确判断"
- **钟摆理论**: 马克斯用"钟摆"比喻投资者心理——它几乎不会停在中间（"合理估值"），而是在"极度乐观"和"极度悲观"之间摆动。2000 年互联网泡沫时钟摆摆到了"极度乐观"（任何带".com"的公司都被炒作），2008 年金融危机时钟摆摆到了"极度悲观"（所有人都认为金融系统要崩溃）。聪明的投资者在钟摆到达极端时"逆向操作"——别人贪婪时恐惧，别人恐惧时贪婪
- **信贷周期**: 马克斯认为信贷周期是所有周期中最重要的。信贷宽松→企业借钱扩张→经济繁荣→资产价格上涨→更多借贷→泡沫→信贷收紧→企业还不起钱→经济衰退→资产价格下跌→信贷进一步收紧。2008 年金融危机就是信贷周期的典型案例——次贷、CDO、CDS 等金融衍生品放大了信贷周期的波动
- **风险与周期**: 马克斯对"风险"有独特的理解——风险不是"波动性"，是"永久损失的可能性"。在周期的顶部（市场过热时），风险最高（因为价格远超价值），但人们觉得风险最低（因为大家都在赚钱）。在周期的底部（市场恐慌时），风险最低（因为价格远低于价值），但人们觉得风险最高（因为大家都在亏钱）。这种"风险感知与真实风险的错位"是逆向投资者的机会
- **橡树资本备忘录**: 马克斯是橡树资本（Oaktree Capital）的联合创始人，他定期写给客户的"投资备忘录"是投资界最珍贵的思想财富。巴菲特说"我第一时间打开并阅读的就是马克斯的备忘录"。《周期》这本书很大程度上是这些备忘录的精华汇编
- **周期的实践应用**: 马克斯给投资者的建议：1）不要试图"逃顶抄底"——没有人能做到；2）但你要在"便宜时多买、贵时少买"——这是可以做到的；3）保持"投资组合的灵活性"——手里留有足够的现金，以便在底部时"有钱可买"；4）"不要加杠杆"——杠杆会在底部时"杀死你"，即使你的判断是对的
- **霍华德·马克斯的背景**: 马克斯（1946-）是价值投资派的代表人物，芝加哥大学经济学硕士，宾夕法尼亚大学沃顿商学院 MBA。1995 年联合创立橡树资本，专注于不良债务投资（在危机中买便宜的债券）。他的投资哲学深受格雷厄姆、巴菲特影响，但加入了"周期"的视角——这是对价值投资理论的重要补充
