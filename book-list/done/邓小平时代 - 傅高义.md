# 邓小平时代

状态：已读

作者：傅高义

一句话评价：傅高义（Ezra Vogel）的"邓小平研究权威著作'"——"'全景展现中国改革开放的历史进程'"。"'发展才是硬道理影响中国走向'"——"'理解当代中国崛起的必读书'"

核心收获：
- **改革开放的启动**: 本书"讲述了改革开放的启动'"。"'背景'"："'文革'结束'"——"'经济'濒临'崩溃'"。"'邓小平'如何'启动'改革'"：1）**解放思想**——"'实践是检验真理的唯一标准'"；2）**工作重心转移**——"'从'阶级斗争'转向'经济建设'"；3）**对外开放**——"'设立'经济特区'"。"'1978 年'十一届三中全会'"——"'改革开放'正式'开始'"
- **经济特区**: 本书"详细介绍了经济特区'"。"'什么'是'经济特区'"——"'特殊'政策'的'地区'"。"'第一批'特区'"："'深圳'"——"'珠海'"——"'汕头'"——"'厦门'"。"'特区'的'政策'"：1）**外资**——"'吸引'外资'"；2）**市场**——"'市场'调节'"；3）**出口**——"'出口'导向'"。"'特区'的'成功'"："'深圳'从'小渔村'变成'大都市'"。"'特区'的'意义'"："'试验田'"——"'成功'后'推广'全国'"
- **一国两制**: 本书"讲述了一国两制的提出'"。"'什么'是'一国两制'"——"'一个'中国'"——"'两种'制度'"。"'针对'什么'"："'香港'"——"'澳门'"——"'台湾'"。"'如何'提出'"："'1980 年代'"——"'邓小平'提出'"。"'香港'回归'"："'1997 年'"——"'一国两制'实施'"。"'一国两制'的'意义'"："'创新'"——"'和平'统一'"
- **南巡讲话**: 本书"重点讲述了南巡讲话'"。"'背景'"："'1990 年代初'"——"'改革'遇到'阻力'"——"'有人'质疑'改革'"。"'邓小平'南巡'"："'1992 年'"——"'视察'南方'"。"'讲话'内容'"：1）**发展才是硬道理**——"'经济'第一'"；2）**胆子要大**——"'敢'试'敢'闯'"；3）**市场经济**——"'市场'不'等于'资本主义'"。"'南巡'的'影响'"："'推动'改革'深入'"——"'经济'快速'发展'"
- **解放思想实事求是**: 本书"强调了解放思想实事求是'"。"'解放思想'"："'不'拘泥'教条'"——"'从'实际'出发'"。"'实事求是'"："'实践'是'检验'真理'的'唯一'标准'"。"'这'是'邓小平'的'思想'方法'"。"'例子'"："'猫论'"——"'不管黑猫白猫，捉到老鼠就是好猫'"——"'实用主义'"
- **中国崛起**: 本书"描述了中国崛起的过程'"。"'从'1978 年'到'2012 年'"——"'中国'经济'增长'"。"'成就'"：1）**GDP**——"'世界'第二'"；2）**脱贫**——"'几亿人'脱贫'"；3）**国际地位**——"'大国'地位'"。"'原因'"："'改革开放'"——"'邓小平'的'领导'"。"'邓小平'的'遗产'"："'中国'的'发展'道路'"
- **作者的背景**: 傅高义（Ezra Vogel, 1930-2020）——美国汉学家、哈佛大学教授——"《邓小平时代》的作者'"。"他'研究'中国'和'日本'"——"'是'著名'的'东亚'专家'"。"《邓小平时代》'2011 年'出版'"——"'英文版'"——"'2013 年'中文版'"。"'他'花'了'10 年'时间'写'这本书'"——"'采访'了'很多'人'"——"'查阅'了'大量'资料'"。"'这本书'是'西方'研究'邓小平'的'权威'著作'"
