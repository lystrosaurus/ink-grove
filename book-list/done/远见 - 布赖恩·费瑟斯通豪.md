# 远见

状态：已读

作者：布赖恩·费瑟斯通豪

一句话评价：布赖恩·费瑟斯通豪（Brian Fetherstonhaugh）的"职业规划经典'"——"'提出用 45 年职业生涯的长远视角来规划当下选择'"。"'将职业分为三个阶段，每个阶段有不同的目标和策略'"。"'职场燃料：可迁移技能、有意义的经验、持久的关系——比薪水更重要'"

核心收获：
- **45 年职业生涯**: 本书提出"'用 45 年视角规划职业'"。"'什么'意思'"——"'不'要'只看'下一份'工作'"——"'要'看'整个'职业生涯'"。"'45 年'从'哪'来'"——"'25 岁'工作'"——"'70 岁'退休'"——"'45 年'"。"'长远'视角'的'好处'"：1）**减少焦虑**——"'不'纠结'短期'得失'"；2）**做出好决策**——"'考虑'长远'影响'"；3）**持续成长**——"'每个'阶段'有'目标'"
- **三阶段模型**: 本书"将职业生涯分为三个阶段'"。"'第一阶段：加添燃料（25-40 岁）**：1）**目标**——"'学习'——'成长'"；2）**策略**——"'尝试'不同'机会'"——"'积累'技能'"；3）**重点**——"'燃料'不'是'薪水'"。"'第二阶段：瞄准甜蜜点（40-55 岁）**：1）**目标**——"'找到'擅长'——'喜欢'——'有'价值'的'事'"；2）**策略**——"'专注'"——"'深耕'"；3）**重点**——"'发挥'优势'"。"'第三阶段：优化长板（55-70 岁）**：1）**目标**——"'传承'——'指导'年轻人'"；2）**策略**——"'优化'经验'"；3）**重点**——"'留下'遗产'"
- **职场燃料**: 本书"提出了职场燃料的概念'"。"'什么'是'职场'燃料'"——"'让'你'职业'发展'的'东西'"。"'三种'燃料'"：1）**可迁移技能**——"'沟通'——'领导'——'解决问题'"——"'任何'工作'都'有用'"；2）**有意义的经验**——"'有'挑战'的'项目'"——"'困难'的'任务'"——"'让'你'成长'"；3）**持久的关系**——"'导师'——'同事'——'客户'"——"'长期'关系'"。"'这'三样'比'薪水'更'重要'"
- **职业决策**: 本书"提供了职业决策的方法'"。"'如何'做'职业'决策'"：1）**长远视角**——"'45 年'视角'"；2）**燃料优先**——"'哪个'机会'给'你'更'多'燃料'"；3）**甜蜜点**——"'擅长'——'喜欢'——'有'价值'"。"'例子'"："'两个'offer'"——"'一个'薪水'高'"——"'一个'能'学习'"。"'长远'看'"——"'选'能'学习'的'"——"'因为'燃料'更'重要'"
- **第一阶段策略**: 本书"详细讲了第一阶段的策略'"。"'第一阶段'是'什么'"——"'25-40 岁'"。"'目标'"：1）**学习**——"'尽可能'多'学习'"；2）**试错**——"'尝试'不同'机会'"；3）**积累燃料**——"'技能'——'经验'——'关系'"。"'建议'"：1）**不要怕跳槽**——"'换'工作'学习'新'东西'"；2）**找好导师**——"'导师'指导'你'"；3）**接受挑战**——"'困难'任务'让'你'成长'"
- **工作与生活的平衡**: 本书"讨论了工作与生活的平衡'"。"'45 年'职业生涯'"——"'不'是'只有'工作'"。"'生活'也'重要'"：1）**家庭**——"'伴侣'——'孩子'"；2）**健康**——"'身体'——'心理'"；3）**兴趣**——"'爱好'"。"'如何'平衡'"：1）**阶段调整**——"'不同'阶段'侧重'不同'"；2）**整合**——"'工作'和'生活'不'对立'"；3）**优先级**——"'知道'什么'最重要'"
- **作者的背景**: 布赖恩·费瑟斯通豪（Brian Fetherstonhaugh）——奥美互动全球 CEO、职业规划专家——"《远见》的作者'"。"他'有'30 多年'职业'规划'经验'"——"'帮助'过'很多'人'"。"'《远见》'2016 年'出版'"——"'畅销'全球'"。"'他'的'方法'"——"'基于'真实'案例'"——"'实用'"。"'他'还'在'领英'写'文章'"——"'分享'职业'规划'建议'"
