# 老残游记

状态：已读

作者：刘鹗

一句话评价：刘鹗的"晚清四大谴责小说之一'"——"'以江湖郎中老残的游历为主线，揭露官场腐败和社会黑暗'"——"'是中国近代小说的里程碑'"。"'谴责小说'的特点'"——"'用游记的形式揭露社会弊端'"——"'笔法辛辣但不失幽默'"。"'老残作为一个旁观者视角'"——"'看到了官场的腐败、百姓的苦难、知识分子的无奈'"

核心收获：
- **谴责小说的特点**: 《老残游记》是"晚清四大谴责小说之一'"。"'谴责小说'的'特点'"：1）**游记形式**——"'以'游历'为'线索'"——"'串联'各种'见闻'"；2）**揭露弊端**——"'官场'——'社会'的'黑暗'"；3）**笔法辛辣**——"'讽刺'——'批判'"——"'但'不失'幽默'"；4）**旁观者视角**——"'主人公'是'局外人'"——"'可以'客观'地'观察'"。"'其他'谴责小说'"——"'《官场现形记》'——'《二十年目睹之怪现状》'——'《孽海花》'"
- **老残的旁观者视角**: 老残是"一个江湖郎中'"——"'他'不是'官员'"——"'不是'士大夫'"——"'他'是'局外人'"。"'这个'视角'让'他'可以'客观'地'观察'"——"'他'看到'了'官员'的'腐败'"——"'他'看到'了'百姓'的'苦难'"——"'他'看到'了'知识分子'的'无奈'"。"'老残'不'参与'政治'"——"'他'只是'游历'"——"'他'记录'他'的'见闻'"——"'这种'旁观者'的'姿态'"——"'让'小说'更'客观'"
- **官场腐败**: 刘鹗"揭露了官场的腐败'"。"'清官'比'贪官'更'可怕'"——"'因为'他们'自以为是'"——"'他们'不顾'百姓'死活'"——"'他们'为了'政绩'"——"'不惜'牺牲'百姓'"。"'书中'的'玉贤'"——"'一个'所谓'清官'"——"'他'为了'抓'强盗'"——"'滥杀'无辜'"——"'他'的'辖区'"——"'人人自危'"。"'刘鹗'说'清官'的'危害'比'贪官'更'大'"——"'因为'他们'有'道德'的'外衣'"
- **百姓苦难**: 刘鹗"描写了百姓的苦难'"。"'黄河水患'"——"'百姓'流离失所'"——"'官员'不'作为'"。"'苛捐杂税'"——"'百姓'负担'沉重'"——"'活不下去'"。"'司法不公'"——"'百姓'被'冤枉'"——"'无处'申冤'"。"'刘鹗'自己'是'水利专家'"——"'他'亲眼目睹'了'黄河水患'"——"'他'对'百姓'的'苦难'有'切身'体会'"
- **知识分子无奈**: 刘鹗"也写了知识分子的无奈'"。"'知识分子'有'理想'"——"'但'无法'实现'"——"'官场'腐败'"——"'他们'同流合污'"——"'否则'被'排挤'"。"'老残'自己'就是'知识分子'"——"'他'选择'做'江湖郎中'"——"'他'不'做官'"——"'他'保持'独立'"——"'但'他'也'无力'改变'什么'"。"'这种'无奈'是'晚清'知识分子'的'共同'心境'"
- **刘鹗的水利专家身份**: 刘鹗"不仅是作家'"——"'也是'水利专家'"。"'他'参与'过'黄河治理'"——"'他'有'切身'体会'"。"'书中'对'黄河水患'的'描写'"——"'基于'他的'亲身经历'"——"'真实'——'生动'"。"'刘鹗'还'是'甲骨文'的'早期'收藏者'"——"'他'有'学问'"——"'但'他'选择'写'小说'"——"'用'小说'来'揭露'社会'问题'"
- **刘鹗的背景**: 刘鹗（1857-1909）——清代小说家、水利专家、甲骨文收藏家——"《老残游记》的作者'"。"他'出身'官宦世家'"——"'他'学过'数学'——'医学'——'水利'"。"他'参与'过'黄河治理'"——"'他'办过'实业'"——"'但'都'失败'了'"。"《老残游记》'1903 年'开始'连载'"——"'轰动'一时'"。"刘鹗'晚年'被'流放'"——"'1909 年'病逝'"——"'他'是'一个'悲剧性'人物'"
