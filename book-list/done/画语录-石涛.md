# 画语录

状态：已读

作者：石涛

一句话评价：清代画僧石涛的"绘画理论经典"——"提出'一画论'和'搜尽奇峰打草稿'等'核心观点'"——"是'中国绘画史上'最重要的'画论之一'"。"'一画论'"——"'万画归一'"——"'一切绘画'都'源于最初的'一画'"——"这是'中国绘画的'哲学根基'"。"'搜尽奇峰打草稿'"——"'画家需要'广泛'观察自然'"——"'但创作时'不能'照搬自然'"——"'要融入'自己的感悟'"。"'无法而法'乃为至法"——"'最高的技法'是'没有固定的技法'"。

核心收获：
- **一画论**: 石涛最核心的观点——"一画论"。"一画论'说'"——"'万画归一'"——"'一切绘画'都'源于最初的'一画'"。"这'一画'"——"'不是'具体的'一笔一画'"——"'是'绘画的'本源'"——"'是'创作者'内心的'那个'最初的冲动'"。"石涛说'一画'是'众有之本'"——"'万象之根'"——"'所有的画'"——"'都'从'一画'开始'"——"'也'都'归于'一画'"。"一画论'把'中国绘画'从'技术层面'提升到了'哲学层面'"
- **搜尽奇峰打草稿**: 石涛提出了"搜尽奇峰打草稿"的观点——"'画家需要'广泛地'观察自然'"——"'看遍'天下的'名山奇峰'"——"'但'创作时'"——"'不能'照搬自然'"——"'要融入'自己的感悟'"。"石涛说'山川'与'我'相遇'"——"'山川'变成了'我心中的'山川'"——"'我'变成了'山川的'我'"——"'这种'交融'就是'创作的'源泉'"。"'搜尽奇峰'是'输入'"——"'打草稿'是'输出'"——"中间'经过了'创作者的'心灵过滤'"
- **无法而法**: 石涛说"'无法而法'乃为至法"——"'最高的技法'是'没有固定的技法'"。"他'反对'当时的'画坛'一味'模仿古人'"——"'认为'每一个'创作者'都'应该'有自己的'风格'"——"'不应该'被'古人的'技法'束缚'"。"石涛说'古人之须眉'不能'长在我之面目上'"——"'古人之肺腑'不能'安入我之腹肠'"——"'我自发我之肺腑'"——"'我自揭我之面目'"。"'无法而法'不是'没有技法'"——"'是不被'技法'束缚'"
- **笔墨当随时代**: 石涛提出了"笔墨当随时代"的观点——"'绘画'应该'反映'当下的'时代精神'"——"'不应该'一味'模仿古人'"。"他说'时代'在'变'"——"'人的'感受'在'变'"——"'绘画'也'应该'变'"。"'如果'一直'模仿古人'"——"'绘画'就会'失去生命力'"——"'变成'一具'空壳'"。"石涛的这个观点'在当时'非常'前卫'"——"'对后世的'中国画'产生了'深远影响'"
- **我自用我法**: 石涛强调"我自用我法"——"'每一个创作者'都'应该'有'自己的'方法'"——"'不应该'被'别人的'方法'束缚'"。"他说'法'是'死的'"——"'我'是'活的'"——"'以'死法'来'约束'活人'"——"'这是'荒谬的'"。"石涛的'艺术观'是'极其'个人主义的'"——"'他'相信'每一个'创作者'都'有'自己独特的'感受'和'表达方式'"——"'关键'是'找到'属于自己的'那一种'"
- **石涛的生平与艺术**: 石涛（1642-约 1707）——清代画家——"原姓朱"——"是'明宗室'后裔"——"明亡后'出家为僧'"——"法名原济"——"字石涛"——"号苦瓜和尚"。"他'一生'漂泊"——"'从广西'到'湖北'"——"'到'南京'"——"'到'扬州'"——"'游历了'大半个中国'"。"他'擅长'山水画'"——"'他的画风'独特'"——"'打破了'当时'摹古的风气'"——"'开创了'一代'新风'"
- **画语录的影响**: 《画语录》是"中国绘画史上'最重要的'画论之一"——"'对后世的'中国画'产生了'深远影响'"。"他的'一画论'"——"'成为了'中国画'的'哲学根基'"——"他的'搜尽奇峰打草稿'"——"'成为了'中国画家'的'座右铭'"——"他的'无法而法'"——"'成为了'中国艺术'的'最高境界'"。"'近现代'的'中国画大师'"——"'如'齐白石'——'黄宾虹'——'傅抱石'"——"'都'深受'石涛的'影响'"
