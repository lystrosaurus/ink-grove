# 廿二史札记

状态：已读

作者：赵翼

一句话评价：清代赵翼（1727-1814）的史学评论集——对从《史记》到《明史》的二十二部正史进行系统梳理和评论，是"中国史学批评史"上的经典之作。赵翼不是"简单地罗列史实"——他分析每部史书的"体例、得失和史家的立场"，找出历史中的"规律和模式"。比如他指出"汉初布衣将相之局"（汉朝开国功臣都是平民出身）、"五代史》中'君臣易位'的频繁"——这种"跨朝代比较"的方法在中国史学史上是开创性的。文笔流畅易读——比一般史学评论更有趣味性。

核心收获：
- **赵翼的史学评论方法**: 赵翼不同于传统史学家（只关注"考证史实"）——他关注"史书本身"。他问的问题不是"这件事是不是真的"——是"史家为什么这样写""这种写法有什么得失""不同史家对同一件事的记载有什么不同"。这种"史学批评"的方法在中国史学史上是"开创性的"——赵翼不是"写历史"——是"写'写历史的人'"
- **跨朝代比较**: 赵翼最有特色的方法是"跨朝代比较"——他不是一部一部地读史书——是把"二十二部正史放在一起比较"。比如他比较了"各朝开国功臣的出身"——汉朝开国功臣多是"布衣"（刘邦、萧何、韩信都是平民），唐朝开国功臣多是"贵族"（李世民家族本身就是贵族），明朝开国功臣多是"底层"（朱元璋是和尚出身）。这种比较揭示了"不同时代的政治生态"
- **汉初布衣将相之局**: 赵翼在书中指出汉朝开国的一个独特现象——"布衣将相"。刘邦是"亭长"（相当于乡长），萧何是"县吏"，韩信是"平民"，张良是"没落贵族"——这些人在秦朝都是"小人物"，但在汉朝开国后成为了"将相"。赵翼说这是"中国历史上第一次平民掌权"——此前的朝代（夏商周）都是"贵族统治"，汉朝开启了"布衣可以封侯"的时代
- **魏晋禅代的模式**: 赵翼分析了从曹魏到西晋到东晋到南北朝的"禅代"模式——每一个朝代的更替都遵循同样的流程：权臣掌权→架空皇帝→封王→加九锡→禅让→新朝建立。赵翼说这种"禅代"不是"真正的让位"——是"披着合法外衣的篡位"。他讽刺说"禅让"两个字被"篡位者"用烂了
- **宋初制禄之厚**: 赵翼指出宋朝的一个特点——"官员俸禄极厚"。宋朝官员的工资是中国历史上最高的——一个宰相的年收入相当于几千石粮食。赵翼分析原因是宋太祖赵匡胤"以文治国"——为了防止武将篡权（他自己就是武将篡权上位的），他"重文轻武"，用"高薪"来笼络文官。但这个政策也带来了副作用——宋朝财政"冗费"严重，成为后来国力衰弱的原因之一
- **明代宦官之祸**: 赵翼对明朝宦官专权的分析极其深刻——从王振（正统朝）到刘瑾（正德朝）到魏忠贤（天启朝），宦官权力一步步膨胀。赵翼说宦官专权的根源是"皇帝的懒惰"——皇帝不想管事，把权力交给了"身边的人"（宦官）。宦官没有后代、没有家族——他们对皇帝的"忠诚度"看起来更高，但他们的"权力欲望"也更不受约束
- **赵翼的文笔**: 《廿二史札记》的文体是"札记"——不是"学术论文"，是"随笔式的评论"。赵翼的文笔"流畅易读"——他用通俗的语言讲深刻的道理，偶尔还带点幽默。比如他评论某个史家"过于轻信"时说"此乃采摭之不审也"（这是采集材料不够谨慎）。这种"不端着"的写法让《廿二史札记》成为了"最有可读性的史学评论"
- **赵翼的背景**: 赵翼（1727-1814）字云崧，号瓯北，江苏阳湖（今常州）人。他是清代乾隆年间的学者、诗人、史学家。赵翼 26 岁中进士，做过翰林院编修、广西镇安府知府等官职，后来辞官回乡，专心著述。他与钱大昕、王鸣盛并称"清代三大史学家"。《廿二史札记》是他 1794 年（67 岁）出版的代表作——此外他还写了《陔余丛考》（考证笔记）和《瓯北诗话》（诗歌评论）
- **清代考据学的背景**: 赵翼生活在清代"考据学"（乾嘉学派）的鼎盛期——当时的学者专注于"考证经典、校勘古籍、辨析真伪"。但赵翼与传统的"考据派"不同——他不做"琐碎的考证"（如"某个字应该怎么写"），他做"宏观的分析"（如"这个朝代的政治制度有什么特点"）。这种"宏观史学"的方法在当时是"非主流的"——但也正因如此，《廿二史札记》具有了"超越时代"的价值
