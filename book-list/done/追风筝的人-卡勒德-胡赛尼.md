# 追风筝的人

状态：已读

作者：卡勒德·胡赛尼

一句话评价：卡勒德·胡赛尼（Khaled Hosseini）的"处女作'"——"'为你，千千万万遍'感动无数读者'"。"'阿米尔与哈桑的友谊与背叛震撼人心'"——"'阿富汗历史变迁下的救赎故事'"。"'全球畅销书'"——"'被翻译成多语言'"

核心收获：
- **阿米尔与哈桑的友谊**: 本书核心是"阿米尔与哈桑的友谊'"。"'阿米尔'是'少爷'"——"'哈桑'是'仆人'的'儿子'"。"'但'他们'一起'长大'"——"'像'兄弟'一样'"。"'哈桑'对'阿米尔'忠诚'"——"'为你，千千万万遍'"。"'但'阿米尔'背叛'了'哈桑'"——"'哈桑'被'欺负'"——"'阿米尔'没有'帮助'"。"'为什么'"——"'阿米尔'懦弱'"——"'阿米尔'嫉妒'哈桑'"。"'这次'背叛'"——"'影响'了'阿米尔'一生'"
- **风筝比赛**: 本书"以风筝比赛为核心事件'"。"'风筝比赛'是'阿富汗'的'传统'"。"'阿米尔'赢得'比赛'"——"'哈桑'去'追'风筝'"。"'但'哈桑'被'欺负'"——"'阿米尔'看到'了'"——"'没有'帮助'"。"'风筝'是'象征'"——"'象征'友谊'"——"'象征'背叛'"——"'象征'救赎'"。"'最后'阿米尔'为'哈桑'的'儿子'追'风筝'"——"'完成'救赎'"
- **背叛与愧疚**: 本书"探讨了背叛与愧疚'"。"'阿米尔'背叛'哈桑'"——"'他'一生'都'愧疚'"。"'愧疚'如何'影响'他'"：1）**逃避**——"'他'逃离'阿富汗'"；2）**压抑**——"'他'不'提'往事'"；3）**补偿**——"'后来'他'帮助'哈桑'的'儿子'"。"'愧疚'的'主题'"——"'人'都'会'犯错'"——"'关键'是'如何'面对'"
- **逃离与回归**: 本书"讲述了逃离与回归的故事'"。"'阿米尔'逃离'阿富汗'"——"'去'美国'"。"'为什么'逃离'"——"'逃避'愧疚'"——"'逃避'过去'"。"'但'过去'追'上'他'"——"'他'必须'面对'"。"'他'回归'阿富汗'"——"'救'哈桑'的'儿子'"。"'回归'是'救赎'的'开始'"——"'只有'面对'过去'"——"'才能'解脱'"
- **救赎之路**: 本书"的主题是救赎'"。"'什么'是'救赎'"——"'弥补'过错'"——"'原谅'自己'"。"'阿米尔'如何'救赎'"：1）**回去**——"'回到'阿富汗'"；2）**救人**——"'救'哈桑'的'儿子'"；3）**追风筝**——"'为'孩子'追'风筝'"——"'像'哈桑'为'他'一样'"。"'最后'阿米尔'说'"——"'为你，千千万万遍'"——"'他'完成'了'救赎'"
- **阿富汗历史**: 本书"背景是阿富汗的历史变迁'"。"'时间'跨度'"——"'1970 年代'到'2000 年代'"。"'事件'"：1）**君主制**——"'1970 年代'和平'"；2）**政变**——"'1978 年'共产主义'政变'"；3）**苏联入侵**——"'1979 年'苏联'入侵'"；4）**塔利班**——"'1990 年代'塔利班'统治'"。"'这些'事件'影响'了'人物'命运'"
- **作者的背景**: 卡勒德·胡赛尼（Khaled Hosseini, 1965-）——阿富汗裔美国作家、医生——"《追风筝的人》的作者'"。"他'出生'于'阿富汗'"——"'1980 年'移居'美国'"。"他'是'医生'"——"'后来'成为'作家'"。"《追风筝的人》'2003 年'出版'"——"'畅销'全球'"——"'销量'超过'5000 万册'"。"'他'的'其他'作品'"——"'《灿烂千阳》'——'《群山回唱》'"——"'都'关注'阿富汗'"
