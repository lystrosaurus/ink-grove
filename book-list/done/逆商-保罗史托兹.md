# 逆商

状态：已读

作者：保罗史托兹

一句话评价：将"面对逆境时的反应能力"量化为可评估的指标，CORE 四维模型有实用价值，但书中案例偏重复，鸡汤味较重。

核心收获：
- 逆商（AQ）由四个维度构成：控制感（Control）、归因（Ownership）、影响范围（Reach）、持续时间（Endurance）
- 高逆商的人把逆境视为暂时的、局部的、可控的挑战；低逆商的人则将其视为永久的、全面的、无能为力的灾难
- LEAD 工具（Listen 倾听反应、Explore 探究责任、Analyze 分析证据、Do 采取行动）是一套实用的应对框架
