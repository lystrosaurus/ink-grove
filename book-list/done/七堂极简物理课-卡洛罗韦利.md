# 七堂极简物理课

状态：已读

作者：卡洛-罗韦利

一句话评价：不到100页把现代物理学讲成诗——广义相对论的美、量子力学的怪、宇宙的浩瀚，全用最朴素的语言说出来。

核心收获：
- 第一堂课：爱因斯坦的广义相对论——空间和时间不是舞台，是演员本身。物质告诉空间怎么弯曲，空间告诉物质怎么运动
- 第二堂课：量子力学——世界不是连续的，能量是一份一份的。微观世界的一切都只能用概率描述
- 第三堂课：宇宙的结构——宇宙在膨胀，始于138亿年前的一次大爆炸
- 第四堂课：基本粒子——标准模型中的夸克、电子、中微子、希格斯玻色子，构成万物的积木
- 第五堂课：量子引力——广义相对论和量子力学的矛盾是物理学最大的未解之谜
- 第六堂课：概率与时间——时间可能不是基本的，而是涌现的
- 第七堂课：我们与宇宙的关系——意识到自己是宇宙的一部分，这是物理学给人类最大的礼物
