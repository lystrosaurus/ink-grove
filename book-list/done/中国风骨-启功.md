# 中国风骨

状态：已读

作者：启功

一句话评价：启功用一辈子练字画画的经验，讲透了什么是"中国风骨"——不是技法，是人格。字如其人，画如其人，功夫在字外。

核心收获：
- "风骨"不是技巧，是人格在笔墨中的自然流露。启功说"字写得好不好，先看做人做得好不好"
- 启功的书法观：临帖是基础，但不能死临。要"入帖"再"出帖"，最后形成自己的风格
- 文人画的核心是"写意"不是"写实"。画竹子不是画竹子长什么样，是画竹子的"气节"
- 启功对碑帖的研究极其深入。他指出很多被奉为经典的碑帖其实是后人伪造的
- "笔墨当随时代"——传统不是僵化的，每个时代都应该有自己的书法语言
