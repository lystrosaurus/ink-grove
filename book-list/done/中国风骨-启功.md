# 中国风骨

状态：已读

作者：启功

一句话评价：启功用一辈子练字画画的经验，讲透了什么是"中国风骨"——不是技法，是人格。字如其人，画如其人，功夫在字外。一个书法大家、画家、学者、诗人用一生的实践告诉你：中国艺术的最高境界是"人"。

核心收获：
- **"风骨"的本质**: 不是技巧，是人格在笔墨中的自然流露。启功说"字写得好不好，先看做人做得好不好"。这不是道德说教，是艺术规律——一个人的性情、学养、品格都会不自觉地体现在他的笔墨中。装不出来的
- **启功的书法观**: 临帖是基础，但不能死临。要"入帖"（把古人的技法学到手）再"出帖"（摆脱古人的束缚），最后形成自己的风格。启功自己的书法就是"入帖出帖"的典范——他学碑帖几十年，但最终形成的"启体"既传统又现代
- **文人画的"写意"**: 文人画的核心不是"写实"，是"写意"。画竹子不是画竹子长什么样，是画竹子的"气节"；画梅花不是画梅花的样子，是画梅花的"傲骨"。中国画的"像"不是"形似"，是"神似"
- **碑帖辨伪的功力**: 启功对碑帖的研究极其深入，他指出很多被奉为经典的碑帖其实是后人伪造的。这不是"打假"，是还原历史的真实——很多"经典"是层层累积的传说，不是事实。启功的辨伪方法是用"内证"（笔法、结构、时代风格）而不是"外证"（题跋、著录）
- **"笔墨当随时代"**: 传统不是僵化的，每个时代都应该有自己的书法语言。启功不反对创新，他反对的是"没有根基的创新"——你必须先学透传统，才有资格谈创新。跳过传统谈创新，那是胡来
- **启功的人生**: 启功是雍正皇帝的九世孙，但家道中落，一生清贫。他没有因为皇族后裔的身份而自傲，也没有因为清贫而自卑。他的书法里有"贵气"但没有"傲气"，有"骨力"但没有"戾气"——这就是"风骨"
- **诗书画的结合**: 启功不仅是书法家，还是画家、诗人、学者。他的诗幽默而深刻，他的画清雅而有趣，他的书法端庄而灵动。三者互相滋养——诗给书法提供意境，画给书法提供构图，学问给书法提供底蕴
- **对中国艺术教育的批评**: 启功对当时的书法教育很不满——太多人追求"速成"，太多人模仿"流行书风"，太多人把书法当成"表演"。他说书法是"慢功夫"，是一辈子的事。"一天不练自己知道，两天不练同行知道，三天不练观众知道"
