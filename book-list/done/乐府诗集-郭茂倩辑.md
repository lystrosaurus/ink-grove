# 乐府诗集

状态：已读

作者：郭茂倩辑

一句话评价：汉魏到唐代的民间歌辞总集——从"孔雀东南飞"到"木兰辞"，装满了普通人的爱情、战争、苦难和欢乐，是中国诗歌最"接地气"的部分。五千多首诗，记录了两千年前普通中国人的声音。

核心收获：
- **乐府的本质**: "乐府" originally 是汉武帝设立的音乐机构，负责采集民间歌辞、配乐、演唱。后来"乐府"成了这类诗歌的名称。乐府诗的本质是"民间歌辞"——不是文人写的，是老百姓唱的。后来被朝廷采集整理，就成了"乐府"
- **乐府 vs 文人诗**: 乐府诗和文人诗的最大区别是直接、朴素、不雕饰。"上邪！我欲与君相知，长命无绝衰。山无陵，江水为竭，冬雷震震，夏雨雪，天地合，乃敢与君绝！"——这种直白的爱情表白在文人诗里几乎看不到。文人诗追求"含蓄"，乐府诗追求"真诚"
- **经典名篇**: 《孔雀东南飞》——中国古代最长的叙事诗（357 句，1785 字），讲焦仲卿和刘兰芝的爱情悲剧，比《罗密欧与朱丽叶》早了一千多年。《木兰辞》——女英雄的故事，"唧唧复唧唧，木兰当户织"是中国人最熟悉的开头之一。《陌上桑》——罗敷拒婚，一个农村女子用智慧拒绝太守的调戏。《十五从军征》——老兵十五岁从军，八十岁归乡，"出门东向看，泪落沾我衣"
- **乐府的分类**: 郊庙歌辞（祭祀用）、燕射歌辞（宴饮用）、相和歌辞（民间合唱）、鼓吹曲辞（军乐）、横吹曲辞（骑兵乐）、杂曲歌辞（miscellaneous）。其中"相和歌辞"和"杂曲歌辞"最有文学价值，因为它们来自民间
- **郭茂倩的编纂**: 郭茂倩是南宋人，他编纂《乐府诗集》收录了从汉到唐的五千多首乐府诗，按类别分为十二大类。这是研究中国古代音乐和文学的最重要文献——很多乐府诗只靠这本书才流传下来
- **乐府的音乐性**: 乐府诗 originally 是"唱"的，不是"读"的。它们的节奏、韵律、重复都是为了配合音乐。后来音乐失传了，只剩下文字——但即使没有音乐，乐府诗的节奏感依然强烈
- **女性的声音**: 乐府诗中女性的声音特别多，而且特别真实。《孔雀东南飞》的刘兰芝、《陌上桑》的罗敷、《木兰辞》的花木兰、《上山采蘼芜》的弃妇——这些女性不是文人笔下的"理想女性"，是真实的、有血有肉的女人
- **乐府对后世的影响**: 乐府诗的传统直接影响了唐代的"新乐府运动"——白居易、元稹提倡"文章合为时而著，歌诗合为事而作"，就是要恢复乐府诗"关注现实、反映民生"的传统。杜甫的"三吏三别"也是乐府精神的延续
