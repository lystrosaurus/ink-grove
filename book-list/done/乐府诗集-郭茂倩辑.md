# 乐府诗集

状态：已读

作者：郭茂倩辑

一句话评价：汉魏到唐代的民间歌辞总集——从"孔雀东南飞"到"木兰辞"，装满了普通人的爱情、战争、苦难和欢乐，是中国诗歌最"接地气"的部分。

核心收获：
- 乐府诗的本质是"民间歌辞"——不是文人写的，是老百姓唱的。后来被朝廷采集整理，就成了"乐府"
- 乐府诗和文人诗的最大区别：乐府诗直接、朴素、不雕饰。"上邪！我欲与君相知，长命无绝衰"——这种直白的爱情表白在文人诗里几乎看不到
- 最著名的几首：《孔雀东南飞》（中国古代最长的叙事诗）、《木兰辞》（女英雄的故事）、《陌上桑》（罗敷拒婚）、《十五从军征》（老兵归乡）
- 乐府诗的分类：郊庙歌辞（祭祀用）、燕射歌辞（宴饮用）、相和歌辞（民间合唱）、鼓吹曲辞（军乐）、横吹曲辞（骑兵乐）、杂曲歌辞（ miscellaneous）
- 郭茂倩在南宋编纂这部诗集，收录了从汉到唐的五千多首乐府诗，是研究中国古代音乐和文学的最重要文献
