# 中国历代政治得失

状态：已读

作者：钱穆

一句话评价：钱穆的"政治史经典之作'"——"'180 页讲透汉唐宋明清五朝制度演变'"。"'三省六部、科举制、郡县制等分析精辟'"。"'制度是死的，人是活的'"——"'任何一项制度，必有其利弊两面'"。"'理解中国治理逻辑的必读之作'"——"'20 世纪中国史学最重要的著作之一'"

核心收获：
- **汉代制度**: 本书核心是"'汉代制度'"的分析'"。"'三公九卿'"——"'郡县制'"——"'察举制'"。"'奠定'了'中国'两千年'的'帝国'框架'"。"'汉代'最'伟大'的'制度'创新'"："'察举制'"——"'让'地方'推荐'人才'到'中央'"——"'打破'了'贵族'世袭'"。"'这'是'科举制'的'前身'"
- **唐代制度**: 本书"分析了唐代制度'"。"'三省六部'"——"'科举制'"。"'制度'成熟'与'开放'"。"'三省'（中书、门下、尚书）'分工'明确'"："'中书'决策'"——"'门下'审核'"——"'尚书'执行'"。"'这种'决策 - 审核 - 执行'的'三分法'"——"'在'现代'政府'中'依然'能'看到'影子'"。"'科举制'在'唐代'成熟'"——"'从此'读书'做官'成为'社会'流动'的'主'渠道'"
- **宋代制度**: 本书"讨论了宋代制度'"。"'文官'政治'"——"'重文'轻武'"。"'富'而'不'强'"。"'宋代'是'中国'历史'上'文人'地位'最高'的'时代'"——"'与'士大夫'治天下'"。"'但'重文'轻武'的'代价'"："'军事'weakness'"——"'被'辽'"——"'金'"——"'西夏'轮番'欺负'"。"'钱穆'的'洞察'"："'宋代'的'问题'不'是'弱'"——"'是'把'资源'全部'投入'内部'稳定'而'非'外部'防御'"
- **明代制度**: 本书"讲述了明代制度'"。"'废'丞相'设'内阁'"——"'皇权'集中'与'僵化'"。"'朱元璋'废除'丞相'制度'后'"——"'皇帝'直接'管'六部'"。"'但'皇帝'管'不'过来'"——"'只好'设'内阁'作为'顾问'机构'"。"'内阁'后来'变成'了'实际'的'行政'中枢'"——"'但'始终'没有'法定'地位'"。"'这'导致'明代'政治'的'混乱'"
- **清代制度**: 本书"分析了清代制度'"。"'军机处'"——"'密折制'"。"'专制'达到'顶峰'"。"'军机处'是'皇帝'的'私人'秘书处'"——"'没有'编制'"——"'没有'品级'"——"'没有'法定'权力'"。"'但'它'掌握'着'帝国'的'最高'决策'"。"'密折制'让'官员'可以'直接'向'皇帝'打'小'报告'"——"'互相'监视'"。"'清代'是'中国'历史'上'皇权'最'集中'的'时代'"
- **科举制度的双面性**: 本书"讨论了科举制度'"。"'科举'是'选拔'人才'的'公平'机制'"——"'维系'了'社会'流动'"。"'朝'为'田舍郎'"——"'暮'登'天子堂'"。"'但'科举'也'固化'了'思想'"——"'读书人'只'读'四书五经'"——"'不'学'科学'技术'"。"'科举'既'是'社会'稳定'的'基石'"——"'也'是'中国'近代'落后'的'原因'之一'"
- **中央与地方的博弈**: 本书"分析了中央与地方的关系'"。"'中国'两千年'政治'的'核心'问题'"："'集权'与'分权'的'动态'平衡'"。"'集权'太多'（如'清代）'导致'地方'没有'活力'"。"'分权'太多'（如'唐代'藩镇）'导致'分裂'"。"'这个'平衡'从来'没有'完美'解决'过'"
- **作者的背景**: 钱穆（1895-1990）——中国历史学家、思想家——"《中国历代政治得失》的作者'"。"'他'是'新儒家'代表'人物'"。"'他'的'其他'著作'"："'《国史大纲》'——'《中国近三百年学术史》'"。"'《中国'历代'政治'得失》'是'他'1950 年代'的'演讲'整理'"——"'180 页'讲透'两千年'制度'"。"'钱穆'的'史观'"："'反对'中国'传统'政治'全是'专制'黑暗'的'说法'"——"'认为'每个'朝代'的'制度'设计'都'有'其'合理性'"
