# 东方快车谋杀案

状态：已读

作者：阿加莎-克里斯蒂

一句话评价：推理小说史上最经典的"封闭空间杀人"——一列因暴雪被困的豪华列车上，十二个乘客每个人都捅了死者一刀，而每个人都无罪。

核心收获：
- 封闭空间叙事的巅峰：暴雪中的东方快车，十二个乘客加波洛，凶手一定在车厢里
- 每个嫌疑人都有不在场证明，但每个不在场证明都有微小破绽——阿加莎故意留了线索
- 结局反转是推理小说史上最震撼的：死者是绑架杀害一个小孩的罪犯，十二个乘客全是受害者家属，每人捅了一刀，是"私刑正义"
- 波洛的道德困境：法律上十二个人都犯了谋杀罪，但从道德角度看他们是替天行道。最终波洛选择了"沉默"
- 阿加莎的结构设计极其精密：线索全部公平地呈现给读者，但所有人都被"十二个人"这个数字误导了
