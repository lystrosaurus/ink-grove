# 虚与实中国绘画语言研究

状态：已读

作者：程抱一

一句话评价：法籍华裔学者程抱一从中西比较视角解读中国绘画的语言体系，以"虚与实"为核心概念，揭示了中国艺术的哲学根基。

核心收获：
- 理解了"虚与实"是中国绘画的核心语言——留白不是空白，而是画面的一部分
- 中国绘画与西方绘画的根本区别：西方追求"再现"，中国追求"传神"
- 诗书画印的统一是中国绘画的独特传统——画面不只是视觉艺术，更是文学和哲学的载体
