# 金钱心理学

状态：已读

作者：丹艾瑞里

一句话评价：行为经济学视角下的金钱决策指南，揭示了人们在消费、投资、储蓄中的非理性行为模式，以及这些行为背后的心理机制。

核心收获：
- 理解了"非理性"在金钱决策中的普遍性——我们以为自己很理性，实际上被各种认知偏差左右
- "心理账户"概念：人们对不同来源的钱有不同的态度（工资 vs 意外之财），但钱本质上是等价的
- 损失厌恶：损失 100 块的痛苦远大于获得 100 块的快乐，这解释了为什么人们会做出非理性的投资决策
