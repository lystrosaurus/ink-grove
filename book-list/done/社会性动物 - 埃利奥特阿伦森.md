# 社会性动物

状态：已读

作者：埃利奥特·阿伦森

一句话评价：埃利奥特·阿伦森（Elliot Aronson）的"社会心理学经典"——"'人是社会性动物，我们的行为和思想深受他人影响'"——"'从众心理'——'服从权威'——'认知失调'——'自我辩护'等概念'深刻揭示'人类行为规律'"——"是'美国社会心理学教材首选'"。"'认知失调让我们痛苦，所以我们学会了自我欺骗'"——"'我们'不是'理性的'动物'"——"'是'理性的'辩护者'"。

核心收获：
- **人是社会性动物**: 阿伦森说"'人是社会性动物'"——"'我们的'行为'和'思想'深受'他人'影响'"。"'我们'以为'自己'是'独立的'"——"'但'实际上'"——"'我们'的'每一个'决定'"——"'都'受到'他人的'影响'"。"'我们'会'从众'"——"'会'服从'权威'"——"'会'改变'自己的'观点'来'适应'群体'"。"阿伦森说'这'不是'弱点'"——"'是'人类的'本能'"——"'在'进化史上'"——"'那些'能够'融入'群体'的'人'"——"'更'容易'生存'"
- **认知失调理论**: 阿伦森是"'认知失调理论'的'重要发展者'"。"'当'你的'行为'和'信念'不一致'时'"——"'你会'感到'痛苦'"——"'这种'痛苦'就是'认知失调'"。"'为了'减少'痛苦'"——"'你'会'改变'你的'信念'"——"'来'让'它'与'行为'一致'"。"阿伦森说'认知失调'让我们'学会了'自我欺骗'"——"'我们'不是'理性的'动物'"——"'是'理性的'辩护者'"——"'我们'会'为'自己的'错误'行为'找到'合理的'借口'"
- **自我辩护**: 阿伦森"详细讲述了'自我辩护'的'机制'"。"'当'我们'做错事'时'"——"'我们'会'找'各种'理由'来'辩护'"——"'我'是'被迫的'"——"'我'是'为了'更大的'利益'"——"'别人'也'这么做'"。"自我辩护'让我们'能够'睡得着觉'"——"'但'也'让我们'无法'从'错误中'学习'"。"阿伦森说'承认错误'是'痛苦的'"——"'但'只有'承认错误'"——"'我们'才能'改变'"
- **从众心理**: 阿伦森"分析了'从众心理'的'原因'"：1）**信息性从众**——"'我们'认为'别人'知道得'更多'"；2）**规范性从众**——"'我们'不想'被'群体'排斥'"。"他'说'从众'不总是'坏的'"——"'有时候'从众'是'聪明的'"——"'比如'在'紧急情况下'"——"'跟着'大多数人'跑'"——"'通常'是'对的'"。"'但'在'需要'独立思考'的'情况下'"——"'从众'就'可能'导致'灾难'"
- **服从权威**: 阿伦森"详细讨论了'服从权威'的'危险'"。"'米尔格拉姆的实验'证明了'"——"'普通人'在'权威命令'下'"——"'可能'做出'非常残忍'的事'"。"阿伦森说'这'不是'因为'他们'是'坏人'"——"'是'因为'他们'进入了'一种'代理状态'"——"'他们'不再'认为'自己'是'行为的'主人'"——"'他们'只是'在执行'命令'"。"理解'这一点'"——"'可以'帮助'我们'抵制'不合理的'权威'"
- **偏见与自我实现预言**: 阿伦森"分析了'偏见'如何'导致'自我实现预言'"。"'如果'老师'认为'某个学生'很'聪明'"——"'老师'会'给'这个学生'更多'关注'"——"'这个学生'就'真的'变'聪明'了'"。"'如果'社会'认为'某个群体'是'危险的'"——"'这个群体'的'成员'会'感受到'歧视'"——"'他们'可能'真的'会'变得'敌对'"。"自我实现预言'让我们'看到了'我们'想看到的'"——"'这'会'强化'我们的'偏见'"
- **埃利奥特·阿伦森的背景**: 埃利奥特·阿伦森（Elliot Aronson, 1932-）——美国社会心理学家——"《社会性动物》的作者"——"这本书'被'称为'美国'社会心理学'的'圣经'"——"已经'更新了'十几次'"——"销量超过'百万册'"。"阿伦森'是'认知失调理论的'重要发展者'"——"他'设计了'很多'经典实验'"——"'如'努力正当化实验'"——"'说慌实验'"。"他'还'发明了'拼图教室'方法'"——"'用来'减少'学生之间的'偏见'和'冲突'"
