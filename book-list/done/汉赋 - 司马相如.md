# 汉赋

状态：已读

作者：司马相如（等）

一句话评价：汉赋是"中国文学史上'铺陈华丽'的韵文巅峰"——司马相如的《子虚赋》《上林赋》以"宏大的规模和繁复的辞藻"展现了"汉代帝国的'恢弘气象'"。"赋"这种文体"介于诗与散文之间"——"铺陈排比、辞藻华丽、以韵文叙事"。汉赋的"体物写志"传统——"先极尽铺陈描写"——"最后归于'讽谏之意'"。

核心收获：
- **赋的文体特点**: "赋"是中国文学中"一种'独特'的文体"——"介于'诗'与'散文'之间"。它的特点是：1）**铺陈排比**——"把一件事'从各个角度'反复描写"——"极尽'铺张'之能事"；2）**辞藻华丽**——"大量使用'生僻字'和'华丽辞藻'"——"让人'目不暇接'"；3）**韵文叙事**——"有'韵律'"——"但不像诗那样'简洁'"——"而是'长篇大论'"；4）**问答结构**——"通常以'两个人对话'的形式展开"——"一问一答"——"逐步展开"
- **司马相如的代表作**: 司马相如是"汉赋'最伟大'的代表"——他的《子虚赋》和《上林赋》是"汉赋的'巅峰之作'"。《子虚赋》描写了"楚国的'子虚先生'和'齐国的乌有先生'"之间的"辩论"——"各自夸耀自己国家的'富庶和壮丽'"。《上林赋》描写了"汉武帝'上林苑'的'宏伟'"——"山川、宫殿、花木、鸟兽、音乐、狩猎"——"无所不包"——"辞藻之华丽"——"规模之宏大"——"在中国文学史上'前无古人'"
- **体物写志的传统**: 汉赋有一个"重要的传统"——"体物写志"——"先'极尽铺陈'地描写'事物'"——"最后'归于讽谏之意'"。比如《上林赋》——前面"大篇幅描写'上林苑的豪华'"——最后"话锋一转"——"说'这样太奢侈了'"——"应该'节俭'"。这种"先扬后抑"的结构是"汉赋的'标准模式'"——"表面上'在赞美'"——"实际上'在规劝'"
- **汉代国力的文化映射**: 汉赋的"铺陈华丽"不是"文学家的'个人偏好'"——是"汉代'国力鼎盛'的'文化映射'"。汉武帝时期——"汉朝是'东亚最强大的帝国'"——"疆域辽阔"——"经济繁荣"——"文化自信"。汉赋的"宏大叙事"——"正是'帝国自信'的体现"——"只有'强大的帝国'才能'孕育'出'如此华丽'的文学"
- **赋的衰落**: 汉赋在"东汉后期开始'衰落'"——原因：1）**过度铺陈**——"辞藻越来越'华丽'"——"内容越来越'空洞'"——"变成了'文字游戏'"；2）**生僻字过多**——"读者'看不懂'"——"失去了'传播性'"；3）**形式僵化**——"千篇一律"——"缺乏'创新'"。后来"魏晋时期的'建安文学'"——"以'简洁有力'取代了'铺陈华丽'"——"赋的时代'结束了'"
- **汉赋的影响**: 尽管汉赋"在后世'备受争议'"——但它对中国文学的"影响是'深远的'"：1）**辞藻的丰富**——"汉赋'创造'了大量'新词汇'"——"丰富了汉语的'表达能力'"；2）**描写技巧**——"汉赋的'铺陈手法'"——"影响了后来的'骈文'和'律诗'"；3）**想象力**——"汉赋的'宏大想象'"——"影响了后来的'浪漫主义文学'"——"李白的诗歌'深受'汉赋影响"
- **司马相如的背景**: 司马相如（约公元前 179-前 118 年）——西汉辞赋家——字长卿——成都人。他是"汉赋'最伟大'的代表"——被誉为"赋圣"。他"以一篇《子虚赋'被汉武帝'赏识'"——"成为了'朝廷的文学侍从'"。他的另一个著名故事是"与卓文君的爱情"——"他'弹琴'（《凤求凰》）'打动'了'富家女'卓文君"——"两人'私奔'"——"成为了'中国古代最著名的'爱情故事之一"
