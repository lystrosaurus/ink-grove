# 西游记

状态：已读

作者：吴承恩

一句话评价："中国神话文学巅峰之作'"——"'九九八十一难对应八十一种心魔'"。"'师徒四人隐喻完整人格（信念/能力/欲望/踏实）'"。"'有背景的妖怪被领走，没背景的被打死辛辣讽刺现实社会'"。"'孙悟空的成长史就是每个人的修行史'"。"'四大名著中最具现代改编价值的经典'"

核心收获：
- **孙悟空的成长史**: 本书核心是"'孙悟空的成长史'"。"'从'石猴'到'美猴王'到'齐天大圣'到'孙行者'到'斗战胜佛'"。"'孙悟空'的'一生'就是'一个'从'野性'到'修行'的'过程'"。"'大闹天宫'是'叛逆期'"——"'有'本事'但'没'规矩'"。"'被压'五行山'是'挫折期'"——"'才华'被'现实'压制'"。"'西天取经'是'成长期'"——"'在'团队'中'找到'自己'的'位置'"
- **师徒四人的隐喻**: 本书"描绘了师徒四人的隐喻'"。"'唐僧'代表'信念'"——"'没有'信念'就'走'不'下去'"。"'孙悟空'代表'能力'"——"'没有'能力'就'克服'不'了'困难'"。"'猪八戒'代表'欲望'"——"'贪吃'好色'但'可爱'"。"'沙僧'代表'踏实'"——"'默默'做事'不'抱怨'"。"'这'四个'人'合起来'就是'完整'的'人'"
- **九九八十一难的本质**: 本书"分析了九九八十一难'"。"'八十一难'不'是'随机'遇到'的'妖怪'"——"'是'修行'的'必修课'"。"'每'一'难'都'对应'一种'心魔'"——"'贪婪'"——"'嫉妒'"——"'傲慢'"——"'恐惧'"——"'执念'"。"'打败'妖怪'不'是'用'武力'"——"'是'降伏'自心'"
- **妖魔鬼怪的社会学**: 本书"讽刺了妖魔鬼怪的社会学'"。"'《西游记》'里'的'妖怪'分'两种'"：1）**有背景的**——"'神仙'的'坐骑'"——"'童子'"——"'亲戚'"——"'打'到'最后'都'被'主人'领走'了'"；2）**没背景的**——"'白骨精'"——"'蜘蛛精'"——"'一棒子'打死'"。"'吴承恩'用'这种'方式'讽刺'了'现实'社会'"
- **修行的主题**: 本书"的主题是修行'"。"'《西游记》'表面'是'打怪升级'"——"'内核'是'修行'"。"'取经'路'就是'修行'路'"。"'唐僧'要'修'定力'"——"'孙悟空'要'修'耐心'"——"'猪八戒'要'修'克制'"——"'沙僧'要'修'主见'"。"'最后'取得'真经'的'不'是'唐僧'一个'人'"——"'是'整个'团队'"
- **幽默与讽刺**: 本书"充满幽默与讽刺'"。"'《西游记》'是'中国'古典'小说'中'最'幽默'的'一部'"。"'吴承恩'的'笔法'极其'讽刺'"。"'玉皇大帝'是'昏君'"——"'太上老君'是'和事佬'"——"'如来'是'老板'"——"'观音'是'HR'"。"'天宫'不'是'神圣'的'"——"'是'官僚'的'"
- **作者的背景**: 吴承恩（约 1500-1582）——明代小说家——"《西游记》的作者'"。"'他'是'江苏'淮安'人'"。"'《西游记》'成书'于'16 世纪'"——"'明代'嘉靖'年间'"。"'现存'最早'的'版本'是'世德堂本'（1592 年）'"。"'鲁迅'"——"'胡适'等'人'通过'考证'认为'是'吴承恩'"。"'但'也'有'学者'认为'是'集体'创作'"
- **当代意义**: 本书"有当代意义'"。"'《西游记》'在'现代'的'影响力'远'超'其他'三大'名著'"。"'电视剧'"——"'动画片'"——"'电影'"——"'游戏'"——"'漫画'"。"'孙悟空'是'最'被'改编'的'中国'文学'形象'"。"'为什么'"："'孙悟空'的'精神'是'反抗'"——"'自由'"——"'成长'"
