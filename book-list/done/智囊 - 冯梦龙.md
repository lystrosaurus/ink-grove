# 智囊

状态：已读

作者：冯梦龙

一句话评价：明代冯梦龙编纂的"智慧故事集"——收录了"从先秦到明代的一千多个智谋故事"——是中国古代"最丰富的'案例式'智慧百科全书"。"智囊"的分类体系：上智、明智、察智、胆智、术智等——每种智慧有"不同的应用场景"。冯梦龙"不只是文学家"（三言二拍）——也是"智慧故事的'搜集整理者'"。

核心收获：
- **智囊的分类体系**: 冯梦龙把"智慧"分为"多个类别"——每一类有"不同的'应用场景'"：1）**上智**——"最高层次的智慧"——"治国安邦的大战略"；2）**明智**——"明智的判断"——"在'复杂情况'下'做出正确选择'"；3）**察智**——"洞察力"——"看透'事物的本质'"——"识破'别人的伪装'"；4）**胆智**——"胆识"——"在'危险'中'果断行动'"；5）**术智**——"权术"——"巧妙的手段和策略"。这种"分类体系"让"智慧"变得"可学习、可应用"
- **政治智慧**: 《智囊》中"最多的故事"是"政治智慧"——"如何在'官场'中'生存和发展'"。比如"晏子使楚"——晏子"用'机智'应对楚王的'侮辱'"——"维护了齐国的尊严"。比如"邹忌讽齐王纳谏"——邹忌"不直接批评齐王"——而是"用自己的'镜子故事'来'暗示'"——让齐王"自己'意识到问题'"。这些故事展示了"中国古代政治家的'智慧'"——"不'硬碰硬'"——"用'巧妙的方式'达到目的"
- **军事智慧**: 《智囊》收录了大量"军事智慧"的故事。比如"孙膑'围魏救赵'"——"不直接'救援赵国'"——而是"攻打魏国的都城"——"让魏军'不得不回防'"——"赵国之围'自然解除'"。比如"韩信的'背水一战'"——"把军队'布置在河边'"——"让士兵'没有退路'"——"从而'激发'了最大的战斗力"。这些故事展示了"中国古代军事家的'战略思维'"
- **商业智慧**: 《智囊》也有"商业智慧"的故事。比如"范蠡'三致千金'"——范蠡"帮助勾践灭吴后"——"辞官经商"——"三次成为'首富'"——又"三次散尽家财"——"重新白手起家"。比如"吕不韦'奇货可居'"——吕不韦看到"在赵国做人质的异人（后来的秦庄襄王）"——认为他是"'奇货'"——"投资了他"——最终"成为了秦国的相国"。这些故事展示了"中国古代商人的'眼光和胆识'"
- **日常智慧**: 《智囊》最"有趣"的部分是"日常生活中的智慧"。比如"如何在'尴尬的场合'"用"幽默化解"——比如"如何在'被误解时'"用"事实澄清"——比如"如何在'对手强大时'"用"智取胜"。冯梦龙说"智慧不只是'大人物的专利'"——"普通人在'日常生活中'也能'展现智慧'"
- **冯梦龙的编纂意图**: 冯梦龙编纂《智囊》的目的"不是'讲故事'"——是"'教人智慧'"。他在序言中说"人'不可无智'"——"智'可以'保身'——可以'济事'——可以'治国'"。他"从'历代文献'中'搜集'智慧故事"——"分类整理"——"加上'自己的评语'"——让"读者在'阅读中'学到'智慧'"
- **冯梦龙的背景**: 冯梦龙（1574-1646）——明代文学家——代表作"三言"（《喻世明言》《警世通言》《醒世恒言》）——是中国"短篇小说的奠基人"。除了"三言"——他还编纂了"《智囊》""《古今谭概》""《情史》"等——是"明代最'多产'的'通俗文学'作家之一"。《智囊》是他"对'智慧文化'的'系统整理'"——在中国"智谋类书籍"中"占有重要地位"
