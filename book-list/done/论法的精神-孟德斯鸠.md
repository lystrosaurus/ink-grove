# 论法的精神

状态：已读

作者：孟德斯鸠

一句话评价：孟德斯鸠（Montesquieu）的"启蒙运动经典政治学著作'"——"'提出'三权分立'理论'"。"'系统分析了法律与政体、气候、地理、宗教、贸易等因素的关系'"。"'立法权、行政权、司法权必须分立，否则自由不复存在'"

核心收获：
- **三权分立**: 孟德斯鸠提出了"'三权分立'"的理论'"。"'什么'是'三权'"：1）**立法权**——"'制定'法律'的'权力'"；2）**行政权**——"'执行'法律'的'权力'"；3）**司法权**——"'解释'法律'的'权力'"。"'为什么'要'分立'"——"'因为'权力'导致'腐败'"——"'绝对'的'权力'导致'绝对'的'腐败'"。"'如果'三权'集中'在'一个人'或'一个'机构'手中'"——"'自由'就'不复'存在'"。"'孟德斯鸠'考察'了'英国'的'制度'"——"'他'认为'英国'的'分权'最'好'"。"'这个'理论'影响'了'美国'宪法'"——"'美国'是'第一个'实行'三权分立'的'国家'"
- **法律的系统性**: 孟德斯鸠"分析了法律与其他因素的关系'"。"'法律'不'是'孤立'的'"——"'法律'与'很多'因素'相关'"：1）**政体**——"'不同'政体'有'不同'法律'"；2）**气候**——"'热带'国家'的'法律'与'寒带'不同'"；3）**地理**——"'山地'国家'与'平原'国家'不同'"；4）**宗教**——"'宗教'影响'法律'"；5）**贸易**——"'商业'国家'的'法律'与'农业'国家'不同'"。"'所以'法律'不'能'照搬'"——"'每个'国家'的'法律'都'要'适合'自己'的'国情'"
- **政体三分**: 孟德斯鸠"把政体分为三类'"。"'共和政体'"：1）**原则**——"'品德'"；2）**特点**——"'人民'当家作主'"；3）**危险**——"'容易'变成'多数人的暴政'"。"'君主政体'"：1）**原则**——"'荣誉'"；2）**特点**——"'君主'统治'"——"'但'有'法律'限制'"；3）**危险**——"'君主'可能'滥用'权力'"。"'专制政体'"：1）**原则**——"'恐惧'"；2）**特点**——"'专制君主'为所欲为'"；3）**危险**——"'最'坏'的'政体'"。"'孟德斯鸠'最'推崇'君主'立宪'"——"'有'君主'"——"'但'有'法律'限制'"
- **自由的概念**: 孟德斯鸠"探讨了什么是自由'"。"'自由'不'是'想'做'什么'就'做'什么'"——"'自由'是'在法律'范围内'做'允许'的'事'"。"'政治'自由'"——"'安全'感'"——"'不'害怕'别人'"。"'如何'保障'自由'"——"'分权'"——"'法治'"——"'制衡'"。"'孟德斯鸠'说'"——"'只有'权力'被'权力'制约'"——"'自由'才'有'保障'"
- **气候与法律**: 孟德斯鸠"分析了气候对法律的影响'"。"'热带'气候'"——"'人'懒惰'"——"'需要'严厉'的'法律'"。"'寒带'气候'"——"'人'勤奋'"——"'可以'有'更'自由'的'法律'"。"'这个'观点'有'争议'"——"'但'孟德斯鸠'的'意图'"——"'是'说明'法律'要'适应'环境'"。"'他'还'分析'了'土壤'——'地形'——'人口'"——"'都'影响'法律'"
- **贸易的影响**: 孟德斯鸠"讨论了贸易的作用'"。"'贸易'让'国家'富裕'"——"'贸易'让'人民'温和'"。"'商业'国家'"——"'人民'计算'利益'"——"'不'喜欢'战争'"。"'所以'贸易'促进'和平'"。"'但'贸易'也'有'坏处'"——"'贸易'让'人'贪婪'"——"'贸易'腐蚀'品德'"。"'孟德斯鸠'的'态度'"——"'支持'贸易'"——"'但'警惕'贸易'的'副作用'"
- **孟德斯鸠的背景**: 孟德斯鸠（Montesquieu, 1689-1755）——法国启蒙思想家、法学家——"《论法的精神》的作者'"。"他'出身'贵族'"——"'他'买'了'波尔多'议会'的'职位'"。"他'游历'了'欧洲'"——"'尤其'是'英国'"——"'他'研究'了'英国'的'制度'"。"《论法的精神》'1748 年'出版'"——"'匿名'出版'"——"'但'立刻'引起'轰动'"。"'它'被'列为'禁书'"——"'但'流传'更'广'"。"这本书'影响'了'美国'建国'者'"——"'杰斐逊'——'麦迪逊'"——"'都'受'它'影响'"
