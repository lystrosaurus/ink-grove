# 设计中的设计

状态：已读

作者：原研哉

一句话评价：日本设计大师原研哉的设计哲学随笔集，阐述了"白""空""虚无"等东方美学理念在设计中的应用，是对"设计是什么"的深度思考。

核心收获：
- 理解了原研哉的核心设计理念：设计不是添加，而是还原——把事物还原到最本质的状态
- "白"不是空无一物，而是包含了所有可能性的"空"——这是东方美学的核心
- 设计的最高境界是让使用者感受到"本来就是这样"——不张扬、不抢眼，但恰到好处
