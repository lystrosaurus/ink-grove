# 逆商

状态：已读

作者：保罗·斯托茨

一句话评价：保罗·斯托茨（Paul Stoltz）的"逆境应对经典'"——"'将"面对逆境时的反应能力"量化为可评估的指标'"。"'CORE 四维模型有实用价值'"——"'控制感、归因、影响范围、持续时间'"。"'高逆商的人把逆境视为暂时的、局部的、可控的挑战'"

核心收获：
- **逆商的定义**: 本书提出"'逆商（AQ，Adversity Quotient）'"的概念'"。"'什么'是'逆商'"——"'面对'逆境'时'的'反应'能力'"。"'逆商'高'的'人'"——"'能'应对'挫折'"——"'能'从'失败'中'恢复'"。"'逆商'低'的'人'"——"'被'挫折'打倒'"——"'放弃'"。"'逆商'可以'测量'"——"'可以'提高'"。"'逆商'与'智商'——'情商'并列'"——"'三'Q'理论'"
- **CORE 四维模型**: 本书"介绍了 CORE 四维模型'"。"'C：控制感（Control）'"：1）**高逆商**——"'觉得'能'控制'"；2）**低逆商**——"'觉得'无力'"。"'O：归因（Ownership）'"：1）**高逆商**——"'承担'责任'"；2）**低逆商**——"'推卸'责任'"。"'R：影响范围（Reach）'"：1）**高逆商**——"'局部'影响'"；2）**低逆商**——"'全面'影响'"。"'E：持续时间（Endurance）'"：1）**高逆商**——"'暂时'的'"；2）**低逆商**——"'永久'的'"
- **高逆商与低逆商的对比**: 本书"对比了高逆商与低逆商'"。"'高逆商'的'人'"：1）**视逆境为挑战**——"'不'是'威胁'"；2）**暂时性**——"'会'过去'"；3）**局部性**——"'不'影响'其他'方面'"；4）**可控**——"'能'做'什么'"。"'低逆商'的'人'"：1）**视逆境为灾难**——"'世界'末日'"；2）**永久性**——"'永远'这样'"；3）**全面性**——"'一切'都'糟'"；4）**无力**——"'做'不了'什么'"
- **LEAD 工具**: 本书"提供了 LEAD 工具'"。"'L：Listen（倾听反应）'"："'觉察'自己'的'反应'"——"'是'高'逆商'还是'低'逆商'"。"'E：Explore（探究责任）'"："'谁'的'责任'"——"'不'是'指责'"——"'是'承担'自己'的'部分'"。"'A：Analyze（分析证据）'"："'证据'支持'什么'"——"'逆境'真的'那么'糟'吗'"。"'D：Do（采取行动）'"："'做'什么'能'改善'情况'"。"'LEAD'是'实用'的'框架'"
- **提高逆商的方法**: 本书"提供了提高逆商的方法'"。"'如何'提高'逆商'"：1）**觉察**——"'知道'自己'的'反应'模式'"；2）**重构**——"'重新'解读'逆境'"——"'是'挑战'不'是'灾难'"；3）**行动**——"'做'一点'什么'"——"'哪怕'很小'"；4）**坚持**——"'练习'"——"'逆商'可以'提高'"。"'例子'"——"'失业'"——"'高'逆商'的'人'"——"'寻找'新'机会'"——"'学习'新'技能'"
- **逆商的应用**: 本书"讨论了逆商的应用场景'"。"'工作'"：1）**面对失败**——"'项目'失败'"——"'不'放弃'"；2）**面对压力**——"'高'压力'"——"'能'应对'"。"'生活'"：1）**面对疾病**——"'生病'"——"'积极'治疗'"；2）**面对关系**——"'关系'问题'"——"'努力'修复'"。"'教育'"："'教'孩子'逆商'"——"'让'他们'能'应对'挫折'"
- **作者的背景**: 保罗·斯托茨（Paul Stoltz）——美国心理学家、逆商研究专家——"《逆商》的作者'"。"他'研究'逆商'20 多年'"——"'提出'了'CORE 模型'"——"'提出'了'LEAD 工具'"。"《逆商》'1999 年'出版'"——"'畅销'全球'"。"'他'还'为'企业'提供'培训'"——"'帮助'员工'提高'逆商'"
