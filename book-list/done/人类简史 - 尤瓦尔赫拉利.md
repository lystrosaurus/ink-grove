# 人类简史

状态：已读

作者：尤瓦尔·赫拉利

一句话评价：赫拉利（Yuval Noah Harari）"用跨学科视角重构 7 万年人类史'"——"'认知革命（虚构故事能力）、农业革命（史上最大骗局）、科学革命（承认无知）三阶段论颠覆传统历史叙事'"。"'金钱、帝国、宗教是推动人类统一的三种力量'"。"'最后追问人类更幸福了吗留下开放性思考'"——"'大历史写作典范'"

核心收获：
- **认知革命（7 万年前）**: 本书核心是"'认知革命'"的分析'"。"'智人'之所以'能'打败'尼安德特人'"——"'走出'非洲'"——"'征服'全球'"——"'不'是'因为'更'强壮'"——"'是'因为'有'了'语言'能力'"。"'更'重要'的'是'八卦'能力'"——"'和'虚构'故事'的'能力'"。"'人类'能'谈论'不'在'眼前'的'第三者'"——"'从而'管理'更大'的'群体'"。"'人类'能'共同'相信'同一个'虚构'的'故事'"——"'神'"——"'国家'"——"'金钱'"——"'公司'"。"'赫拉利'说'"："'人类'和'黑猩猩'之间'真正'不同'的'地方'就'在'于'那些'虚构'的'故事'"
- **农业革命（1.2 万年前）**: 本书"提出了农业革命的观点'"。"'农业革命'是'史上'最大'的'骗局'"。"'赫拉利'的'核心'论断'"："'农业'并'没有'让'人类'生活'得'更'好'"——"'而'是'让'人'更'苦'了'"。"'采集'时代'的'人类'每天'工作'3-5 小时'"——"'饮食'多样'"。"'农业'时代'的'人类'每天'工作'10 小时'以上'"——"'饮食'单一'"。"'但'农业'让'人口'暴增'"——"'数量'战胜'了'质量'"
- **人类的统一进程**: 本书"讨论了人类的统一进程'"。"'赫拉利'说'人类'历史'的'大'方向'"："'统一'"——"'从'分散'的'部落'到'城邦'到'帝国'到'全球化'"。"'推动'统一'的'三种'力量'"：1）**金钱**——"'经济'统一'"；2）**帝国**——"'政治'统一'"；3）**宗教**——"'文化'统一'"。"'这'三种'力量'看似'不同'"——"'本质'都'是'虚构'的'故事'"
- **科学革命（500 年前）**: 本书"分析了科学革命'"。"'科学'革命'的'核心'不'是'知识'"——"'是'承认'无知'"。"'前'现代'的'传统'认为'所有'重要'的'知识'都'已经'知道'了'"。"'科学'革命'说'我们'不'知道'"——"'这种'承认'无知'的'态度'让'人类'开始'系统'地'观察'"——"'实验'"——"'积累'知识'"。"'科学'加'帝国'加'资本'"——"'等于'现代'世界'"
- **资本主义与信用的力量**: 本书"讨论了资本主义'"。"'资本主义'的'核心'不'是'贪婪'"——"'是'信用'（credit）'"。"'信用'是'对'未来'的'信任'"——"'相信'明天'会'比'今天'更'好'"。"'这种'相信'让'人类'敢于'借钱'"——"'敢于'投资'"——"'敢于'扩张'"。"'赫拉利'说'"："'资本主义'不'是'零和'游戏'"——"'是'做大'蛋糕'的'游戏'"
- **帝国与宗教**: 本书"分析了帝国与宗教'"。"'帝国'是'政治'统一'的'引擎'"——"'宗教'是'文化'统一'的'引擎'"。"'赫拉利'对'帝国'的'定义'很'宽泛'"——"'任何'通过'征服'建立'的'多'民族'政权'都'是'帝国'"。"'他'说'帝国'虽然'靠'暴力'建立'"——"'但'最终'促进'了'文化'融合'"。"'宗教'方面'"："'佛教'"——"'基督教'"——"'伊斯兰教'"——"'民族主义'"——"'自由主义'都'视为'宗教'"
- **幸福的疑问**: 本书"最后提出了幸福的疑问'"。"'全书'最后'一章'问'了'一个'问题'"："'人类'比'以前'更'幸福'了'吗'"。"'赫拉利'的'答案'是'不'确定'"。"'物质'条件'确实'改善'了'"——"'但'幸福'不'是'物质'的'函数'"。"'它'是'期望'与'现实'的'差值'"。"'现代'人'的'期望'比'以前'高'得'多'"。"'他'建议'关注'冥想'和'内省'"
- **作者的背景**: 尤瓦尔·赫拉利（Yuval Noah Harari, 1976-）——以色列历史学家、作家——"《人类简史》的作者'"。"'他'是'牛津'大学'博士'"——"'现'任'耶路撒冷'希伯来'大学'教授'"。"《人类'简史》'2011 年'出版'"——"'被'翻译'成'60 多种'语言'"。"'他'的'写作'方法'"："'跨'学科'"——"'历史学'"——"'生物学'"——"'人类学'"——"'经济学'"——"'心理学'"。"'他'不'追求'细节'准确'"——"'追求'大图景'清晰'"
