## 核心总结

《精益创业》提出科学创业方法论：开发 - 测量 - 认知的反馈循环，MVP 最小可行产品，验证式学习，创新核算。核心是降低创业失败率，用科学方法管理创新。

---

## 深入理解

### 核心循环

**开发 - 测量 - 认知**：
```
假设 → MVP → 数据 → 学习 → 坚持或转型
```

### 关键概念

| 概念 | 说明 |
|------|------|
| MVP | 最小可行产品，快速验证 |
| 验证式学习 | 实验验证假设 |
| 创新核算 | 衡量创新进展 |
| 转型 | 方向调整 |

### 与传统创业对比

| 维度 | 传统创业 | 精益创业 |
|------|---------|---------|
| 规划 | 详细商业计划 | 假设驱动 |
| 开发 | 完美产品 | MVP |
| 发布 | 大规模 | 小范围测试 |
| 指标 | 收入/利润 | 学习进展 |
| 决策 | 直觉 | 数据 |

## 实践方法

### MVP 设计

1. **识别核心假设**
2. **设计最小测试**
3. **快速发布**
4. **收集数据**
5. **学习调整**

### 创新核算

1. **建立基准**
2. **调整引擎**
3. **决定坚持或转型**
