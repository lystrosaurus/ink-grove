# 中华上下五千年

状态：已读

作者：多人

一句话评价：中华文明的通俗读本——从黄帝到辛亥革命，用讲故事的方式把五千年历史串成一条线，适合入门但不适合深究。

核心收获：
- 以故事为线索串联历史：大禹治水、商鞅变法、赤壁之战、安史之乱、鸦片战争……每个时代挑最有代表性的事件讲
- 优点是"好读"——没有学术术语，没有史料考证，就是讲故事。缺点是"太简"——复杂的历史被简化为好人坏人的对立
- 对中国历史的分期比较传统：上古（先秦）→ 中古（秦汉至隋唐）→ 近古（宋元明清）→ 近代（鸦片战争以后）
- 对重要人物的评价基本沿袭传统：秦始皇暴但统一有功、汉武帝雄才大略但穷兵黩武、唐太宗开明、宋徽宗艺术天才但政治白痴
- 作为"历史入门读物"价值很高，但读完之后需要读更专业的通史（如吕思勉、钱穆）来纠正简化叙事带来的偏差
