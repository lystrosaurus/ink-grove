# 春秋

状态：已读

作者：孔子

一句话评价：中国第一部编年体史书，记载了鲁国从隐公元年到哀公十四年共 242 年的历史，孔子据鲁史修订，开创"春秋笔法"的史学传统。

核心收获：
- 理解了"春秋笔法"：用极简的文字隐含褒贬——"微言大义"，一个字的选用就有道德评判
- 春秋时期是礼崩乐坏的时代——周天子权威衰落，诸侯争霸，孔子用《春秋》来维护道义秩序
- "春秋三传"（左传、公羊传、谷梁传）是对《春秋》的注解——《春秋》本身太简略，需要传文来解释
