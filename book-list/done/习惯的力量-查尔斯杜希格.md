# 习惯的力量

状态：已读

作者：查尔斯-杜希格

一句话评价：揭示了习惯的"暗示-惯常行为-奖赏"循环——改变习惯不是靠意志力，是找到循环中的关键节点并替换它。

核心收获：
- 习惯循环三要素：暗示（cue）→ 惯常行为（routine）→ 奖赏（reward）。暗示触发行为，行为带来奖赏，奖赏强化暗示
- 改变习惯的黄金法则：保留暗示和奖赏，替换惯常行为。比如暗示是"压力大"，奖赏是"放松"，把"抽烟"替换为"散步"
- 关键习惯（keystone habit）：某些习惯的改变会引发连锁反应。比如运动习惯的养成会自然带动饮食、睡眠、工作效率的改善
- 意志力是可耗尽的资源，所以不能靠意志力改变习惯。要靠设计环境和流程
- 组织也有"习惯"——公司的惯例和文化本质上就是集体习惯。改变组织习惯比改变个人习惯更难
