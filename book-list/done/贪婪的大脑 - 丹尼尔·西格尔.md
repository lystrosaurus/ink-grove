# 贪婪的大脑

状态：已读

作者：丹尼尔·西格尔

一句话评价：丹尼尔·西格尔（Daniel Siegel）的"神经科学科普佳作'"——"'从神经科学角度解析人类欲望的起源'"。"'解释了为什么大脑天生不知满足'"——"'多巴胺不是快乐分子，而是渴望分子'"。"'这种贪婪推动了人类文明的进步'"

核心收获：
- **多巴胺的作用**: 本书核心是"'多巴胺'"的解释'"。"'什么'是'多巴胺'"——"'大脑'中'的'神经'递质'"。"'传统'观点'"——"'多巴胺'是'快乐'分子'"——"'让'人'快乐'"。"'但'本书'说'不对'"——"'多巴胺'是'渴望'分子'"——"'驱动'人'去'追求'"。"'比如'你'想吃'蛋糕'"——"'多巴胺'分泌'"——"'你'去'买'蛋糕'"。"'但'吃'蛋糕'的'快乐'"——"'不'是'多巴胺'"——"'是'其他'递质'"。"'多巴胺'的'作用'"——"'让'你'想要'更多'"
- **不知满足的机制**: 本书"解释了人类不知满足的机制'"。"'为什么'人'永远'不'满足'"——"'因为'大脑'的'奖励'系统'"。"'你'得到'想要'的'东西'"——"'快乐'很'短'暂'"——"'很快'就'想'要'更多'"。"'这'是'进化'的'结果'"——"'祖先'如果'满足'"——"'就'不'会'继续'寻找'食物'"——"'会'饿死'"。"'所以'不'满足'是'生存'机制'"。"'但'在'现代'社会'"——"'这'个'机制'导致'问题'"——"'过度'消费'"——"'成瘾'"
- **贪婪的进化意义**: 本书"分析了贪婪的进化意义'"。"'贪婪'不'是'道德'缺陷'"——"'贪婪'是'进化'赋予'的'机制'"。"'在'原始'社会'"——"'资源'稀缺'"——"'多'收集'资源'的'人'"——"'更'可能'生存'"。"'所以'贪婪'的'基因'"——"'被'保留'下来'"。"'但'在'现代'社会'"——"'资源'丰富'"——"'贪婪'导致'问题'"——"'肥胖'"——"'债务'"——"'环境'破坏'"。"'理解'了'进化'意义'"——"'能'更好'地'控制'贪婪'"
- **文明的推动力**: 本书"提出贪婪推动了文明进步'"。"'人类'的'进步'"——"'源于'永远'想要'更多'"。"'想要'更多'食物'"——"'发明'农业'"。"'想要'更多'舒适'"——"'发明'技术'"。"'想要'更多'知识'"——"'发展'科学'"。"'所以'贪婪'有'积极'的'一面'"。"'但'也'有'消极'的'一面'"——"'过度'贪婪'导致'问题'"。"'关键'是'平衡'"——"'利用'贪婪'的'动力'"——"'但'不'被'贪婪'控制'"
- **成瘾的神经机制**: 本书"解释了成瘾的神经机制'"。"'什么'是'成瘾'"——"'大脑'的'奖励'系统'被'劫持'"。"'毒品'——'酒精'——'赌博'"——"'都'刺激'多巴胺'分泌'"。"'大脑'适应'了'"——"'需要'更多'才能'有'相同'效果'"。"'这'就是'耐受性'"。"'成瘾'的'治疗'"——"'不'是'意志力'"——"'是'重新'训练'大脑'"。"'如何'做'到'"：1）**避免触发**——"'远离'诱惑'"；2）**替代奖励**——"'找'健康'的'奖励'"；3）**正念**——"'觉察'欲望'"
- **控制欲望**: 本书"提供了控制欲望的方法'"。"'如何'控制'欲望'"：1）**觉察**——"'知道'自己'的'欲望'"——"'不'被'它'控制'"；2）**延迟满足**——"'等'一'等'"——"'欲望'会'减弱'"；3）**设定目标**——"'有'更'高'的'目标'"——"'欲望'就'不'那么'重要'"；4）**正念练习**——"'观察'欲望'"——"'不'评判'"——"'欲望'会'自动'消失'"。"'这些'方法'"——"'基于'神经'科学'"——"'有效'"
- **作者的背景**: 丹尼尔·西格尔（Daniel Siegel）——美国神经科学家、精神科医生——"《贪婪的大脑》的作者'"。"他'在'加州大学'任教'"——"'研究'大脑'与'行为'"。"'他'的'写作'风格'"——"'通俗'"——"'用'例子'解释'复杂'的'神经'科学'"。"'这本书'的'目的'"——"'让'普通人'理解'大脑'"——"'更好'地'控制'自己'"
