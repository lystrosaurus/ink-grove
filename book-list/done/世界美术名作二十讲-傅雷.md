# 世界美术名作二十讲

状态：已读

作者：傅雷

一句话评价：傅雷用文学的语言讲美术——二十件西方艺术史上的代表作，从乔托到罗丹，每一件都讲透了它为什么伟大。

核心收获：
- 二十讲覆盖文艺复兴到19世纪：乔托、波提切利、达芬奇、米开朗基罗、拉斐尔、提香、伦勃朗、鲁本斯、罗丹
- 傅雷的讲法不是"美术史"，是"作品欣赏"。他不讲流派演变，讲每一幅画、每一座雕塑"为什么好"
- 达芬奇那讲最精彩：《最后的晚餐》的构图秘密、《蒙娜丽莎》的微笑为什么神秘——傅雷用文学语言解释了视觉艺术的魅力
- 米开朗基罗的西斯廷天顶画：一个人四年仰卧在脚手架上画了三百多个人物。傅雷说这是"人类意志力的极限展示"
- 傅雷的翻译家身份让他的美术评论有独特的跨文化视角——他用中国美学的语言讲西方艺术
