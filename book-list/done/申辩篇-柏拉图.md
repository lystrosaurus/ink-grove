# 申辩篇

状态：已读

作者：柏拉图

一句话评价：柏拉图的"西方哲学史上'最具震撼力的'文本之一"——"'记录了'苏格拉底'在雅典法庭上'为自己辩护的'全过程'"——"他'以死亡'捍卫了'哲学的尊严'和'思想自由'"。"'无知之知'"——"'我唯一知道的'就是'我一无所知'"——"'这种谦逊'是'哲学的起点'"。"苏格拉底'面对死亡的态度'"——"'宁可死'也'不放弃'哲学追问"——"'未经省察的人生没有价值'"。

核心收获：
- **无知之知**: 苏格拉底最著名的观点——"无知之知"——"'我唯一知道的'就是'我一无所知'"。"他说'自己'不是'智者'"——"'智者'声称'自己知道'很多"——"'但'苏格拉底说'我'知道自己'不知道'"——"'这就是'我'比他们'聪明的地方'"。"无知之知'不是'否定一切'"——"'是'承认'自己的局限'"——"'只有'承认'自己不知道'"——"'你'才能'开始学习'"——"'开始追问'"——"'开始思考'"
- **面对死亡的态度**: 苏格拉底"面对死亡的态度'令人震撼'"。"雅典法庭'判处他'死刑'"——"'罪名是'腐蚀青年'和'不敬神'"——"他'有机会'逃跑'"——"他'有机会'道歉'以求'宽恕'"——"但他'都拒绝了'"。"他说'宁可死'也'不放弃'哲学追问"——"'如果'你们'释放我'的条件'是'不再追问'"——"'那我'宁愿'选择死亡'"。"苏格拉底说'死亡'可能'是一件好事'"——"'因为'死后'要么'是'永恒的睡眠'"——"'要么'是'去到另一个世界'"——"'两种'都'不可怕'"
- **民主的警示**: 申辩篇是"对民主制度的'深刻警示'"。"雅典是'当时'最'民主的'城邦'"——"'但'正是'这个民主的城邦'"——"'以多数投票'判处了'苏格拉底死刑'"。"柏拉图通过这件事'得出了一个结论'"——"'民主'也可能'迫害真理'"——"'多数人'不一定'是对的'"——"'有时候'多数人'是'被'情绪和偏见'左右的'"。"这个警示'在今天'依然'有效'"——"'网络暴力'"——"'舆论审判'"——"'都是'多数人的暴政'"
- **未经省察的人生**: 苏格拉底说"'未经省察的人生没有价值'"——"'这是'他'一生'的'座右铭'"。"他说'一个人'如果'不反思'自己的生活'"——"'不追问'自己的'信念'"——"'不质疑'自己的'行为'"——"'那他'就是'在'梦游'"——"'他没有'真正地'活着'"。"苏格拉底的'哲学方法'"——"'就是'不断地'追问'"——"'你相信什么'"——"'为什么相信'"——"'这个信念'有'根据吗'"——"'如果'没有'根据'"——"'你'应该'放弃它'"
- **苏格拉底的使命**: 苏格拉底说"他的'使命'是'来自神的'"。"'神谕说'苏格拉底'是'雅典'最聪明的人'"——"'他不相信'"——"'所以他'去'找'那些'被认为是'聪明的人'"——"'政治家'——'诗人'——'工匠'"——"'发现'他们'都'不知道'自己'在说什么'"。"苏格拉底说'神'让他'做'一件事'"——"'就是'不断地'追问'雅典人'"——"'让他们'意识到'自己'的'无知'"——"'从而'开始'追求'真正的'智慧'"
- **苏格拉底的死亡**: 苏格拉底"最终'被判处'死刑'"——"'饮鸩而亡'"。"他在'临死前'还'在和朋友'讨论'哲学'"——"'讨论'灵魂不朽'"——"'讨论'死后的世界'"——"'他'没有'恐惧'"——"'没有'后悔'"——"'他'平静地'喝下了'毒药'"。"柏拉图说'苏格拉底的死'"——"'是'哲学史上'最'悲壮的'时刻'"——"'一个'追求真理的人'"——"'被'他深爱的城邦'判处了'死刑'"——"'但他'没有'背叛'自己的'信念'"
- **柏拉图的背景**: 柏拉图（Plato, 约公元前 428-前 348 年）——古希腊哲学家——"苏格拉底的学生"——"'申辩篇'是他'早期'的作品'"——"'记录了'苏格拉底'在法庭上的'辩护词'"——"'虽然'不是'逐字记录'"——"'但'基本反映了'苏格拉底的'思想和'人格'"。"'申辩篇'与'克里同篇'——'斐多篇'一起"——"'构成了'柏拉图'关于苏格拉底之死的'三部曲'"
