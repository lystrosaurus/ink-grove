# 资治通鉴

状态：已读

作者：司马光

一句话评价：司马光的"编年体通史巨著'"——"'鉴于往事，有资于治道'"。"'与史记并称史学双璧'"——"'治国理政与历史教训的百科全书'"。"'中国史学巅峰之作'"——"'从战国到五代，1362 年历史'"

核心收获：
- **编年体例**: 《资治通鉴》"采用编年体例'"。"'什么'是'编年体'"——"'按'时间'顺序'记录'事件'"。"'与'纪传体'不同'"——"'史记'是'纪传体'"——"'以'人物'为'中心'"。"'编年体'的'优点'"：1）**清晰**——"'时间'线索'清晰'"；2）**连贯**——"'事件'发展'连贯'"；3）**便于**借鉴——"'容易'看到'因果'"。"'缺点'"：1）**分散**——"'一个'人'的'事迹'分散'在'多年'"；2）**枯燥**——"'不如'纪传体'生动'"。"'司马光'选择'编年体'"——"'因为'他'的'目的'是'借鉴'"
- **时间跨度**: 本书"涵盖了从战国到五代的历史'"。"'起始'"——"'公元前 403 年'"——"'三家分晋'"。"'结束'"——"'公元 959 年'"——"'宋朝'建立'前'"。"'总共'1362 年'历史'"。"'为什么'从'三家分晋'开始'"——"'司马光'认为'这是'礼制'崩溃'的'开始'"。"'为什么'到'五代'结束'"——"'因为'宋朝'是'他'的'当代'"——"'他'不'写'当代'史'"
- **治国理政**: 本书的核心是"'治国理政'"的经验'"。"'司马光'写'这本书'的'目的'"——"'给'皇帝'看'"——"'让'皇帝'从'历史'中'学习'"。"'主要内容'"：1）**兴衰**——"'王朝'如何'兴起'"——"'如何'衰亡'"；2）**用人**——"'如何'识别'人才'"——"'如何'使用'人才'"；3）**制度**——"'什么'制度'好'"——"'什么'制度'坏'"；4）**道德**——"'君臣'关系'"——"'父子'关系'"。"'司马光'的'观点'"——"'德'比'才'重要'"——"'德者，才之帅也'"
- **历史教训**: 本书"提供了大量历史教训'"。"'比如'"：1）**秦亡**——"'暴政'导致'灭亡'"；2）**汉兴**——"'休养生息'"——"'文景之治'"；3）**唐衰**——"'安史之乱'"——"'藩镇割据'"。"'司马光'的'方法'"——"'不'只是'记录'事件'"——"'是'分析'原因'"——"'总结'教训'"。"'他'的'评论'"——"'臣光曰'"——"'表达'他'的'观点'"
- **人物评价**: 本书"评价了大量历史人物'"。"'司马光'的'评价'标准'"：1）**德**——"'品德'第一'"；2）**功**——"'功绩'第二'"；3）**忠**——"'忠诚'于'国家'"。"'他'赞赏'的'人'"——"'诸葛亮'"——"'魏徵'"——"'忠诚'——'有德'"。"'他'批评'的'人'"——"'曹操'"——"'有才'无德'"。"'司马光'的'评价'"——"'影响'了'后世'对'这些'人'的'看法'"
- **政治智慧**: 本书"蕴含了丰富的政治智慧'"。"'比如'"：1）**制衡**——"'权力'要'制衡'"——"'不'能'集中'"；2）**用人**——"'用人'不疑'"——"'疑人不用'"；3）**民心**——"'得'民心'者'得'天下'"；4）**改革**——"'改革'要'渐进'"——"'不'能'激进'"。"'这些'智慧'"——"'来自'历史'经验'"——"'对'后世'有'借鉴'意义'"
- **司马光的背景**: 司马光（1019-1086）——北宋政治家、史学家——"《资治通鉴》的作者'"。"他'是'保守派'代表'"——"'反对'王安石变法'"。"《资治通鉴》'写了'19 年'"——"'从'1066 年'到'1084 年'"。"'他'有'一个'团队'"——"'刘恕'——'刘攽'——'范祖禹'"。"'宋神宗'命名'为'《资治通鉴》'"——"'意思是'鉴于'往事'"——"'有'资'于'治道'"。"'这本书'的'地位'"——"'中国'史学'巅峰'"——"'与'史记'并称'双璧'"
