# 第三帝国的兴亡

状态：已读

作者：威廉·夏伊勒

一句话评价：威廉·夏伊勒（William L. Shirer）的"二战史经典巨著"——"以'亲身经历'和'纳粹解密档案'为基础'"——"'全景式记录了'纳粹德国'从崛起到覆灭的'完整历史'"。"'纳粹崛起'的社会土壤"——"'一战后的屈辱'——'经济崩溃'——'民主制度脆弱'——'民众渴望强人'"。"希特勒的上台'不是偶然的'"——"'他是'利用了'民主制度的漏洞'和'民众的恐惧与愤怒'"——"'历史的最大教训'"——"'当民主制度失去警惕时'"——"'极权主义就会趁虚而入'"

核心收获：
- **纳粹崛起的社会土壤**: 夏伊勒"详细分析了'纳粹崛起的社会土壤'"。"'一战后'的'德国'"——"'被迫'接受了'凡尔赛条约'"——"'割地'——'赔款'——'裁军'"——"'德国人'感到'被羞辱了'"。"'1920 年代'的'恶性通货膨胀'"——"'让'中产阶级'的'积蓄'化为乌有'"——"'1930 年代'的'大萧条'"——"'让'失业率'飙升到 30%'"。"'魏玛共和国'的'民主制度'很'脆弱'"——"'政党林立'"——"'政府频繁更迭'"——"'民众'对'民主'失去了'信心'"——"'他们'渴望'一个'强人'来'拯救德国'"
- **希特勒的上台**: 夏伊勒"讲述了'希特勒如何上台'"。"希特勒'不是'通过'政变'上台的'"——"'是'通过'选举'"——"'纳粹党'在'1932 年'的'选举中'成为'第一大党'"。"但'希特勒'没有'获得多数'"——"'是'保守派'政客'以为'可以'控制他'"——"'任命'他'为'总理'"——"'结果'被'他'架空了'议会'"——"'建立了'独裁政权'"。"夏伊勒说'希特勒的上台'是'民主制度的失败'"——"'民主'给了'他'权力'"——"'他'用'这个权力'摧毁了'民主'"
- **极权主义的建立**: 夏伊勒"描述了'纳粹如何建立极权主义'"。"1）**取缔反对党**——"'其他政党'被'禁止'"——"'议员'被'逮捕'"；2）**控制媒体**——"'报纸'——'广播'——'电影'都'被'纳粹'控制'"——"'宣传'戈培尔的'谎言'"；3）**建立秘密警察**——"'盖世太保'可以'随意'逮捕'任何人'"——"'集中营'里'关满了'政治犯'"；4）**控制司法**——"'法官'必须'是'纳粹党员'"——"'法律'成为'镇压工具'"；5）**控制教育**——"'学校'教'种族主义'"——"'青少年'被'灌输'纳粹思想'"
- **大屠杀的真相**: 夏伊勒"揭露了'大屠杀的真相'"。"'纳粹'的'种族主义'认为'雅利安人'是'优等种族'"——"'犹太人'是'劣等种族'"——"'必须'被'消灭'"。"'1941 年'的'万湖会议'"——"'纳粹'正式'决定'实施'最终解决方案'"——"'系统性地'屠杀'犹太人'"。"'奥斯维辛'——'特雷布林卡'——'达豪'等'集中营'"——"'用'毒气室'和'焚尸炉'屠杀'了'600 万'犹太人'"。"夏伊勒说'这是'人类历史上'最'黑暗的'篇章'"
- **战争的进程**: 夏伊勒"详细记录了'二战的进程'"。"'1939 年'入侵波兰'"——"'英法'对德'宣战'"——"'二战爆发'"。"'1940 年'闪电战'击败法国'"——"'德国'控制了'大半个欧洲'"。"'1941 年'入侵苏联'"——"'初期'进展顺利'"——"'但'在'莫斯科'和'斯大林格勒'被'阻挡'"——"'这是'转折点'"。"'1944 年'诺曼底登陆'"——"'盟军'从'西线'进攻'"——"'苏联'从'东线'进攻'"——"'1945 年'德国'投降'"
- **纳粹覆灭的原因**: 夏伊勒"分析了'纳粹覆灭的原因'"。"1）**两线作战**——"'德国'同时'与'英法美'和'苏联'作战'"——"'兵力分散'"；2）**资源不足**——"'德国'的'资源'不足以'支撑'长期战争'"——"'石油'——'钢铁'——'粮食'都'短缺'"；3）**战略失误**——"'入侵苏联'是'致命错误'"——"'低估了'苏联的'抵抗能力'"；4）**道德破产**——"'纳粹的'暴行'让'全世界'都'反对'它'"——"'它'没有'盟友'"——"'只有'仆从国'"
- **威廉·夏伊勒的背景**: 威廉·夏伊勒（William L. Shirer, 1904-1993）——美国记者、历史学家——"《第三帝国的兴亡》的作者"。"他'在'1930 年代'作为'CBS 记者'驻'柏林'"——"'亲眼目睹了'纳粹的'崛起'"——"'1940 年'被迫'离开'德国'"。"战后'他'利用'缴获的'纳粹档案'"——"'结合'自己的'亲身经历'"——"'写了'这本'巨著'"——"'1960 年出版'"——"'获得了'美国国家图书奖'"——"'销量超过'百万册'"——"'是'二战史'最'经典的'著作之一'"
