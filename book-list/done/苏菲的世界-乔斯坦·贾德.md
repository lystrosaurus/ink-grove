# 苏菲的世界

状态：已读

作者：乔斯坦·贾德

一句话评价：乔斯坦·贾德（Jostein Gaarder）的"哲学启蒙经典'"——"'通过苏菲的信件讲述西方哲学史'"。"'你是谁？''世界从何而来？'开篇引人深思'"——"'最适合初学者的哲学入门书'"——"'小说与哲学的完美结合'"

核心收获：
- **哲学的基本问题**: 《苏菲的世界》"从两个基本问题开始'"。"'你是谁？'"——"'这是'自我认知'的'问题'"——"'你'不'只是'名字'"——"'不'只是'身体'"——"'你'是'什么'"。"'世界从何而来？'"——"'这是'宇宙论'的'问题'"——"'世界'是'永恒'的'"——"'还是'被'创造'的'"。"'这两个'问题'"——"'是'哲学'的'起点'"——"'每个'人'都'应该'问'自己'"。"'孩子'常'问'这些'问题'"——"'成人'忘记'了'这些'问题'"——"'哲学'就是'重新'学会'提问'"
- **西方哲学史脉络**: 贾德"通过苏菲的信件，系统讲述了西方哲学史'"。"'古代'——'神话'解释'世界'"。"'前苏格拉底'——'自然'哲学家'寻找'世界'的'本源'"——"'水'——'气'——'火'——'原子'"。"'苏格拉底'——'从'自然'转向'人'"——"'认识'你自己'"。"'柏拉图'——'理念论'——'洞穴'比喻'"。"'亚里士多德'——'逻辑'——'分类'——'经验'"。"'中世纪'——'基督教'哲学'——'信仰'与'理性'"。"'文艺复兴'——'人文主义'"。"'理性主义'——'笛卡尔'——'我思故我在'"。"'经验主义'——'洛克'——'休谟'"。"'启蒙运动'——'伏尔泰'——'卢梭'——'康德'"。"'浪漫主义'——'黑格尔'——'马克思'"——"'存在主义'——'萨特'"
- **苏菲与艾伯特**: 小说有"两条线索'"。"'第一条'是'苏菲'收到'神秘'信件'"——"'一个'叫'艾伯特'的'人'给'她'写'信'"——"'教'她'哲学'"。"'第二条'是'苏菲'和'母亲'的'生活'"——"'她们'发现'一些'奇怪'的'事情'"。"'两条'线索'逐渐'交汇'"——"'最后'揭示'一个'惊人'的'真相'"。"'苏菲'和'艾伯特'"——"'他们'是'谁'"——"'他们'存在'吗'"——"'这个'问题'本身'就是'哲学'"
- **哲学与童话**: 贾德"把哲学比作童话'"。"'童话'里'有'魔法'"——"'哲学'里'有'思想'"。"'童话'让'我们'想象'另一个'世界'"——"'哲学'让'我们'思考'这个'世界'"。"'孩子'喜欢'童话'"——"'因为'他们'还'有'好奇心'"。"'成人'应该'读'哲学'"——"'因为'他们'需要'重新'获得'好奇心'"。"'哲学'不'是'枯燥'的'理论'"——"'哲学'是'一场'冒险'"——"'是'对'存在'的'探索'"
- **存在与意识**: 小说"探讨了存在与意识的关系'"。"'我们'如何'知道'自己'存在'"——"'笛卡尔'说'我思故我在'"。"'我们'如何'知道'世界'存在'"——"'是'真实'的'"——"'还是'幻觉'"。"'苏菲'发现'自己'可能'是'书中'的'人物'"——"'她'的'世界'是'别人'创造'的'"。"'这个'想法'让人'不安'"——"'但'它'提出'了'深刻'的'问题'"——"'我们'的'意识'是'什么'"——"'存在'的'本质'是'什么'"
- **哲学的意义**: 贾德"最后探讨了哲学的意义'"。"'哲学'不'能'给'我们'答案'"——"'但'它'能'给'我们'问题'"。"'问题'比'答案'更'重要'"——"'因为'问题'让'我们'思考'"。"'哲学'的'目的'"：1）**认识自己**——"'理解'自己'的'局限'"；2）**理解世界**——"'用'理性'看'世界'"；3）**过好的生活**——"'哲学'是'生活'的'指南'"。"'所以'哲学'不'是'学术'"——"'哲学'是'每个人'都'需要'的'"
- **贾德的背景**: 乔斯坦·贾德（Jostein Gaarder, 1952-）——挪威作家——"《苏菲的世界》的作者'"。"他'曾是'高中'哲学'教师'"——"'他'懂'如何'向'年轻人'解释'哲学'"。"《苏菲的世界》'1991 年'出版'"——"'全球'销量'超过'4000 万册'"——"'被'翻译成'60 多种'语言'"。"这本书'的'成功'"——"'因为'它'把'复杂'的'哲学'讲'得'通俗'"——"'用'小说'的'形式'吸引'读者'"。"贾德'的'其他'作品'"——"'《纸牌的秘密》'——'《橙色女孩》'"——"'也'是'哲学'主题'"
