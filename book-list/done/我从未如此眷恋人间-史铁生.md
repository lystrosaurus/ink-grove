# 我从未如此眷恋人间

状态：已读

作者：史铁生

一句话评价：史铁生散文精选集，以"地坛"为核心意象，记录了他在双腿瘫痪后对生命、死亡、亲情和人间百态的深刻思考，文字沉静而有力量。

核心收获：
- 理解了史铁生最核心的主题：在最绝望的处境中找到眷恋世界的理由
- 《我与地坛》是中国当代散文的巅峰之作，一个残疾青年与一座废弃古园的对话
- 史铁生对母亲的描写催人泪下——"我从来没有想过，母亲为我做过的那些事，会在我心里留下这么深的印子"
