# 机器人叛乱

状态：已读

作者：基思斯坦诺维奇

一句话评价：认知科学家斯坦诺维奇用"机器人叛乱"隐喻探讨人类理性，提出人类需要像机器人反抗主人一样反抗自身的认知偏差和基因本能。

核心收获：
- 理解了"机器人叛乱"的隐喻：人类需要反抗自身的认知偏差和基因本能，就像机器人反抗创造者
- 基因和模因（meme）是人类的"创造者"——它们设计了我们的认知系统，但并不总是服务于我们的最佳利益
- 理性思维是人类反抗"生物程序"的唯一武器
