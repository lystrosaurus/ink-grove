# 有限与无限的游戏

状态：已读

作者：詹姆斯卡斯

一句话评价：一本薄薄的小书却提出了一个极具解释力的世界观框架：世上只有两种游戏，有限游戏以赢为目的，无限游戏以延续游戏为目的。这个框架影响了从商业到人生的诸多领域。

核心收获：
- 理解了两种游戏的根本区别：有限游戏有明确的规则、边界和终点，目的是赢；无限游戏的规则可以改变、边界可以扩展，目的是让游戏继续
- 商业、婚姻、文化、战争都可以用这个框架重新理解
- "无限游戏玩家不追求赢，而是追求让游戏继续"——这是一种完全不同的思维方式
