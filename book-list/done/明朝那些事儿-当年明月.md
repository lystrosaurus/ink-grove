# 明朝那些事儿

状态：已读

作者：当年明月

一句话评价：当年明月（本名石悦）的"历史通俗读物"——用"幽默笔法"讲述"明朝三百年兴衰"——从"朱元璋崛起"到"崇祯亡国"——让"历史变得有趣"。全书 7 部——2400 多页——是"中国最畅销的历史读物"之一。"成功只有一个——就是用自己的方式度过一生。"这是全书"最著名的结尾"——也是"当年明月对人生的理解"。

核心收获：
- **朱元璋的崛起**: 当年明月用"极其生动的笔触"讲述了"朱元璋的传奇人生"。朱元璋"从一个'放牛娃'到'和尚'到'乞丐'到'皇帝'"——是"中国历史上'最励志'的逆袭"。当年明月说"朱元璋的成功"靠的是"三个品质"：1）**坚韧**——"在'最绝望'的时候'不放弃'"；2）**识人**——"能'看出'谁值得信任"（如徐达、常遇春、李善长）；3）**狠**——"对敌人'毫不留情'"——"对功臣'也毫不留情'"。朱元璋的"复杂性"——"既是英雄"——又是"暴君"——是当年明月"刻画得最成功的角色"之一
- **朱棣夺权**: 朱棣（永乐大帝）是"明朝最'有争议'的皇帝"。他是"朱元璋的第四子"——被封为"燕王"——镇守北平。朱元璋死后——"皇位传给了'长孙'朱允炆（建文帝）"——朱棣"不服"——发动"靖难之役"——"夺取了皇位"。当年明月说"朱棣是'明朝最有作为'的皇帝之一"——他"迁都北京"、"修了紫禁城"、"派郑和下西洋"、"编了《永乐大典》"——但他的"皇位来路不正"——"成为了他'一生的心结'"
- **土木堡之变**: 当年明月讲述了"明朝最大的'军事灾难'"。明英宗朱祁镇"亲征瓦剌"——"五十万大军'全军覆没'"——皇帝"被俘虏"——这是"明朝由盛转衰的'转折点'"。但故事没有"到此为止"——于谦"力挽狂澜"——"拥立新皇帝"——"坚守北京"——"击退了瓦剌"——"挽救了明朝"。当年明月说"于谦是'明朝最伟大的'忠臣'"——他"在国家'最危险'的时候'挺身而出'"——但后来"被'恩将仇报'的朱祁镇杀害"——"让人扼腕"
- **张居正改革**: 张居正是"明朝最'有能力'的首辅"——他在"万历皇帝年幼时'摄政'"——推行了"一系列改革"：1）**一条鞭法**——"简化税制"——"把各种税'合并为一条'"；2）**考成法**——"考核官员'政绩'"——"不合格的'罢免'"；3）**整顿边防**——"重用戚继光'"——"巩固了'北方防线'"。当年明月说"张居正的改革'让明朝'起死回生'"——但"张居正死后"——"万历皇帝'清算了他'"——"改革成果'化为乌有'"
- **魏忠贤乱政**: 魏忠贤是"明朝最'臭名昭著'的宦官"——他"在天启年间'权倾朝野'"——"东林党人'被他大批杀害'"。当年明月说"魏忠贤的崛起"不是"因为他'有能力'"——是"因为他'迎合了'天启皇帝的'懒惰'"。天启皇帝"喜欢做木工"——"把朝政'交给了'魏忠贤"——魏忠贤"借机'铲除异己'"——"建立了一个'特务帝国'"。崇祯皇帝即位后"迅速除掉了魏忠贤"——但"明朝的'元气'已经'大伤'"
- **崇祯亡国**: 当年明月对"崇祯皇帝"的描写"令人唏嘘"。崇祯"不是'昏君'"——他"很勤政"——"每天早起上朝"——"节俭到'穿打补丁的衣服'"——但他"生性多疑"——"17 年换了 50 个内阁大学士"——"杀了'几十个'大臣"——"没有人敢'对他说真话'"。当年明月说"崇祯的悲剧"是"'越是努力——越是出错'"——"他的'勤政'反而'加速了'明朝的灭亡"。1644 年——李自成攻破北京——崇祯"在煤山自缢"——"明朝灭亡"
- **成功只有一个**: 当年明月在全书的"最后一段"写道："我写这么多——其实只想说一件事——成功只有一个——就是用自己的方式度过一生。"这句话是"全书的'点睛之笔'"——它"超越了历史"——成为了"对'每个人'的人生建议"。不是"功成名就"——不是"青史留名"——是"'按照自己的意愿'活着"
- **当年明月的背景**: 当年明月（本名石悦，1979-）——中国历史作家——《明朝那些事儿》是他"唯一的长篇作品"——但"足以让他'载入中国出版史'"。这本书 2006 年开始"在天涯论坛连载'"——"迅速走红'"——后来"出版成书'"——"累计销量超过 500 万册"。当年明月后来"进入仕途'"——但他的"作家身份'远比'官员身份'更为人所知'"
