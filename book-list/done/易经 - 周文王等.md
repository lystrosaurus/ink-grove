# 易经

## 基本信息
- **书名**: 易经
- **作者**: 周文王等
- **出版社**: 中华书局
- **出版年份**: 2020
- **页数**: 480
- **分类**: 国学经典

## 选书理由
中国最古老的经典，群经之首。用六十四卦符号系统表达宇宙变化规律，"易"有三义：变易（万事万物都在变化）、不易（变化本身是不变的规律）、简易（复杂现象背后有简单原理）。虽然常被用于占卜，但其哲学内涵（阴阳、刚柔、时位）影响深远。希望理解中国式的变化哲学，在不确定性中把握规律。
