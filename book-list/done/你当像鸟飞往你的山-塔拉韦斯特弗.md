# 你当像鸟飞往你的山

状态：已读

作者：塔拉-韦斯特弗

一句话评价：一个在废料场长大的女孩，靠教育逃离原生家庭，飞到剑桥和哈佛——但教育的代价是与过去的自己彻底决裂。

核心收获：
- 教育不只是获取知识，更是"重新认识自己"和"重新定义世界"的过程
- 塔拉17岁前没上过学，在家里废料场帮父亲干活。父亲是极端宗教信仰者，不相信政府、医院、学校
- 通过自学考上杨百翰大学，然后一路到剑桥博士、哈佛访学。每一步都在撕裂原来的世界
- 教育的双刃剑：它给了她自由，也让她失去了家庭。她在两个世界之间找不到归属
- "你当像鸟飞往你的山"出自《圣经·诗篇》，意思是逃离，也意味着找到属于自己的地方
