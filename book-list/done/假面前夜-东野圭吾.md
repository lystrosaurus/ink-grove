# 假面前夜

状态：已读

作者：东野圭吾

一句话评价：《假面饭店》的前传短篇集，四个故事从不同视角铺垫了"科尔特西亚"酒店的谜团，人物动机比正传更立体。

核心收获：
- 四个短篇分别从酒店员工、客人、侦探、凶手的视角展开，拼出完整的事件前史
- 东野圭吾最擅长的是"动机的合理性"——每个看似不可理喻的行为背后都有一个让人同情的理由
- 酒店是完美的悬疑场景：每天接待陌生人、每个人都有秘密、服务行业的"微笑面具"天然适合隐藏真实身份
- 新田浩介和山岸尚美这对搭档在本书中还是"前传状态"，能看到他们性格形成的线索
- 短篇的局限：故事之间关联性不够强，更像"设定集"而非独立作品
