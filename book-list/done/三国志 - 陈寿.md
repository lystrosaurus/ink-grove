# 三国志

状态：已读

作者：陈寿

一句话评价：二十四史中最"好看"的一部——人物鲜活、叙事紧凑，裴松之的注解比正文还精彩。陈寿用极其精炼的笔法记录了从黄巾之乱到三国归晋的百年历史，是了解三国历史的第一手材料。

核心收获：
- **以魏为正统**: 陈寿生活在晋朝，晋承魏统，所以《三国志》以《魏书》为正统，曹操、曹丕入"纪"，刘备、孙权只能入"传"。但这不影响他对蜀吴的公正——诸葛亮、周瑜、陆逊的传记一样精彩
- **曹操的复杂性**: "治世之能臣，乱世之奸雄"——这是《三国志》对曹操最精炼的概括。陈寿不美化也不丑化，他写曹操的雄才大略（官渡之战、统一北方），也写他的残忍多疑（杀吕伯奢、杀杨修）。《三国志》里的曹操是历史上最立体的政治人物之一
- **诸葛亮的真实形象**: 陈寿评价诸葛亮"治戎为长，奇谋为短，理民之干，优于将略"——军事才能不如政治才能，"鞠躬尽瘁"比"神机妙算"更准确。草船借箭、空城计、借东风全是《三国演义》虚构的。真实的诸葛亮是一个伟大的政治家和行政管理者，不是"算无遗策"的军师
- **裴松之注解**: 南朝宋裴松之为《三国志》做的注解是史学史上的奇迹——注解的篇幅是正文的三倍。他补充了大量陈寿没有采用的史料，包括《魏略》《江表传》《汉晋春秋》等。很多后来被《世说新语》《后汉书》引用的典故都来自裴注。读三国志不读裴注，等于只读了一半
- **人物传记的精彩**: 《三国志》的好看不在于"叙事技巧"，在于"人物本身就好看"。关羽的"刮骨疗毒"、张飞的"据水断桥"、周瑜的"谈笑间樯橹灰飞烟灭"、陆逊的"火烧连营"——陈寿用最少的字写出了最鲜活的人
- **三国志 vs 三国演义**: 这是每个读三国志的人必须搞清楚的区别。三国志是"史"（陈寿著），三国演义是"小说"（罗贯中著）。桃园三结义是虚构的，三英战吕布是虚构的，草船借箭是虚构的，空城计是虚构的——但虚构不等于不好。《三国演义》之所以成为经典，不是因为"忠实于历史"，是因为它把历史人物塑造成了文化符号
- **陈寿的史德**: 陈寿写史有"曲笔"——他为司马氏避讳、为晋朝开国找合法性。但总体而言，《三国志》是二十四史中史料价值最高的一部之一。司马光写《资治通鉴》的三国部分，大量引用《三国志》
- **三国历史的教训**: 三国的百年动荡告诉我们什么？人才决定成败（曹操的用人、刘备的三顾茅庐、孙权的知人善任），但制度决定持久（三国的继承问题都没解决好，最终都被司马氏收割）。个人英雄主义可以赢一场战争，但赢不了一个时代
