# 史记

状态：已读

作者：司马迁

一句话评价：司马迁"纪传体通史开山之作'"——"'史家之绝唱，无韵之离骚'"。"'五体结构开创史学先河'"。"'项羽本纪、刺客列传等篇章人物鲜活'"。"'人固有一死激励千古'"。"'中国史学与文学的双重巅峰'"

核心收获：
- **五体结构**: 本书核心是"'五体结构'"。"'本纪'（帝王）'"——"'表'（大事'年表）'"——"'书'（典章'制度）'"——"'世家'（诸侯）'"——"'列传'（人物）'"。"'这'套'体例'影响'了'此后'两千年'的'中国'史书'"
- **项羽本纪**: 本书"记载了项羽本纪'"。"'司马迁'把'失败'的'项羽'写进'了'本纪'"——"'这'是'史学家'的'公正'"——"'也'是'文学家'的'同情'"。"'项羽'的'悲剧'是'性格'的'悲剧'"
- **孔子世家**: 本书"记载了孔子世家'"。"'把'孔子'这样'一个'没有'封地'的'学者'列入'世家'"——"'司马迁'对'知识'和'道德'的'最高'敬意'"
- **刺客列传**: 本书"记载了刺客列传'"。"'荆轲'"——"'专诸'"——"'聂政'"——"'豫让'"。"'司马迁'对'士为知己者死'的'浪漫化'书写'"
- **货殖列传**: 本书"记载了货殖列传'"。"'为'中国'商人'立传'"——"'司马迁'的'经济'思想'超越'时代'"。"'他'认为'追求'财富'是'人之常情'"——"'不'应该'被'道德'批判'"
- **太史公自序**: 本书"有太史公自序'"。"'司马迁'自述'写作'动机'"："'究'天人'之际'"——"'通'古今'之变'"——"'成'一家'之言'"。"'一个'被'宫刑'的'人'"——"'用'一部'书'完成'了'自我'救赎'"
- **司马迁的史德**: 本书"体现了司马迁的史德'"。"'不'避讳'"——"'不'粉饰'"——"'不'偏袒'"。"'他'写'刘邦'的'无赖'"——"'汉武帝'的'迷信'"——"'吕后'的'残忍'"。"'不'虚美'"——"'不'隐恶'"
- **作者的背景**: 司马迁（约公元前 145-前 86）——西汉史学家、文学家——"《史记》的作者'"。"'他'父亲'司马谈'是'太史令'"。"'他'继承'父业'"——"'立志'写'一部'通史'"。"'李陵'之祸'"："'他'为'李陵'辩护'"——"'被'汉武帝'处以'宫刑'"。"'他'忍辱'负重'"——"'用'13 年'完成'《史记》'"。"'《史记》'130 篇'"："'12 本纪'"——"'10 表'"——"'8 书'"——"'30 世家'"——"'70 列传'"
