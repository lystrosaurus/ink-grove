# 千家诗

状态：已读

作者：多人

一句话评价：中国古代最流行的启蒙诗歌选本，收录唐宋名家诗歌二百余首，按春夏秋冬四季编排。与《唐诗三百首》相比，千家诗更浅显、更生活化，是古代孩童学诗的"第一本书"。

核心收获：
- **千家诗的编纂**: 千家诗不是一次编成的，是南宋刘克庄先从唐诗中选出"千家诗"，后来不断增补，到明清时期定型为 226 首。选诗标准不是"文学价值最高"，是"最适合启蒙"——语言浅显、意境优美、朗朗上口
- **四季编排的智慧**: 千家诗按"春、夏、秋、冬"四季编排，每个季节选几十首诗。春天是"春眠不觉晓""好雨知时节"，夏天是"接天莲叶无穷碧"，秋天是"停车坐爱枫林晚"，冬天是"千山鸟飞绝"。这种编排让孩童在学诗的同时感受四季的变化
- **千家诗 vs 唐诗三百首**: 唐诗三百首是蘅塘退士（孙洙）在清代编的，选诗 310 首，更注重"文学价值"和"风格多样"。千家诗选诗 226 首，更注重"启蒙"和"通俗"。千家诗里有很多"不太出名但很好懂"的诗，唐诗三百首里有很多"出名但需要一定功底才能懂"的诗
- **经典名篇**: 千家诗收录了许多中国人最熟悉的诗：孟浩然《春晓》、杜甫《绝句》（两个黄鹂鸣翠柳）、王安石《元日》、苏轼《题西林壁》、陆游《示儿》。很多人"会背"这些诗，但不知道它们出自千家诗
- **启蒙教育的理念**: 古代启蒙教育讲究"先识字后明理"，千家诗的作用不仅是学诗，是学汉字、学韵律、学审美、学人情世故。一首诗里有自然（"春风又绿江南岸"）、有人情（"慈母手中线"）、有历史（"商女不知亡国恨"）、有哲理（"不识庐山真面目"）
- **千家诗的"俗"与"雅"**: 千家诗被一些文人批评为"太俗"——选的诗不够"高深"。但正是这种"俗"让它成为了最普及的诗选。王国维说"一切景语皆情语"，千家诗选的景语都是最贴近日常生活的——春花秋月、晨昏雨雪、田园山水
- **千家诗的流传**: 千家诗在明清时期是最畅销的"童书"，几乎家家户户都有一本。它的版本极多，有插图本、注释本、音韵本、女版（专门给女孩读的）。近代以后逐渐被《唐诗三百首》取代，但在民间的影响力依然巨大
- **当代意义**: 在今天这个快节奏的时代，千家诗的价值反而凸显——它教孩子（和大人）慢下来，感受一朵花、一场雨、一轮月。"熟读唐诗三百首，不会作诗也会吟"——千家诗的作用不是"成为诗人"，是"成为有诗意的人"
