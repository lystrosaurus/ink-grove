# 长安的荔枝

状态：已读

作者：马伯庸

一句话评价：马伯庸的"历史小说佳作'"——"'以"一骑红尘妃子笑"为引子'"。"'写了一个唐朝小吏如何在极限条件下完成"从岭南运鲜荔枝到长安"的不可能任务'"。"'既是精彩的历史小说，也是一堂生动的项目管理课'"。"'小人物在大系统中的困境'"——"'就算失败，我也想知道，自己倒在距离终点多远的地方'"

核心收获：
- **小人物的困境**: 本书核心是"'小人物在大系统中的困境'"。"'主角'李善德'"："'一个'小吏'"——"'九品'官'"。"'他'接到'任务'"："'从'岭南'运'鲜'荔枝'到'长安'"。"'为什么'是'不可能'任务'"：1）**距离远**——"'5000 里'"；2）**保鲜难**——"'荔枝'一日'而'色变'"；3）**时间短**——"'必须'在'贵妃'生日'前'送到'"。"'李善德'的'困境'"："'接'任务'"——"'死'"——"'不'接'"——"'也'死'"
- **项目管理视角**: 本书"用现代项目管理视角展开'"。"'李善德'如何'完成'任务'"：1）**路线规划**——"'计算'路线'"——"'选择'最快'的'"；2）**保鲜实验**——"'实验'各种'保鲜'方法'"；3）**资金筹措**——"'找'钱'"——"'预算'"；4）**跨部门协调**——"'找'各个'部门'帮忙'"。"'读起来'像'古代'版'项目管理'"——"'有'计划'"——"'有'执行'"——"'有'风险'管理'"
- **极限条件下的创新**: 本书"展现了极限条件下的创新'"。"'李善德'如何'创新'"：1）**双层瓮**——"'用'双层'瓮'保鲜'"；2）**冰块**——"'用'冰'降温'"；3）**接力**——"'用'快马'接力'"。"'这些'创新'"："'在'当时'是'先进'的'"。"'创新'的'动力'"："'生存'压力'"
- **官场的描绘**: 本书"描绘了唐代官场'"。"'官场'的'特点'"：1）**推诿**——"'各部门'推诿'"；2）**腐败**——"'官员'贪污'"；3）**权力**——"'权力'决定'一切'"。"'李善德'在'官场'中'"："'小'人物'"——"'被'利用'"——"'被'牺牲'"。"'但'他'也'利用'官场'"："'找'关系'"——"'找'资源'"
- **名句的力量**: 本书"有震撼人心的名句'"。"'"就算失败，我也想知道，自己倒在距离终点多远的地方'"。"'这句话'的'意义'"：1）**坚持**——"'不'放弃'"；2）**探索**——"'知道'极限'在哪'"；3）**尊严**——"'即使'失败'"——"'也'有'尊严'"。"'这句话'是'全书'最'燃'的'瞬间'"
- **历史与虚构**: 本书"结合了历史与虚构'"。"'历史'部分'"："'唐朝'背景'"——"'杨贵妃'"——"'荔枝'故事'"。"'虚构'部分'"："'李善德'这个'人物'"——"'具体'过程'"。"'马伯庸'的'手法'"："'大'历史'真实'"——"'小'细节'虚构'"。"'这种'手法'"："'让'历史'生动'"
- **作者的背景**: 马伯庸（1980-）——中国当代作家——"《长安的荔枝》的作者'"。"'他'擅长'历史'小说'"——"'《风起陇西》'——'《古董局中局》'"。"'他'的'风格'"："'历史'+'悬疑'"——"'历史'+'现代'视角'"。"《长安的荔枝》'2022 年'出版'"——"'畅销'"。"'他'的'作品'"："'被'改编'成'影视'剧'"
