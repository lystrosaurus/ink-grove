# 谁在看中国画

状态：已读

作者：柯律格

一句话评价：艺术史佳作，柯律格跳出"名作赏析"的传统框架，从观看者的社会身份出发，揭示中国绘画在不同时代、不同人群中的功能和意义。

核心收获：
- 理解了"谁在看"比"看什么"更重要——同一幅画，帝王、文人、商人、工匠看到的东西完全不同
- 中国画的鉴赏不是纯审美活动，而是社会身份的展演和文化资本的较量
- 艺术市场的形成、收藏品的流通、画作的复制与传播，构成了一个完整的社会网络
