# 后汉书

状态：已读

作者：范晔

一句话评价：南朝宋范晔编纂的纪传体断代史，记载东汉近两百年历史，与《史记》《汉书》《三国志》并称"前四史"。范晔以一人之力修撰本纪十卷、列传八十卷（志为后人补入），文辞华美、议论深刻，尤其《党锢列传》《逸民列传》记录了东汉知识分子的风骨，是了解东汉历史的必读之作。

核心收获：
- **东汉的历史框架**: 《后汉书》记载了从光武帝刘秀重建汉室（公元 25 年）到汉献帝禅位给曹丕（公元 220 年）的近两百年历史。东汉的特点是"外戚与宦官交替专权"——皇帝年幼时太后临朝、外戚掌权，皇帝成年后依靠宦官夺权，宦官又成新贵。这个循环贯穿了整个东汉，最终导致黄巾起义和群雄割据
- **光武中兴**: 刘秀是东汉开国皇帝，也是历史上少有的"完美皇帝"——出身皇室但家道中落，起兵反莽时表现出色的军事才能，称帝后"退功臣而进文吏"（不杀功臣但也不让他们掌权），实行"休养生息"政策。范晔对刘秀的评价极高，称他"虽身济大业，兢兢如不及"
- **外戚与宦官的权力斗争**: 这是《后汉书》最核心的叙事线索。外戚（如窦宪、梁冀）和宦官（如"十常侍"）交替掌权，皇帝往往是"傀儡"。范晔在《宦者列传》中痛斥宦官"手握王爵，口含天宪"——把国家权力变成了私人工具。这种权力结构的腐败最终导致了东汉的灭亡
- **党锢之祸**: 《后汉书》最动人的篇章之一是《党锢列传》。东汉末年，太学生和士大夫联合批评宦官专权，被称为"党人"。两次"党锢之祸"中，数百名知识分子被杀、被囚、被流放。但这些人"宁为玉碎不为瓦全"——李膺、杜密、范滂等人的气节令人敬佩。范滂临刑前对母亲说"弟仲博孝敬，足以供养母亲，我随父亲去地下，也算死得其所"——这种知识分子的风骨是中国文化的精神脊梁
- **《逸民列传》的价值**: 范晔首创《逸民列传》，记录了那些不愿做官、隐居山林的知识分子。这在中国史学史上是第一次——以前的史书只记录"有功名的人"，范晔开始关注"有品格的人"。严光（严子陵）拒绝光武帝的征召、继续隐居钓鱼——范晔对他的评价是"不降其志，不辱其身"。这种对"隐士"的尊重体现了范晔的价值观
- **范晔的史学才华**: 范晔（398-445）是南朝宋的史学家、文学家。他写《后汉书》时参考了十多家东汉史书（如《东观汉记》《后汉记》），但"不袭前人"——有自己的裁剪和评判。范晔的文辞华美，擅长"论赞"——每篇末尾的"论曰""赞曰"是他对历史人物和事件的评价。他自己说"吾杂传论皆有精意深旨"——不是随便写的，是认真思考后的判断
- **《后汉书》的志**: 范晔只写了本纪和列传，没有写"志"（记载典章制度、天文地理等）。后来梁朝的刘昭把司马彪《续汉书》的八志（律历、礼仪、祭祀、天文、五行、郡国、百官、舆服）补入《后汉书》，形成了今天看到的 120 卷版本。所以《后汉书》是"两个人的作品"——范晔的本纪和列传、司马彪的志
- **前四史的比较**: 《史记》是"通史"（从黄帝到汉武帝），《汉书》是"断代史"（西汉一代），《后汉书》是"断代史"（东汉一代），《三国志》是"断代史"（三国时期）。《史记》最"文学"，《汉书》最"严谨"，《后汉书》最"有观点"，范晔的议论和评判是四史中最鲜明的
- **《后汉书》的局限**: 范晔因为参与谋反被杀（445 年），《后汉书》没有最终完成。他写的是"初稿"——有些地方"草创未就"。但即使是不完整的《后汉书》，也因其"文采"和"见识"而成为"前四史"之一。范晔的死是中国史学的一大损失——如果他完成了，这部书可能会更加精彩
