# 中国通史

状态：已读

作者：吕思勉

一句话评价：吕思勉用极简的笔法写完了从上古到民国的中国历史——不堆砌史料，不罗列帝王，而是抓住每个时代的社会结构、经济基础和文化精神。上编讲"文化现象"的写法在 1940 年代极其超前，至今仍是通史写法的典范。

核心收获：
- **独特的双编结构**: 上编讲"文化现象"（婚姻、族制、政体、阶级、财产、官制、选举、赋税、兵制、刑法、实业、货币、衣食、住行、教育、语文、宗教），下编按朝代讲政治变迁。这种"先社会后政治"的写法在中国史学中极其罕见——大多数通史从帝王将相讲起，吕思勉从普通人的生活讲起
- **上编的超前性**: 不讲帝王将相，讲普通人的生活。婚姻制度怎么演变、财产怎么分配、教育怎么发展——这些才是历史的真正底色。吕思勉在 1940 年代就用社会学、经济学的眼光看历史，比"新史学"浪潮早了半个世纪
- **历史分期的洞见**: 封建社会（先秦）→ 郡县社会（秦汉至清）→ 近代转型。他明确指出"封建"只适用于先秦——秦以后是"郡县制"，不是"封建"。这个区分在今天的学术界依然是共识，但吕思勉在 80 年前就说清楚了
- **儒家与法家的关系**: "外儒内法"是中国两千年政治的核心密码。儒家是"理想"——仁政、德治、礼乐；法家是"现实"——严刑峻法、中央集权、官僚控制。表面上是儒家的天下，实际上运作靠的是法家的手段。这个洞察至今适用
- **对历代政治的评价**: 吕思勉不美化任何一个朝代。他说汉代"外宽内忌"、唐代"前期开明后期腐败"、宋代"富而不强"、明代"专制而混乱"、清代"集权而封闭"。每个朝代的优点和缺点他都一针见血
- **经济史的眼光**: 吕思勉特别关注经济基础——土地制度、赋税制度、货币制度。他认为中国历史的很多政治变动背后是经济原因：土地兼并导致农民起义、赋税过重导致王朝崩溃、货币混乱导致社会动荡。这种"经济决定论"的视角在传统史学中很少见
- **对近代转型的分析**: 吕思勉写到民国，他对近代中国的分析极其冷静——不是简单地说"西方侵略导致中国落后"，而是分析中国内部的结构性问题：科举废除后知识分子去哪了、传统社会结构解体后新的秩序怎么建立、现代化是不是等于西方化
- **吕思勉的史学地位**: 与钱穆、陈寅恪、陈垣并称为"现代史学四大家"。吕思勉的特点是"通"——他不是某个断代的专家，他是整个中国历史的通家。他的《中国通史》《白话本国史》《先秦史》《秦汉史》《三国史话》构成了完整的中国历史写作体系
