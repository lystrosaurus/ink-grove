# 中国通史

状态：已读

作者：吕思勉

一句话评价：吕思勉用极简的笔法写完了从上古到民国的中国历史——不堆砌史料，不罗列帝王，而是抓住每个时代的社会结构、经济基础和文化精神。

核心收获：
- 吕思勉的通史分上下两编：上编讲"文化现象"（婚姻、族制、政体、阶级、财产、官制、选举、赋税、兵制、刑法、实业、货币、衣食、住行、教育、语文、宗教），下编按朝代讲政治变迁
- 上编的写法极其超前——不讲帝王将相，讲普通人的生活。婚姻制度怎么演变、财产怎么分配、教育怎么发展——这些才是历史的真正底色
- 吕思勉对中国历史分期的看法：封建社会（先秦）→ 郡县社会（秦汉至清）→ 近代转型。他明确指出"封建"只适用于先秦，秦汉以后是"郡县制"
- 对儒家和法家的分析极其精辟：儒家是"理想"，法家是"现实"。中国两千年的政治一直是"外儒内法"
- 吕思勉写于1940年代，但他对历史的理解远超同时代人。他用社会学、经济学的眼光看历史，不是传统的"治乱兴衰"叙事
