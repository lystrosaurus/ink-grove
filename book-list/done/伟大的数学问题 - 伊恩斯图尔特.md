# 伟大的数学问题

状态：已读

作者：伊恩·斯图尔特

一句话评价：把数学史上最美的未解之谜和已解之谜用讲故事的方式呈现，让不懂数学的人也能感受到"数学之美"。从费马大定理到庞加莱猜想，每一个问题背后都是人类智力的极限挑战。

核心收获：
- **费马大定理**: "x^n + y^n = z^n 在 n>2 时没有整数解。"费马在 1637 年写下这个猜想，说"我发现了一个绝妙的证明，但空白太小写不下"。此后 350 年，无数数学家前赴后继，直到 1995 年安德鲁·怀尔斯用椭圆曲线和模形式证明了它。这是人类毅力的极限考验
- **四色定理**: "任何地图只需要四种颜色，就能让相邻区域颜色不同。"这个猜想看似简单，但证明极其困难。1976 年阿佩尔和哈肯用计算机穷举了 1936 种情况才证明。这是数学家第一次承认"我证不出来，但机器能"——引发了关于"什么是证明"的哲学争论
- **庞加莱猜想**: "三维空间中任何封闭的曲面都等价于球面。"这是拓扑学的核心问题，困扰了数学家近一个世纪。2003 年俄罗斯数学家佩雷尔曼证明了它，然后拒绝了菲尔兹奖和百万美元奖金，说"我对钱和名誉不感兴趣"
- **哥德尔不完备定理**: "任何足够复杂的数学系统里，都存在'既不能证明也不能证伪'的命题。"哥德尔 1931 年证明了这个定理，粉碎了希尔伯特"数学是完备且一致"的梦想。数学不是万能的——总有一些真理是我们无法用系统内的规则证明的
- **黎曼猜想**: "黎曼ζ函数的所有非平凡零点都位于复平面的实部为 1/2 的直线上。"这是数学界最重要的未解之谜，和素数分布密切相关。如果证明了，密码学、数论、物理学都会发生地震。克雷数学研究所悬赏 100 万美元
- **P vs NP 问题**: "如果一个问题的解可以在多项式时间内验证，那它是否也可以在多项式时间内求解？"这是计算机科学的圣杯。如果 P=NP，意味着所有"难解"的问题（如密码破解、旅行商问题）其实都有快速解法。大多数人相信 P≠NP，但没人能证明
- **连续统假设**: "是否存在一个集合，它的元素数量严格介于自然数和实数之间？"康托尔提出了这个问题，但哥德尔和科恩证明了它"在标准集合论中既不能证明也不能证伪"。这意味着数学的基础比我们想象的更灵活
- **数学问题的价值**: 斯图尔特说数学问题的价值不在于"有没有用"（很多数学问题几百年后才找到应用），而在于"它是否揭示了深层的结构"。费马大定理的证明发展出了椭圆曲线理论，后来成为了现代密码学的基础。数学的美在于"无用之用"
