# 人性论

状态：已读

作者：休谟

一句话评价：经验主义哲学的巅峰之作，用"印象-观念-推理"的框架解构人类认知，休谟的怀疑论至今仍在挑战每一个自以为理性的人。

核心收获：
- "印象"（直接感知）和"观念"（印象的副本）区分了经验的两个层次
- 因果关系不是客观规律，而是"恒常联结"产生的心理习惯——这是休谟最著名的论断
- 理性是激情的奴隶，不是主人。人类行为由情感驱动，理性只是工具
- 归纳法没有逻辑基础——过去每天太阳升起，不能逻辑地推出明天太阳一定升起
- "是"推不出"应该"——事实判断和价值判断之间有不可跨越的鸿沟（休谟法则）
