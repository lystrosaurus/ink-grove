# 万物简史

状态：已读

作者：比尔-布莱森

一句话评价：用幽默的笔调把宇宙大爆炸到人类文明的历史讲成段子合集——科学的进步不是英雄史诗，而是一群古怪的人不断犯错、纠正、再犯错的故事。

核心收获：
- 宇宙有多大：可观测宇宙直径约930亿光年，但大部分是空的。如果把原子核放大到足球场大小，电子在看台最外圈
- 地球多老：45.5亿年。这个数字是通过测量陨石中铅的同位素比例算出来的
- 生命多偶然：如果地球的历史压缩成24小时，人类只在最后3秒出现。绝大多数时间里地球上只有微生物
- 科学家的故事：卡文迪许害羞到不和女性说话，却在实验室发现了氢气；富兰克林冒险放风筝证明闪电是电
- 科学的本质不是"正确"，是"不断修正错误"。每一个伟大发现背后都有十个被遗忘的错误理论
