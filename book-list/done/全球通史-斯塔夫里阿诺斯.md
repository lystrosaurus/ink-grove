# 全球通史

状态：已读

作者：斯塔夫里阿诺斯

一句话评价：用"全球视角"重写人类历史，跳出"西方中心论"和"国别史"的局限，第一次把人类当作一个整体来讲述。

核心收获：
- 历史不是"各国的历史"，而是"人类的历史"。文明的交流、碰撞、融合才是主线
- 史前史的价值：农业革命是人类最重要的转折点，比任何王朝更替都重要
- 欧亚大陆的优势不是"人更聪明"，是地理条件（东西轴向、可驯化动植物多）
- 1500年是全球史的分水岭——从此世界真正连成一个整体
- 看待今天的全球化，要放在"500年全球化进程"的大框架下理解
