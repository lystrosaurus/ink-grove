# 论语

状态：已读

作者：孔子弟子及再传弟子

一句话评价：孔子言行的记录，儒家思想的集大成者——"不是哲学论文，是一个老人在和他的学生聊天，但聊出了中国文化的底色'"。"'仁'是论语的核心'"——"'学而时习之'的'习'是实践'"——"'因材施教'的教育方法'"——"'君子'是品格不是地位'"

核心收获：
- **仁的多重解释**: "仁'是论语的核心'"。"'孔子'对'仁'的'解释'每次'都'不同'"。"'对'颜回'说'"——"'克己复礼为仁'"——"'克制'自己'"——"'符合'礼'"。"'对'樊迟'说'"——"'爱人'"——"'仁'就是'爱'别人'"。"'对'子贡'说'"——"'己所不欲，勿施于人'"——"'推己及人'"。"'不'是'孔子'没有'定论'"——"'是'仁'本身'就是'情境化'的'"。"'仁'不'是'一个'概念'"——"'仁'是'一种'实践'"——"'在'不同'情境'下'有'不同'表现'"
- **学而时习之**: 论语开篇"学而时习之，不亦说乎'"。"'习'不'是'复习'"——"'习'是'实践'"。"'学'了'之后'去'做'了'"——"'这'才'是'习'"。"'所以'孔子'说'学而时习之，不亦说乎'"——"'因为'实践'比'读书'更'有'成就感'"。"'孔子'重视'实践'"——"'他'说'听其言而观其行'"——"'不'是'听'人'说'什么'"——"'是'看'人'做'什么'"。"'这个'观点'影响'了'中国'的'教育'传统'"——"'知行合一'"
- **因材施教**: 孔子"的教育方法是因材施教'"。"'同样'的'问题'"——"'孔子'给'不同'的'答案'"。"'子路'问'闻斯行诸'"（听到'了'就'做'吗）'"——"'孔子'说'有'父兄'在'——'先'问问'他们'"。"'冉有'问'同样'的'问题'"——"'孔子'说'闻斯行之'"（听到'了'就'做）。"'公西华'不'理解'"——"'孔子'解释'"——"'子路'太'冲动'"——"'所以'压'一'压'他'"——"'冉有'太'犹豫'"——"'所以'推'一'推'他'"。"'教育'不'是'给'答案'"——"'教育'是'补'不足'"
- **君子的重新定义**: 孔子"重新定义了君子'"。"'君子'原来'是'地位'概念'"——"'君子'是'贵族'"。"'孔子'把'君子'重新定义'为'品格'概念'"——"'君子'是'有德之人'"。"'君子'的'特点'"：1）**仁**——"'有'仁爱之心'"；2）**义**——"'做'应该'做'的'事'"；3）**礼**——"'遵守'礼仪'"；4）**智**——"'有'智慧'"；5）**信**——"'守'信用'"。"'这个'转变'很'关键'"——"'从'此'以后'"——"'君子'不'再'是'出身'决定'"——"'是'修养'决定'"
- **论语的语言**: 论语的"语言极其精炼'"。"'几乎'每句'都'可以'当'格言'用'"。"'比如'"：1）**三人行，必有我师**——"'虚心'学习'"；2）**知之为知之，不知为不知**——"'诚实'对待'知识'"；3）**温故而知新**——"'复习'中'发现'新'东西'"。"'但'格言'不'等于'真理'"——"'需要'在'具体'情境'中'理解'"。"'论语'不'是'系统'的'哲学'"——"'论语'是'片段'的'智慧'"
- **孔子的教育**: 孔子"是中国第一个私人教师'"。"'他'有'三千'弟子'"——"'七十二'贤人'"。"'他'的'教育'内容'"：1）**诗**——"'《诗经》'"；2）**书**——"'《尚书》'"；3）**礼**——"'礼仪'"；4）**乐**——"'音乐'"；5）**易**——"'《易经》'"；6）**春秋**——"'历史'"。"'他'的'教育'目的'"——"'培养'君子'"——"'培养'治国平天下'的'人'"。"'孔子'的'教育'思想'"——"'有教无类'"——"'不管'出身'"——"'都'可以'学习'"
- **论语的成书**: 论语"不是孔子写的'"。"'论语'是'孔子'弟子'及'再传弟子'记录'的'"。"'记录'了'孔子'的'言行'"——"'也'记录'了'一些'弟子'的'言行'"。"'论语'成书'于'战国'初期'"——"'距离'孔子'去世'约'100 年'"。"'论语'有'20 篇'"——"'每篇'有'若干'章'"。"'《论语》'的'地位'"——"'四书'之一'"——"'中国'人'的'必读'书'"——"'影响'了'两千'多年'"
