# 呐喊

状态：已读

作者：鲁迅

一句话评价：鲁迅"第一部小说集'"——"'狂人日记开创中国现代白话文学'"。"'阿 Q 正传的国民性批判至今振聋发聩'"。"'孔乙己、药等篇篇经典'"。"'14 篇小说勾勒出中国人的精神图谱'"。"'现代文学必读之作'"

核心收获：
- **狂人日记**: 本书收录"'狂人日记'"。"'中国'第一'篇'现代'白话'小说'"。"'狂人'看穿'了'礼教'的'本质'"——"'吃人'"。"'谁'是'狂人'"——"'清醒'的'人'在'疯癫'的'社会'里'就是'疯子'"
- **阿 Q 正传**: 本书收录"'阿 Q 正传'"。"'阿 Q 的精神胜利法'是'中国'国民性'最'深刻'的'诊断'"。"'被'打'后'说'儿子'打'老子'"。"'临死'前'担心'画圈'不'够'圆'"。"'鲁迅'的'讽刺'入木三分'"
- **孔乙己**: 本书收录"'孔乙己'"。"'一个'穿'长衫'但'站'着'喝酒'的'人'"——"'旧'知识分子'的'悲剧'"。"'窃书'不'能'算'偷'是'孔乙己'最后'的'尊严'"——"'也'是'最大'的'悲哀'"
- **药**: 本书收录"'药'"。"'华老栓'用'革命者'夏瑜'的'血'蘸'馒头'给'儿子'治病'"——"'愚昧'与'革命'的'双重'悲剧'"。"'革命者'的'血'被'当成'药'"——"'这'是'最'深刻'的'讽刺'"
- **明天**: 本书收录"'明天'"。"'单四嫂子'失去'儿子'后'的'绝望'"——"'底层'妇女'的'无声'悲剧'"
- **一件小事**: 本书收录"'一件小事'"。"'车夫'撞到'老妇人'后'主动'承担'责任'"。"'鲁迅'少有'的'正面'书写'"——"'普通人'身上'的'道德'光辉'"
- **鲁迅的笔法**: 本书"体现了鲁迅的笔法'"。"'冷峻'"——"'辛辣'"——"'精准'"。"'他'不'写'好人'坏人'"——"'只'写'人'"。"'人性'的'复杂'与'荒诞'"
- **作者的背景**: 鲁迅（1881-1936）——中国作家、思想家、革命家——"《呐喊》的作者'"。"'他'原名'周树人'"——"'浙江'绍兴'人'"。"'《呐喊》'1923 年'出版'"——"'收录'14 篇'小说'"。"'他'的'其他'著作'"："'《彷徨》'"——"'《朝花夕拾》'"——"'《野草》'"。"'鲁迅'的'主题'"："'国民性'批判'"——"'启蒙'"——"'反封建'"。"'他'的'影响'"："'中国'现代'文学'奠基人'"
