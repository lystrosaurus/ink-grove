# 祈祷落幕时

状态：已读

作者：东野圭吾

一句话评价：东野圭吾"加贺恭一郎系列的收官之作"——"通过'一桩命案'揭开了'两个家庭'跨越数十年的'悲剧'"——"展现了'父母之爱的'极致与沉重'"。"东野圭吾'用'推理小说的外壳'讲述'亲情故事'"——"'谜底'不是'谁是凶手'"——"'是'为什么杀人'"。"父母之爱'可以走到'多极端'"——"'为了保护孩子'"——"'可以'牺牲自己的一生'"。"加贺恭一郎的'个人身世'与案件'产生了共鸣'"——"'这是'整个系列'最动人的'终章'"

核心收获：
- **加贺系列的终章**: 《祈祷落幕时》是"加贺恭一郎系列的'最后一部'"。"在这个系列中'"——"加贺'从一个'大学生侦探'"——"成长为'一个'成熟的'刑警'"——"他'解决'了'很多'案件'"——"但'这个'案件'"——"与'他'自己的'身世'产生了'共鸣'"。"加贺'一直'在'寻找'母亲'离开的'真相'"——"'在这个案件中'"——"'他'终于'找到了'答案'"——"'这'是'整个系列'的'收官'——"'也是'加贺'个人故事'的'完结'"
- **两个家庭的悲剧**: 东野圭吾"讲述了'两个家庭'跨越数十年的'悲剧'"。"一个家庭'"——"'母亲'为了'保护女儿'"——"'隐瞒'了'一个'秘密'"——"'这个秘密'毁了'她'的'一生'"。"另一个家庭'"——"'父亲'为了'保护儿子'"——"'牺牲'了'自己的'一切'"——"'甚至'生命'"。"这两个家庭'本来是'没有'交集的'"——"'但'因为'一个'共同的'秘密'"——"'他们'的'命运'交织在了一起'"
- **父母之爱的极致**: 东野圭吾"探讨了'父母之爱'可以走到'多极端'"。"'为了'保护'孩子'"——"'父母'可以'做什么'"——"'可以'隐瞒真相'"——"'可以'牺牲自己的幸福'"——"'可以'杀人'"——"'可以'被杀'"。"东野圭吾说'父母之爱是'最伟大的'爱'"——"'但'也'可能是'最沉重的'爱'"——"'因为'它'要求'父母'付出'一切'"——"'而'孩子'可能'永远'不知道'"
- **为什么杀人**: 东野圭吾"在这本书中'"——"'谜底'不是'谁是凶手'"——"'是'为什么杀人'"。"'凶手'不是'坏人'"——"'是'一个'被'逼到'绝境的'普通人'"——"'他'杀人'"——"'不是'为了'自己'"——"'是'为了'保护'所爱的人'"。"东野圭吾说'有时候'杀人'不是'因为'恨'"——"'是'因为'爱'"——"'这种'爱'扭曲了'"——"'变成了'杀人的'动机'"
- **加贺的个人故事**: 加贺恭一郎"在这个案件中'"——"'找到了'自己'母亲'离开的'真相'"。"'他的'母亲'百合子'"——"'在'他'很小的时候'就'离开了家'"——"'加贺'一直'不理解'"——"'为什么'母亲'要'离开'"。"'在'这个案件中'"——"'加贺'发现'"——"'母亲'离开'是'因为'她'得了'重病'"——"'她'不想'拖累'家人'"——"'她'选择'一个人'面对'死亡'"。"加贺'终于'理解了'母亲'"——"'他'的'心结'解开了'"
- **落幕的意象**: 书名"祈祷落幕时"有"双重含义"：1）**舞台剧的落幕**——"书中'有一个'重要的'场景'是'在'剧院'"——"'舞台剧'的'落幕'"——"'象征着'一个'时代的'结束"；2）**生命的落幕**——"书中'的'人物'"——"'一个'一个'地'离开'"——"'他们的'生命'落幕了'"——"'但'他们的'爱'留了下来'"。"东野圭吾用'落幕'这个意象'"——"'表达了'对'生命的'敬畏'"——"'对'爱的'赞美'"
- **东野圭吾的背景**: 东野圭吾（1958-）——日本最畅销的推理小说家——"加贺恭一郎系列是他的'代表作之一'"——"《祈祷落幕时》是'系列的'第 10 部'"——"'也是'最后一部'"——"2013 年出版"——"2018 年'被'改编为'电影'"——"由'阿部宽'主演"——"获得了'很高的'评价"。"东野圭吾说'加贺系列'是他'最'用心'的'系列'"——"'因为'加贺'是'一个'有'温度'的'刑警'"——"'他'不'只是'在'破案'"——"'他'在'理解'人性'"
