# 金钱心理学

状态：已读

作者：丹·艾瑞里

一句话评价：丹·艾瑞里（Dan Ariely）的"行为经济学经典'"——"'从行为经济学视角解析金钱决策'"。"'揭示了人们在消费、投资、储蓄中的非理性行为模式'"。"'心理账户概念：人们对不同来源的钱有不同态度'"——"'损失厌恶：损失 100 块的痛苦远大于获得 100 块的快乐'"

核心收获：
- **非理性的普遍性**: 本书核心是"'非理性'在金钱决策中的普遍性'"。"'我们'以为'自己'很'理性'"——"'但'实际'被'认知'偏差'左右'"。"'例子'"：1）**价格锚定**——"'先'看到'高'价格'"——"'觉得'低'价格'便宜'"；2）**免费的力量**——"'免费'东西'"——"'即使'不'需要'"——"'也'要'"；3）**相对性**——"'不'看'绝对'值'"——"'看'相对'值'"。"'这些'偏差'"——"'让'我们'做出'非理性'决策'"
- **心理账户**: 本书"提出了心理账户的概念'"。"'什么'是'心理'账户'"——"'人们'对'不同'来源'的'钱'有'不同'态度'"。"'例子'"：1）**工资**——"'省着'花'"；2）**奖金**——"'随意'花'"；3）**意外之财**——"'很快'花掉'"。"'但'钱'本质上是'等价'的'"——"'100 元'工资'"——"'和'100 元'奖金'"——"'价值'一样'"。"'心理'账户'的'问题'"："'导致'非理性'消费'"
- **损失厌恶**: 本书"讨论了损失厌恶'"。"'什么'是'损失'厌恶'"——"'损失'100 块'的'痛苦'"——"'远'大于'获得'100 块'的'快乐'"。"'研究'表明'"："'损失'的'痛苦'"——"'是'获得'快乐'的'2 倍'"。"'损失'厌恶'的'影响'"：1）**投资**——"'不愿'卖出'亏损'股票'"；2）**决策**——"'避免'损失'"——"'不'追求'收益'"；3）**保险**——"'愿意'买'保险'"——"'避免'损失'"
- **支付痛苦**: 本书"分析了支付的痛苦'"。"'支付'是'痛苦'的'"——"'花钱'时'感到'痛苦'"。"'如何'减轻'痛苦'"：1）**信用卡**——"'不'直接'看到'钱'"——"'痛苦'小'"；2）**预付费**——"'先'付'钱'"——"'后'消费'"——"'痛苦'小'"；3）**订阅**——"'自动'扣款'"——"'不'感觉'到'"。"'商家'利用'这一点'"——"'鼓励'用'信用卡'"——"'推广'订阅'"
- **期望的力量**: 本书"讨论了期望对金钱决策的影响'"。"'期望'影响'体验'"。"'例子'"：1）**价格期望**——"'贵'的'酒'"——"'觉得'好喝'"；2）**品牌期望**——"'名牌'"——"'觉得'好'"；3）**安慰剂**——"'贵'的'药'"——"'觉得'有效'"。"'期望'的'力量'"："'改变'我们'的'感知'"
- **社会规范与市场规范**: 本书"区分了社会规范与市场规范'"。"'社会'规范'"："'人情'——'礼物'——'帮助'"——"'不'谈'钱'"。"'市场'规范'"："'价格'——'工资'——'交易'"——"'谈'钱'"。"'问题'"："'把'社会'规范'变成'市场'规范'"——"'给'父母'钱'"——"'父母'生气'"。"'金钱'的'引入'"——"'改变'关系'"
- **作者的背景**: 丹·艾瑞里（Dan Ariely, 1967-）——以色列裔美国行为经济学家、杜克大学教授——"《金钱心理学》的作者'"。"他'研究'行为'经济学'"——"'非理性'行为'"。"'他'的'其他'著作'"——"'《怪诞行为学》'——'《助推》'"。"'《金钱心理学》'的'目的'"——"'帮助'人们'理解'自己'的'金钱'行为'"——"'做出'更好'决策'"
