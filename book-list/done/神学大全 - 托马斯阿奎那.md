# 神学大全

状态：已读

作者：托马斯·阿奎那

一句话评价：托马斯·阿奎那（Thomas Aquinas）的"中世纪经院哲学的巅峰之作"——"系统整合了'亚里士多德哲学'与'基督教神学'"——"是'西方思想史上'最重要的'著作之一'"。"'五路证明'"——"阿奎那'用'理性'论证'上帝存在'的'五种方式'"。"'信仰与理性的关系'"——"'信仰'不排斥'理性'"——"'理性'可以'为'信仰'服务'"。"'自然法理论'"——"'人类'可以'通过'理性'认识'道德法则'"——"'这是'现代人权思想的'源头之一'"

核心收获：
- **五路证明**: 阿奎那提出了"'五路证明'"——"用'理性'论证'上帝存在'的'五种方式'"：1）**运动的证明**——"'万物'都在'运动'"——"'运动'必须'有'一个'第一推动者'"——"'这个'第一推动者'就是'上帝'"；2）**因果的证明**——"'万物'都'有'原因'"——"'因果链'不能'无限追溯'"——"'必须'有'一个'第一因'"——"'这个'第一因'就是'上帝'"；3）**偶然性与必然性的证明**——"'万物'都'是'偶然的'"——"'但'必须'有'一个'必然的存在'"——"'这个'必然的存在'就是'上帝'"；4）**完美等级的证明**——"'万物'都'有'完美'的'等级'"——"'必须'有'一个'最完美的'存在'"——"'这个'最完美的'存在'就是'上帝'"；5）**目的论的证明**——"'万物'都'有'目的'"——"'必须'有'一个'设计者'"——"'这个'设计者'就是'上帝'"
- **信仰与理性**: 阿奎那"系统阐述了'信仰与理性的关系'"。"'他'说'信仰'不排斥'理性'"——"'理性'可以'为'信仰'服务'"。"'有些'真理'可以'通过'理性'获得'"——"'如'上帝存在'"——"'有些'真理'只能'通过'信仰'获得'"——"'如'三位一体'"。"阿奎那'反对'两种'极端'"：1）**唯理主义**——"'认为'理性'可以'解释'一切'"——"'这'会'导致'否定信仰'"；2）**唯信主义**——"'认为'信仰'不需要'理性'"——"'这'会'导致'盲从'"。"阿奎那说'信仰与理性'是'互补的'"——"'它们'都'来自'上帝'"——"'所以'不会'矛盾'"
- **自然法理论**: 阿奎那提出了"'自然法理论'"——"'人类'可以'通过'理性'认识'道德法则'"。"'自然法'是'上帝'赋予'人类'的'道德本能'"——"'它'告诉'我们'什么'是'对的'"——"'什么'是'错的'"。"阿奎那说'自然法'的'第一原则'"——"'行善避恶'"——"'这是'所有'道德法则'的'基础'"。"'从'这个'原则'出发'"——"'人类'可以'推导出'具体的'道德规范'"——"'如'不可杀人'"——"'不可偷盗'"——"'不可奸淫'"。"自然法理论'是'现代人权思想的'源头之一'"——"'它'认为'道德'不是'任意的'"——"'是'可以'通过'理性'认识的'"
- **存在与本质**: 阿奎那"区分了'存在'与'本质'"。"'本质'是'一个'事物'是'什么'"——"'存在'是'一个'事物'存在'这个'事实'"。"'在'上帝'那里'"——"'存在'与'本质'是'同一的'"——"'上帝'的'本质'就是'存在'"——"'上帝'就是'存在本身'"。"'在'其他'事物'那里'"——"'存在'与'本质'是'分离的'"——"'一个'事物'可以'有'本质'"——"'但'不'存在'"——"'如'独角兽'"。"这个区分'对'后世的'形而上学'产生了'深远影响'"
- **德性伦理**: 阿奎那"发展了'德性伦理'"——"'他'区分了'三种'德性'"：1）**理智德性**——"'智慧'——'理解'——'知识'"——"'通过'学习'获得'"；2）**道德德性**——"'节制'——'勇敢'——'正义'"——"'通过'习惯'获得'"；3）**神学德性**——"'信'——'望'——'爱'"——"'通过'上帝的'恩典'获得'"。"阿奎那说'最高的'德性'是'爱'"——"'爱'是'所有'德性'的'顶峰'"——"'爱'上帝'"——"'爱'邻人'"——"'这是'基督教'伦理'的'核心'"
- **政教关系**: 阿奎那"讨论了'政教关系'"——"'他'认为'教会'与'国家'是'两个'不同的'领域'"。"'国家'负责'世俗事务'"——"'教会'负责'精神事务'"——"'但'教会'的'权威'高于'国家'"——"'因为'精神'高于'世俗'"。"阿奎那说'如果'国家'的'法律'与'自然法'冲突'"——"'公民'没有'义务'遵守'"——"'这'为'后世的'公民不服从'提供了'理论依据'"
- **托马斯·阿奎那的背景**: 托马斯·阿奎那（Thomas Aquinas, 约 1225-1274）——意大利神学家、哲学家——"经院哲学的'集大成者'"——"《神学大全》是他'最伟大'的作品'"——"写了'近 20 年'"——"未完成'就'去世'了'"。"阿奎那'把'亚里士多德的'哲学'与'基督教'神学'结合'起来'"——"'创造了'一个'庞大的'思想体系'"——"'这个'体系'成为了'天主教'的'官方哲学'"——"'影响'了'西方'思想'近 800 年'"
