# 哲学家们都干了些什么

状态：已读

作者：欣德尔

一句话评价：欣德尔"哲学入门通俗读物'"——"'用幽默笔调讲述从苏格拉底到萨特的思想历程'"。"'把哲学家写成有血有肉的人'"。"'哲学不能当饭吃但能让你知道为什么吃饭精准概括哲学价值'"。"'适合初学者的哲学史框架书'"

核心收获：
- **苏格拉底之死**: 本书核心是"'苏格拉底之死'"。"'雅典'民主'投票'处死'了'最'智慧'的'人'"。"'苏格拉底'宁可'死'也'不'逃亡'"。"'因为'他'尊重'法律'"。"'这'是'哲学'史上'最'悲壮'的'一幕'"
- **柏拉图的理想国**: 本书"讨论了柏拉图的理想国'"。"'哲学家'应该'当'国王'"。"'柏拉图'对'理想'政治'的'想象'"。"'但'他'的'理想国'其实'是'个'极权主义'蓝图'"
- **笛卡尔的我思故我在**: 本书"介绍了笛卡尔的思想'"。"'怀疑'一切'"。"'最后'发现'唯一'不'能'怀疑'的'是'我'在'怀疑'这个'事实'"。"'现代'哲学'的'起点'"
- **康德的纯粹理性批判**: 本书"分析了康德的思想'"。"'人类'认识'世界'的'能力'是'有'边界'的'"。"'我们'只能'认识'现象'"。"'永远'无法'认识'物自体'"。"'这'是'哲学'史上'最'深刻'的'自我'反思'"
- **尼采的超人**: 本书"讲述了尼采的思想'"。"'上帝'已死'"。"'传统'价值'崩溃'了'"。"'人'必须'自己'创造'价值'"。"'尼采'不'是'虚无主义者'"。"'他'是'价值'的'重建者'"
- **存在主义**: 本书"讨论了存在主义'"。"'萨特'说'存在'先'于'本质'"。"'人'没有'预设'的'目的'"。"'人'是'什么'取决于'他'做'了'什么'"。"'自由'是'最大'的'负担'"
- **哲学的意义**: 本书"探讨了哲学的意义'"。"'哲学'不'能'当饭'吃'"。"'但'能'让'你'知道'为什么'吃饭'"。"'它'不'是'答案'"——"'是'问题'"。"'好'的'哲学'问题'比'坏'的'答案'更'有'价值'"
- **作者的背景**: 欣德尔——德国作家——"《哲学家们都干了些什么》的作者'"。"'他'用'幽默'的'笔调'写'哲学史'"。"'《哲学家'都'干'了'什么》'2011 年'出版'"。"'他'把'哲学家'写成'有'血'有'肉'的'人'"。"'这'本书'是'哲学'入门'的'通俗'读物'"。"'适合'初学者'阅读'"
