# 社会契约论

状态：已读

作者：卢梭

一句话评价：启蒙运动最重要的政治哲学著作之一，提出"人生而自由，却无往不在枷锁之中"，奠定了现代民主理论的基础。

核心收获：
- 理解了"社会契约"的核心：合法的政治权威来自于人民的共同约定，而不是君权神授
- "公意"（General Will）概念：人民的共同意志不同于个人意志的简单相加，而是指向公共利益的
- 主权在民：人民是最高主权者，政府只是人民的代理人——这个观点直接影响了法国大革命
