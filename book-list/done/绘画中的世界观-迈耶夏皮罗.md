# 绘画中的世界观

状态：已读

作者：迈耶夏皮罗

一句话评价：艺术史大家迈耶·夏皮罗的艺术评论集，从绘画、雕塑到建筑，探讨了艺术形式与思想观念之间的深层联系。

核心收获：
- 理解了艺术形式不是单纯的审美选择，而是世界观的表达——每一种风格背后都有一种看待世界的方式
- 夏皮罗的分析方法：把艺术作品放在历史、社会、哲学的语境中解读，而不是孤立的审美评判
- 现代艺术的革命不仅仅是形式上的突破，更是对"艺术是什么"这个问题的重新思考
