# 乡土中国

状态：已读

作者：费孝通

一句话评价：费孝通的"社会学奠基之作'"——"'差序格局概念精准刻画中国社会关系（以己为中心的波纹结构）'"。"'礼治秩序、无讼、男女有别揭示乡土社会运行逻辑'"。"'薄薄七万字是理解中国传统社会和当代人情社会的必读钥匙'"

核心收获：
- **乡土本色**: 本书核心是"'乡土本色'"的分析'"。"'从'基层'上'看去'"——"'中国'社会'是'乡土性'的'"。"'中国'基层'社会'的'本质'"："'乡土'"——"'农民'依附'于'土地'"——"'安土'重迁'"——"'生'于'斯'"——"'长'于'斯'"——"'死'于'斯'"。"'乡土'社会'是'熟人'社会'"——"'每个'人'认识'每个'人'"——"'每个'人'的'行为'都'被'所有'人'看着'"。"'这'和'城市'的'陌生人'社会'完全'不同'"
- **差序格局**: 本书"提出了差序格局概念'"。"'费孝通'最'核心'的'概念'"——"'中国'社会'结构'像'石头'丢进'水面'的'波纹'"。"'以'己'为'中心'"——"'一圈'一圈'推'出去'"。"'离'自己'近'的'（家人、亲戚）'关系'紧'"——"'离'自己'远'的'（朋友、熟人、陌生人）'关系'松'"。"'这'解释'了'为什么'中国'人'看人'下菜碟'"——"'道德'和'法律'都'因'所施'的'对象'和'自己'的'关系'而'加以'程度'上'的'伸缩'"
- **礼治秩序**: 本书"分析了礼治秩序'"。"'乡土'社会'靠'礼'维持'秩序'"——"'不'是'靠'法'"。"'礼'不'是'礼貌'"——"'是'传统'"——"'代代'相传'的'行为'规范'"。"'礼治'不'需要'警察'和'法院'"——"'靠'的'是'教化'"——"'和'敬畏'"。"'在'乡土'社会'"——"'打官司'是'可耻'的'"——"'因为'这'意味着'你'不'懂礼'"
- **无讼**: 本书"讨论了无讼观念'"。"'乡土'社会'排斥'诉讼'"——"'追求'和谐'"。"'费孝通'记录'了'一个'案例'"："'一个'丈夫'打'妻子'"——"'妻子'要'离婚'"。"'调解人'不'是'判'对错'"——"'是'劝和'"——"'因为'家和'万事'兴'"。"'在'乡土'社会'"——"'理'不'是'法律'"——"'是'情理'"——"'合乎'人情'的'道理'"
- **家族**: 本书"讲述了家族的作用'"。"'乡土'社会'的'基本'单位'不'是'个人'"——"'是'小家族'"。"'家族'不仅'是'血缘'群体'"——"'是'事业'组织'"。"'生产'"——"'消费'"——"'教育'"——"'养老'都'在'家族'内部'完成'"。"'家族'的'延续'比'个人'的'幸福'更'重要'"——"'所以'有'不孝'有'三'"——"'无后'为'大'"
- **男女有别**: 本书"分析了男女关系'"。"'乡土'社会'中'男女'分工'明确'"——"'感情'淡漠'"。"'不'是'因为'不'爱'"——"'是'因为'不'需要'爱'"。"'婚姻'的'首要'目的'"："'生育'"——"'和'家族'延续'"。"'不'是'爱情'"。"'男女'之间'保持'距离'"——"'反而'减少'了'冲突'"。"'男女'有别'是'乡土'社会'稳定'的'基础'"
- **名实分离**: 本书"讨论了名实分离现象'"。"'社会'变迁'中'"——"'名义'和'实际'经常'脱节'"。"'费孝通'说'"："'乡土'社会'在'变迁'中'有'一种'缓冲'机制'"。"'表面'上'维持'传统'（名）'"——"'实际'上'已经'改变'（实）'"。"'比如'"："'孝'的'名义'还'在'"——"'但'实际'内容'已经'从'绝对'服从'变成'了'互相'尊重'"
- **作者的背景**: 费孝通（1910-2005）——中国社会学家、人类学家——"《乡土中国》的作者'"。"'他'是'中国'社会学'的'奠基人'之一'"。"'《乡土'中国》'1947 年'出版'"——"'基于'他'1930-40 年代'的'田野'调查'"。"'他'的'其他'著作'"："'《江村'经济》'——'《生育'制度》'"。"'他'的'贡献'"："'将'西方'社会学'理论'与'中国'实际'结合'"。"'《乡土'中国》'的'当代'意义'"："'今天'的'中国'已经'高度'城市化'"——"'但'乡土性'依然'深刻'影响'着'中国'人'的'行为'"
