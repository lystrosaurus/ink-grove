# 写作是最好的自我投资

状态：已读

作者：Spenser

一句话评价：用互联网思维讲写作——写作不是文学修养，是个人品牌放大器。好的文字能让你的影响力呈指数级增长。

核心收获：
- 写作的本质是"思考的整理"——写不清楚就是想不清楚
- 好文章的标准不是"美"，是"有用"——读者看完能带走什么
- 标题决定打开率，开头决定读完率，结尾决定转发率
- 写作是最好的自我投资：成本低、复利高、可积累。一篇好文章能持续带来影响力
- 互联网时代写作的核心技能：抓注意力、讲故事、制造共鸣、促进行动
