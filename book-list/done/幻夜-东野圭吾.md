# 幻夜

状态：已读

作者：东野圭吾

一句话评价：东野圭吾的长篇悬疑小说，以大阪地震为背景，讲述一个女人为达目的不择手段的黑暗故事，是东野圭吾最绝望的作品之一。

核心收获：
- 理解了东野圭吾的"恶女"书写——女主角美冬是东野笔下最冷酷、最精于算计的女性角色
- 大阪地震后的社会混乱为故事提供了独特的背景——灾难中的人性扭曲
- 与《白夜行》的对比：同样是两人共谋犯罪，但《幻夜》更加绝望，没有救赎
- 东野圭吾的叙事结构——两条时间线并行，最终交汇到一个令人窒息的真相
