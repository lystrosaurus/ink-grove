# 搜神记

状态：已读

作者：干宝

一句话评价：中国志怪小说的鼻祖，东晋干宝搜集整理的神鬼灵异故事集，开创了中国志怪文学的传统。四百多则故事，既是文学珍品，也是研究魏晋社会信仰和民间文化的宝贵史料。

核心收获：
- **志怪小说的起源**: 志怪小说不是"编故事"，是古人对自己无法解释的现象的"认真记录"。雷电、地震、疾病、死亡——在科学不发达的时代，人们用神鬼来解释这些现象。《搜神记》就是这种"认真记录"的集大成
- **干宝的编纂动机**: 干宝说"发明神道之不诬"——他相信这些神鬼故事是真实的。他不是"创作"，是"搜集"。这种态度让《搜神记》不同于后世的"故意编造"，它更接近"民间口述历史的记录"
- **经典故事**: "董永与七仙女"（后来演变为黄梅戏《天仙配》）、"韩凭夫妇"（化蝶故事的原型，比梁祝早）、"李寄斩蛇"（少女智勇除妖）、"干将莫邪"（铸剑师的血泪故事）。这些故事后来成为中国文学和戏剧的重要素材
- **人鬼关系**: 《搜神记》中的鬼不是"恐怖"的代名词，而是"有情"的存在。鬼会报恩、会复仇、会恋爱、会思念亲人。"人鬼殊途"但"情无殊途"——这是中国鬼故事和西方恐怖故事最大的区别
- **善恶报应**: 《搜神记》中大量的"善有善报、恶有恶报"故事。这不是简单的道德说教，是古人对"正义"的渴望——现实中好人不一定有好报，但在神鬼的世界里，正义最终会实现
- **女性形象**: 《搜神记》中的女性形象特别丰富——有智勇双全的李寄，有忠贞不渝的韩凭妻，有善良温柔的七仙女，也有复仇的女鬼。在男权社会中，志怪小说反而给了女性更多的"能动性"
- **《搜神记》的流传**: 原书三十卷已散佚，现存二十卷是后人辑录的。明朝胡应麟从《法苑珠林》《太平御览》等书中辑出佚文，才有了今天看到的版本。原书散佚本身就是一个"文献失而复得"的故事
- **对后世文学的影响**: 《搜神记》直接影响了唐代的传奇小说、宋元的话本、明清的《聊斋志异》。蒲松龄写《聊斋》时说"才非干宝，雅爱搜神"——直接致敬干宝。没有《搜神记》，就没有中国志怪文学的传统
