# 现代艺术

状态：已读

作者：150-年-威尔贡培兹

一句话评价：最好的现代艺术入门书之一，贡培兹以泰特美术馆前高管的身份，用幽默通俗的语言梳理了从印象派到当代艺术的 150 年发展历程，让"看不懂"的现代艺术变得可以理解。

核心收获：
- 理解了现代艺术的发展脉络：印象派→后印象派→野兽派→立体主义→达达主义→超现实主义→抽象表现主义→波普艺术→观念艺术
- 现代艺术的核心转向是从"画得像"到"表达什么"——形式让位于观念
- 杜尚的"现成品"艺术（小便池）彻底重新定义了什么是艺术
