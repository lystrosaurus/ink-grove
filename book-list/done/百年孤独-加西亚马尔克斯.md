# 百年孤独

状态：已读

作者：加西亚·马尔克斯

一句话评价：加西亚·马尔克斯（Gabriel García Márquez）的"魔幻现实主义巅峰之作"——"布恩迪亚家族七代人的兴衰"——"马孔多小镇的百年历史"——"拉丁美洲文学的巨著"。"多年以后，面对行刑队，奥雷里亚诺·布恩迪亚上校将会回想起父亲带他去见识冰块的那个遥远的下午。"——"这是文学史上最著名的开篇之一"。"生命中曾经有过的所有灿烂，原来终究，都需要用寂寞来偿还。"——"孤独与循环命运是全书的核心主题"。

核心收获：
- **布恩迪亚家族**: 布恩迪亚家族"七代人的故事"是"全书的主线"。"从何塞·阿尔卡蒂奥·布恩迪亚和乌尔苏拉建立马孔多"——"到家族的最后一代被蚂蚁吃掉"——"七代人"——"每一代人'都'重复着'同样的名字'"——"奥雷里亚诺"——"何塞·阿尔卡蒂奥"——"这种'重复'象征着'命运的循环'"。"家族中的'每一个男人'"——"都'带着'孤独的'气质"——"他们'无法'真正地'爱别人'"——"他们'注定'要'孤独地'死去"
- **马孔多小镇**: 马孔多是"马尔克斯虚构的一个小镇"——"它是'拉丁美洲的'缩影"。"马孔多'从'一个'与世隔绝的'小村庄'"——"到'被'香蕉公司'入侵'"——"到'最终'被'飓风'抹去"——"它'经历了'拉丁美洲的'整个现代化过程'"。"马孔多的'命运'"——"就是'拉丁美洲的'命运"——"被'殖民'"——"被'剥削'"——"被'遗忘'"
- **魔幻现实主义**: 马尔克斯的"魔幻现实主义"是"本书最显著的特征"。"在'马孔多'"——"死人'会'回来'"——"美人'会'飞上天'"——"雨'可以'下'四年'十一个月'零'两天'"——"这些'魔幻的'事件"——"被'当作'日常的'事情'来'描写'"。"马尔克斯说'魔幻现实主义'不是'发明'"——"是'拉丁美洲的'现实'"——"在'拉丁美洲'"——"'现实'本身就'是'魔幻的'"
- **孤独主题**: "孤独"是"全书的核心主题"。"布恩迪亚家族的'每一个成员'"——"都'带着'深深的'孤独'"。"奥雷里亚诺上校'发动了'32 次起义'"——"'全部失败'"——"他'最终'躲进'作坊'"——"'制作'小金鱼'"——"'做好了'就'熔化'"——"'熔化了'再'重做'"——"这就是'孤独'"。"乌尔苏拉'活了一百多岁'"——"'看着'家族'一代一代'逝去'"——"她'最终'也'在'孤独中'死去"
- **循环命运**: 马尔克斯"反复强调'命运的循环'"。"布恩迪亚家族的'七代人'"——"每一代人'都'重复着'上一代人的'错误'"——"每一代人'都'逃不脱'孤独的'命运"。"奥雷里亚诺'这个名字'"——"在'每一代人'中'都'出现"——"每一个'叫'奥雷里亚诺的'人'"——"都'有'相似的'性格'"——"都'有'相似的'结局"。"马尔克斯说'时间'不是'线性的'"——"是'循环的'"——"'历史'在'不断重复'"
- **拉丁美洲历史**: 马尔克斯"通过马孔多的故事'"——"写了'拉丁美洲的'历史"。"殖民时期"——"独立战争"——"内战"——"香蕉公司的入侵"——"大屠杀"——"这些'都是'拉丁美洲'真实发生过的'事件"。"马尔克斯说'拉丁美洲的'历史'就是'一部'被遗忘的'历史'"——"世界'总是'忘记'拉丁美洲'"——"拉丁美洲'也'在'忘记'自己的'历史"
- **加西亚·马尔克斯的背景**: 加西亚·马尔克斯（Gabriel García Márquez, 1927-2014）——哥伦比亚作家——"魔幻现实主义的代表人物"——"1982 年诺贝尔文学奖得主"。《百年孤独》"1967 年出版"——"引起了'世界文坛的'轰动"——"被'翻译成'40 多种语言"——"销量超过'5000 万册'"。"马尔克斯的'其他作品'"包括"《霍乱时期的爱情》"——"《没有人给他写信的上校》"——"《一桩事先张扬的凶杀案》"
