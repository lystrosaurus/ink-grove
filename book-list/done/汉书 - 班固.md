# 汉书

状态：已读

作者：班固

一句话评价：班固的"中国第一部纪传体断代史"——记述"西汉从汉高祖到王莽'两百多年'的历史"——与《史记》并称"史汉"——是"二十四史中'最重要的'正史之一"。《汉书》的"体例"（本纪、表、志、列传）"被'后世正史'沿用"。"十志"是"中国历史上'第一部'系统性的'制度史'"——食货志、地理志、艺文志等——"开创了'制度史'的先河"。

核心收获：
- **纪传体断代史的体例**: 班固确立了"纪传体断代史"的"体例"——"本纪"（帝王的"编年记录"）、"表"（大事年表）、"志"（制度史）、"列传"（人物传记）。这种"体例"被"后世所有正史"沿用——从"《后汉书》"到"《明史》"——"都是'汉书体例'的'延续'"。班固的贡献是"把'通史'变成了'断代史'"——"每一代'都有自己的史'"——"不'混在一起'"
- **十志的价值**: 《汉书》的"十志"是"全书'最有价值'的部分"——它们是"中国历史上'第一部'系统性的'制度史'"：1）**食货志**——"经济史"——"土地制度、赋税制度、货币制度"；2）**地理志**——"地理史"——"行政区划、人口、物产"；3）**艺文志**——"文献史"——"先秦到西汉的'所有书籍'的分类和简介"——是"了解'先秦学术'的'关键资料'"；4）**刑法志**——"法律史"；5）**天文志**——"天文史"；6）**五行志**——"灾异史"；7）**郊祀志**——"祭祀史"；8）**沟洫志**——"水利史"
- **与史记的区别**: 班固的史观与司马迁"截然不同"。司马迁重"通"——"通古今之变"——"把'几千年'的历史'连在一起'看"。班固重"断"——"一代之史"——"只写'西汉'一代"。司马迁"敢于'批评'皇帝"——"把项羽写进'本纪'"——"把陈涉写进'世家'"。班固"更'保守'"——"以'正统'为纲"——"维护'汉朝的合法性'"。但班固的"严谨"让"《汉书》在'史料价值'上'超过'了《史记》"——"很多'史记没有记载'的细节"——"《汉书》都有"
- **重要人物**: 《汉书》记载了"西汉'最重要的人物'"：1）**刘邦**——"从'亭长'到'皇帝'"——"中国历史上'最成功'的'草根逆袭'"；2）**汉武帝**——"开疆拓土"——"罢黜百家独尊儒术"——"派张骞出使西域"——"让汉朝'成为'东亚最强大的帝国"；3）**司马迁**——"班固在《汉书》中为司马迁'立传'"——"记录了《史记》的创作过程"；4）**王莽**——"篡汉建立'新朝'"——"是'西汉'到'东汉'的'过渡人物'"
- **班固的遭遇**: 班固自己也有"悲剧性的一生"。他"花了'二十多年'写《汉书》"——但"没有写完"——"就'被卷入'了政治斗争"——"被'下狱'"——"死在了'狱中'"——享年 61 岁。《汉书》的"最后部分"是"他的妹妹班昭'续写的'"——"班昭是中国历史上'最著名的'女学者之一"。班固的" unfinished"命运"让人唏嘘"——"他写了'两百年的历史'"——"却没有'看到'自己的书'完成'"
- **艺文志的意义**: 《汉书·艺文志》是"中国'最早的'图书目录"——它"记录了'先秦到西汉'的所有书籍"——"分为'六艺、诸子、诗赋、兵书、数术、方技'六大类"——"共'596 家、13269 卷'"。"很多书'后来失传了'"——"只有通过'艺文志'才知道'它们'曾经存在过'"。"艺文志"是"研究'中国古代学术史'的' indispensable'资料"
- **班固的背景**: 班固（32-92）——东汉史学家——字孟坚——扶风安陵人（今陕西咸阳）。他的父亲班彪"已经开始写'西汉历史'"——"但没有完成"——班固"继承了父亲的'遗志'"——花了"二十多年"写成了"《汉书》的大部分"。《汉书》是"二十四史'中'第二部'"——"仅次于《史记》"——但"在'史料严谨性'上'超过了'《史记》"
