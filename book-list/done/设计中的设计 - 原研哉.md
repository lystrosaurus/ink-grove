# 设计中的设计

状态：已读

作者：原研哉

一句话评价：原研哉（Kenya Hara）的"设计哲学随笔集'"——"'阐述了"白""空""虚无"等东方美学理念在设计中的应用'"。"'设计不是添加，而是还原'"——"'把事物还原到最本质的状态'"。"'是对"设计是什么"的深度思考'"

核心收获：
- **设计的本质**: 原研哉提出"'设计不是添加，而是还原'"。"'什么'是'设计'"——"'设计'不'是'把'东西'做得'更'华丽'"——"'设计'是'把'东西'还原'到'最'本质'的'状态'"。"'比如'无印良品'的'产品'"——"'没有'多余'的'装饰'"——"'只有'必要'的'功能'"。"'这种'设计'理念'"——"'源于'东方'的'美学'"——"'少即是多'"。"'设计'的'目的'"——"'不'是'吸引'注意'"——"'是'让'人'感觉'"本来'就是'这样'"
- **白的概念**: 原研哉"详细阐述了"白"的概念'"。"'白'不'是'空无一物'"——"'白'是'包含'了'所有'可能性'的'空'"。"'白'像'一张'白纸'"——"'它'可以'变成'任何'东西'"。"'白'也'像'雪'"——"'雪'覆盖'一切'"——"'但'雪'下'有'生命'"。"'这种'白'是'东方'美学'的'核心'"——"'西方'的'白'是'空白'"——"'东方'的'白'是'丰富'的'空'"。"'原研哉'的'设计'"——"'大量'使用'白'"——"'让'人'想象'"——"'让'人'参与'"
- **空与虚无**: 原研哉"探讨了空与虚无'"。"'空'不'是'没有'"——"'空'是'有'的'另一种'形式'"。"'比如'日本'的'茶室'"——"'茶室'很'小'"——"'很'简单'"——"'但'茶室'有'意境'"。"'虚无'也'一样'"——"'虚无'不'是'不存在'"——"'虚无'是'潜在'的'存在'"。"'设计'中'的'空'"——"'留白'"——"'让'使用者'想象'"——"'让'使用者'参与'创造'"。"'这'与'西方'设计'不同'"——"'西方'设计'填满'空间'"——"'东方'设计'保留'空间'"
- **感觉的觉醒**: 原研哉"强调感觉的觉醒'"。"'现代人'的'感觉'钝化'了'"——"'被'信息'轰炸'"——"'被'物质'包围'"。"'设计'的'任务'"——"'唤醒'人'的'感觉'"。"'如何'唤醒'"：1）**简化**——"'减少'刺激'"——"'让'感觉'恢复'敏感'"；2）**材质**——"'用'自然'材质'"——"'纸'——'木'——'布'"——"'让'人'触摸'"；3）**留白**——"'给'感觉'空间'"。"'原研哉'的'展览'"——"'Exformation'"——"'就是'为了'唤醒'感觉'"
- **无印良品的理念**: 原研哉"作为无印良品的艺术总监，阐述了其理念'"。"'无印良品'的'理念'"：1）**无品牌**——"'不'追求'品牌'"；2）**简约**——"'设计'简约'"；3）**自然**——"'用'自然'材质'"；4）**合理**——"'价格'合理'"。"'无印良品'的'广告'"——"'原研哉'设计'的'海报'"——"'常常'是'空白'"——"'只有'一'两个'字'"。"'这'种'理念'"——"'反对'消费主义'"——"'反对'过度'包装'"——"'反对'品牌'崇拜'"
- **信息与感知**: 原研哉"讨论了信息与感知的关系'"。"'信息'不'是'知识'"——"'信息'是'原料'"——"'知识'是'加工'后'的'产品'"。"'现代'社会'信息'过剩'"——"'但'知识'不'一定'增加'"。"'设计'可以'帮助'人'理解'信息'"——"'把'信息'组织'好'"——"'让'人'容易'感知'"。"'比如'地图'"——"'地图'不'是'复制'现实'"——"'地图'是'简化'现实'"——"'让'人'容易'理解'"
- **原研哉的背景**: 原研哉（Kenya Hara, 1958-）——日本平面设计师、无印良品艺术总监——"《设计中的设计》的作者'"。"他'在'武藏野美术大学'学习'设计'"——"'后'来'在'日本'设计'中心'工作'"。"2002 年'"——"'他'成为'无印良品'的'艺术'总监'"。"他'的'设计'风格'"——"'极简'——'白'——'空'"。"《设计中的设计》'2003 年'出版'"——"'畅销'亚洲'"。"他'还'策划'了'很多'展览'"——"'长野'冬奥会'开幕式'节目'册'也'是'他'设计'的'"
