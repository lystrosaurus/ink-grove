# 乌合之众

状态：已读

作者：勒庞

一句话评价：勒庞（Gustave Le Bon）的"群体心理学奠基之作'"——"'1895 年出版至今仍是理解群体行为最直接的入口'"。"'群体智商低于个体'"——"'揭示冲动、传染、暗示三大特征'"。"'断言、重复、传染是领袖的三大手段'"。"'在社交媒体时代比在 19 世纪更有解释力'"——"'社会心理学必读经典'"

核心收获：
- **群体特征**: 本书核心是"'群体特征'"的分析'"。"'勒庞'的'核心'论断'"："'群体'不'是'个体'的'简单'相加'"——"'而'是'一个'新'的'心理'群体'"。"'它'有'不同于'个体'的'特征'"。"'群体'的'智力'永远'低于'个体'中'最'聪明'的'那个'人'"——"'但'群体'的'力量'永远'大于'个体'"。"'群体'的'特征'"："'冲动'"——"'易变'"——"'急躁'"
- **匿名性与责任分散**: 本书"讨论了匿名性的作用'"。"'群体'中'个人'责任感'消失'"——"'行为'失控'"。"'勒庞'说'"："'孤立'的'个人'很'清楚'"——"'在'孤身'一人'时'他'不'能'放火'烧'宫殿'"。"'但'在'群体'中'"——"'他'就'会'毫不犹豫'地'参与'纵火'"。"'匿名'感'让'人'觉得'自己'不'会'被'追责'"——"'于是'释放'了'平时'被'压抑'的'本能'"
- **情绪传染**: 本书"描述了情绪传染'"。"'情绪'和'行为'在'群体'中'快速'传播'"——"'像'病毒'一样'"。"'一个'人'愤怒'"——"'周围'的'人'都'愤怒'"——"'愤怒'在'传播'中'被'不断'放大'"。"'社交媒体'时代'的'热搜'"——"'网暴'"——"'就是'情绪'传染'的'极致'表现'"
- **暗示与轻信**: 本书"分析了群体的暗示性'"。"'勒庞'说'"："'群体'既'容易'接受'暗示'"——"'也'容易'把'暗示'变成'行动'"。"'这'就是'为什么'谣言'在'群体'中'传播'最快'"——"'群体'不'验证'信息'的'真实性'"——"'只'感受'信息'的'情绪'"
- **领袖的力量**: 本书"讨论了领袖的作用'"。"'群体'需要'领袖'"。"'领袖'的'三大'手段'"：1）**断言**——"'简洁'有力'的'结论'"——"'不'需要'论证'"；2）**重复**——"'同样'的'话说'一百遍'就'成'真理'"；3）**传染**——"'让'情绪'在'群体'中'扩散'"。"'这'三种'手段'在'今天'的'政治'宣传'和'广告'中'依然'被'广泛'使用'"
- **形象思维**: 本书"指出群体用形象思考'"。"'群体'用'形象'思考'"——"'而'非'理性'推理'"。"'群体'不'理解'复杂'的'论证'"——"'但'能'被'一个'画面'"——"'一个'故事'"——"'一个'口号'打动'"。"'这'就是'为什么'让'美国'再次'伟大'比'任何'政策'白皮书'都'有'力量'"
- **道德的两极**: 本书"讨论了群体道德'"。"'群体'既'可'犯罪'"——"'也'可'牺牲'"——"'取决于'暗示'"。"'法国'大革命'中'的'群众'"："'既'能'屠杀'贵族'"——"'也'能'为'革命'献出'生命'"。"'群体'没有'固定'的'道德'"——"'只有'被'暗示'的'道德'"
- **作者的背景**: 古斯塔夫·勒庞（Gustave Le Bon, 1841-1931）——法国社会心理学家——"《乌合之众》的作者'"。"'他'写'《乌合之众》'时'（1895 年）'"——"'法国'刚'经历'了'普法'战争'和'巴黎'公社'"。"'他'目睹'了'群体'的'暴力'"——"'对'群体'的'态度'总体'是'恐惧'和'鄙视'"。"'他'认为'群体'是'文明'的'威胁'"。"'现代'心理学'认为'勒庞'夸大'了'群体'的'非理性'"——"'忽略'了'群体'也'能'产生'集体'智慧'"
