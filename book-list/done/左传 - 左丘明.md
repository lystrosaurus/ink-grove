# 左传

状态：已读

作者：左丘明

一句话评价：中国第一部叙事完整的编年体史书（记载公元前 722 年 - 前 468 年，春秋时期），是对孔子《春秋》的详细注解和补充。《春秋》太简略（每年只有一两句话），《左传》补充了大量细节、背景、人物对话和历史故事。以"精彩的战争描写"和"生动的人物刻画"著称——城濮之战、崤之战、鞍之战每一场都写得跌宕起伏。"烛之武退秦师""曹刿论战""郑伯克段于鄢"都是千古名篇。

核心收获：
- **左传与春秋的关系**: 孔子编纂的《春秋》是"中国最早的编年史"——但极其简略（242 年的历史只有 18000 字），很多条目只有几个字（如"夏五月，郑伯克段于鄢"）。读者根本不知道"发生了什么""为什么发生""结果怎样"。《左传》就是对《春秋》的"详细注解"——它补充了事件的起因、经过、结果，以及人物的对话和心理活动。没有《左传》，《春秋》就是一本"天书"
- **郑伯克段于鄢**: 《左传》开篇最著名的故事——郑庄公的弟弟共叔段在母亲武姜的支持下企图夺权。郑庄公明知弟弟在扩张势力，却"纵容"他——"多行不义必自毙，子姑待之"（他做了太多不义之事，自然会灭亡，你等着看吧）。最终共叔段叛乱被镇压，郑庄公把母亲安置在城颍并发誓"不及黄泉，无相见也"（不到地下不再见面）。后来后悔了，在颖考叔的建议下"掘地及泉"——在地道中与母亲相见，恢复了母子关系。这个故事展现了"政治的冷酷"和"人性的复杂"
- **曹刿论战**: 齐鲁长勺之战——弱小的鲁国击败了强大的齐国。曹刿是一个"普通人"——他去见鲁庄公说"肉食者鄙，未能远谋"（当官的人目光短浅，我来帮你）。他指挥鲁军在齐军"三鼓"（三次击鼓冲锋）后才出击——"一鼓作气，再而衰，三而竭"。齐军士气耗尽后鲁军一举击溃。战后曹刿解释了他的战术——这是中国最早的"军事心理学"案例
- **烛之武退秦师**: 秦晋联军围攻郑国——郑国派老臣烛之武去游说秦穆公。烛之武对秦穆公说了一段精彩的外交辞令——"郑国灭亡了对秦国有什么好处？郑国隔着晋国，秦国管不到。灭了郑国，地盘都归晋国。晋国强了，秦国就弱了。不如留下郑国做您东边的朋友。"秦穆公被说服了——撤军并留下军队保护郑国。晋国见秦军撤了，也撤了。烛之武凭借"三寸不烂之舌"救了一个国家。这是中国外交史上最经典的"说辞"之一
- **战争描写的精彩**: 《左传》的战争描写是中国古代史书中最精彩的。城濮之战（晋楚争霸）——晋文公"退避三舍"（履行当年流亡楚国时的诺言），然后出其不意地攻击楚军两翼，大获全胜。崤之战——秦军远征郑国未果，归途中在崤山被晋军伏击，全军覆没。鞍之战——齐晋交战，晋军将领郤克受伤但仍坚持击鼓指挥，最终获胜。每一场战争都写得"有起因、有经过、有结果、有人物、有对话、有评论"——不是"干巴巴的记录"，是"精彩的故事"
- **人物刻画的生动**: 《左传》中的人物个个"有血有肉"——晋文公（重耳）流亡十九年最终称霸的坚韧、楚庄王"一鸣惊人"的雄才大略、子产的智慧和仁政、晏婴的机智和幽默。《左传》不仅记录"做了什么"，还记录"说了什么"——人物的对话是"个性化的"——你能从对话中听出一个人的性格、智慧、品德
- **"君子曰"的评论**: 《左传》中经常出现"君子曰"——这是作者（或引用他人）对事件和人物的道德评判。比如"多行不义必自毙"（做多了坏事必然会灭亡）、"居安思危"（在安全的时候要想到危险）、"信，国之宝也"（信用是一个国家最宝贵的财富）。这些"君子曰"是《左传》的"价值观"——它不只是一个"历史记录"，是一个"道德教科书"
- **左丘明的背景**: 左丘明是春秋末期鲁国人，与孔子同时代或稍晚。司马迁在《史记》中说"左丘失明，厥有《国语》"——传说左丘明后来失明，但他凭记忆写成了《左传》和《国语》。关于《左传》的作者有争议——有学者认为不是左丘明一个人写的，是"战国初期的多人作品"。但不管作者是谁，《左传》的文学价值和历史价值都是毋庸置疑的
- **左传的地位**: 《左传》是"十三经"之一，是中国古代"科举考试"的必读经典。它不仅是中国"最早的叙事史书"——也是中国"最早的文学杰作"之一。韩愈说"左氏浮夸"（文采飞扬），司马迁写《史记》大量引用了《左传》的材料。直到今天，"退避三舍""一鼓作气""多行不义必自毙""居安思危""信国之宝"等成语都源自《左传》——它已经融入了中国人的语言和文化
