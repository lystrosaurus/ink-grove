# 孟子

状态：已读

作者：孟子

一句话评价：儒家核心经典之一，与《论语》《大学》《中庸》并称"四书"。孟子继承并发展了孔子的思想，提出"性善论""四端说""仁政""民贵君轻""浩然之气"等核心概念，与孔子并称"孔孟"。《孟子》七篇，以对话体写成，文辞雄辩、气势磅礴，是先秦散文中最有说服力和感染力的著作之一。

核心收获：
- **性善论**: 孟子最著名的命题——"人性之善也，犹水之就下也。"人性本善——不是因为"人天生就是好人"，是因为"人天生就有向善的倾向"，就像水天然往下流。孟子说每个人都有"四端"（四种善的萌芽）：恻隐之心（同情心）、羞恶之心（羞耻心）、辞让之心（谦让心）、是非之心（判断心）。这"四端"是"仁义礼智"的根源——只要你"培养"它们，你就能成为君子。如果不去培养，它们就会"枯萎"
- **四端说**: 孟子举了一个经典例子——任何人看到一个小孩快掉到井里，都会"心中一惊"——不是因为"想从孩子的父母那里得到好处"，不是"想博取名声"，是"本能的不忍"。这种"不忍"就是"恻隐之心"——是"仁"的萌芽。孟子说这个例子证明了"人性本善"——即使最坏的人，在特定情境下也会表现出"善的倾向"
- **仁政**: 孟子的政治理想——统治者应该"以仁治国"，不是"以力治国"。"仁政"的核心是"民本"——关心百姓的生活（"制民之产"——让老百姓有地种、有饭吃、有衣穿），减轻赋税，不滥用民力。孟子说如果一个君主能做到"仁政"，百姓会"归之如流水"——他不需要用武力统一天下，天下人会自动拥护他
- **民贵君轻**: 孟子最大胆的政治命题——"民为贵，社稷次之，君为轻。"在君主专制的时代，这种"民本"思想极其激进。孟子说君主的权力不是"天授的"——如果君主"不仁"，百姓可以"推翻他"。齐宣王问"汤伐桀、武王伐纣，有这回事吗？臣弑其君可乎？"孟子回答："贼仁者谓之贼，贼义者谓之残。残贼之人谓之一夫。闻诛一夫纣矣，未闻弑君也。"——杀死一个"暴君"不是"弑君"，是"诛独夫"。这种"革命"思想在中国政治思想史上是独一无二的
- **浩然之气**: 孟子说自己"善养吾浩然之气"——一种"至大至刚"的精神力量。"浩然之气"不是"天生的"，是"通过道德修养积累"的——"以直养而无害，则塞于天地之间。"当你做的事情都是"正义的"，你的内心就会充满"浩然之气"——一种无所畏惧的道德勇气。这种"气"后来成为了中国士人精神的象征——文天祥的《正气歌》就是对"浩然之气"的继承
- **义利之辨**: 孟子见梁惠王，王曰："叟不远千里而来，亦将有以利吾国乎？"孟子对曰："王何必曰利？亦有仁义而已矣。"孟子反对"以利为先"——如果一个国家从上到下都在追求"利益"，那这个国家就危险了。"仁义"才是治国的根本——不是"不要利益"，是"先讲仁义，利益自然会来"
- **君子人格**: 孟子对"君子"（理想人格）的描述——"富贵不能淫，贫贱不能移，威武不能屈，此之谓大丈夫。"君子在"富贵"时不骄奢淫逸，在"贫贱"时不改变自己的志向，在"威武"（威胁）面前不屈服。这种"大丈夫"精神是中国知识分子的精神脊梁——从屈原到文天祥到谭嗣同，都是这种精神的传承者
- **孟子的辩论技巧**: 《孟子》七篇几乎全是对话体——孟子与各国君主（梁惠王、齐宣王、滕文公）、弟子（公孙丑、万章）、论敌（告子）的辩论。孟子的辩论技巧极高——他善于用类比（"水之就下"）、比喻（"揠苗助长"）、反问（"不为者与不能者之形何以异"）来说服对方。他的文辞"气势磅礴"——不是"冷冰冰的逻辑推理"，是"充满感情的雄辩"
- **孟子的背景**: 孟子（约公元前 372 年 - 约公元前 289 年）名轲，邹国（今山东邹城）人。他是孔子之孙子思的门人的学生——所以儒家传统说"孔子传曾子，曾子传子思，子思传孟子"，这就是"道统"。孟子周游列国（梁、齐、宋、滕），向各国君主推销他的"仁政"理念——但没有一个君主采纳。晚年回到邹国，与弟子万章等人一起著书——这就是《孟子》七篇。孟子在世时并不成功——他的思想太"理想主义"了，战国时代的君主需要的是"富国强兵"，不是"仁义道德"。但两千多年后，孟子成为了"亚圣"——地位仅次于孔子
