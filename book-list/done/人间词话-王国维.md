# 人间词话

状态：已读

作者：王国维

一句话评价：中国古典词学批评的巅峰之作，"境界说"三个字道破了诗词好坏的根本标准，至今无人超越。

核心收获：
- "境界"是评判诗词的最高标准——有境界则自成高格，无境界则流于雕饰
- "昨夜西风凋碧树，独上高楼，望尽天涯路"——人生三境界的第一重：立志
- "衣带渐宽终不悔，为伊消得人憔悴"——第二重：执着
- "众里寻他千百度，蓦然回首，那人却在灯火阑珊处"——第三重：顿悟
- 隔与不隔：好诗词让读者直接感受，差的诗词隔了一层典故和修辞
- 造境与写境：主观创造和客观描写的区分
