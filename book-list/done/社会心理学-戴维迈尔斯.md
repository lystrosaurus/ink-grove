# 社会心理学

状态：已读

作者：戴维·迈尔斯

一句话评价：戴维·迈尔斯（David Myers）的"全球最畅销的心理学教材"——"用'通俗语言'和'大量实验'讲述了'人们在社会环境中'如何'思考、影响和相互关联'"。"'基本归因错误'"——"'我们'倾向于'把别人的行为'归因于'性格'"——"'而把'自己的行为'归因于'情境'"。"'从众心理'的力量'远超想象'"——"'即使'明显是'错的'答案'"——"'大多数人'也会'在群体压力下'附和'"。"'认知失调'"——"'当行为和信念不一致时'"——"'人会改变信念'来'减少心理不适'"

核心收获：
- **基本归因错误**: 迈尔斯说"'基本归因错误'是'最常见的'认知偏差"。"'当'别人'做错事'时'"——"'我们'倾向于'认为'是'他'性格有问题'"——"'他'太'粗心'"——"'他'太'自私'"。"'但'当'我们'自己做错事'时'"——"'我们'会'归因于'情境'"——"'我'是'因为'太忙了'"——"'我'是'因为'压力大'"。"这种'双重标准'"——"'让我们'对别人'更苛刻'"——"'对自己'更宽容'"——"'这'会'导致'很多'人际冲突'"
- **从众心理**: 迈尔斯介绍了"'阿希的从众实验'"——"'被试'被'要求'判断'两条线'的'长度'"——"'这'是一个'非常简单'的任务"——"'但'当'其他人都'给出'错误答案'时'"——"'75% 的'被试'至少'有一次'附和了'错误答案'"。"从众的'原因'有两个"：1）**信息性影响**——"'我们'认为'别人'知道得'更多'"；2）**规范性影响**——"'我们'不想'被'群体'排斥'"。"从众'不是'弱点'"——"'是'人类'的'本能'"——"'它'帮助'我们'在'不确定的'情况下'做出'决策'"
- **认知失调**: 迈尔斯说"'认知失调'是'社会心理学'最重要的'理论之一'"。"'当'你的'行为'和'信念'不一致'时'"——"'你会'感到'不舒服'"——"'这种'不舒服'就是'认知失调'"。"'为了'减少'失调'"——"'你'会'改变'你的'信念'"。"比如'你'知道'吸烟有害健康'"——"'但'你'还在'吸烟'"——"'这'会产生'失调'"——"'你'会'说服自己'"——"'吸烟'的'危害'被'夸大了'"——"'我'的'爷爷'吸烟'活到 90 岁'"。"认知失调'让我们'学会了'自我欺骗'"
- **服从权威**: 迈尔斯介绍了"'米尔格拉姆的服从实验'"——"'被试'被'要求'给'另一个人'施加'电击'"——"'电击'强度'逐渐增加'"——"'那个人'会'惨叫'"——"'甚至'说'自己有心脏病'"——"'但'当'实验者'说'继续'时'"——"'65% 的'被试'会'服从'到'最高'电压'"。"这个实验'说明了'"——"'普通人'在'权威命令'下'"——"'可能'做出'非常残忍'的事'"——"'这不是'因为'他们'是'坏人'"——"'是'因为'他们'服从了'权威'"
- **偏见与歧视**: 迈尔斯"详细分析了'偏见和歧视'的'根源'"：1）**社会不平等**——"'优势群体'会'贬低'弱势群体'"——"'来'合理化'不平等'"；2）**替罪羊**——"'当'人们'受挫'时'"——"'会'把'愤怒'发泄到'弱势群体'身上'"；3）**社会规范**——"'在某些'社会'中'"——"'偏见'是'被'鼓励的'"。"迈尔斯说'偏见'不是'天生的'"——"'是'后天'习得的'"——"'所以'可以'通过'教育'来'减少'"
- **人际吸引**: 迈尔斯"研究了'什么因素'让人'相互吸引'"：1）**接近性**——"'离得近'的人'更容易'成为朋友'"；2）**外表吸引力**——"'漂亮的人'更'受欢迎'"——"'这是'不公平的'"——"'但'是'事实'"；3）**相似性**——"'相似的人'更容易'相互吸引'"——"'价值观、兴趣、背景'相似'"；4）**互惠**——"'喜欢'那些'喜欢我们'的人'"。"迈尔斯说'爱情'不是'神秘的'"——"'是'可以'用'科学'来'理解的'"
- **戴维·迈尔斯的背景**: 戴维·迈尔斯（David Myers）——美国心理学家——"密歇根霍普学院教授"——"《社会心理学》的作者"——"这本书'已经'更新了'十几次'"——"被'全球'数百所大学'采用'"。"迈尔斯'擅长'用'通俗的语言'讲解'心理学'"——"'他'的'书中'有'大量的'案例'和'实验'"——"'让'非专业'读者'也能'轻松理解'"。"他'还写了'很多'科普书'"——"'如'《探索心理学》'——'《快乐心理学》'"
