# 什么是数学

状态：已读

作者：柯朗

一句话评价：把数学从公式和计算中解放出来，还原为"思维的体操"——数学不是做题，是理解世界的语言。

核心收获：
- 数学的本质不是计算，而是"关系的发现"和"结构的理解"
- 数论的美：质数分布、费马大定理、哥德巴赫猜想——最纯的数学往往最无用也最美
- 几何的直觉：拓扑学把"连续变形"的概念推到极致，咖啡杯和甜甜圈是同一种东西
- 微积分的核心思想是"无限小"和"极限"，这是人类思维最了不起的跳跃之一
- 数学和应用的关系：纯数学家看不起应用，但所有应用数学最终都会找到用武之地
