# 深度思维

状态：已读

作者：叶修

一句话评价：叶修的"方法论书籍"——"探讨如何从'浅层思考'升级到'深度思考'"——"提出'信息处理'——'逻辑推理'——'系统思维'等'进阶路径'"。"深度思维的核心——'不是'想得多——'是'想得深——'看到问题的本质'而不是表象"。"信息处理能力'决定了'思考的深度"——"输入质量'决定了'输出质量"。

核心收获：
- **深度思维的核心**: 叶修说"深度思维'不是'想得多"——"是'想得深'"。"想得多'是'发散'"——"想得深'是'聚焦'"。"看到一个'现象'"——"浅层思考'只看到'表象'"——"深度思考'会追问'为什么'"——"再追问'为什么的为什么'"——"直到'找到'根本原因"。"比如'公司业绩下滑'"——"浅层思考说'要加强营销'"——"深度思考说'是产品问题'还是'市场问题'还是'团队问题'"——"找到'真正的'原因"——"才能'解决'问题"
- **信息处理能力**: 叶修说"信息处理能力'决定了'思考的深度"——"输入质量'决定了'输出质量'"。"如果你'输入的信息'是'片面的'"——"你的'思考结果'也会'是片面的'"。"如果你'输入的信息'是'高质量的'"——"你的'思考结果'也会'是高质量的'"。"提升信息处理能力的方法"：1）**广泛阅读**——"不要'只看'一个领域"——"要'跨领域'学习"；2）**批判性思考**——"不要'轻易相信'"——"要'追问'信息的来源'和'逻辑"；3）**建立知识框架**——"把'碎片化的'信息'组织起来'"——"形成'系统性的'理解"
- **逻辑推理**: 叶修"详细讲解了'逻辑推理'的方法"。1）**演绎推理**——"从'一般到特殊'"——"如果 A 则 B"——"A 成立"——"所以 B 成立"；2）**归纳推理**——"从'特殊到一般'"——"观察'多个案例'"——"总结出'规律'"；3）**类比推理**——"从'一个领域'到'另一个领域'"——"A 和 B'有相似之处'"——"所以 A 的方法'可以应用到'B"。叶修说"逻辑推理是'深度思维的基础'"——"没有'逻辑'"——"思考就是'空中楼阁'"
- **系统思维**: 叶修说"系统思维是'最高层次的'思考方式"。"系统思维'是看到'事物之间的关联'"——"而不是'孤立的点'"。"一个'问题'"——"不是'孤立存在'的"——"它是'系统的一部分'"。"比如'交通拥堵'"——"不是'路不够宽'"——"是'城市规划'——'公共交通'——'人口分布'——'出行习惯'等多个因素'共同作用'的结果"。"用系统思维看问题"——"你'不会'只解决'一个点'"——"你会'解决'整个'系统'"
- **从浅到深的进阶路径**: 叶修"提出了'从浅到深的'进阶路径"：1）**信息层**——"收集'信息'"——"这是'最浅的'"；2）**逻辑层**——"分析'信息之间的'逻辑关系'"——"这'深了一层'"；3）**系统层**——"看到'整个系统'"——"这'更深了'"；4）**本质层**——"找到'问题的本质'"——"这是'最深的'"。"大多数人'停留在'信息层'"——"他们'收集了很多信息'"——"但'没有'深入分析"。"深度思考者'会'一路'走到'本质层'"
- **叶修的背景**: 叶修——中国思维训练专家——"《深度思维》的作者"。他"长期'从事'思维训练'和'学习方法'的研究'"——"他的'方法'被'广泛应用'在'教育'和'企业管理'中"。叶修的"写作风格'非常接地气'"——"他'用大量的'案例'来解释'抽象的概念'"——"让'普通读者'也能'理解和应用'"
