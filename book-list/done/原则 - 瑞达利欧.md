# 原则

状态：已读

作者：瑞·达利欧

一句话评价：瑞·达利欧（Ray Dalio）"桥水基金创始人自传'"——"'极度求真、极度透明原则震撼人心'"。"'痛苦 + 反思=进步'公式简洁有力'"。"'可信度加权决策是管理创新'"。"'560 页的生活与工作方法论'"——"'商业领袖必读'"

核心收获：
- **极度求真**: 本书核心是"'极度求真'"的原则'"。"'桥水'的'文化'核心'"——"'不'说'客套话'"——"'直面'问题'"。"'这'听起来'反直觉'"——"'但'达利欧'用'40 年'证明'有效'"
- **极度透明**: 本书"讨论了极度透明'"。"'所有'会议'录音'"——"'所有'员工'互评'"。"'这'是'一种'激进'的'组织'管理'方式'"
- **五步流程**: 本书"介绍了五步流程'"。"'目标'"——"'问题'"——"'诊断'"——"'方案'"——"'执行'"。"'简单'但'大多数'人'做'不'到'第一'步'"："'诚实'地'面对'目标'"
- **可信度加权决策**: 本书"提出了可信度加权决策'"。"'不'是'一人一票'"——"'而'是'谁'在'这个'问题'上'过去'判断'更'准'"——"'谁'的'权重'更'大'"
- **创意择优**: 本书"强调创意择优'"。"'最好'的'想法'胜出'"——"'不'管'来自'实习生'还是'CEO'"
- **痛苦加反思等于进步**: 本书"提出了核心成长公式'"。"'痛苦'加'反思'等于'进步'"——"'达利欧'最'核心'的'成长'公式'"。"'痛苦'本身'没有'价值'"——"'但'对'痛苦'的'反思'能'产生'原则'"

- **作者的背景**: 瑞·达利欧（Ray Dalio, 1949-）——美国投资者、对冲基金经理——"《原则》的作者'"。"'他'创立'了'桥水'基金'"——"'全球'最大'的'对冲'基金'"。"'《原则》'2017 年'出版'"——"'总结'了'他'40 年'的'经验'"。"'他'的'投资'理念'"："'全球'宏观'投资'"——"'风险'平价'"。"'他'的'管理'哲学'"："'极度'求真'"——"'极度'透明'"——"'可信度'加权'决策'"
