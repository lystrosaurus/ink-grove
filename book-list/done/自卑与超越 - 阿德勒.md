# 自卑与超越

状态：已读

作者：阿尔弗雷德·阿德勒

一句话评价：阿尔弗雷德·阿德勒（Alfred Adler）的"个体心理学奠基之作'"——"'提出'自卑感是人类进步的源动力'"——"'将心理学从弗洛伊德的本能决定论中解放出来'"。"'人的行为被未来目标牵引'"——"'目的论 vs 因果论'"。"'所有问题都是人际关系问题'"——"'健康的人际关系是心理健康的基础'"

核心收获：
- **自卑感是进步的动力**: 阿德勒提出了"'自卑感是人类进步的源动力'"的核心观点'"。"'什么是自卑感'"——"'人'生来'就'有'不足感'"——"'身体'的'——'心理'的'——'社会'的'"。"'婴儿'依赖'成人'"——"'感到'无力'"——"'这是'自卑'的'起源'"。"'自卑感'不是'病'"——"'是'正常'的'"——"'是'人类'共有'的'"。"'因为'感到'不足'"——"'所以'追求'超越'"——"'追求'更好'——'更强'"。"'文明'的'进步'"——"'科学'的'发展'"——"'都'源于'自卑感'"——"'人类'想'克服'自然'的'限制'"。"'问题'不是'有'自卑感'"——"'是'如何'应对'自卑感'"
- **追求优越**: 阿德勒认为"人都有'追求优越'（Striving for Superiority）的本能'"。"'追求优越'不是'想要'比别人'强'"——"'是'想要'从'不足'状态'变成'更好'状态'"。"'追求优越'的'方向'"：1）**建设性**——"'通过'努力'变得'更好'"——"'学习'——'成长'"；2）**破坏性**——"'通过'贬低'别人'来'显得'自己'强'"——"'这是'错误'的'"。"'健康'的'追求优越'"——"'是'自我'超越'"——"'不是'压倒'别人'"。"'自卑情结'是'什么'"——"'当'人'无法'应对'自卑感'时'"——"'就'形成'自卑情结'"——"'自暴自弃'——'逃避'挑战'"
- **目的论 vs 因果论**: 阿德勒"与弗洛伊德的根本分歧在于目的论与因果论'"。"'弗洛伊德'是'因果论'"——"'过去'决定'现在'"——"'童年'创伤'决定'成年'行为'"——"'人'是'过去'的'奴隶'"。"'阿德勒'是'目的论'"——"'现在'被'未来'目标'牵引'"——"'人'选择'行为'来'达成'目的'"——"'人'是'自由'的'"。"'比如'社交恐惧'"——"'弗洛伊德'说'因为'童年'被'嘲笑'"——"'阿德勒'说'因为'不想'社交'的'目的'"——"'所以'制造'恐惧'情绪'"。"'目的论'的'意义'"——"'人'可以'改变'"——"'只要'改变'目的'"——"'行为'就'改变'"
- **社会兴趣**: 阿德勒提出了"'社会兴趣'（Social Interest）"的概念'"。"'社会兴趣'是'对'他人'的'关心'"——"'是'与'社会'的'连接'"。"'阿德勒'认为'"——"'所有'问题'都'是'人际关系'问题'"——"'工作'——'交友'——'爱情'"——"'都'是'人际关系'"。"'心理健康'的'标准'"——"'有'社会兴趣'"——"'关心'他人'"——"'贡献'社会'"。"'自卑情结'的'人'"——"'缺乏'社会兴趣'"——"'只'关注'自己'"——"'把'别人'当'敌人'"。"'如何'培养'社会兴趣'"——"'合作'"——"'贡献'"——"'信任'他人'"
- **生活风格**: 阿德勒提出了"'生活风格'（Style of Life）"的概念'"。"'生活风格'是'人'的'行为模式'"——"'是'人'应对'生活'的'方式'"。"'生活风格'在'童年'形成'"——"'4-5 岁'就'基本'定型'"。"'但'可以'改变'"——"'通过'自我'觉察'"——"'通过'重新'选择'目的'"。"'生活风格'的'类型'"：1）**支配型**——"'想'控制'别人'"；2）**索取型**——"'依赖'别人'"；3）**回避型**——"'逃避'问题'"；4）**社会有益型**——"'合作'——'贡献'"。"'只有'社会有益型'是'健康'的'"
- **早期记忆**: 阿德勒"重视早期记忆的分析'"。"'早期记忆'是'人'最早'的'记忆'"——"'通常是'3-5 岁'"。"'早期记忆'不是'偶然'的'"——"'是'人'选择'记住'的'"——"'反映'了'人'的'生活风格'"。"'分析'早期记忆'"——"'可以'理解'人'的'行为模式'"。"'比如'一个人'记得'被'欺负'"——"'他'的'生活风格'可能'是'防御'的'"——"'他'可能'不'信任'别人'"。"'改变'早期记忆'的'解读'"——"'可以'改变'生活风格'"
- **阿德勒的背景**: 阿尔弗雷德·阿德勒（Alfred Adler, 1870-1937）——奥地利心理学家——"个体心理学创始人'"。"他'是'犹太人'"——"'出身'贫寒'"——"'童年'体弱多病'"——"'有'自卑感'"。"他'曾是'弗洛伊德'的'追随者'"——"'但'后来'分道扬镳'"——"'因为'不'认同'性本能'理论'"。"阿德勒'的'理论'更'积极'"——"'强调'人'的'自由'和'选择'"。"《自卑与超越》'1930 年'出版'"——"'是'他'的'代表作'"。"阿德勒'的'影响'深远'"——"'人本主义心理学'——'认知行为疗法'都'受'他'影响'"
