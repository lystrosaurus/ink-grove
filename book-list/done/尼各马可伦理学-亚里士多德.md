# 尼各马可伦理学

状态：已读

作者：亚里士多德

一句话评价：亚里士多德的伦理学核心著作，系统探讨了幸福、德性、中庸、友谊等核心伦理概念，是西方伦理学的奠基之作。

核心收获：
- 理解了亚里士多德的幸福观——幸福（eudaimonia）不是情绪而是"灵魂合乎德性的实现活动"
- "中庸之道"——德性是两个极端之间的中道，如勇敢是懦弱和鲁莽之间的中道
- 友谊的三种类型——功利型、快乐型、德性型，只有德性型友谊才是真正的友谊
