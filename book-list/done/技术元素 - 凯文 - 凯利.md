# 技术元素

## 基本信息
- **书名**: 技术元素
- **作者**: 凯文 - 凯利
- **出版社**: 电子工业出版社
- **出版年份**: 2019
- **页数**: 480
- **分类**: 科学探索/西方哲学

## 选书理由
KK 对"技术"概念的哲学重构——技术不是工具，而是一个有自身进化逻辑的"第七生命王国"。他提出"技术元素"（technium）概念，认为技术像生物一样有自主演化的趋势。这种视角超越了技术乐观主义与悲观主义的对立，帮助我们理解人与技术的共生关系。希望建立对技术本质的深层认知框架。
