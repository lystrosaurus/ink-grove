# 真希望你也喜欢自己

状态：已读

作者：房琪

一句话评价：房琪的"个人成长随笔集"——"用'温暖的文字'鼓励读者'接纳自己、爱自己'"——"是'写给'所有'不够自信的人'的'一封情书'"。"'喜欢自己'是'爱别人的'前提'"——"'只有'先'接纳自己'"——"'才能'真正'接纳别人'"。"房琪用'自身经历'告诉读者"——"'每个人'都有'自己的'闪光点'"——"'不需要'跟别人'比较'"——"'自信'不是'天生的'"——"是'可以'通过'日常的'小行动'慢慢建立的'"

核心收获：
- **喜欢自己的重要性**: 房琪说"'喜欢自己'是'爱别人的'前提'"。"很多人'都'在'努力'去'爱别人'"——"'努力'去'讨好别人'"——"'但'他们'从来'没有'努力'去'喜欢自己'"。"一个'不喜欢自己'的人'"——"'是'无法'真正地'去'爱别人的'"——"'因为'他'的'爱'是'带着'讨好'的'"——"'是'带着'乞求'的'"。"房琪说'你'只有'先'喜欢自己'"——"'你'的'爱'才会'是'健康的'"——"'你'的'关系'才会'是'平等的'"
- **接纳自己的不完美**: 房琪"反复强调'接纳自己的不完美'"。"'每个人'都有'缺点'"——"'每个人'都有'不足'"——"'但'这些'缺点'和'不足'"——"'不'影响'你'成为一个'好的人'"。"房琪'说'她'自己'也'有很多'缺点'"——"'她'会'焦虑'"——"'她'会'自卑'"——"'她'会'嫉妒'"——"'但'她'学会了'接纳'这些'缺点'"——"'她'说'缺点'也是'我'的一部分'"
- **不需要比较**: 房琪说"'不需要'跟别人'比较'"。"'每个人'都有'自己的'节奏'"——"'每个人'都有'自己的'路'"——"'你'的'路'和'别人'的'路'"——"'不一样'"——"'所以'没有'可比性'"。"房琪'说'她'曾经'也'很'爱'比较'"——"'看到'别人'成功'"——"'她'就'焦虑'"——"'看到'别人'漂亮'"——"'她'就'自卑'"——"'后来'她'明白了'"——"'比较'是'痛苦的'根源'"——"'不比较'"——"'才'会'快乐'"
- **自信的建立**: 房琪说"'自信'不是'天生的'"——"'是'可以'通过'日常的'小行动'慢慢建立的'"。"她'提出了'几个'方法'"：1）**每天记录三件好事**——"'感恩'你'拥有的'东西'"；2）**停止自我批评**——"'不要'总是'说'自己'不好'"；3）**做让自己骄傲的事**——"'哪怕'是'很小的'事'"——"'完成'了'就'给自己'点赞'"；4）**和积极的人在一起**——"'远离'那些'总是'打击你的人'"
- **房琪的经历**: 房琪"在书中'分享了'她的'成长经历'"。"她'曾经'很'自卑'"——"'她'觉得'自己'不够'漂亮'"——"'不够'聪明'"——"'不够'优秀'"——"'她'总是'躲在'角落里'"——"'不敢'表达'自己'"。"后来'她'开始'做'短视频'"——"'她'用'自己的'方式'去'表达'"——"'她'发现'原来'自己'也'有'闪光点'"——"'原来'自己'也'可以'被'很多人'喜欢'"
- **给不够自信的人**: 房琪说"这本书'是'写给'所有'不够自信的人'的'"。"'你'可能'觉得'自己'不够'好'"——"'不够'漂亮'"——"'不够'聪明'"——"'不够'有钱'"——"'但'房琪'说'"——"'你'已经'很好了'"——"'你'只是'没有'发现'而已'"。"'房琪'希望'每个'读者'都'能'喜欢自己'"——"'因为'你'值得'被'喜欢'"——"'首先'被'自己'喜欢'"
- **房琪的背景**: 房琪（1993-）——中国旅行博主、作家——"《真希望你也喜欢自己》是她'第一本'书'"——"2022 年出版"——"在'年轻人'中'引起了'很大的'共鸣"。她"通过'短视频'分享'旅行'和'生活'"——"'她的'文字'温暖'而'治愈'"——"'她'用'自己的'经历'告诉'年轻人'"——"'你'可以'不'完美'"——"'但'你'要'喜欢自己'"
