# 自私的基因

状态：已读

作者：理查德·道金斯

一句话评价：理查德·道金斯（Richard Dawkins）的"进化生物学经典'"——"'基因中心论'颠覆生命认知'"——"'我们是基因的生存机器'"——"'震撼人心'"。"'模因理论'开创文化传播研究'"——"'进化稳定策略'解释动物行为'"——"'利他行为的基因解释'"

核心收获：
- **基因中心论**: 道金斯提出了"'基因中心论'的进化观'"。"'传统'进化观'是'个体中心'的'"——"'认为'进化'是'为了'物种'或'个体'的'利益'"。"'道金斯'说'不对'"——"'进化'的'基本单位'是'基因'"——"'不是'个体'"——"'不是'物种'"。"'基因'是'复制子'（Replicator）'"——"'它'的'唯一'目的'是'复制'自己'"。"'个体'是'生存机器'（Survival Machine）'"——"'基因'制造'个体'来'保护'自己'"——"'来'复制'自己'"。"'所以'我们'是'基因的'生存机器'"——"'我们'的'身体'"——"'我们'的'行为'"——"'都'是'基因为了'复制'自己'而'设计'的'"
- **成功基因的特性**: 道金斯"分析了成功基因的特性'"。"'成功基因'的'一个'显著'特性'是'无情的自私性'"。"'自私'不是'道德'概念'"——"'是'描述'基因'的'行为'"——"'基因'只'关心'自己'的'复制'"。"'但'自私'的'基因'可以'产生'利他'的'行为'"——"'比如'父母'为'孩子'牺牲'"——"'因为'孩子'有'父母'的'基因'"。"'亲缘选择'（Kin Selection）'"——"'帮助'有'亲缘关系'的'个体'"——"'等于'帮助'自己'的'基因'"。"'互惠利他'（Reciprocal Altruism）'"——"'我'帮助'你'"——"'你'帮助'我'"——"'双方'都'受益'"——"'基因'都'受益'"
- **进化稳定策略**: 道金斯"介绍了'进化稳定策略'（ESS, Evolutionarily Stable Strategy）'"。"'ESS'是'一种'策略'"——"'当'大多数'个体'采用'时'"——"'任何'其他'策略'都'无法'入侵'"。"'鹰鸽博弈'"——"'鹰'是'攻击性'策略'"——"'鸽'是'和平'策略'"——"'纯'鹰'或'纯'鸽'都'不'稳定'"——"'混合'策略'才'是'ESS'"。"'ESS'的'应用'"：1）**动物行为**——"'解释'动物'的'争斗'——'求偶'——'育雏'"；2）**人类社会**——"'解释'道德'——'规范'——'合作'"。"'ESS'的'意义'"——"'进化'不是'随机'的'"——"'有'稳定'的'模式'"
- **利他行为的解释**: 道金斯"解释了利他行为的进化起源'"。"'利他行为'看似'与'自私'矛盾'"——"'但'可以'用'基因'解释'"。"'亲缘利他'"——"'帮助'亲属'"——"'因为'亲属'有'相似'基因'"——"'汉密尔顿法则'"——"'rB > C'"（r=亲缘系数，B=受益，C=成本）。"'互惠利他'"——"'帮助'非亲属'"——"'期望'将来'回报'"——"'你'帮我'挠背'"——"'我'帮你'挠背'"。"'群体选择'"——"'利他'的'群体'比'自私'的'群体'更'成功'"——"'但'群体选择'有'争议'"。"'利他'不是'无私'"——"'是'基因'的'自私'策略'"
- **模因理论**: 道金斯提出了"'模因'（Meme）"的概念'"。"'模因'是'文化'的'复制子'"——"'类似'基因'"。"'模因'的'例子'"——"'旋律'——'口号'——'时尚'——'宗教'"——"'任何'可以'复制'的'文化'元素'"。"'模因'通过'模仿'传播'"——"'从'一个'大脑'到'另一个'大脑'"。"'模因'也'有'进化'"——"'好'的'模因'传播'快'"——"'坏'的'模因'消失'"。"'模因'可能'与'基因'冲突'"——"'比如'独身'的'宗教'模因'"——"'不利'基因'复制'"——"'但'有利'模因'传播'"。"'模因理论'开创'了'文化传播'研究'"——"'是'文化'进化'的'基础'"
- **生存机器**: 道金斯"描述了基因如何制造生存机器'"。"'基因'不'直接'控制'行为'"——"'基因'编码'蛋白质'"——"'蛋白质'构建'身体'"——"'身体'产生'行为'"。"'大脑'是'基因'的'计算机'"——"'基因'预设'程序'"——"'大脑'执行'程序'"。"'但'大脑'有'学习'能力'"——"'可以'超越'基因'的'预设'"。"'人类'是'唯一'能'反抗'基因'的'物种'"——"'避孕'"——"'不'生育'"——"'这是'基因'的'叛徒'"。"'道金斯'说'我们'可以'反抗'基因'的'暴政'"——"'用'理性'和'文化'"
- **道金斯的背景**: 理查德·道金斯（Richard Dawkins, 1941-）——英国进化生物学家、科普作家——"《自私的基因》的作者'"。"他'在'牛津大学'任教'"——"'他'是'英国'皇家学会'院士'"。"《自私的基因》'1976 年'出版'"——"'轰动'一时'"——"'销量'超过'百万册'"。"道金斯'是'无神论者'"——"'他'写'了'《上帝错觉》'"——"'批判'宗教'"。"他'是'科普'大师'"——"'文笔'优美'"——"'善用'比喻'"——"'把'复杂'的'科学'讲'得'通俗'"。"道金斯'的'影响'深远'"——"'他'的'基因中心论'成为'进化生物学'的'主流'观点'"
