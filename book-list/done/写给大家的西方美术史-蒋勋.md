# 写给大家的西方美术史

状态：已读

作者：蒋勋

一句话评价：从希腊雕塑到当代艺术，蒋勋把西方美术史讲成了一场"视觉革命"的连续剧——每一次风格转变都是对前一次的反叛。

核心收获：
- 希腊艺术追求"理想美"——人体是最完美的几何形式
- 文艺复兴是把"人"从"神"的阴影中解放出来。达芬奇、米开朗基罗、拉斐尔的共同点是"把人当人画"
- 印象派的革命性：不画"知道的颜色"，画"看到的颜色"。阴影不是黑的，是紫色的
- 现代艺术的本质不是"画得丑"，是"不再追求美"——而是追求真实、表现、观念
- 杜尚的小便池是艺术史的转折点：从此"什么是艺术"的定义权从作品转移到了观念
