# 全球通史

状态：已读

作者：斯塔夫里阿诺斯

一句话评价：用"全球视角"重写人类历史，跳出"西方中心论"和"国别史"的局限，第一次把人类当作一个整体来讲述——不是"各国的历史"，而是"人类的历史"。

核心收获：
- **全球史观**: 历史不是"各国的历史拼在一起"，而是"人类的历史"。文明的交流、碰撞、融合才是主线。丝绸之路、蒙古帝国、大航海——这些不是某个国家的故事，是人类的故事
- **史前史的价值**: 大多数历史书从文明写起，斯塔夫里阿诺斯从人类起源写起。农业革命是人类最重要的转折点——比任何王朝更替、任何帝王将相都重要。农业革命改变了人类的生活方式、社会结构、甚至基因
- **欧亚大陆的优势**: 为什么是欧洲人发现了美洲，而不是美洲人发现了欧洲？斯塔夫里阿诺斯的答案不是"欧洲人更聪明"，是地理条件——欧亚大陆是东西轴向（纬度相近，气候相似，作物易传播），拥有最多可驯化的动植物（小麦、水稻、牛、马）
- **1500 年的分水岭**: 1500 年是全球史最重要的分界线——哥伦布 1492 年到达美洲，达伽马 1498 年绕过好望角，从此世界真正连成一个整体。1500 年之前是"区域文明"，1500 年之后是"全球文明"
- **西方的崛起**: 斯塔夫里阿诺斯不认为西方崛起是"必然"或"优越"，而是一系列偶然因素的叠加——地理大发现、科学革命、工业革命。这些因素在 1500 年之前并不明显，一旦叠加就产生了碾压性的优势
- **殖民主义的双重性**: 殖民主义带来了灾难（奴隶贸易、种族灭绝、资源掠夺），但也带来了全球化（贸易网络、技术传播、文化交流）。斯塔夫里阿诺斯不美化殖民主义，但他也不简化——历史的复杂性在于好与坏常常纠缠在一起
- **全球化的 500 年**: 今天的全球化不是"新鲜事"，而是 1500 年以来 500 年进程的延续。理解今天的全球化，要放在这个大框架下——它不是从 iPhone 开始的，是从哥伦布开始的
- **历史的教育意义**: 斯塔夫里阿诺斯写这本书的目的是让读者"从全球角度看问题"——当你学会用全球视角看历史，你也就学会了用全球视角看今天的新闻、冲突和合作
