# 终身成长

状态：已读

作者：卡罗尔·德韦克

一句话评价：卡罗尔·德韦克（Carol Dweck）的"成长型思维经典'"——"'区分'固定型思维'与'成长型思维'"——"'固定型思维'认为'能力'是'天生的'"——"'成长型思维'认为'能力'可以'通过'努力'培养'"——"'失败'不是'终点'"——"'是'成长的机会'"

核心收获：
- **固定型思维 vs 成长型思维**: 德韦克区分了"'两种'思维模式'"。"'固定型思维'"——"'认为'能力'是'天生的'"——"'你'要么'聪明'"——"'要么'不'聪明'"——"'努力'是'没用'的'"——"'如果'你'需要'努力'"——"'说明'你'不'聪明'"。"'成长型思维'"——"'认为'能力'可以'通过'努力'培养'"——"'大脑'像'肌肉'"——"'越'锻炼'越'强壮'"——"'失败'是'学习的机会'"——"'挑战'是'成长的'机会'"
- **努力的意义**: 德韦克说"'在'成长型思维'中'"——"'努力'是'积极的'"。"'固定型思维'认为'"——"'如果你'聪明'"——"'你'不'需要'努力'"——"'如果你'需要'努力'"——"'说明'你'不'聪明'"。"'成长型思维'认为'"——"'努力'是'通往'精通'的'路径'"——"'没有'努力'"——"'天赋'也'会'浪费'"——"'天赋'加'努力'"——"'等于'成功'"
- **面对失败**: 两种思维模式"面对失败'的态度'完全不同'"。"'固定型思维'"——"'失败'是'对'能力的'否定'"——"'我'失败'了'"——"'说明'我'不'行'"——"'我'放弃'"。"'成长型思维'"——"'失败'是'学习的机会'"——"'我'失败'了'"——"'说明'我'需要'改进'"——"'我'尝试'新的'方法'"。"德韦克'研究'发现'"——"'成长型思维'的'学生'在'失败后'更'努力'"——"'固定型思维'的'学生'在'失败后'放弃'"
- **表扬方式**: 德韦克说"'表扬'会'影响'孩子的'思维模式'"。"'表扬'天赋'"——"'你'真'聪明'"——"'会'培养'固定型思维'"——"'孩子'会'害怕'挑战'"——"'因为'挑战'可能'证明'他们'不'聪明'"。"'表扬'过程'"——"'你'很'努力'"——"'你'用'了'很好的'策略'"——"'会'培养'成长型思维'"——"'孩子'会'喜欢'挑战'"——"'因为'挑战'是'学习的机会'"。"德韦克'建议'"——"'不要'表扬'孩子'聪明'"——"'要'表扬'他们'的'努力'——'策略'——'进步'"
- **思维转换**: 德韦克说"'思维模式'可以'改变'"。"'如何'培养'成长型思维'"：1）**认识'固定型思维'"——"'当'你'想'放弃'时'"——"'当'你'害怕'挑战'时'"——"'意识'到'这是'固定型思维'在'作祟'"；2）**对话'自己'"——"'告诉'自己'"——"'我'还'不会'"——"'但'我'可以'学习'"；3）**设定'学习'目标'"——"'不'是'证明'自己'"——"'是'学习'新东西'"；4）**庆祝'进步'"——"'不'是'庆祝'成功'"——"'是'庆祝'比'昨天'更好'"
- **持续学习**: 成长型思维"强调'持续学习'"。"'在'成长型思维'中'"——"'学习'不'是'为了'考试'"——"'是'为了'成长'"。"'终身学习'的'人'"——"'不'害怕'新事物'"——"'他们'认为'学习'是'一生的'事情'"。"'如何'培养'终身学习'的'习惯'"——"'保持'好奇心'"——"'挑战'自己'"——"'从'失败中'学习'"——"'相信'自己'可以'成长'"
- **卡罗尔·德韦克的背景**: 卡罗尔·德韦克（Carol Dweck）——美国心理学家——"斯坦福大学教授'"——"'成长型思维'的'创始人'"。"她'研究'了'几十年'"——"'发现'思维模式'对'成功'的'影响'"。"《终身成长》'2006 年'出版'"——"'被'翻译成'多种语言'"——"'销量'超过'百万册'"。"她的'理论'被'广泛应用'于'教育'——'企业'——'体育'"——"'很多'学校'和'公司'都'在'使用'成长型思维'培训'"
