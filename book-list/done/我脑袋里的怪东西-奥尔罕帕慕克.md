# 我脑袋里的怪东西

状态：已读

作者：奥尔罕帕慕克

一句话评价：帕慕克以伊斯坦布尔街头小贩麦夫鲁特为主角，用四十年的人生跨度描绘了一座城市的变迁，是充满人情味的城市史诗。

核心收获：
- 理解了帕慕克的叙事技巧——多视角、多时间线交织，把一个人的故事变成一座城市的故事
- 麦夫鲁特卖酸奶的执着——在快速城市化的时代，一个普通小贩如何保持自己的尊严和生活方式
- 伊斯坦布尔的城市变迁——从传统到现代，从乡村到城市，从东方到西方的撕裂
