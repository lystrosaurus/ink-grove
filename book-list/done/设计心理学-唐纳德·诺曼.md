# 设计心理学

状态：已读

作者：唐纳德·诺曼

一句话评价：唐纳德·诺曼（Donald Norman）的"设计领域开山之作'"——"'提出"以用户为中心"的设计理念'"。"'揭示了好设计与坏设计的区别在于是否符合人类的认知习惯'"。"'物品的外观应该暗示它的用法'"——"'用户的错误往往是设计的失败'"

核心收获：
- **示能（Affordance）**: 诺曼提出了"'示能'"的概念'"。"'什么'是'示能'"——"'物品'的'外观'应该'暗示'它'的'用法'"。"'比如'门把手'"——"'圆形'的'把手'"——"'暗示'要'转动'"——"'平板'的'把手'"——"'暗示'要'推'"。"'如果'门'需要'推'"——"'但'装'了'圆形'把手'"——"'人'会'拉'"——"'这'就'是'设计'失败'"。"'好'的'示能'"——"'让'人'一看'就'知道'怎么'用'"——"'不'需要'说明书'"。"'示能'不'是'属性'"——"'示能'是'人'与'物品'的'关系'"
- **以用户为中心**: 诺曼"提出了以用户为中心的设计理念'"。"'传统'设计'"——"'设计师'决定'产品'长'什么样'"——"'用户'适应'产品'"。"'以'用户'为'中心'"——"'设计师'研究'用户'"——"'产品'适应'用户'"。"'如何'做'到'"：1）**观察用户**——"'看'用户'怎么'用'产品'"；2）**理解认知**——"'理解'人'的'认知'习惯'"；3）**迭代测试**——"'不断'测试'"——"'不断'改进'"。"'这'种'理念'"——"'现在'是'主流'"——"'UX 设计'——'交互设计'都'基于'这'个'理念'"
- **错误的归因**: 诺曼"讨论了错误的归因'"。"'当'用户'用'错'产品'"——"'人们'往往'怪'用户'"——"'说'用户'笨'"。"'但'诺曼'说'"——"'用户'的'错误'往往'是'设计'的'失败'"。"'为什么'"——"'因为'设计'没有'考虑'人'的'认知'局限'"。"'人'的'认知'局限'"：1）**注意力有限**——"'人'不'能'同时'注意'很多'东西'"；2）**记忆有限**——"'人'的'短期'记忆'只有'7±2 个项目'"；3）**容易分心**——"'人'容易'被'干扰'"。"'好'的'设计'"——"'要'考虑'这些'局限'"——"'让'产品'容错'"
- **概念模型**: 诺曼"介绍了概念模型的概念'"。"'什么'是'概念'模型'"——"'人'对'产品'如何'工作'的'理解'"。"'比如'恒温器'"——"'人'的'概念'模型'"——"'温度'调'高'"——"'房间'热'得'快'"。"'但'实际'不'是'这样'"——"'恒温器'只'是'开关'"。"'概念'模型'不'匹配'"——"'人'就'会'用'错'"。"'设计'的'任务'"——"'让'产品'的'实际'模型'"——"'匹配'人'的'概念'模型'"——"'或者'改变'人'的'概念'模型'"
- **反馈与约束**: 诺曼"强调了反馈与约束的重要性'"。"'反馈'"——"'产品'要'告诉'用户'"它'在'做'什么'"。"'比如'按钮'按下'后'"——"'要'有'声音'或'光'"——"'告诉'用户'按'成功'了'"。"'约束'"——"'限制'用户'的'操作'"——"'防止'错误'"。"'比如'SIM 卡'托盘'"——"'只'有'一个'方向'能'放'进去'"——"'这'是'物理'约束'"。"'好'的'设计'"——"'有'充分'的'反馈'"——"'有'合理'的'约束'"
- **七宗罪**: 诺曼"列举了设计的七宗罪'"。"1）**骄傲**——"'设计师'自以为是'"；2）**懒惰**——"'不'愿意'测试'"；3）**贪婪**——"'添加'太多'功能'"；4）**嫉妒**——"'模仿'别人'"；5）**暴食**——"'功能'过多'"；6）**愤怒**——"'对'用户'不耐烦'"；7）**色欲**——"'追求'美观'牺牲'好用'"。"'这'些'罪'"——"'都'导致'坏'设计'"。"'好'的'设计师'"——"'要'避免'这些'罪'"
- **诺曼的背景**: 唐纳德·诺曼（Donald Norman, 1935-）——美国认知心理学家、设计师——"《设计心理学》的作者'"。"他'在'MIT'学习'工程'"——"'后'来'转'心理学'"。"他'在'加州大学'任教'"——"'他'也'在'企业'工作'"——"'HP'——'苹果'"。"《设计心理学》'1988 年'出版'"——"'原名'《日常用品的心理学》'"。"'这本书'开创'了'用户体验'领域'"。"'诺曼'还'写'了'《情感化设计》'"——"'提出'设计'的'三个'层次'"——"'本能'——'行为'——'反思'"
