# 三国演义

状态：已读

作者：罗贯中

一句话评价：中国历史演义的巅峰——七分史实三分虚构，把权谋、忠义、背叛、智慧写到了极致。"天下大势分久必合合久必分"是中国人对历史最经典的概括，三国演义塑造的人物形象至今定义着中国人对"忠""义""智""奸"的理解。

核心收获：
- **"分久必合，合久必分"**: 开篇第一句就是全书的题眼。这是中国人对历史周期律最精炼的概括——统一的帝国终将分裂，分裂的乱世终将统一。这种循环史观是中国历史思维的核心
- **人物的脸谱化**: 演义的特点也是局限。刘备仁德、关羽忠义、曹操奸诈、诸葛亮智慧、张飞莽撞、赵云完美。脸谱化让人物成为文化符号（提到"忠"就想到关羽，提到"奸"就想到曹操），但也简化了历史的复杂性。《三国志》里的曹操远比《三国演义》里的立体
- **赤壁之战**: 全书最高潮。连环计（庞统）、反间计（蒋干盗书）、苦肉计（黄盖）、借东风（诸葛亮）——计中有计，环环相扣。这场战争是中国军事文学的巅峰，后来无数战争描写都在模仿赤壁的叙事节奏
- **忠义的主题**: 关羽是"忠义"的化身——千里走单骑、过五关斩六将、华容道义释曹操。但关羽的败亡恰恰是因为"义"变成了"执"——他放走曹操是"义"，但他刚愎自用、看不起东吴也是"义"的延伸。罗贯中写出了一个深刻道理：美德走到极端就变成缺点
- **诸葛亮的"神化"**: 《三国演义》把诸葛亮塑造成"半仙"——借东风、草船借箭、空城计、七擒孟获、六出祁山。这些大多不是史实，但罗贯中需要诸葛亮成为一个"完美军师"的文化符号。这个符号比真实的诸葛亮更有影响力
- **七分史实三分虚构**: 桃园结义是虚构的（三国志只说三人"恩若兄弟"），草船借箭是虚构的（历史上是孙权的事），空城计是虚构的（三国志没有记载），借东风是虚构的。但虚构不等于不好——《三国演义》之所以成为经典，是因为它把历史变成了"故事"，把人物变成了"符号"
- **三国的悲剧感**: 三国演义最打动人的不是"谁赢了"，是"所有人都在努力但所有人都失败了"。刘备想复兴汉室，失败了；曹操想统一天下，没完成；诸葛亮想北伐成功，星落五丈原；孙权想守成，后代依然被灭。三国的结局是"三家归晋"——拼了三代人的命，最后便宜了司马懿。这种"努力了但没用"的悲剧感是三国演义最深沉的情感底色
- **三国演义的文化影响**: 中国人对三国的理解 90% 来自《三国演义》而非《三国志》。"说曹操曹操到""三个臭皮匠顶个诸葛亮""既生瑜何生亮"——这些成语和俗语塑造了中国人的思维方式。三国演义不只是小说，是中国文化的一部分
