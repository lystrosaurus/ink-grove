# 三国演义

状态：已读

作者：罗贯中

一句话评价：中国历史演义的巅峰——七分史实三分虚构，把权谋、忠义、背叛、智慧写到了极致。"天下大势分久必合合久必分"是中国人对历史最经典的概括。

核心收获：
- "天下大势，分久必合，合久必分"——开篇第一句就是全书的题眼。这是中国人对历史周期律最精炼的概括
- 人物的脸谱化是演义的特点也是局限：刘备仁德、关羽忠义、曹操奸诈、诸葛亮智慧。脸谱化让人物容易记住，但也简化了历史的复杂性
- 权谋描写的巅峰：赤壁之战是全书高潮，连环计、反间计、苦肉计、借东风，计中有计，每一环都扣人心弦
- 忠义的主题：关羽是"忠义"的化身，但他也有傲慢和刚愎。他的败亡恰恰是因为"义"变成了"执"
- 七分史实三分虚构：桃园结义、草船借箭、空城计、借东风全是虚构，但虚构得比史实更精彩
