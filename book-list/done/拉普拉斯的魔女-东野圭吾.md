# 拉普拉斯的魔女

状态：已读

作者：东野圭吾

一句话评价：东野圭吾融合量子物理学与推理的科幻悬疑小说，以"拉普拉斯妖"为概念核心，探讨了预测未来与自由意志的哲学问题。

核心收获：
- 理解了"拉普拉斯妖"的思想实验——如果有一个存在知道宇宙中所有粒子的位置和速度，它是否能预测未来
- 东野圭吾尝试将硬科学（量子物理、流体力学）融入推理小说——这是一个大胆的跨界尝试
- 科学与伦理的冲突：如果人类能够预测未来，这种能力应该被如何使用
