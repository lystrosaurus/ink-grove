# 三国志

状态：已读

作者：陈寿

一句话评价：二十四史中最"好看"的一部——人物鲜活、叙事紧凑，裴松之的注解比正文还精彩，是了解三国历史的第一手材料。

核心收获：
- 三国志分《魏书》《蜀书》《吴书》三部分，以魏为正统，但蜀吴人物一样生动
- 曹操的复杂性：既是"治世之能臣"，又是"乱世之奸雄"。三国志对他的刻画最立体
- 诸葛亮的真实形象：军事才能不如政治才能，"鞠躬尽瘁"比"神机妙算"更准确
- 裴松之注解的价值：补充了大量三国志正文没有的史料，很多后来被《世说新语》《后汉书》引用
- 三国志与三国演义的区别：三国志是"史"，演义是"小说"。草船借箭、空城计、借东风全是虚构
