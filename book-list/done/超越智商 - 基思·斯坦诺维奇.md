# 超越智商

状态：已读

作者：基思·斯坦诺维奇

一句话评价：基思·斯坦诺维奇（Keith Stanovich）的"认知心理学经典'"——"'提出"理性障碍"概念'"。"'高智商不等于理性，聪明人也会做出愚蠢决策'"。"'智力和理性是两套不同的认知系统'"——"'教育的重点应该培养理性思维'"

核心收获：
- **理性障碍**: 本书核心是"'理性障碍'"的概念'"。"'什么'是'理性障碍'"——"'高'智商'的'人'"——"'也'可能'缺乏'理性'思考'能力'"。"'例子'"：1）**聪明人信伪科学**——"'科学家'相信'占星术'"；2）**高 IQ 做蠢事**——"'高'智商'的'人'投资'被骗'"；3）**专家犯错**——"'专家'做出'明显'错误'的'决策'"。"'理性障碍'的'启示'"——"'智商'高'不'保证'理性'"
- **智力与理性的区别**: 本书"区分了智力和理性'"。"'智力'是'什么'"：1）**计算能力**——"'算'得'快'"；2）**记忆力**——"'记得'多'"；3）**逻辑推理**——"'推理'能力强'"。"'理性'是'什么'"：1）**避免认知偏差**——"'不'被'偏见'左右'"；2）**做出好决策**——"'考虑'长远'"；3）**根据证据更新信念**——"'愿意'改变'想法'"。"'智力'和'理性'是'两套'不同'的'能力'"——"'相关'但'不'相同'"
- **双系统理论**: 本书"介绍了双系统理论'"。"'系统 1'"：1）**快**——"'自动'反应'"；2）**直觉**——"'不'需要'思考'"；3）**容易错**——"'有'偏见'"。"'系统 2'"：1）**慢**——"'需要'思考'"；2）**理性**——"'分析'问题'"；3）**准确**——"'但'费力'"。"'聪明'人'的'问题'"——"'他们'依赖'系统 1'"——"'因为'他们'聪明'"——"'觉得'自己'对'"。"'但'系统 1'容易'出错'"
- **认知偏差**: 本书"列举了常见的认知偏差'"。"'1）**确认偏误**——"'只'看'支持'自己'的'证据'"；"'2）**锚定效应**——"'被'第一'个'数字'影响'"；"'3）**可得性启发**——"'容易'想到'的'事'"——"'觉得'更'可能'"；"'4）**框架效应**——"'同样'的'信息'"——"'不同'表述'"——"'决策'不同'"。"'聪明'人'也'有'这些'偏差'"——"'甚至'更'严重'"——"'因为'他们'更'自信'"
- **如何提升理性**: 本书"提供了提升理性的方法'"。"'如何'提升'理性'"：1）**意识到偏差**——"'知道'自己'有'偏差'"；2）**慢思考**——"'用'系统 2'"——"'不'要'太快'下结论'"；3）**考虑反面**——"'主动'寻找'反面'证据'"；4）**概率思维**——"'用'概率'思考'"——"'不'是'非黑即白'"；5）**元认知**——"'思考'自己'的'思考'"。"'这些'方法'可以'学习'"——"'理性'是'可以'培养'的'"
- **教育的启示**: 本书"讨论了教育的启示'"。"'传统'教育'"——"'重视'智力'"——"'教'知识'"——"'教'技能'"。"'但'本书'认为'"——"'教育'应该'重视'理性'"。"'如何'做'"：1）**教批判性思维**——"'质疑'——'分析'"；2）**教概率**——"'理解'不确定性'"；3）**教认知偏差**——"'让'学生'知道'偏差'"；4）**教科学方法**——"'根据'证据'更新'信念'"。"'理性'教育'"——"'比'智力'教育'更'重要'"
- **作者的背景**: 基思·斯坦诺维奇（Keith Stanovich）——加拿大心理学家、多伦多大学教授——"《超越智商》的作者'"。"他'研究'理性'和'决策'"——"'30 多年'"。"'他'提出'了'理性障碍'概念'"——"'提出'了'双系统'理论'"。"'《超越智商》'2009 年'出版'"——"'畅销'全球'"。"'他'的'其他'著作'"——"'《什么是理性》'——'《推理与理性》'"
