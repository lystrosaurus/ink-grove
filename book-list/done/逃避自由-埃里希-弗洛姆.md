# 逃避自由

## 基本信息
- **书名**: 逃避自由
- **作者**: 埃里希 - 弗洛姆
- **出版社**: 国际文化出版公司
- **出版年份**: 2020
- **页数**: 256
- **分类**: 心理与沟通/西方哲学

## 选书理由
弗洛姆从精神分析角度剖析现代人面对自由时的矛盾心理：既渴望自由，又恐惧自由带来的孤独与责任。书中提出的"逃避自由"机制（权威主义、破坏欲、机械趋同）对理解极权主义兴起、现代人焦虑根源有深刻启示。希望理解自由的心理学代价，以及如何在保持独立的同时避免孤立。
