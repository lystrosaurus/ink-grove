# 大学

状态：已读

作者：曾子

一句话评价：儒家修身治国的纲领性文献，全文仅一千七百余字，却用"三纲领八条目"构建了一个从个人认知到天下治理的完整逻辑链条。"修身齐家治国平天下"成为中国知识分子两千年的精神坐标。

核心收获：
- **三纲领**: "大学之道，在明明德，在亲民，在止于至善。"这是《大学》的总目标。"明明德"是发扬人内在的光明德性；"亲民"（或作"新民"）是使人日新又新、不断进步；"止于至善"是达到最完善的境界。三者是递进关系：先修己，再化人，最后达到完美
- **八条目**: 格物→致知→诚意→正心→修身→齐家→治国→平天下。这是实现三纲领的八个步骤。前四个是"内修"（认知和道德），后四个是"外治"（社会和政治）。"修身"是中间枢纽——内修的终点，外治的起点
- **"格物致知"的争议**: "格物致知"是八条目的起点，但《大学》原文没有解释什么叫"格物"。朱熹认为"格物"是"穷究事物之理"（通过研究外物获得知识），王阳明认为"格物"是"正其不正以归于正"（纠正内心的不正）。两种解释代表了理学和心学的根本分歧
- **"诚意正心"的道德防线**: "所谓诚其意者，毋自欺也。"诚意就是不欺骗自己——像厌恶恶臭、喜欢美色一样自然地厌恶恶、喜欢善。"正心"是不让愤怒、恐惧、好乐、忧患等情绪干扰判断。诚意正心是修身的核心——认知清楚了，道德纯粹了，才能管好自己
- **"修身齐家"的逻辑**: "其家不可教而能教人者，无之。"连自己的家人都管不好，怎么可能治理好国家？"齐家"不是"管住家人"，是用自己的德行感化家人——"身修而后家齐"。家和国的逻辑是一样的：孝对应忠，悌对应顺，慈对应仁
- **"治国平天下"的政治伦理**: 《大学》的治国之道是"絜矩之道"——"所恶于上，毋以使下；所恶于下，毋以事上。"自己不喜欢的对待方式，不要用来对待别人。这就是儒家的"黄金法则"（己所不欲，勿施于人）在政治中的应用
- **《大学》的成书**: 传统认为是曾子所作，但现代学者多认为是战国至秦汉之际儒家学者的作品。朱熹把《大学》从《礼记》中抽出，与《中庸》《论语》《孟子》合为"四书"，并作《大学章句》。从此《大学》成为科举考试的第一必读书
- **"修身为本"的当代意义**: 《大学》的核心逻辑是"自天子以至于庶人，壹是皆以修身为本"。不管你是谁，修身是根本。这个逻辑在现代依然成立——领导力始于自我管理，社会治理始于公民素质。"修身"不是"独善其身"，是"兼济天下"的前提
