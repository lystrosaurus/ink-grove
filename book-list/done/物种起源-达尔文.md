# 物种起源

状态：已读

作者：查尔斯·达尔文

一句话评价：查尔斯·达尔文（Charles Darwin）的"进化论奠基之作"——"'物竞天择，适者生存'"——"改变了'人类对自身与自然的'认知"。"自然选择理论'解释了'物种是如何'演化'的"——"'所有生物'都有'共同的祖先'"。达尔文"用'大量的证据'"——"地质记录、生物地理分布、人工选择、形态学比较"——"构建了一个'完整的'进化论框架"——"这是'生物学史上'最具影响力的著作"。

核心收获：
- **自然选择**: 达尔文的核心理论——"自然选择"（natural selection）。"在一个物种中"——"'个体之间存在差异'"——"有些差异'让个体更适应'环境"——"这些个体'更可能'存活和繁殖"——"它们的'有利特征'就会'传给后代'"。"经过'很多代'"——"这些'有利特征'就会'在种群中'扩散"——"最终'导致'物种的'演化'"。达尔文说"自然选择'不需要'设计者"——"它'是一个'自动的过程'"
- **适者生存**: "适者生存"（survival of the fittest）——"不是'最强壮的'生存'"——"是'最适应环境的'生存'"。"在'一个岛上'"——"喙大的鸟'适合吃'硬种子'"——"喙小的鸟'适合吃'软果实'"——"当'干旱来临'"——"硬种子'变多了'"——"喙大的鸟'就'更容易活下来'"。"适者生存'不是'道德判断'"——"是'自然规律'"
- **共同祖先**: 达尔文提出了"共同祖先"（common descent）理论——"所有生物'都来自'一个'共同的祖先'"。"从'细菌'到'人类'"——"从'鱼'到'鸟'"——"从'树'到'花'"——"我们'都是'亲戚'"。达尔文用"形态学比较"（不同物种的'相似结构'）、"胚胎学"（不同物种的'胚胎早期'非常相似）、"化石记录"（过渡物种的发现）来'证明'共同祖先理论
- **地质记录**: 达尔文"用'地质记录'来'证明'进化论"。"化石'显示'"——"'越古老的地层'"——"'生物越简单'"——"'越新的地层'"——"'生物越复杂'"。"达尔文也'承认'地质记录'不完整'"——"因为'化石的形成'需要'特殊条件'"——"很多'过渡物种'的化石'还没有'被发现'"——"但他'预测'未来'会发现'更多'过渡化石'"——"后来的'考古发现'证实了'他的预测'"
- **生物地理分布**: 达尔文"在'加拉帕戈斯群岛'的观察"是"自然选择理论的'关键证据'"。"他发现'不同岛屿上'的'雀鸟'喙形不同'"——"因为'每个岛屿上的'食物不同'"——"雀鸟的喙'逐渐适应'了'各自岛屿的'食物"。"达尔文还发现'"——"'岛屿上的'物种'和'附近大陆上的'物种'很相似'"——"但'不完全相同'"——"这说明'它们来自'共同的祖先'"——"但'在岛上'独立演化了"
- **进化证据**: 达尔文"列举了'大量的'进化证据"：1）**人工选择**——"人类通过'选择性育种'"——"把'野生植物'变成了'农作物'"——"把'野生动物'变成了'家畜'"——"这证明'选择'可以'改变物种'"；2）**退化器官**——"鲸鱼'有退化的'后肢骨'"——"人类'有退化的'尾骨'"——"这些器官'没有用'"——"但它们证明'这些物种'的祖先'曾经'有'完整的'这些器官"；3）**胚胎发育**——"不同物种的胚胎'在早期'非常相似"——"这证明'它们有'共同的祖先"
- **达尔文的背景**: 查尔斯·达尔文（Charles Darwin, 1809-1882）——英国博物学家——"进化论的奠基人"。他"1831 年'乘坐'小猎犬号（HMS Beagle）'"——"进行了'为期五年'的'环球考察'"——"在'加拉帕戈斯群岛'等地'收集了大量的'标本和观察数据"。他"1859 年'出版了'《物种起源》'"——"引发了'科学界和宗教界'的'巨大震动'"。"直到今天"——"进化论'仍然是'生物学'最核心的'理论"
