# 纯粹理性批判

状态：已读

作者：康德

一句话评价：西方哲学史上最重要的著作之一，康德对"人类能知道什么"的系统追问，开创了批判哲学传统，调和了经验主义和理性主义。

核心收获：
- 理解了"先天综合判断"：有些知识既是必然的（先天的）又是关于世界的（综合的），如数学和物理学
- "物自体"与"现象"的区别：我们只能认识事物呈现给我们的样子（现象），不能认识事物本身（物自体）
- 人类认知的局限：时间、空间、因果律等不是客观世界的属性，而是我们感知世界的"滤镜"
