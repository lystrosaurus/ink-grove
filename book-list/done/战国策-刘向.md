# 战国策

状态：已读

作者：刘向（编）

一句话评价：战国时期纵横家言论和策略的汇编——由西汉刘向整理编辑——记录了苏秦、张仪、范雎、邹忌等策士的"游说辞令和政治谋略"——是了解"战国时代外交智慧和语言艺术"的珍贵文献。"合纵连横"——弱国联合抗强 vs 强国拉拢弱国各个击破——这些"国际博弈"的策略在两千多年后的今天"仍然适用"。策士们的游说技巧堪称"古代说服力大师课"——善用比喻、寓言、数据来打动君主。

核心收获：
- **合纵连横的大格局**: 战国策的核心背景是"七国争霸"——秦、楚、齐、燕、赵、魏、韩。"合纵"是"弱国联合抗强"——苏秦说服六国"联合起来对抗秦国"——"六国合纵"让秦国"十五年不敢出函谷关"。"连横"是"强国拉拢弱国各个击破"——张仪代表秦国"逐个瓦解六国联盟"——"连横"最终让秦国"统一天下"。这两种战略是"国际关系"中最基本的"博弈模式"——直到今天仍然在"全球政治"中上演
- **苏秦的合纵**: 苏秦是"合纵"的代表人物——他出身"寒微"——最初"游说秦国失败"——回家后"妻不以为夫、嫂不以为叔、父母不以为子"——他"发愤读书"——"锥刺股"（用锥子刺大腿来保持清醒）。然后他"游说六国"——用"精准的局势分析"和"生动的比喻"说服了每一国的君主——"六国合纵"——苏秦"身佩六国相印"——成为"当时最有权势的人"。苏秦的成功秘诀是"了解每个君主'最怕什么'和'最想要什么'"——然后"对症下药"
- **张仪的连横**: 张仪是"连横"的代表人物——他是苏秦的"同学"（都师从鬼谷子）——但选择了"完全相反"的策略。张仪代表秦国——他的目标是"瓦解六国联盟"——让六国"各自为战"。他的方法是"逐个击破"——对 A 国说"B 国要攻打你"——对 B 国说"C 国不可信"——让六国"互不信任"。张仪最著名的故事是"楚怀王被骗"——张仪许诺"割让六百里土地"给楚国——结果只给了"六里"——楚怀王"气得发兵攻秦"——结果"大败"
- **游说技巧**: 战国策最精彩的部分是"策士们的游说辞令"——他们的"说服技巧"堪称"古代大师课"：1）**善用比喻**——邹忌讽齐王纳谏——用"我和城北徐公谁美"来比喻"大王被蒙蔽"；2）**善用寓言**——"画蛇添足"、"狐假虎威"、"南辕北辙"——这些成语都出自战国策；3）**善用数据**——策士们会精确地告诉君主"我们有多少兵力、多少粮食、多少财富"——让君主"有'安全感'"；4）**善用恐惧**——"如果不这样做——后果是什么"——让君主"感到'紧迫'"
- **范雎的远交近攻**: 范雎是秦国的重要谋士——他提出了"远交近攻"的战略——"和远处的国家交朋友——攻打近处的国家"。这个策略的精髓是"不要同时树敌太多"——"集中力量打最近的敌人"——"同时和远处的国家保持友好"——避免"两面作战"。"远交近攻"是"秦国统一天下"的关键战略之一——也是中国外交思想的"经典原则"
- **触龙说赵太后**: 这是战国策中"最著名的篇章之一"——触龙是一位"老臣"——赵太后"不肯让她的儿子去做人质"——所有大臣"直接劝"都失败了。触龙的方法是"迂回"——他先和赵太后"聊家常"——"身体好不好？饭量如何？"——然后聊到"父母对孩子的爱"——"真正爱孩子——是为他'长远打算'"——"让长安君去齐国做人质"——"是为他'建立功勋'"——"以后才能在赵国立足"。赵太后被"说服了"。触龙的成功在于"不直接说'你应该怎么做'"——是"让赵太后自己'得出结论'"
- **战国时代的国际关系**: 战国策记录的"国际关系"与"当今世界"有"惊人的相似之处"：1）**联盟与背叛**——六国"合纵"——但"很快就因为'内部分歧'而瓦解"；2）**均势与博弈**——每个国家都在"计算'力量对比'"——寻找"对自己最有利的'联盟'"；3）**外交欺诈**——张仪"骗楚怀王"——"外交中的'欺骗'是常态"；4）**信息战**——策士们"散布假情报"——"制造'假盟友'"。这些"博弈逻辑"在两千多年后"没有任何变化"
- **刘向的背景**: 刘向（公元前 77-前 6 年）——西汉经学家、目录学家——汉高祖刘邦之弟楚元王刘交的四世孙。战国时期的策士言论"散见于各种文献"——刘向将它们"收集整理"——编成了《战国策》33 篇。刘向还编了《说苑》《新序》《列女传》等。《战国策》不是"历史书"——是"策略汇编"——它的价值不在于"记录了'发生了什么'"——在于"记录了'人们如何说服别人'"
