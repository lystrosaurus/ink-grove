# 战国策

状态：已读

作者：刘向

一句话评价：战国时期纵横家言论和策略的汇编，记录了苏秦、张仪、范雎等策士的游说辞令和政治谋略，是了解战国时代外交智慧和语言艺术的珍贵文献。

核心收获：
- 理解了战国时期"合纵连横"的外交大格局——弱国联合抗强（合纵）vs 强国拉拢弱国各个击破（连横）
- 策士们的游说技巧堪称古代"说服力大师课"——善用比喻、寓言、数据来打动君主
- 战国时代的国际关系与当今世界有惊人的相似之处——联盟、背叛、均势、博弈
