# 镜花缘

状态：已读

作者：李汝珍

一句话评价：李汝珍的"清代奇书'"——"'以唐敖出海游历为主线'"。"'经过君子国、女儿国、无肠国等奇异国度'"。"'讽刺社会现实的同时展现了对女性才华的肯定'"。"'镜花水月暗示了书中世界的虚幻性'"——"'每个国家都对应一种社会现象的夸张化'"

核心收获：
- **游历的结构**: 《镜花缘》"采用游历的结构'"。"'什么'结构'"：1）**唐敖出海**——"'主角'出海'"；2）**经过各国**——"'君子国'——'女儿国'——'无肠国'等'"；3）**返回**——"'最后'返回'"。"'这种'结构'"："'类似'《格列佛游记》'"——"'通过'游历'"——"'讽刺'现实'"。"'每个'国家'"："'代表'一种'社会'现象'"
- **女性主义视角**: 本书"有女性主义视角'"。"'什么'视角'"："'肯定'女性'才华'"。"'书中'有'一百个'才女'"："'各'有'所长'"——"'文学'——'艺术'——'科学'"。"'女儿国'"："'女性'当'权'"——"'男性'被'压迫'"——"'讽刺'男权'社会'"。"'这种'视角'"："'在'清代'小说'中'少见'"
- **各国讽刺**: 本书"通过各国讽刺社会现象'"。"'君子国'"："'讽刺'虚伪'"——"'人人'都'是'君子'"——"'但'实际'不'是'"。"'女儿国'"："'讽刺'男权'"——"'男女'地位'颠倒'"——"'暴露'男权'的'问题'"。"'无肠国'"："'讽刺'贪婪'"——"'人'没有'肠子'"——"'吃'了'就'拉'"——"'比喻'贪官'"。"'每个'国家'"："'对应'一种'社会'问题'"
- **镜花水月的隐喻**: 本书"标题有隐喻'"。"'什么'隐喻'"："'镜中'花'"——"'水中'月'"——"'虚幻'"。"'暗示'"："'书中'世界'是'虚幻'的'"——"'是'现实'的'倒影'"。"'作者'的'意图'"："'不'是'写'奇幻'"——"'是'讽刺'现实'"。"'这种'手法'"："'让'读者'思考'"
- **才女的形象**: 本书"刻画了才女的形象'"。"'一百个'才女'"："'各'有'特长'"。"'比如'"：1）**文学**——"'会'写诗'"；2）**艺术**——"'会'画画'"；3）**科学**——"'懂'算学'"。"'这些'才女'"："'证明'女性'有'才华'"——"'不'比'男性'差'"。"'作者'的'观点'"："'男女'平等'"——"'女性'应该'有'机会'"
- **清代小说的特点**: 本书"是清代小说的代表'"。"'清代'小说'的'特点'"：1）**讽刺**——"'讽刺'社会'"；2）**奇幻**——"'有'奇幻'元素'"；3）**才学**——"'展示'作者'才学'"。"'《镜花缘》'的'特点'"："'融合'了'这些'特点'"。"'作者'李汝珍'"："'博学'"——"'书中'展示'了'很多'知识'"
- **作者的背景**: 李汝珍（约 1763-1830）——清代小说家——"《镜花缘》的作者'"。"'他'博学'多才'"——"'精通'音韵'——'算学'——'医学'"。"'《镜花缘》'成书'于'19 世纪'初'"——"'嘉庆'年间'"。"'这本书'的'特点'"："'讽刺'"——"'奇幻'"——"'才学'"。"'李汝珍'的'其他'作品'"："'不'多'"——"'《镜花缘》'是'代表作'"
