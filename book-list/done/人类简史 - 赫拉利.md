# 人类简史

## 基本信息
- **作者**：尤瓦尔·赫拉利
- **分类**：历史/人类学

## 核心总结

赫拉利以宏大叙事讲述人类历史，提出三大革命：认知革命（语言与想象）、农业革命（定居与阶级）、科学革命（承认无知）。核心论点是：人类统治地球靠的是虚构故事的能力——国家、货币、宗教、公司都是想象的概念，但能大规模协作。

## 关键洞见

1. **认知革命**：虚构故事使人能大规模协作
2. **农业骗局**：小麦驯化了人类，而非相反
3. **统一趋势**：金钱、帝国、宗教趋向统一
4. **科学革命**：承认无知是进步起点
5. **未来挑战**：基因工程、AI、不平等
