# 随园诗话

状态：已读

作者：袁枚

一句话评价：袁枚的"清代诗话佳作'"——"'以"性灵说"为核心'"。"'主张诗贵真情、反对拟古'"——"'品评历代诗人见解独到'"。"'文字洒脱有趣'"——"'诗话这种文体本身就是中国文论的独特形式'"

核心收获：
- **性灵说的内涵**: 本书核心是"'性灵说'"。"'什么'是'性灵'"：1）**真性情**——"'诗'应'抒发'真'情感'"；2）**自然**——"'不'做作'"——"'自然'流露'"；3）**个性**——"'有'自己'的'风格'"。"'袁枚'反对'什么'"：1）**拟古**——"'模仿'古人'"；2）**格律**——"'拘泥'格律'"；3）**说教**——"'诗'不'是'道德'说教'"。"'性灵说'的'意义'"："'解放'诗歌'"——"'让'诗歌'回归'真情'"
- **品评历代诗人**: 本书"品评了历代诗人'"。"'袁枚'的'品评'"："'见解'独到'"——"'不'盲从'主流'"。"'比如'"：1）**杜甫**——"'肯定'其'成就'"——"'但'也'指出'缺点'"；2）**李白**——"'赞赏'其'才华'"；3）**苏轼**——"'评价'其'诗'"。"'袁枚'的'标准'"："'有'无'性灵'"——"'不'是'名气'"
- **诗话的文体**: 本书"是诗话文体的代表'"。"'什么'是'诗话'"："'中国'文论'的'独特'形式'"。"'特点'"：1）**随笔式**——"'不'系统'"——"'片段'式'"；2）**引诗**——"'引'诗句'"——"'点评'"；3）**议论**——"'发表'观点'"。"'诗话'的'好处'"："'灵活'"——"'有趣'"。"'《随园诗话》'的'地位'"："'清代'诗话'的'代表作'"
- **袁枚的诗论**: 本书"阐述了袁枚的诗论'"。"'袁枚'的'观点'"：1）**诗主情**——"'情感'第一'"；2）**反对模仿**——"'不'要'模仿'古人'"；3）**重视创新**——"'要有'自己'的'风格'"；4）**男女平等**——"'女性'也'能'写诗'"。"'这些'观点'"："'在'当时'是'进步'的'"
- **女性诗人**: 本书"特别关注女性诗人'"。"'袁枚'收'女'弟子'"："'很多'女性'诗人'"。"'他'品评'女性'诗'"："'肯定'女性'的'才华'"。"'例子'"："'他'的'妹妹'——'他'的'学生'"。"'这种'态度'"："'在'清代'是'少见'的'"
- **文字风格**: 本书"的文字风格独特'"。"'袁枚'的'文字'"："'洒脱'"——"'有趣'"——"'不'拘谨'"。"'他'的'点评'"："'幽默'"——"'犀利'"。"'读'《随园诗话》'"："'像'听'一位'智者'聊天'"。"'这种'风格'"："'让'诗话'不'枯燥'"
- **作者的背景**: 袁枚（1716-1797）——清代诗人、文学评论家——"《随园诗话》的作者'"。"'他'24 岁'中'进士'"——"'后来'辞官'"——"'隐居'随园'"。"'随园'在'南京'"："'他'的'园林'"。"'袁枚'的'诗'"："'性灵'派'代表'"。"'《随园诗话》'24 卷'"："'品评'历代'诗人'"。"'他'还'收'女'弟子'"："'提倡'女性'写诗'"
