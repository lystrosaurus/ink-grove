# 昭明文选

状态：已读

作者：萧统（编）

一句话评价：中国"现存最早的诗文总集"——南朝梁太子萧统（昭明太子）主持编纂——选录"先秦至梁的诗文辞赋"——是"研究中国古代文学的重要文献"。《文选》的选文标准是"事出于沉思——义归乎翰藻"——文章"既要有思想深度"——又要有"文学美感"。《文选》"确立了后世'文'的正统"——"唐诗宋词的经典范式'很大程度上'受《文选》影响"。"文选学"是"唐代最重要的学问之一"——杜甫教导儿子"熟精文选理"。

核心收获：
- **选文标准**: 萧统的选文标准是"事出于沉思——义归乎翰藻"——意思是"文章的内容要'经过深思熟虑'"——"文章的意义要'归于优美的文辞'"。这个标准"兼顾了'思想'和'文采'"——不是"只看'辞藻华丽'"——也不是"只看'思想深刻'"——是"'两者兼备'"。这个标准"影响了后世千年的文学评价"
- **内容范围**: 《文选》选录了"先秦至梁"的"700 多篇作品"——分为"37 个类别"：赋、诗、骚、七、诏、册、令、教、文、表、上书、启、弹事、笺、奏记、书、移、檄、对问、设论、辞、序、颂、赞、符命、史论、史述赞、论、连珠、箴、铭、诔、哀、碑文、墓志、行状、吊文、祭文。"赋"是"最重要的类别"——占了"约三分之一"——包括"司马相如的《子虚赋》""班固的《两都赋》""左思的《三都赋》"
- **文学正统的确立**: 《文选》最重要的意义是"确立了'文'的正统"。在萧统之前——"文学没有'明确的'评价标准'"——萧统通过"选录"和"排除"——告诉后人"'什么是好文章'"。他"不选经书"（经书是'圣人'的——不需要'选'"）——"不选子书"（子书是'思想'的——不是'文学'"）——"不选史书"（史书是'记录'的——不是'创作'"）——只选"'纯文学'的作品"。这种"'文'与'非文'的区分"是"中国文学批评史上的'重大突破'"
- **对后世的影响**: 《文选》对"后世文学的影响"是"深远的"。唐代的"诗人和散文家"几乎"人手一册《文选》"——"杜甫教导儿子'熟精文选理'"——"李白说'清水出芙蓉——天然去雕饰'"——这种"审美取向"就来自《文选》。宋代的"古文运动"虽然"反对'骈文'"——但"韩愈、柳宗元"的"文学功底"仍然"建立在《文选》之上"。"文选学"在唐代成为了"一门专门的学问"——研究《文选》的人"被称为'选学'"
- **萧统的背景**: 萧统（501-531）——南朝梁武帝萧衍的"长子"——被封为"昭明太子"。他"自幼聪慧"——"博览群书"——"爱好文学"——"广招文学之士"——在东宫"编纂了《文选》"。可惜他"英年早逝"——年仅 30 岁——"没有'继承皇位'"——但他的"文学贡献"远远超过了"政治贡献"。"昭明"是他的"谥号"——"《昭明文选》"因此得名
- **骈文的典范**: 《文选》收录的大量"骈文"——"对仗工整、辞藻华丽、音韵和谐"——是"中国文学'形式主义'的巅峰"。骈文的特点是"四六句式"——"上下句对仗"——"声律和谐"——"大量用典"。虽然"后来的古文运动'反对'骈文的'过度形式化'"——但"骈文的'美学价值'"是"不可否认的"——《文选》中的骈文代表作"至今仍是'文学欣赏的'经典"
- **《文选》的版本**: 《文选》在"流传过程中"出现了"多个版本"——最著名的是"李善注本"——唐代李善为《文选》做了"详尽的注释"——"引用了大量'已经失传的'文献"——使得"李善注本"成为了"研究'先秦至南北朝文学'的'重要工具'"。后来又有"五臣注本"——"注释'更简洁'"——但"学术价值'不如'李善注本"
