# 远见

状态：已读

作者：布赖恩费瑟斯通豪

一句话评价：职业规划领域的经典之作，提出用 45 年职业生涯的长远视角来规划当下选择，将职业分为三个阶段，每个阶段有不同的目标和策略。

核心收获：
- 理解了职业生涯的三阶段模型：第一阶段（加添燃料）、第二阶段（瞄准甜蜜点）、第三阶段（优化长板）
- 职业决策不应该只看下一份工作，而应该看整个 45 年的职业生涯
- "职场燃料"概念：可迁移技能、有意义的经验、持久的关系——这三样东西比薪水更重要
