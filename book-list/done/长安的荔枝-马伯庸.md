# 长安的荔枝

状态：已读

作者：马伯庸

一句话评价：马伯庸以"一骑红尘妃子笑"为引子，写了一个唐朝小吏如何在极限条件下完成"从岭南运鲜荔枝到长安"的不可能任务，既是精彩的历史小说，也是一堂生动的项目管理课。

核心收获：
- 理解了"小人物在大系统中的困境"——李善德接到的是一个注定要掉脑袋的任务
- 小说用现代项目管理的视角展开：路线规划、保鲜实验、资金筹措、跨部门协调，读起来像在看古代版"生死时速"
- "就算失败，我也想知道，自己倒在距离终点多远的地方"——这句话是全书最燃的瞬间
