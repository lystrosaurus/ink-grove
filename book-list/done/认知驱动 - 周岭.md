# 认知驱动

状态：已读

作者：周岭

一句话评价：周岭的"《认知觉醒》续作'"——"'从"知道"走向"做到"'"。"'聚焦如何将认知转化为行动力'"——"'案例贴近中国读者生活'"。"'认知升级不是目的，行动和改变才是'"——"'真正驱动行为的不是道理，而是情感体验和即时反馈'"

核心收获：
- **知道与做到的鸿沟**: 本书的核心是"'知道'与'做到'的鸿沟'"。"'为什么'我们'知道'很多'道理'"——"'但'依然'过'不好'一生'"。"'因为'知道'和'做到'之间'有'巨大'的'鸿沟'"。"'如何'跨越'鸿沟'"：1）**情感体验**——"'道理'不'能'驱动'行为'"——"'情感'能'"——"'你'要'让'自己'体验'到'改变'的'好处'"；2）**即时反馈**——"'人'喜欢'即时'反馈'"——"'但'改变'往往'是'长期'的'"——"'要'设计'即时'反馈'"；3）**小步快跑**——"'不'要'想'一步'到位'"——"'要'小步'前进'"——"'每'一步'都'有'成就感'"
- **价值输出**: 本书提出了"'价值输出'"的概念'"。"'价值'输出'是'什么'"——"'通过'帮助'他人'来'倒逼'自己'成长'"。"'比如'你'想'学习'"——"'你'可以'写'文章'分享'"——"'你'可以'教'别人'"。"'为了'输出'"——"'你'必须'输入'"——"'你'必须'理解'"。"'价值'输出'的'好处'"：1）**倒逼学习**——"'为了'输出'"——"'你'会'更'努力'学习'"；2）**获得反馈**——"'别人'的'反馈'"——"'帮助'你'改进'"；3）**建立影响力**——"'输出'价值'"——"'别人'认可'你'"——"'你'有'影响力'"。"'所以'成长'的'核心'"——"'不'是'输入'"——"'是'输出'"
- **情感驱动**: 本书"强调了情感的作用'"。"'人'不'是'理性'的'动物'"——"'人'是'情感'的'动物'"。"'驱动'行为'的'"——"'不'是'道理'"——"'是'情感'"。"'如何'利用'情感'"：1）**创造积极体验**——"'让'改变'变得'愉快'"——"'不'是'痛苦'"；2）**关联意义**——"'把'行为'与'意义'关联'"——"'这'不'是'跑步'"——"'这'是'健康'"；3）**社交激励**——"'加入'社群'"——"'别人'的'鼓励'"——"'让'你'坚持'"。"'所以'想'改变'"——"'不'要'讲'道理'"——"'要'创造'情感'体验'"
- **反馈系统**: 本书"讨论了反馈系统的重要性'"。"'人'需要'反馈'"——"'没有'反馈'"——"'人'不'知道'自己'做得'对不对'"——"'人'会'放弃'"。"'如何'设计'反馈'"：1）**可视化**——"'让'进步'可见'"——"'比如'打卡'——'记录'"；2）**即时性**——"'反馈'要'及时'"——"'不'要'等'很久'"；3）**正向**——"'多'关注'进步'"——"'不'要'只'关注'不足'"。"'游戏'为什么'让人'上瘾'"——"'因为'游戏'有'即时'反馈'"。"'把'学习'设计'成'游戏'"——"'让'学习'上瘾'"
- **身份认同**: 本书"探讨了身份认同的作用'"。"'身份'是'什么'"——"'你'认为'自己'是'什么'人'"。"'身份'影响'行为'"——"'你'认为'自己'是'读书'的'人'"——"'你'就'会'读书'"。"'如何'改变'身份'"：1）**假装**——"'先'假装'"——"'慢慢'就'成'了'真的'"；2）**环境**——"'改变'环境'"——"'让'环境'塑造'你'的'身份'"；3）**标签**——"'给'自己'贴'标签'"——"'我是'运动'的'人'"。"'身份'改变'了'"——"'行为'就'自动'改变'"——"'不'需要'意志力'"
- **长期主义**: 本书"倡导长期主义'"。"'很多'人'失败'"——"'因为'他们'不'能'坚持'"。"'为什么'不'能'坚持'"——"'因为'他们'想'快速'成功'"。"'但'真正'的'成长'"——"'是'长期'的'"——"'是'积累'的'"。"'如何'坚持'"：1）**降低期望**——"'不'要'想'一'夜'暴富'"——"'接受'慢'成长'"；2）**享受过程**——"'不'是'为了'结果'"——"'是'享受'过程'"；3）**建立习惯**——"'习惯'不'需要'意志力'"——"'习惯'是'自动'的'"。"'长期主义'的'核心'"——"'不'是'坚持'"——"'是'热爱'"
- **周岭的背景**: 周岭——中国个人成长作家、知识博主——"《认知驱动》的作者'"。"他'的'《认知觉醒》'2020 年'出版'"——"'畅销'百万册'"。"《认知驱动》'2021 年'出版'"——"'是'《认知觉醒》'的'续作'"。"'《认知觉醒》'讲'认知'"——"'《认知驱动》'讲'行动'"。"他'的'写作'风格'"——"'通俗'——'实用'"——"'大量'案例'"——"'贴近'中国'读者'"。"他'还'运营'公众号'"——"'分享'个人'成长'方法'"——"'有'大量'粉丝'"
