# 财务自由之路

状态：已读

作者：博多·舍费尔

一句话评价：博多·舍费尔（Bodo Schäfer）的"个人理财入门经典'"——"'语言通俗、方法实操性强'"。"'先支付自己'和'让钱为你工作'"——"'简单但有效'"。"'适合财务小白建立基本认知'"——"'财务自由的四个阶段'"

核心收获：
- **先支付自己**: 本书核心是"'先支付自己'"的原则'"。"'什么'意思'"——"'每月'收入'的'固定'比例'"——"'先'存'起来'投资'"——"'剩下'的'才'用于'消费'"。"'很多'人'的'习惯'"——"'先'消费'"——"'剩下'的'才'存'"——"'结果'存'不'下'钱'"。"'先'支付'自己'"——"'强制'储蓄'"——"'让'你'养成'习惯'"。"'比例'是'多少'"——"'至少'10%'"——"'最好'20%'"。"'这'笔钱'"——"'不'能'动'"——"'只'能'用于'投资'"
- **资产与负债**: 本书"区分了资产和负债'"。"'资产'是'什么'"——"'把'钱'放进'你'的'口袋'的'东西'"。"'比如'股票'"——"'债券'"——"'出租'的'房子'"。"'负债'是'什么'"——"'把'钱'从'你'的'口袋'拿走'的'东西'"。"'比如'自住房'"——"'车'"——"'信用卡'债务'"。"'富人'的'策略'"——"'买'资产'"——"'让'资产'产生'收入'"。"'穷人'的'策略'"——"'买'负债'"——"'以为'是'资产'"。"'财务'自由'的'关键'"——"'增加'资产'"——"'减少'负债'"
- **财务自由四阶段**: 本书"提出了财务自由的四个阶段'"。"'第一阶段：债务清零**——"'还清'所有'消费'债务'"——"'信用卡'——'车贷'"。"'第二阶段：建立安全垫**——"'存'6 个月'的'生活费'"——"'应急'用'"。"'第三阶段：持续投资**——"'把'钱'投入'资产'"——"'股票'——'基金'——'房产'"。"'第四阶段：被动收入覆盖支出**——"'投资'收入'"——"'租金'收入'"——"'够'生活'开支'"。"'到'了'第四'阶段'"——"'你'就'财务'自由'了'"
- **让钱为你工作**: 本书"强调了让钱为你工作'"。"'什么'意思'"——"'不'要'只'靠'工资'"——"'要'让'钱'生钱'"。"'如何'做到'"：1）**储蓄**——"'先'存'钱'"；2）**投资**——"'让'钱'投入'资产'"；3）**复利**——"'让'时间'发挥'作用'"。"'富人'不'工作'"——"'他们'的'钱'在'工作'"。"'穷人'工作'"——"'他们'的'钱'不'工作'"。"'财务'自由'的'本质'"——"'让'钱'为你'工作'"
- **投资者类型**: 本书"区分了四种投资者类型'"。"'1）不投资**——"'钱'放'银行'"——"'被'通胀'吃掉'"；"'2）借款人**——"'借钱'消费'"——"'付'利息'"；"'3）储蓄者**——"'存'钱'"——"'但'不'投资'"；"'4）投资者**——"'让'钱'生钱'"。"'你'要'成为'投资者'"。"'如何'成为'投资者'"：1）**学习**——"'学习'投资'知识'"；2）**行动**——"'开始'投资'"；3）**坚持**——"'长期'投资'"
- **梦想的力量**: 本书"强调了梦想的作用'"。"'为什么'要'财务'自由'"——"'为了'实现'梦想'"。"'你'要'有'清晰'的'梦想'"——"'买'什么'房子'"——"'去'哪里'旅行'"——"'过'什么'生活'"。"'梦想'给你'动力'"——"'让'你'坚持'储蓄'"——"'让'你'抵制'消费'诱惑'"。"'舍费尔'建议'"——"'写'下'你'的'梦想'"——"'每天'看'"——"'想象'实现'后'的'感觉'"
- **舍费尔的背景**: 博多·舍费尔（Bodo Schäfer, 1960-）——德国理财作家、演讲家——"《财务自由之路》的作者'"。"他'26 岁'时'破产'"——"'负债'累累'"。"'他'遇到'一位'导师'"——"'导师'教'他'理财'"——"'他'用'4 年'时间'还清'债务'"——"'实现'财务'自由'"。"'后来'他'写'了'这本书'"——"'分享'他'的'经验'"。"'这本书'畅销'欧洲'"——"'被'翻译成'多'种'语言'"
