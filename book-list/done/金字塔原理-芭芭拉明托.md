# 金字塔原理

状态：已读

作者：芭芭拉明托

一句话评价：麦肯锡经典沟通方法论，提出"结论先行、以上统下、归类分组、逻辑递进"的金字塔结构，是职场写作和表达的必备指南。

核心收获：
- 理解了金字塔原理的四大核心：结论先行、以上统下、归类分组、逻辑递进
- "MECE 法则"（Mutually Exclusive, Collectively Exhaustive）——不重叠、不遗漏地拆解问题
- 写作的本质不是"写"而是"想"——先理清思路，再组织语言
