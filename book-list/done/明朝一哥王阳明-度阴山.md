# 明朝一哥王阳明

状态：已读

作者：度阴山

一句话评价：王阳明传记的通俗读物，以幽默风趣的笔调讲述了王阳明从叛逆少年到一代宗师的传奇人生，"知行合一"和"致良知"的思想讲解清晰，适合入门。

核心收获：
- 理解了王阳明"心学"的核心：心即理、知行合一、致良知
- 王阳明的"龙场悟道"是中国思想史上最重要的顿悟时刻之一——在被贬贵州的绝境中悟出"心外无物"
- "知行合一"不是"知道了就要做到"，而是"知和行本来就是一件事"——真正的知必然包含行
