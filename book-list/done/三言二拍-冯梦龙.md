# 三言二拍

状态：已读

作者：冯梦龙-凌濛初

一句话评价：明代市民文学的巅峰——120篇短篇小说，写的是商人、妓女、书生、官吏的市井生活，但每一篇都在讲人性的复杂。

核心收获：
- "三言"是冯梦龙编的（喻世明言、警世通言、醒世恒言），"二拍"是凌濛初写的（初刻拍案惊奇、二刻拍案惊奇），合称三言二拍
- 商人成为主角是文学史上的重大转折。之前的文学写帝王将相、才子佳人，三言二拍写卖油郎、丝绸商、当铺老板
- 女性形象极其丰富：有贞烈的、有风流的、有精明的、有狠毒的。最精彩的是那些"不守妇道"但让人同情的角色
- "劝善惩恶"是表面主题，但很多故事的道德判断极其模糊——好人没好报、坏人活得滋润，这才是真实的人性
- 叙事技巧极高：每篇开头用一首诗点题，中间悬念迭起，结尾常常出人意料但又合乎情理
