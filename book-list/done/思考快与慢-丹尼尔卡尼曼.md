# 思考快与慢

状态：已读

作者：丹尼尔卡尼曼

一句话评价：诺贝尔经济学奖得主卡尼曼的集大成之作，系统阐述了人类思维的两个系统——快速直觉和缓慢理性，揭示了认知偏差的根源。

核心收获：
- 理解了系统 1（快思考）和系统 2（慢思考）的区别——大多数错误决策源于过度依赖系统 1
- 前景理论：人们对损失的敏感度远高于收益——这是行为经济学的基石
- 锚定效应、可用性偏差、确认偏误——这些认知偏差无处不在，了解它们是避免它们的第一步
