# 模型思维

状态：已读

作者：斯科特佩奇

一句话评价：系统介绍多学科思维模型的工具书，涵盖正态分布、幂律分布、网络模型、博弈论、系统动力学等数十种模型，是查理·芒格"多元思维模型"理念的学术化展开。

核心收获：
- 理解了"所有模型都是错的，但有些是有用的"——模型的价值不在于完美，在于有用
- 不同类型的分布（正态、幂律）决定了不同的思维方式和管理策略
- 多模型思维比单一模型更强大——不同模型从不同角度切割现实，叠加后逼近真相
