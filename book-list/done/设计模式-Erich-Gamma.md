# 设计模式

状态：已读

作者：Erich-Gamma

一句话评价：GoF 四人合著的软件工程经典，总结了 23 种面向对象设计模式，是程序员提升代码质量的必读之作。

核心收获：
- 理解了设计模式的本质：不是发明新东西，而是总结前人解决常见问题的最佳实践
- 三大类模式：创建型（如何创建对象）、结构型（如何组合对象）、行为型（对象之间如何协作）
- 模式的价值在于提供通用语言——当团队都用"工厂模式""观察者模式"交流时，沟通效率大幅提升
