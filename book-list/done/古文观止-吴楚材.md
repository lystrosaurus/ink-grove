# 古文观止

状态：已读

作者：吴楚材

一句话评价：中国古代散文最佳选本——收录从先秦到明代 222 篇散文名作，"观止"意为"读到这些文章就够了"。吴楚材叔侄的选文眼光极高，每篇附有简明评注，是学习古文和理解中国文学传统的必读之书。

核心收获：
- **中国散文的发展脉络**: 通过选文可以清晰看到中国散文的演变轨迹。先秦是"叙事与雄辩"的时代——《左传》的战争叙事（曹刿论战、烛之武退秦师）精准凝练，《战国策》的游说辞令（邹忌讽齐王纳谏、触龙说赵太后）机智雄辩。汉代是"史传与政论"的时代——司马迁的《史记》选段（报任安书）悲壮慷慨，贾谊的《过秦论》气势磅礴。唐宋是"文采与思想"的黄金时代——韩愈的《师说》、柳宗元的《山水游记》、苏轼的《赤壁赋》、欧阳修的《醉翁亭记》，篇篇都是千古名作
- **经典名篇的思想价值**: 《古文观止》中的文章不仅是文学经典，更是思想经典。诸葛亮的《出师表》——"鞠躬尽瘁，死而后已"的忠诚；李密的《陈情表》——"乌鸟私情，愿乞终养"的孝道；范仲淹的《岳阳楼记》——"先天下之忧而忧，后天下之乐而乐"的家国情怀；苏轼的《赤壁赋》——"寄蜉蝣于天地，渺沧海之一粟"的宇宙观。这些文章塑造了中国人的精神世界
- **吴楚材的选文眼光**: 吴楚材和吴调侯（叔侄二人）是清代康熙年间的普通塾师，但他们的选文眼光远超同时代学者。222 篇文章涵盖了叙事、议论、抒情、写景各种文体，兼顾了思想性和文学性。他们不选"难懂的"，选"好的"——让初学者也能欣赏古文的魅力
- **评注的价值**: 每篇文章后附有简明的评注——解释难点、点评妙处、提示背景。这些评注不是"学术论文"，是"老师的口吻"——亲切、简洁、一针见血。比如评《滕王阁序》："通篇以'望'字为骨，以'秋'字为魂。"一句话点出全文的精髓
- **"观止"的含义**: "观止"出自《左传》——吴国公子季札在鲁国观看乐舞，看到《韶箾》时说"观止矣"——意思是"看到这里就够了，后面的不用再看了"。吴楚材用"观止"做书名，意思是"这些文章是最好的，读了这些就够了"。这既是自信，也是对读者的承诺
- **学习古文的方法**: 《古文观止》不仅是"读本"，是"教材"。古人学写文章的方法就是"熟读古文观止"——先背诵，再模仿，再创作。这个"输入→内化→输出"的学习路径在今天依然有效。很多现代作家（如鲁迅、周作人、汪曾祺）的古文功底都来自《古文观止》
- **《古文观止》的版本**: 最初刊刻于 1695 年（康熙三十四年），三百多年来一直是"最畅销的古文选本"。它的生命力在于"选得精、评得准、编得顺"——选文精当、评注精准、编排顺畅（按时代顺序，从先秦到明代）。后来的《古文辞类纂》《经史百家杂钞》等选本都未能取代它的地位
- **当代意义**: 在"碎片化阅读"和"网络用语"盛行的今天，《古文观止》提供了一种"深度阅读"的体验——每一篇文章都值得一字一句地读、一遍又一遍地读。好的古文不是"古董"，是"活的"——它们在千年后依然能打动你，因为人性没有变
