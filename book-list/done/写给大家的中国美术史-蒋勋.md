# 写给大家的中国美术史

状态：已读

作者：蒋勋

一句话评价：蒋勋用最温柔的语言把中国美术史讲成了"美的巡礼"——从彩陶到当代艺术，每一次风格转变背后都是时代精神的折射。

核心收获：
- 中国美术的核心是"意境"而非"写实"——不求画得像，求画出"味道"
- 山水画的黄金时代（五代至北宋）是中国艺术的高峰。范宽、郭熙、李成的山水不只是风景，是宇宙观
- 文人画的兴起：从"画工"到"文人"，画画从手艺变成修养。苏轼的"论画以形似，见与儿童邻"是转折点
- 书法是中国独有的艺术形式——线条的美学在书法中达到了极致
- 蒋勋的叙事方式：不堆砌学术名词，用感性的语言让每一件作品"活"起来
