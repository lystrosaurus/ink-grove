# 中庸

状态：已读

作者：子思

一句话评价：儒家最精微的哲学文献——"中庸"不是和稀泥，是在极端之间找到最恰当的平衡点，是儒家最高的人格理想。

核心收获：
- "中庸"的含义：中是不偏不倚，庸是平常之道。中庸就是在日常生活中做到不偏不倚
- "喜怒哀乐之未发，谓之中；发而皆中节，谓之和"——情绪管理的最高境界不是没有情绪，是情绪表达恰到好处
- "致中和，天地位焉，万物育焉"——个人修养到了"中和"的境界，就能和宇宙的运行规律同频
- "君子中庸，小人反中庸"——君子能做到中庸，小人要么过度要么不足
- 中庸和"和而不同"是相通的：不是所有人都一样，而是在差异中找到和谐
