# 世说新语

状态：已读

作者：刘义庆

一句话评价：魏晋风度的最佳读本——一千多则小故事，装下了一个时代名士的真性情、怪癖、智慧和浪漫。

核心收获：
- 三十六门分类：德行、言语、政事、文学、方正、雅量、识鉴、赏誉、品藻、规箴等，每门都有精彩故事
- 魏晋风度的核心是"真"——真性情、真感受、真表达。不装、不做作、不虚伪
- 名士的"清谈"不是闲聊，是哲学辩论。讨论"有无""名教与自然""言意之辨"，水平极高
- 最打动我的几个故事：王子猷雪夜访戴（"乘兴而行，兴尽而返"）、刘伶醉酒（"我以天地为栋宇"）、阮籍穷途之哭
- 世说新语的文风极其精炼，往往一两句对话就勾勒出一个完整的人物形象
