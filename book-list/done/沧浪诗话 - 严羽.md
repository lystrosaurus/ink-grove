# 沧浪诗话

状态：已读

作者：严羽

一句话评价：严羽的"宋代最重要的诗论著作"——"以禅喻诗"——提出"妙悟说"和"兴趣说"——对"后世诗歌创作和批评'产生了'深远影响"。"妙悟"——"诗歌的'最高境界'"不是"技巧和学问"——是"直觉和感悟"——"类似于禅宗的'顿悟'"。"诗有别材——非关书也——诗有别趣——非关理也"——"诗歌需要的是'才情和感受力'"——"不是'学问和道理'"。

核心收获：
- **妙悟说**: 严羽最核心的观点——"妙悟"。"妙悟"来自"禅宗"——"禅宗说'开悟'不需要'读书'"——"需要'直觉'"。严羽说"诗歌的'最高境界'也是'妙悟'"——"不是'学出来的'"——是"'悟出来的'"。"你读了'一千首诗'"——"不一定会'写诗'"——"但如果你'悟了'"——"你就能'写出好诗'"。"妙悟"不是"神秘的东西"——是"一种'对美的'直觉感知能力"——"你能'感受到'什么是'好的诗'"——"你'自己也能'写出'好的诗'"
- **兴趣说**: 严羽的另一个核心观点——"兴趣"——"诗歌要'有韵味'"——"有'言外之意'"——"有'弦外之音'"。"好的诗'不只是'字面上的意思'"——"它'还'有'更深层的'感受'"——"读完之后"——"你'会'有'余味'"——"就像'好茶'"——"喝完之后"——"'口中还有'香气'"。严羽说"唐诗'有'兴趣'"——"宋诗'没有'兴趣"——"因为宋诗'太注重'说理'"——"少了'韵味'"
- **以禅喻诗**: 严羽"用禅宗的概念来解释诗歌"——"诗禅相通"。"禅宗说'不立文字'——'直指人心'"——严羽说"好诗也'不靠'文字'堆砌'"——"靠'直觉的'表达'"。"禅宗说'顿悟'"——严羽说"写诗也'需要'顿悟'"。"禅宗说'平常心是道'"——严羽说"好诗也'来自'平常的'感受'"。这种"以禅喻诗"的方法"开创了中国诗论的'新传统'"——"后来的'王士禛'（神韵说）'深受'其影响"
- **尊唐抑宋**: 严羽"极力推崇唐诗"——"批评宋诗"。他说"唐诗'有'兴趣'——'有'气象'——'有'韵味'"——"宋诗'只有'道理'"。"唐诗'像'一幅画'"——"宋诗'像'一篇论文'"。严羽的"尊唐抑宋"立场"奠定了后世'几百年的'诗学传统"——"明代'前后七子'"——"清代'王士禛'"——"都'继承了'严羽的'尊唐'立场"。但"这种立场也有'偏颇'"——"宋诗的'说理性'也有'其价值'"
- **诗有别材**: 严羽说"诗有别材——非关书也"——"写诗需要的'才情'"——"不是'读书多'就有的"。"一个人'可以读万卷书'"——"但'不一定能'写出一首'好诗'"——"因为他'缺少'诗人的'感受力'"。"另一个'读书不多'"——"但'天生敏感'"——"他'能感受到'别人感受不到的'美'"——"他就能'写出好诗'"。严羽的这个观点"挑战了'宋代'以'学问'论诗的传统"
- **严羽对唐诗的分类**: 严羽"把唐诗分为'几种风格'"——"高古"（如"陈子昂"）、"飘逸"（如"李白"）、"沉郁"（如"杜甫"）、"悲壮"（如"王昌龄"）、"自然"（如"王维"）、"清奇"（如"孟浩然"）。他认为"每种风格'都有'其价值'"——"但'最高境界'是'妙悟'"——"不管'哪种风格'"——"只要'有妙悟'"——"就是'好诗'"
- **严羽的背景**: 严羽——南宋诗论家——字仪卿——号沧浪逋客——福建邵武人。《沧浪诗话》是他"唯一的著作"——但"影响极大"——"是中国'最重要的'诗论著作之一"。严羽"在宋代'诗坛'的地位"是"独特的"——"他'不是诗人'"——"是'诗论家'"——"他对'诗歌的'理解'超过了'大多数'写诗的人"
