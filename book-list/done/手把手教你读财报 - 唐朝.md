# 手把手教你读财报

状态：已读

作者：唐朝

一句话评价：唐朝面向"普通投资者"的财报分析入门书——用"大白话"讲解"资产负债表、利润表、现金流量表"的阅读方法——结合"A 股实例"——"实操性强"。核心逻辑："资产负债表看家底——利润表看能力——现金流量表看健康"。财务分析不是"算精确的数"——是"判断一家公司的商业模式是否健康"。这本书最大的价值是"让'非财务专业'的投资者也能'看懂财报'"——不再"被专业术语吓到"。

核心收获：
- **三大报表的核心逻辑**: 唐朝把三大报表的核心用最简单的话概括：1）**资产负债表看"家底"**——"公司有多少钱、欠多少钱、值多少钱"；2）**利润表看"能力"**——"公司赚不赚钱、怎么赚钱、赚了多少"；3）**现金流量表看"健康"**——"公司有没有'真金白银'进账"。这三张表"缺一不可"——只看利润表——你可能被"纸面利润"骗了。只看资产负债表——你不知道"公司能不能赚钱"。只看现金流量表——你不知道"这些钱'从哪来'"
- **资产负债表**: 唐朝说资产负债表的核心是"资产 = 负债 + 股东权益"——这个等式"永远成立"。关键看什么：1）**资产质量**——"现金、应收账款、存货、固定资产"——"现金最值钱"——"应收账款和存货多了说明'货卖不出去'"；2）**负债结构**——"短期负债"和"长期负债"——"短期负债太多"说明"公司资金链紧张"；3）**有息负债**——"需要付利息的负债"（银行贷款、债券）——"越少越好"——因为"利息是'固定成本'"——"经济不好时'会要命'"
- **利润表**: 利润表的核心是"收入 - 费用 = 利润"。唐朝教你看几个关键指标：1）**毛利率**——"（收入 - 成本）/ 收入"——毛利率高说明"产品有竞争力"；2）**净利率**——"净利润 / 收入"——净利率高说明"公司'整体赚钱能力强'"；3）**费用率**——"销售费用、管理费用、财务费用"占收入的比例——"费用率太高"说明"公司'花钱效率低'"；4）**非经常性损益**——"卖资产、政府补贴"等"一次性收入"——"要剔除"——因为"不可持续"
- **现金流量表**: 唐朝说现金流量表是"最不容易造假"的报表——因为"现金是'真金白银'"——"不像利润可以'做账'"。关键看：1）**经营活动现金流**——"公司'主营业务'产生的现金"——"必须为正"——而且"应该大于净利润"（如果经营现金流持续小于净利润——说明"利润是'纸面的'"）；2）**投资活动现金流**——"公司'花钱投资'的现金"——"为负是正常的"——因为"公司在'扩张'"；3）**筹资活动现金流**——"公司'借钱或还钱'的现金"——"如果经营现金流为负、筹资现金流为正"——说明"公司在'借钱过日子'"
- **危险信号**: 唐朝列出了财报中"最需要警惕的信号"：1）**应收账款暴增**——"收入增长了但'钱没收回来'"——可能是"放宽信用'冲业绩'"；2）**存货积压**——"存货增长速度超过收入增长"——说明"产品'卖不动了'"；3）**经营现金流持续为负**——"公司在'赔钱运营'"——"迟早会'资金链断裂'"；4）**商誉过高**——"并购花了太多钱"——"一旦并购对象'业绩不达预期'"——"商誉减值会'吃掉利润'"；5）**关联交易过多**——"公司和'自己人'做生意"——"价格'不透明'"——"容易'操纵利润'"
- **商业模式判断**: 唐朝说"财务分析的最终目的"不是"算精确的数"——是"判断一家公司的'商业模式是否健康'"。好的商业模式特征：1）**高毛利率**——"产品有定价权"；2）**低资本支出**——"不需要'持续砸钱'来维持"；3）**现金流好**——"先收钱后交货"或"现款现货"；4）**轻资产**——"不需要大量固定资产"；5）**复购率高**——"客户会'反复购买'"。坏的商业模式：1）**靠'烧钱'维持**；2）**客户"一次性"消费**；3）**应收账款"永远收不回来"**
- **唐朝的背景**: 唐朝——中国投资者、财经博主——在雪球等平台上以"价值投资"和"财报分析"著称。《手把手教你读财报》是他最受欢迎的作品——"用最通俗的语言"讲解"最枯燥的财务知识"——适合"非财务专业的普通投资者"。他的核心理念是"财报是'了解一家公司'的'基本功'"——"不看财报就投资"——等于"盲人摸象"
