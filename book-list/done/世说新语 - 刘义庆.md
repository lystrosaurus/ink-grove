# 世说新语

状态：已读

作者：刘义庆

一句话评价：魏晋风度的最佳读本——一千多则小故事，装下了一个时代名士的真性情、怪癖、智慧和浪漫。中国文学中最有"个性"的一本书，每个人物都像是从你朋友圈里走出来的。

核心收获：
- **三十六门分类**: 德行、言语、政事、文学、方正、雅量、识鉴、赏誉、品藻、规箴、捷悟、夙惠、豪爽、容止、自新、企羡、伤逝、栖逸、贤媛、术解、巧艺、宠礼、任诞、简傲、排调、轻诋、假谲、黜免、俭啬、汰侈、忿狷、谗险、尤悔、纰漏、惑溺、仇隙。三十六个主题，几乎涵盖了人性的所有面向
- **魏晋风度的核心是"真"**: 真性情、真感受、真表达。不装、不做作、不虚伪。在礼教森严的中国古代，魏晋名士的"真"是极其罕见的——他们敢说、敢做、敢哭、敢笑、敢醉、敢疯
- **清谈的哲学深度**: 名士的"清谈"不是闲聊，是哲学辩论。讨论"有无"（王弼的"贵无论"）、"名教与自然"（嵇康的"越名教而任自然"）、"言意之辨"（言能否尽意）。这些辩论的深度不亚于同时期西方的哲学讨论
- **经典故事**: 王子猷雪夜访戴（"乘兴而行，兴尽而返，何必见戴"）——这是中国文学中最浪漫的"说走就走的旅行"。刘伶醉酒（"我以天地为栋宇，屋室为裈衣，诸君何为入我裈中"）——醉酒后的荒诞逻辑反而最真诚。阮籍穷途之哭——走到路的尽头大哭而返，是对人生困境最诗意的表达
- **文风的精炼**: 世说新语的文风极其精炼，往往一两句对话就勾勒出一个完整的人物形象。"小时了了，大未必佳"（孔融）、"管中窥豹，时见一斑"、"望梅止渴"、"东床快婿"——这些成语都来自《世说新语》
- **名士的"怪"**: 魏晋名士的怪癖是中国文化中最有趣的部分。刘伶裸奔、阮籍青白眼（喜欢的人用正眼看，不喜欢的人用白眼看）、嵇康临刑前弹一曲《广陵散》、王羲之东床坦腹——这些"怪"不是疯，是对世俗规范的拒绝
- **刘义庆的编纂**: 刘义庆是南朝宋的宗室，不是魏晋人。他编《世说新语》时，魏晋名士的时代已经过去了一百多年。这本书某种程度上是对一个"回不去的时代"的怀念——那个可以自由表达、可以怪、可以真的时代，在刘义庆的时代已经不存在了
- **对中国文化的影响**: 《世说新语》是后世所有写"名士风流"的源头。苏轼、王安石、李贽、袁枚——这些人身上都有魏晋名士的影子。"魏晋风度"这个词本身就来自《世说新语》
