# 菜根谭

状态：已读

作者：洪应明

一句话评价：洪应明的"明代处世格言集'"——"'融合儒释道三家智慧'"——"'用精炼的语录体讲述修身、处世、治家的道理'"。"'咬得菜根，则百事可做'"——"'能吃苦的人什么都能做'"。"'中国古代最受欢迎的"人生指南"之一'"——"'中庸之道——不过分、不偏激、不急功近利'"

核心收获：
- **菜根的寓意**: 《菜根谭》的"书名就有深意'"。"'菜根'是'什么'"——"'菜根'是'蔬菜'的'根'"——"'味苦'"——"'难以下咽'"。"'但'能'咬'菜根'"——"'说明'能'吃苦'"。"'咬得菜根，则百事可做'"——"'能'吃苦'的'人'"——"'什么'事'都'能'做'"。"'这'是'一种'人生'态度'"——"'不'追求'享受'"——"'能'忍受'艰苦'"。"'洪应明'用'菜根'比喻'人生'"——"'人生'如'菜根'"——"'苦'中'有'味'"——"'需要'慢慢'品味'"
- **儒释道融合**: 《菜根谭》"融合了儒释道三家智慧'"。"'儒家'的'修身'"——"'修身齐家治国平天下'"——"'重视'道德'修养'"。"'道家'的'自然'"——"'顺其自然'"——"'无为'而治'"——"'不'强求'"。"'佛家'的'超脱'"——"'看破'红尘'"——"'放下'执念'"——"'心'无'挂碍'"。"'这三家'在'菜根谭'中'和谐'共存'"——"'不'冲突'"。"'比如'"——"'儒家'说'积极'入世'"——"'道家'说'消极'出世'"——"'菜根谭'说'入世'但'不'执着'"——"'出世'但'不'逃避'"
- **中庸之道**: 《菜根谭》的"核心原则是中庸'"。"'中庸'是'什么'"——"'不'过分'"——"'不'偏激'"——"'不'走'极端'"。"'比如'"：1）**名利**——"'不'追求'名利'"——"'也'不'刻意'回避'"；2）**得失**——"'得'不'喜'"——"'失'不'忧'"；3）**进退**——"'该'进'则'进'"——"'该'退'则'退'"。"'中庸'不'是'平庸'"——"'是'智慧'"——"'是'把握'分寸'"。"'洪应明'说'凡事'留'余地'"——"'话'不'说'满'"——"'事'不'做'绝'"
- **修身格言**: 《菜根谭》"有很多修身的格言'"。"'关于'自律'"：1）**慎独**——"'独处'时'也'要'自律'"；2）**反省**——"'每天'反省'自己'"；3）**克制**——"'克制'欲望'"。"'关于'品德'"：1）**诚信**——"'做人'要'诚信'"；2）**谦虚**——"'不'骄傲'"；3）**宽容**——"'宽容'别人'"。"'关于'心态'"：1）**平静**——"'心'要'平静'"；2）**知足**——"'知足'常乐'"；3）**豁达**——"'看'开'得失'"。"'这些'格言'简短'"——"'但'意味'深长'"
- **处世智慧**: 《菜根谭》"讲了很多处世的智慧'"。"'如何'与人'相处'"：1）**谦和**——"'对人'谦和'"——"'不'盛气凌人'"；2）**宽容**——"'宽容'别人'的'缺点'"；3）**真诚**——"'真诚'待人'"——"'不'虚伪'"。"'如何'对待'名利'"：1）**淡泊**——"'淡泊'名利'"；2）**随缘**——"'不'强求'"；3）**知足**——"'知足'常乐'"。"'如何'面对'逆境'"：1）**坚韧**——"'逆境'中'要'坚韧'"；2）**乐观**——"'乐观'面对'"；3）**反省**——"'从'逆境'中'学习'"
- **语录体风格**: 《菜根谭》"采用语录体风格'"。"'语录体'是'什么'"——"'一条'一条'的'格言'"——"'每条'独立'"——"'不'成'系统'"。"'这种'风格'的'优点'"：1）**简短**——"'容易'记住'"；2）**灵活**——"'随时'可以'读'"；3）**实用**——"'每条'都'能'指导'生活'"。"'但'缺点'是'不'系统'"——"'需要'自己'领悟'"。"'《菜根谭》'有'300 多条'格言'"——"'每条'都'值得'品味'"
- **洪应明的背景**: 洪应明——明代思想家——"《菜根谭》的作者'"。"他'的'生平'不'详'"——"'只'知道'他'是'明代'万历'年间'人'"。"他'曾'做过'官'"——"'后'来'隐居'"。"《菜根谭》'成书'于'1590 年代'"——"'但'长期'不'受'重视'"。"'清末'后'"——"'《菜根谭》'开始'流行'"——"'在'日本'尤其'受欢迎'"。"'现在'"——"'《菜根谭》'被'视为'中国'处世'智慧'的'经典'"——"'被'翻译成'多'种'语言'"
