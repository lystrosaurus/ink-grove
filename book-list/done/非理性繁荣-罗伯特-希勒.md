# 非理性繁荣

状态：已读

作者：罗伯特·希勒

一句话评价：罗伯特·希勒（Robert Shiller）的"行为金融学奠基之作'"——"'以详实数据揭示股市和楼市泡沫的心理机制'"。"'书名来自格林斯潘的名言'"——"'资产价格不仅由基本面决定，更受投资者心理、媒体叙事和社会情绪的驱动'"。"'对理解历次金融危机极具参考价值'"

核心收获：
- **行为金融学的视角**: 本书核心是"行为金融学的视角'"。"'什么'是'行为'金融学'"："'传统'金融学'假设'人'是'理性'的'"——"'但'行为'金融学'说'不'"。"'人'是'非理性'的'"——"'受'心理'因素'影响'"。"'希勒'的'观点'"："'资产'价格'不'只'由'基本面'决定'"——"'更'受'投资者'心理'影响'"。"'心理'因素'"：1）**过度自信**——"'觉得'自己'比'别人'聪明'"；2）**从众心理**——"'别人'买'我也'买'"；3）**叙事**——"'故事'驱动'投资'"
- **泡沫形成的机制**: 本书"分析了泡沫形成的机制'"。"'泡沫'如何'形成'"：1）**价格上涨**——"'初始'上涨'"——"'吸引'注意'"；2）**媒体放大**——"'媒体'报道'"——"'更多'人'知道'"；3）**新投资者涌入**——"'新手'进入'"——"'推动'价格'"；4）**正反馈**——"'价格'进一步'上涨'"——"'循环'"。"'例子'"："'1929 年'大萧条'"——"'2000 年'互联网'泡沫'"——"'2008 年'房地产'泡沫'"。"'泡沫'的'特征'"："'价格'远离'价值'"——"'人人'都'在'谈论'"
- **CAPE 指标**: 本书"提出了 CAPE 指标'"。"'什么'是'CAPE'"："'周期'调整'市盈率'"——"'Cyclically Adjusted Price-to-Earnings ratio'"。"'如何'计算'"："'价格'除以'过去'10 年'的'平均'盈利'"。"'为什么'用'10 年'"："'平滑'经济'周期'"。"'CAPE'的'作用'"："'判断'市场'是否'过热'"。"'历史'数据'"："'CAPE'高'时'"——"'未来'回报'低'"。"'2000 年'互联网'泡沫'"："'CAPE'达到'44 倍'"——"'历史'最高'"——"'后来'崩盘'"
- **心理因素的作用**: 本书"详细分析了心理因素'"。"'哪些'心理'因素'"：1）**过度自信**——"'投资者'高估'自己'的'能力'"；2）**代表性启发**——"'近期'表现'好'"——"'觉得'会'持续'"；3）**锚定**——"'被'某个'价格'锚定'"；4）**损失厌恶**——"'不愿'卖出'亏损'股票'"。"'这些'因素'"："'导致'非理性'行为'"——"'推动'泡沫'"
- **媒体的角色**: 本书"讨论了媒体的作用'"。"'媒体'如何'影响'市场'"：1）**放大**——"'放大'利好'消息'"；2）**叙事**——"'创造'故事'"——"'新时代'——'永久'繁荣'"；3）**从众**——"'媒体'都'说'一样'的'话'"。"'例子'"："'1929 年'媒体'说'永久'繁荣'"——"'2000 年'媒体'说'互联网'改变'一切'"。"'媒体'的'问题'"："'追求'吸引'眼球'"——"'不'是'真相'"
- **政策建议**: 本书"提供了政策建议'"。"'如何'防止'泡沫'"：1）**监管**——"'加强'金融'监管'"；2）**投资者教育**——"'教育'投资者'"——"'理解'风险'"；3）**预警**——"'建立'预警'系统'"——"'CAPE'等'指标'"；4）**结构性改革**——"'改革'金融'体系'"。"'希勒'的'观点'"："'泡沫'不'可'完全'避免'"——"'但'可以'减轻'"
- **作者的背景**: 罗伯特·希勒（Robert Shiller, 1946-）——美国经济学家、耶鲁大学教授——"《非理性繁荣》的作者'"。"他'2013 年'获得'诺贝尔经济学奖'"——"'行为'金融学'贡献'"。"'《非理性繁荣》'2000 年'出版'"——"'预测'了'互联网'泡沫'破裂'"。"'他'的'其他'贡献'"："'CAPE 指标'"——"' Case-Shiller 房价指数'"。"'希勒'的'研究'领域'"："'行为'金融学'"——"'房地产'"——"'市场'波动'"
