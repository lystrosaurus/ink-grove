# 非暴力沟通

状态：已读

作者：马歇尔·卢森堡

一句话评价：马歇尔·卢森堡（Marshall Rosenberg）的"沟通经典'"——"'观察、感受、需要、请求四步法实用有效'"。"'不带评论的观察是人类智力的最高形式'"——"'改善人际关系与化解冲突的指南'"。"'同理心沟通的核心'"——"'关注需要而非批评'"

核心收获：
- **四步法模型**: 本书核心是"'非暴力沟通四步法'"。"'什么'是'四步法'"：1）**观察**——"'说出'看到'的'事实'"——"'不'评价'"；2）**感受**——"'表达'自己'的'感受'"——"'不'指责'"；3）**需要**——"'说出'自己'的'需要'"——"'不'隐藏'"；4）**请求**——"'提出'具体'请求'"——"'不'命令'"。"'例子'"："'你'说'我'从'不'关心'你'"——"'这'是'评价'"。"'改为'"："'这'周'我'加班'5 天'（观察）'"——"'我'感到'疲惫'（感受）'"——"'我'需要'休息'（需要）'"——"'周末'能'让'我'休息'吗'（请求）'"
- **观察与评价的区分**: 本书"强调观察与评价的区分'"。"'什么'是'观察'"："'事实'"——"'不'带'评判'"。"'什么'是'评价'"："'判断'"——"'带'偏见'"。"'例子'"："'你'很'懒'"——"'这'是'评价'"。"'你'这周'3 天'没'洗碗'"——"'这'是'观察'"。"'为什么'要'区分'"："'评价'引发'防御'"——"'观察'不'引发'防御'"。"'卢森堡'说'"："'不带'评论'的'观察'是'人类'智力'的'最高'形式'"
- **感受的表达**: 本书"教导如何表达感受'"。"'什么'是'感受'"："'情绪'"——"'快乐'"——"'悲伤'"——"'愤怒'"。"'不'是'感受'"："'想法'"——"'你'不'爱'我'"——"'这'是'想法'"。"'例子'"："'你'说'我'被'忽视'"——"'这'是'想法'"。"'改为'"："'我'感到'孤独'"——"'这'是'感受'"。"'感受'词汇表'"："'书'中'提供'了'感受'词汇'"——"'帮助'识别'感受'"
- **需要的识别**: 本书"强调需要的识别'"。"'什么'是'需要'"："'人类'共有'的'"——"'安全'"——"'爱'"——"'尊重'"——"'自主'"。"'感受'来自'需要'"："'需要'满足'"——"'正面'感受'"。"'需要'未满足'"——"'负面'感受'"。"'例子'"："'我'生气'"——"'因为'我'需要'尊重'"。"'识别'需要'的'好处'"："'理解'自己'"——"'理解'别人'"。"'所有'行为'都'是'试图'满足'需要'"
- **请求的艺术**: 本书"讲解了如何提出请求'"。"'什么'是'请求'"："'具体'"——"'可操作'"——"'不'是'命令'"。"'请求'vs 命令'"："'命令'"——"'不'接受'就'惩罚'"。"'请求'"——"'接受'拒绝'"。"'例子'"："'你'必须'改'"——"'这'是'命令'"。"'你'愿意'改'吗'"——"'这'是'请求'"。"'如何'提出'请求'"：1）**具体**——"'不'要'抽象'"；2）**正面**——"'说'要'什么'"——"'不'说'不'要'什么'"；3）**可操作**——"'对方'能'做到'"
- **同理心倾听**: 本书"强调同理心倾听'"。"'什么'是'同理心'倾听'"："'倾听'对方'的'感受'和'需要'"——"'不'给'建议'"——"'不'安慰'"。"'例子'"："'朋友'说'我'很'沮丧'"。"'错误'回应'"："'别'难过'"——"'这'有'什么'"。"'正确'回应'"："'你'感到'沮丧'"——"'因为'你'需要'支持'吗'"。"'同理心'的'力量'"："'让'对方'感到'被'理解'"——"'化解'冲突'"
- **作者的背景**: 马歇尔·卢森堡（Marshall Rosenberg, 1934-2015）——美国心理学家、和平主义者——"《非暴力沟通》的作者'"。"他'师从'人本主义'心理学家'罗杰斯'"。"'他'创立'了'非暴力'沟通'中心'"——"'在'全球'推广'非暴力'沟通'"。"《非暴力沟通》'1999 年'出版'"——"'被'翻译成'30 多种'语言'"。"'他'的'应用'"："'学校'"——"'监狱'"——"'企业'"——"'战区'"。"'卢森堡'的'理念'"："'暴力'来自'未满足'的'需要'"——"'满足'需要'"——"'消除'暴力'"
