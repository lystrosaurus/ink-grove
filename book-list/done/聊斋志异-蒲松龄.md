# 聊斋志异

状态：已读

作者：蒲松龄

一句话评价：蒲松龄的"清代志怪小说巅峰之作'"——"'用狐仙鬼怪的故事写人间百态'"——"'既有浪漫的想象，也有对社会的深刻讽刺'"。"'聊斋的双重性'"——"'表面是狐仙鬼怪的奇幻故事'"——"'内核是对人性、科举、官场的批判'"。"'蒲松龄笔下的女妖往往比人更有情有义'"——"'这是对封建礼教的含蓄反叛'"

核心收获：
- **聊斋的双重性**: 《聊斋志异》有"双重性'"。"'表面'是'狐仙鬼怪'的'奇幻故事'"——"'花妖狐魅'——'书生小姐'"——"'浪漫'——'离奇'"。"'内核'是'对'人性、科举、官场'的'批判'"——"'讽刺'——'揭露'"——"'深刻'"。"'比如'《画皮》'"——"'表面'是'鬼怪'故事'"——"'内核'是'告诫'人们'不要被'外表'迷惑'"。"'比如'《促织》'"——"'表面'是'蟋蟀'故事'"——"'内核'是'批判'皇帝'的'荒淫'"
- **女妖比人更有情**: 蒲松龄笔下的"女妖往往比人更有情有义'"。"'婴宁'"——"'一个'狐女'"——"'她'天真'——'烂漫'"——"'她'爱'笑'"——"'她'的'笑'感染'了'所有人'"。"'聂小倩'"——"'一个'女鬼'"——"'她'被'迫'害人'"——"'但'她'内心'善良'"——"'她'爱'上'了'宁采臣'"——"'她'最终'获得'了'新生'"。"'这些'女妖'比'人'更'有情有义'"——"'这是'对'封建礼教'的'含蓄反叛'"——"'在'封建社会'"——"'女性'被'压迫'"——"'但'在'聊斋'中'"——"'女妖'有'自由'"——"'有'爱情'"
- **科举批判**: 蒲松龄"批判了科举制度'"。"'蒲松龄'自己'考'了一辈子'科举'"——"'到'71 岁'才'中'举人'"——"'他'对'科举'有'切肤之痛'"。"'书中'很多'故事'批判'科举'"：1）**《司文郎》**——"'考官'是'瞎子'"——"'他'用'鼻子'闻'文章'"——"'讽刺'考官'有眼无珠'"；2）**《叶生》**——"'一个'才子'"——"'考'不中'"——"'死后'才'中'"——"'讽刺'科举'埋没'人才'"；3）**《王子安》**——"'一个'人'中'举'后'疯了'"——"'讽刺'科举'对人'的'摧残'"
- **官场揭露**: 蒲松龄"也揭露了官场黑暗'"。"'《梦狼》**——"'一个'官员'梦见'自己'变成'狼'"——"'吃人'"——"'醒来'后'他'真的'吃人'"——"'揭露'官员'如'狼'"。"'《席方平》**——"'一个'人'为'父亲'申冤'"——"'他'告'到'阴间'"——"'阴间'的'官员'也'腐败'"——"'讽刺'人间'阴间'一样'黑暗'"。"'蒲松龄'说'官场'是'吃人'的'"——"'官员'是'狼'"——"'百姓'是'羊'"
- **文言小说的巅峰**: 《聊斋志异》是"文言小说的巅峰'"。"'文言'是'古代'的'书面语'"——"'简练'——'含蓄'"——"'但'也'难懂'"。"'蒲松龄'的'文言'"——"'优美'——'生动'"——"'他'用'文言'写'故事'"——"'但'不'晦涩'"。"'聊斋'的'语言'"——"'有'诗意'"——"'有'节奏'"——"'比如'描写'女妖'"——"'她'们'的'美貌'"——"'用'诗词'般的'语言'"。"'聊斋'把'文言小说'推向了'新高度'"——"'后世'无法'超越'"
- **浪漫的想象**: 蒲松龄有"浪漫的想象'"。"'他'的'故事'中'"——"'人'可以'与'狐'相恋'"——"'人'可以'与'鬼'相爱'"——"'人'可以'死而复生'"——"'物体'可以'成精'"。"'这种'想象'是'浪漫主义'的'"——"'它'超越'了'现实'的'束缚'"。"'蒲松龄'的'想象'不是'逃避'"——"'是'对'现实'的'超越'"——"'在'想象'中'"——"'他'实现'了'现实'中'无法'实现'的'理想'"
- **蒲松龄的背景**: 蒲松龄（1640-1715）——清代小说家——"《聊斋志异》的作者'"。"他'出身'寒微'"——"'他'考'了一辈子'科举'"——"'71 岁'才'中'举人'"。"他'一生'贫困'"——"'他'在'乡下'教书'"——"'他'收集'民间'故事'"——"'写'成'《聊斋志异》'"。"'聊斋'是'他'的'书房'名'"——"'他'在'那里'写'了'近 500 篇'故事'"。"'蒲松龄'说'写'聊斋'是'为了'抒发'心中的'愤懑'"——"'他'用'鬼怪'来'讽刺'人间'"
