# 查拉图斯特拉如是说

状态：已读

作者：弗里德里希·尼采

一句话评价：尼采（Friedrich Nietzsche）的"哲学代表作"（1883-1885 年出版）——"超人哲学"与"永恒轮回"思想的"文学表达"。"上帝死了"——"震撼了西方思想"。"人是连接动物与超人的'一根绳索'"——"绳索悬在'深渊之上'"。查拉图斯特拉是"尼采思想的'代言人'"——他"从山上下来"——向"世人传授'超人'的道理"。

核心收获：
- **超人哲学**: 尼采最核心的概念——"超人"（Übermensch）。"超人"不是"超级英雄"——是"超越了'普通人'的人"。"人是'一根绳索'"——"连接在'动物'和'超人'之间"——"悬在'深渊之上'"。尼采说"人的'意义'不在于'他是人'"——在于"他是'通往超人的'桥梁'"。超人"不遵循'传统道德'"——"他'创造'自己的价值观"——"他是'自己的主人'"
- **永恒轮回**: 尼采最"令人不安"的思想——"永恒轮回"（eternal recurrence）。"假设你的一生——每一个细节——都会'无限次地重复'"——"你会'感到绝望'还是'感到狂喜'"？如果你"感到绝望"——说明"你没有'真正地活着'"。如果你"感到狂喜"——说明"你'热爱'你的生命"——"这就是'超人的态度'"。永恒轮回是"对生命最极端的'肯定'"——"不是'接受'生命"——是"'热爱'生命"——"热爱到'愿意'让它'无限重复'"
- **权力意志**: 尼采说"万物的'根本驱动力'"不是"生存"——是"权力意志"（will to power）。"权力意志"不是"政治权力"——是"生命力"——是"想要'变得更强大'的冲动"。"植物向阳光生长"——"这是'权力意志'"。"艺术家创作"——"这是'权力意志'"。"哲学家思考"——"这也是'权力意志'"。尼采说"生命的'本质'不是'活着'"——是"'变得更强大'"
- **骆驼、狮子、婴儿三变**: 尼采用"三种变形"描述了"精神的成长历程"。1）**骆驼**——"负重前行"——"接受'别人给的任务'"——"服从'传统和权威'"；2）**狮子**——"反抗"——"对'一切既定价值'说'不'"——"打破'旧的规则'"；3）**婴儿**——"创造"——"说'是'"——"创造'新的价值'"。尼采说"大多数人'停留在'骆驼阶段'"——"少数人'成为'狮子'"——"只有极少数人'成为'婴儿'"——"成为'超人'"
- **上帝死了**: 尼采最著名的宣言——"上帝死了"。这不是"无神论的'欢呼'"——是"对'西方文明危机'的'诊断'"。尼采说"基督教价值观'统治了'西方两千年"——"但当'上帝死了'"——"这些价值观就'失去了根基'"——"西方将面临'虚无主义'的危机"——"如果没有上帝——一切'还有意义'吗？"尼采的答案是"有"——"意义不在'上帝'那里"——在"'你自己'那里"——"你'创造'自己的意义"
- **末人**: 尼采说"超人的'对立面'"是"末人"（the last man）——"末人"是"最'渺小'的人"——"他'没有理想'"——"他'不冒险'"——"他'追求舒适'"——"他说'我们发明了幸福'"——"然后'眨着眼睛'"。尼采说"末人是'现代人的'写照'"——"追求'安逸'"——"害怕'风险'"——"没有'激情'"——"没有'创造力'"
- **尼采的背景**: 弗里德里希·尼采（1844-1900）——德国哲学家——"西方思想史上'最具争议'的人物之一"。《查拉图斯特拉如是说》（Thus Spoke Zarathustra）是他"最重要的作品"——用"诗意的语言"表达"哲学思想"——而不是"用'论证'的方式"。尼采 1889 年"精神崩溃"——此后"十年间'失去理智'"——1900 年去世——享年 55 岁。他的思想"影响了'存在主义'（萨特、加缪）"——"影响了'后现代主义'（福柯、德里达）"——"影响了'心理学'（弗洛伊德、荣格）"
