# 了凡四训

状态：已读

作者：袁了凡

一句话评价：中国版"命运可自主"的启蒙书，袁了凡用亲身经历讲透"改命"的底层逻辑——信、愿、行缺一不可。全文仅万余字，却融合了儒释道三家的修身智慧，是明清以来流传最广的劝善书。

核心收获：
- **"命由我作，福自己求"**: 了凡的核心命题是"命运不是预设轨道，而是可修可改的因果系统"。他被算命先生算定了一生的功名、寿命、子嗣，起初一一应验，于是"心如死灰"。后来遇到云谷禅师，告诉他"命由我作"——命运是可以通过行善改过的。从此他 actively 行善改过，最终改变了算命先生的预言
- **立命之学**: 知命而不认命。了凡不否认"命"的存在——他确实被算准了前半生。但他认为"命"是起点，不是终点。改过迁善是改变命运的唯一路径。这和儒家的"尽人事"、佛家的"因果"、道家的"我命在我不在天"完全契合
- **积善之方**: 善有真假、端曲、阴阳、是非之分。"真善"是出于本心、不求回报；"假善"是做给别人看、求名声。"端善"是正直无私；"曲善"是别有用心。"阴善"是做了好事不张扬；"阳善"是到处宣扬。了凡强调"阴善"最有力量——因为它是纯粹的
- **谦德之效**: 谦卑是承载福报的容器，骄者必败。了凡说"满招损，谦受益"——骄傲的人装不下福报，谦卑的人能容纳更多。他观察科举中举的人，发现中举前都有一个共同特征：变得谦卑。这是"谦德之效"的经验证明
- **改过之法**: 从事上改、从理上改、从心上改三层递进。"从事上改"是强制自己不做坏事（最浅）；"从理上改"是想通了为什么不该做（中等）；"从心上改"是连坏的念头都不生了（最高）。"心上改"是根本——心改了，行自然改
- **功过格**: 了凡用"功过格"记录每天的善恶行为——做一件善事记一功，做一件恶事记一过。每天统计，看功多还是过多。这不是"道德记账"，是"自我觉察"的工具。通过量化，他能清楚地看到自己的进步和退步
- **了凡的人生**: 袁了凡（1533-1606）原名袁黄，字坤仪。他被算定只能活 53 岁、无子、只做小官。结果他活到 74 岁、有儿子、做到了兵部职方司主事。他用自己的亲身经历证明了"命可改"。《了凡四训》是他 69 岁时写给儿子的家训
- **当代意义**: 在"躺平"和"内卷"并存的今天，《了凡四训》提供了一个中间路径——不认命，也不焦虑。"知命"是接受现实，"改命"是积极行动。这不是"鸡汤"，是一套可操作的方法论：觉察→改过→积善→谦逊→持续
