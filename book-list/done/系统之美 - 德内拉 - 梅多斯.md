# 系统之美

## 基本信息
- **书名**: 系统之美
- **作者**: 德内拉 - 梅多斯
- **出版社**: 浙江人民出版社
- **出版年份**: 2020
- **页数**: 400
- **分类**: 认知思维/管理学

## 选书理由
系统思考领域的入门经典，梅多斯用通俗语言解释了系统的核心概念（存量、流量、反馈回路、延迟）和常见陷阱（目标侵蚀、舍本逐末、增长上限）。在复杂互联的世界中，系统思维是理解问题根源、避免意外后果的关键能力。希望建立系统思考的基本框架，在生活和工作中识别杠杆点，实现更有效的改变。
