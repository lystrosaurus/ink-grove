# 清醒思考的艺术

状态：已读

作者：罗尔夫·多贝里

一句话评价：罗尔夫·多贝里（Rolf Dobelli）的"认知偏差百科"——"总结了 52 种常见的思维偏差"——"用'简洁的案例'说明了'人类在日常决策中'如何被'认知偏误'所误导"。"'确认偏误'"——"'幸存者偏差'"——"'沉没成本谬误'"——"'社会认同'"——"'锚定效应'"。"知道自己有哪些思维偏差是避免它们的第一步"——"觉察是改变的起点"。

核心收获：
- **确认偏误**: 多贝里说"确认偏误"是"所有认知偏差中最普遍的"。"人们倾向于'寻找支持自己观点的信息'"——"'忽视反驳自己的证据'"。"比如你'买了一只股票'"——"你会'专门去看'看好这只股票的文章"——"而'忽略'看空的观点"。"确认偏误'让'我们'越来越'极端'"——"因为'我们只看'支持自己的信息"——"最终'活在自己的'信息茧房里"
- **幸存者偏差**: 多贝里说"幸存者偏差"是"最容易让人产生幻觉的偏差"。"我们'只看到'成功者"——"'忽略'了失败者"——"所以'严重高估'了成功的概率"。"比如'大家都说'创业能成功'"——"但你'只看到了'那几个'成功的企业家'"——"你'没看到'的'是'成千上万'破产的人'"。"幸存者偏差'让你'做出错误的决策'"——"因为'你看到的'数据'是不完整的'"
- **沉没成本谬误**: 多贝里说"沉没成本谬误"是"让人最痛苦的偏差"。"你已经'投入了'时间、金钱、精力'"——"但'项目失败了'"——"你'不愿意放弃'"——"因为'你觉得'放弃了'之前的投入'就'白费了'"。但"沉没成本'已经沉没了'"——"它'不应该'影响你的'未来决策'"。"正确的做法是'问自己'"——"'如果'我现在'从零开始'"——"我'还会'投入'这个项目'吗？"——"如果'不会'"——"那就'立刻放弃'"
- **社会认同**: 多贝里说"社会认同"是"让人从众的心理机制"。"当'你不确定'该怎么做时"——"你会'看别人'怎么做"——"然后'跟着做'"。"在'大多数情况下'"——"从众是'安全的'"——"但'在投资领域'"——"从众是'最危险的'"。"当'所有人都在'买某只股票时"——"你'也会想买'"——"但这'往往就是'见顶的信号"
- **锚定效应**: 多贝里说"锚定效应"是"让人被'第一个数字'左右的偏差"。"商家'先标一个'高价'"——"然后'打折'"——"你'觉得很便宜'"——"但'那个高价'就是一个'锚'"——"它'让你的'判断'被'扭曲了'"。"锚定效应'在'谈判'中'很常见'"——"谁'先出价'"——"谁就'设定了锚'"——"后续的'讨论'都'围绕这个锚'进行"
- ** hindsight 偏差**: 多贝里说"后见之明偏差"是"让人'自以为是'的偏差"。"事情'发生后'"——"你'会说'"——"'我早就知道会这样'"——"但'实际上'"——"你'之前并'不知道'"。"后见之明偏差'让你'过度自信'"——"你'以为'自己能'预测未来'"——"但'实际上'你'只是'在'事后找理由'"
- **罗尔夫·多贝里的背景**: 罗尔夫·多贝里（Rolf Dobelli, 1966-）——瑞士作家、企业家——"《清醒思考的艺术》的作者"。他"是'一家'全球性企业俱乐部'的'创始人'"——"他的'写作风格'非常'简洁明了'"——"每个偏差'用'2-3 页'就能'讲清楚'"——"配'一个'生活中的'案例'"——"让'普通读者'也能'轻松理解'"
