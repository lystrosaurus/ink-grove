# 九诗心

状态：已读

作者：黄晓丹

一句话评价：用九位古代诗人的生命故事讲"诗心"是什么——不是写诗的技巧，是对世界的敏感、对生命的深情、对美的执着。

核心收获：
- 九位诗人：从陶渊明到苏轼，从李清照到纳兰性德，每个人代表一种"诗心"
- 黄晓丹的解读方式不是"文学分析"，是"生命共鸣"。她不讲格律和典故，讲这些诗人怎么活、怎么爱、怎么痛
- 陶渊明的"归去来兮"不是逃避，是清醒地选择了自己想要的生活
- 李清照的词从少女的明媚到丧夫后的凄凉，一个人的一生就是一部词史
- 苏轼的"一蓑烟雨任平生"——不是不在乎，是经历了所有在乎之后的放下
