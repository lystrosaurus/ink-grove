# 深度学习入门

状态：已读

作者：斋藤康毅

一句话评价：最好的深度学习入门书之一，从零开始用 Python 实现神经网络，不依赖高级框架，让读者真正理解底层原理，图解清晰，代码完整。

核心收获：
- 通过从零实现感知机、神经网络、CNN、RNN，理解了深度学习不是黑箱而是可理解的数学
- 反向传播（误差反向传播法）是深度学习训练的核心机制，用计算图的方式讲解非常清晰
- 理解了过拟合、学习率、权重初始化等关键概念的实际含义和解决方法
