# 零到一

## 基本信息
- **作者**：彼得·蒂尔
- **分类**：创业/科技

## 核心总结

蒂尔提出从 0 到 1（垂直进步）比从 1 到 n（水平进步）更重要。核心是创造垄断而非竞争，通过技术创新实现 10 倍改进。

## 关键洞见

1. **垄断优势**：竞争是失败者的游戏
2. **10 倍改进**：渐进改进不够
3. **秘密**：相信还有未发现的真理
4. **幂次法则**：投资回报高度集中
5. **确定乐观**：对未来有计划
