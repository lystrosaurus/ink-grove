# 假面饭店

状态：已读

作者：东野圭吾

一句话评价：用酒店做舞台的悬疑小说，把"服务行业的微笑"变成最大的悬疑道具——你永远不知道面具后面藏着什么。

核心收获：
- 连续杀人案的线索指向东京"科尔特西亚"酒店，刑警新田浩介伪装成前台卧底
- 酒店员工山岸尚美是唯一知道新田真实身份的人，两人从互不信任到默契配合
- 每个客人都有嫌疑，每个人都有秘密。酒店的天然是悬疑的最佳舞台
- 东野圭吾的叙事节奏把控极佳：悬疑感持续累积，直到最后一章才揭开真相
- "假面"的双重含义：客人的面具和酒店员工的面具。服务人员必须微笑，但微笑下面是什么？
