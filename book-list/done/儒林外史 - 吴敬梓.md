# 儒林外史

状态：已读

作者：吴敬梓

一句话评价：中国古代讽刺小说巅峰之作，以科举制度为核心刻画了范进、严监生等一群被功名腐蚀的读书人形象，笔法辛辣而含蓄——吴敬梓不骂人，他让角色自己暴露自己的可笑。

核心收获：
- **范进中举**: 全书最震撼的场景。范进考了 20 多次科举，54 岁终于中举，听到消息后疯了——"噫！好了！我中了！"这一声喊里有多少辛酸、多少压抑、多少被科举制度扭曲的人生。吴敬梓写范进不是写笑话，是写悲剧
- **严监生**: 吝啬到了极致——临死前伸着两根手指不肯闭眼，家人猜了半天才明白他是嫌灯芯费油。吴敬梓写吝啬不写"他很小气"，写一个动作、一个细节，让读者自己得出结论
- **杜少卿**: 书中少有的亮色。散尽家财、广交好友、蔑视礼法——某种程度上是吴敬梓的自我投射。在满书功名利禄中，杜少卿代表了一种超越的生活态度
- **讽刺的最高境界**: 吴敬梓不直接批判科举，不骂读书人，他只是"呈现"——让角色用自身的荒诞暴露可笑。范进疯了，严监生死了，匡超人变质了——读者自己会得出结论：这个制度有问题
- **"外史"的含义**: 不是正统历史，而是被正史忽略的知识分子众生相。正史写帝王将相，外史写普通读书人；正史写丰功伟绩，外史写人性百态。吴敬梓选择"外史"是对正统叙事的反叛
- **科举制度的解剖**: 科举不是坏的制度——它比世袭制公平得多。但它的问题是把"读书"变成"敲门砖"，一旦门敲开了，砖就扔了。范进中举后再也不读书了——读书对他只是手段，不是目的
- **结构特点**: 全书没有贯穿始终的主角，由一个又一个独立故事串联。这种"串珠式"结构影响了后世的《官场现形记》《二十年目睹之怪现状》等谴责小说。每个人物出场几章就消失，但形象刻得极深
- **吴敬梓的处境**: 他自己出身书香门第，年轻时也参加科举，后来家道中落，看透功名。《儒林外史》是他用半生经历写成的"科举受害者自白书"
