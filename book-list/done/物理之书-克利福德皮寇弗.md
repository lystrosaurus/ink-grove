# 物理之书

状态：已读

作者：克利福德皮寇弗

一句话评价：用 250 个里程碑事件讲述物理学发展史，从古希腊的自然哲学到量子力学和宇宙学，是一本图文并茂的物理学入门百科。

核心收获：
- 理解了物理学的发展不是线性进步，而是范式转换——从亚里士多德到牛顿再到爱因斯坦，每一次都是对旧框架的颠覆
- "250 个里程碑"的编排让读者可以快速浏览整个物理学史，不需要从头读到尾
- 物理学史的本质是人类认知边界的不断拓展——每一个突破都来自"不可能"的质疑
