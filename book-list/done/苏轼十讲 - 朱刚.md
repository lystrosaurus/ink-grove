# 苏轼十讲

状态：已读

作者：朱刚

一句话评价：朱刚的"苏轼研究佳作'"——"'以十个主题切入苏轼的人生与创作'"。"'学术功底扎实但不枯燥'"——"'将文学分析与历史语境结合'"。"'呈现出一个立体的苏轼——文学家、政治家、思想家、生活家'"。"'儒释道三家在他身上融合为一种独特的人生态度'"

核心收获：
- **思想的多元融合**: 朱刚"分析了苏轼思想的多元性'"。"'苏轼'的'思想'来源'"：1）**儒家**——"'入世'——'治国平天下'——'他的'政治'理想'"；2）**道家**——"'出世'——'顺其自然'——'他的'人生态度'"；3）**佛家**——"'空'——'无'——'他的'精神'慰藉'"。"'这三家'在'苏轼'身上'不是'简单'拼凑'"——"'是'融合'"——"'形成'了'一种'独特'的'人生态度'"。"'顺境'时'他'是'儒家'"——"'逆境'时'他'是'道家'——'佛家'"。"'这种'融合'让'他'能'在'任何'处境'下'都'保持'内心'的'平静'"
- **乌台诗案**: 朱刚"详细分析了'乌台诗案'"。"'1079 年'"——"'苏轼'因'诗'获罪'"——"'被'下狱'"——"'几乎'被'杀'"。"'这是'宋代'文字狱'的'标志性'事件'"。"'苏轼'的'诗'被'解读'为'讽刺'新法'"——"'新党'要'整'他'"。"'最后'苏轼'被'贬'黄州'"——"'这是'他'人生'的'转折点'"。"'乌台诗案'后'"——"'苏轼'的'创作'风格'改变'"——"'从'豪放'转向'旷达'"——"'从'外在'转向'内心'"。"'黄州'时期'"——"'苏轼'写'了'最'伟大'的'作品'"
- **贬谪与创作**: 朱刚"探讨了贬谪对苏轼创作的影响'"。"'苏轼'一生'三次'大'贬'"：1）**黄州**——"'1080-1084'"——"'写'了'《赤壁赋》'——'《念奴娇·赤壁怀古》'"；2）**惠州**——"'1094-1097'"——"'写'了'大量'诗词'"；3）**儋州**——"'1097-1100'"——"'海南'当时'是'蛮荒之地'"。"'贬谪'是'不幸'"——"'但'成就'了'苏轼'的'文学'"。"'在'贬谪中'"——"'苏轼'远离'政治'"——"'亲近'自然'"——"'思考'人生'"——"'写出'了'最'深刻'的'作品'"。"'所以'说'国家不幸诗家幸'"
- **文学成就**: 朱刚"系统评价了苏轼的文学成就'"。"'诗'"——"'苏轼'的'诗'题材'广泛'"——"'风格'多样'"——"'与'黄庭坚'并称'苏黄'"。"'词'"——"'苏轼'开创'豪放派'"——"'扩大'了'词'的'题材'"——"'与'辛弃疾'并称'苏辛'"。"'文'"——"'苏轼'是'唐宋八大家'之一'"——"'他的'散文'自然'流畅'"——"'《赤壁赋》'是'千古'名篇'"。"'书法'"——"'苏轼'是'宋四家'之首'"——"'《黄州寒食帖》'是'天下'第三'行书'"。"'绘画'"——"'苏轼'提倡'文人画'"——"'诗中有画，画中有诗'"
- **人生态度**: 朱刚"分析了苏轼的人生态度'"。"'苏轼'的'人生态度'"：1）**旷达**——"'不'被'得失'左右'"——"'宠辱不惊'"；2）**乐观**——"'在'逆境'中'也'能'找到'乐趣'"——"'一蓑烟雨任平生'"；3）**包容**——"'能'容纳'不同'思想'"——"'能'理解'不同'处境'的'人'"；4）**真诚**——"'不'虚伪'"——"'不'做作'"——"'真性情'"。"'这种'人生态度'"——"'让'苏轼'在'任何'处境'下'都'能'活得'精彩'"。"'苏轼'是'中国'文人'的'理想'人格'典范'"
- **历史语境**: 朱刚"把苏轼放在历史语境中分析'"。"'苏轼'的'时代'"——"'北宋'中后期'"——"'王安石变法'"——"'新旧党争'"。"'苏轼'夹'在'两派'之间'"——"'他'不'完全'支持'新法'"——"'也'不'完全'反对'"——"'他'主张'渐进'"——"'所以'两派'都'不'喜欢'他'"。"'这种'独立'立场'"——"'让'他'一生'坎坷'"——"'也'让'他'更'清醒'"。"'理解'苏轼'"——"'要'理解'他'的'时代'"——"'理解'他'的'选择'"
- **朱刚的背景**: 朱刚——复旦大学中文系教授——"《苏轼十讲》的作者'"。"他'是'宋代文学'研究'专家'"——"'尤其'擅长'苏轼'研究'"。"《苏轼十讲》'2019 年'出版'"——"'获得'好评'"。"这本书'的'特点'"——"'学术'与'通俗'结合'"——"'既有'研究'深度'"——"'又'适合'普通'读者'"。"'十讲'的'形式'"——"'每讲'一个'主题'"——"'结构'清晰'"——"'易于'理解'"
