# 围城

状态：已读

作者：钱钟书

一句话评价：钱钟书"唯一长篇小说'"——"'围城隐喻成为中国人最熟悉的人生困境表述'"。"'方鸿渐的优柔寡断折射知识分子的精神软弱'"。"'三闾大学篇讽刺学术圈入木三分'"。"'语言精妙到几乎每页都有可摘抄的句子'"。"'1947 年出版即成经典'"——"'中国现代文学必读之作'"

核心收获：
- **方鸿渐的困境**: 本书核心是"'方鸿渐的困境'"。"'一个'无用'但'不坏'的'知识分子'"。"'留学'欧洲'买'了'个'假'博士'学位'"。"'回国'后'在'爱情'"——"'事业'"——"'婚姻'中'处处'被动'"。"'他'不'是'坏人'"——"'只是'太'软弱'"——"'太'犹豫'"——"'太'不'主动'"。"'这种'人'在'现实'中'比比皆是'"
- **围城隐喻**: 本书"提出了围城隐喻'"。"'婚姻'是'围城'"——"'事业'是'围城'"——"'人生'也'是'围城'"。"'里面'的'人'想'出去'"——"'外面'的'人'想'进来'"。"'这'不'是'婚姻'的'悲剧'"——"'是'人类'的'普遍'困境'"——"'永远'不'满足'"——"'永远'在'追求'另一种'生活'"
- **苏文纨的虚荣**: 本书"刻画了苏文纨的形象'"。"'女'博士'"——"'爱'方鸿渐'但'放'不'下'身段'"。"'最后'嫁'给'了'自己'不'爱'的'人'"。"'她'的'悲剧'是'高傲者'的'悲哀'"——"'太'把'自己'当'回事'"——"'结果'错过'了'真正'爱'的'人'"
- **孙柔嘉的心机**: 本书"描绘了孙柔嘉的形象'"。"'看似'温柔'实则'精明'"——"'用'手段'嫁'给'方鸿渐'"。"'但'婚后'发现'这个'围城'并'不'如'想象'中'美好'"。"'她'的'悲剧'是'精明人'的'悲哀'"——"'算'得到'婚姻'"——"'算'不'到'幸福'"
- **唐晓芙的错过**: 本书"讲述了唐晓芙的故事'"。"'方鸿渐'的'真爱'"——"'但'因为'误会'和'双方'的'不'主动'"——"'最终'错过'"。"'这'是'全书'最'令人心疼'的'一对'"——"'明明'相爱'"——"'却'因为'都'不'肯'先'开口'而'终身'遗憾'"
- **三闾大学的荒诞**: 本书"讽刺了三闾大学'"。"'方鸿渐'去'内地'大学'任教'"——"'见识'了'知识分子'圈'的'虚伪'"——"'派系'斗争'"——"'勾心斗角'"。"'钱钟书'写'大学'不'是'写'象牙塔'"——"'是'写'小社会'"。"'学术'圈'的'名利场'一点'都'不'比'外面'干净'"
- **作者的背景**: 钱钟书（1910-1998）——中国作家、学者——"《围城》的作者'"。"'他'是'清华'大学'教授'"——"'精通'多国'语言'"。"'《围城》'1947 年'出版'"——"'是'他'唯一'的'长篇小说'"。"'他'的'其他'著作'"："'《谈艺录》'——'《管锥编》'"。"'钱钟书'的'风格'"："'讽刺'"——"'幽默'"——"'比喻'精妙'"
- **语言的巅峰**: 本书"语言达到巅峰'"。"'中国'现代'文学'中'语言'最'精妙'的'小说'之一'"。"'钱钟书'的'比喻'"——"'反讽'"——"'双关'"——"'几乎'每'一页'都'有'可以'摘抄'的'句子'"。"'但'语言'不'是'炫技'"——"'是'为'人物'和'主题'服务'"
