# 论语

状态：已读

作者：孔子弟子及再传弟子

一句话评价：孔子言行的记录，儒家思想的集大成者——不是哲学论文，是一个老人在和他的学生聊天，但聊出了中国文化的底色。

核心收获：
- "仁"是论语的核心。孔子对"仁"的解释每次都不同——对颜回说"克己复礼"，对樊迟说"爱人"，对子贡说"己所不欲勿施于人"。不是孔子没有定论，是"仁"本身就是情境化的
- "学而时习之"——"习"不是复习，是实践。学了之后去做了，这才是"习"。所以孔子说"学而时习之，不亦说乎"，因为实践比读书更有成就感
- 因材施教：同样的问题（"闻斯行诸"），孔子对子路说"你先问问父兄"，对冉有说"听到了就做"。因为子路太冲动，冉有太犹豫。教育不是给答案，是补不足
- "君子"不是地位，是品格。孔子把"君子"从"贵族"重新定义为"有德之人"。这是中国思想史上最关键的转变之一
- 论语的语言极其精炼，几乎每句都可以当格言用。但格言不等于真理——需要在具体情境中理解
