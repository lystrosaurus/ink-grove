# 布鲁克林有棵树

状态：已读

作者：贝蒂史密斯

一句话评价：20 世纪最温暖的成长小说之一，讲述贫民窟女孩弗兰西在贫困中通过阅读和教育改变命运的故事，文字质朴但充满力量，"愿你是那棵在水泥地里也能长成的树"。

核心收获：
- 理解了教育和阅读如何成为底层孩子突破阶层的最有效工具
- 弗兰西的母亲凯蒂是全书最强大的角色——她不识字但有着钢铁般的意志和清醒的头脑
- 贫困不是堕落的借口，而是一种需要被超越的处境——尊严可以在任何环境中保持
