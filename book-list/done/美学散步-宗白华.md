# 美学散步

状态：已读

作者：宗白华

一句话评价：中国现代美学的经典之作，宗白华用散文诗般的笔法讲述中国艺术的意境之美，从诗歌、绘画到书法、园林，展现了中国美学的独特传统。

核心收获：
- 理解了"意境"是中国美学的核心概念——不是客观事物的再现，而是主客交融的精神境界
- 中国艺术与西方艺术的根本区别：西方追求"再现"（形似），中国追求"传神"（神似）
- "散步"是宗白华的美学方法论——用自由的、非体系化的方式感悟美，而不是用逻辑推理分析美
