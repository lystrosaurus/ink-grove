# 财务自由第一课

状态：已读

作者：帅张

一句话评价：帅张的"面向普通人的投资理财入门指南'"——"'用通俗语言讲解资产配置、复利思维、指数基金定投等核心概念'"。"'帮助建立正确的金钱观'"——"'财务自由的本质是被动收入覆盖生活支出'"

核心收获：
- **财务自由的本质**: 本书提出"'财务自由'的本质'"。"'什么'是'财务'自由'"——"'不'是'有'很多'钱'"——"'是'被动'收入'覆盖'生活'支出'"。"'被动'收入'是'什么'"——"'不'需要'你'工作'就'有'的'收入'"——"'房租'"——"'股息'"——"'利息'"。"'如果'你'每月'支出'1 万'"——"'你'的'被动'收入'也'有'1 万'"——"'你'就'财务'自由'了'"。"'所以'财务'自由'不'是'数字'游戏'"——"'是'收入'结构'的'改变'"
- **复利思维**: 本书"强调了复利思维'"。"'什么'是'复利'"——"'利滚利'"——"'收益'再投资'"。"'复利'的'公式'"——"'本金×(1+ 收益率)^时间'"。"'时间'是'关键'"——"'时间'越'长'"——"'复利'效果'越'明显'"。"'例子'"——"'每月'定投'1000 元'"——"'年化'10%'"——"'30 年'后'是'200 万'"。"'复利'的'启示'"：1）**早开始**——"'时间'是'朋友'"；2）**持续**——"'不'中断'"；3）**耐心**——"'等待'复利'发挥作用'"
- **指数基金定投**: 本书"推荐指数基金定投'"。"'什么'是'指数'基金'"——"'跟踪'市场'指数'的'基金'"——"'比如'沪深 300'"。"'为什么'推荐'"：1）**分散风险**——"'买'一篮子'股票'"；2）**低成本**——"'管理费'低'"；3）**长期上涨**——"'市场'长期'是'上涨'的'"。"'定投'是'什么'"——"'定期'定额'投资'"。"'定投'的'好处'"：1）**不用择时**——"'不'需要'预测'市场'"；2）**平摊成本**——"'高'时'买'得'少'"——"'低'时'买'得'多'"；3）**强制储蓄**——"'自动'扣款'"
- **资产配置**: 本书"讲解了资产配置'"。"'什么'是'资产'配置'"——"'把'钱'分散'到'不同'资产'"。"'为什么'"——"'不'同'资产'的'表现'不'同'"——"'分散'可以'降低'风险'"。"'如何'配置'"：1）**股票**——"'高'收益'——'高'风险'"；2）**债券**——"'低'收益'——'低'风险'"；3）**现金**——"'流动性'好'"。"'比例'如何'定'"——"'年轻'人'多'配'股票'"——"'年纪'大'多'配'债券'"
- **金钱观**: 本书"帮助建立正确的金钱观'"。"'金钱'不'是'万能的'"——"'但'金钱'是'工具'"。"'金钱'的'作用'"：1）**选择自由**——"'有'钱'有'更多'选择'"；2）**安全保障**——"'应对'意外'"；3）**实现梦想**——"'支持'你'的'理想'"。"'不要'成为'金钱'的'奴隶'"——"'要'做'金钱'的'主人'"。"'正确'的'金钱观'"——"'金钱'是'手段'"——"'不'是'目的'"
- **投资心理**: 本书"讨论了投资心理'"。"'投资'的'最大'敌人'"——"'自己'的'情绪'"。"'贪婪'"——"'涨'时'想'更'多'"——"'追高'"。"'恐惧'"——"'跌'时'害怕'"——"'割肉'"。"'如何'克服'"：1）**长期思维**——"'不'看'短期'波动'"；2）**定投**——"'自动'化'投资'"——"'减少'情绪'干扰'"；3）**学习**——"'理解'市场'"——"'减少'恐惧'"
- **帅张的背景**: 帅张——中国理财博主、投资人——"《财务自由第一课》的作者'"。"他'在'互联网'行业'工作'"——"'后'来'实现'财务'自由'"。"'他'写'了'这本书'"——"'分享'他'的'经验'"。"'他'的'写作'风格'"——"'通俗'"——"'实用'"——"'适合'普通人'"。"'他'还'运营'公众号'"——"'分享'理财'知识'"——"'有'大量'粉丝'"
