# 第三选择

状态：已读

作者：史蒂芬柯维

一句话评价：史蒂芬·柯维继《七个习惯》后的又一力作，提出在冲突中跳出"你的方法"和"我的方法"，寻找双方都满意的"第三选择"。

核心收获：
- 理解了"第三选择"的核心：不是妥协（双方各退一步），而是协同（找到比各自方案都好的新方案）
- 四步法：我看到自己（跳出自我）、我看到你（把对方当人而非障碍）、我找到你（倾听理解）、我和你协同（共创第三选择）
- 第三选择适用于所有冲突场景——家庭、职场、社会、国际关系
