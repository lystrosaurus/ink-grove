# 第三选择

状态：已读

作者：史蒂芬·柯维

一句话评价：史蒂芬·柯维（Stephen Covey）"继《七个习惯》后的又一力作'"——"'提出在冲突中跳出'你的方法'和'我的方法'"——"'寻找双方都满意的'第三选择'"。"'第三选择'的核心"——"'不是妥协'（双方各退一步）'"——"'是协同'（找到比各自方案都好的新方案）'"。"'四步法'"——"'我看到自己'（跳出自我）'——'我看到你'（把对方当人而非障碍）'——'我找到你'（倾听理解）'——'我和你协同'（共创第三选择）'"

核心收获：
- **第三选择的核心理念**: 柯维提出"'第三选择'的核心'"——"'不是妥协'"——"'是协同'"。"'妥协'是'双方各退一步'"——"'每个人都'牺牲'一部分'利益'"——"'结果'是'双赢'但'都不满意'"。"'协同'是'找到'一个'新的方案'"——"'这个方案'比'各自的方案'都'好'"——"'结果是'双赢'且'都满意'"。"柯维说'第三选择'不是'你的方法'或'我的方法'"——"'是'我们的方法'"——"'一个'更好的'方法'"
- **四步法**: 柯维提出了"'实现第三选择的'四步法'"：1）**我看到自己**——"'跳出'自我'"——"'我'不是'我的'观点'"——"'我'可以'选择'我的'回应'"；2）**我看到你**——"'把对方'当作'人'"——"'不是'障碍'"——"'尊重'对方'的'观点'和'感受'"；3）**我找到你**——"'倾听'对方'"——"'理解'对方'的'需求'和'关切'"——"'不是'为了'反驳'"——"'是'为了'理解'"；4）**我和你协同**——"'共创'第三选择'"——"'头脑风暴'"——"'寻找'比'各自方案'都'好的'方案'"
- **我看到自己**: 柯维说"'我看到自己'是'第一步'"——"'跳出'自我'"。"'我们'在'冲突中'"——"'容易'被'情绪'控制'"——"'我们'认为'我的观点'就是'我'"——"'如果'你'反对'我的观点'"——"'就是'反对'我'"。"'我看到自己'意味着'"——"'我'意识到'我'不是'我的观点'"——"'我'可以'选择'我的'回应'"——"'我'可以'跳出'情绪'"——"'理性地'看待'问题'"
- **我看到你**: 柯维说"'我看到你'是'第二步'"——"'把对方'当作'人'"。"'在冲突中'"——"'我们'容易'把对方'当作'障碍'"——"'敌人'"——"'问题'"。"'我看到你'意味着'"——"'我'看到'对方'也是'一个'人'"——"'有'自己的'需求'——'感受'——'观点'"——"'我'尊重'对方'"——"'即使'我'不同意'他'"
- **我找到你**: 柯维说"'我找到你'是'第三步'"——"'倾听理解'"。"'倾听'不是'为了'反驳'"——"'是'为了'理解'"。"'你'要'问'对方'"——"'你的需求是什么'"——"'你的关切是什么'"——"'什么对你最重要'"——"'然后'你'要'真正地'听'"——"'不要'打断'"——"'不要'评判'"——"'只是'理解'"。"柯维说'倾听'是'最难的一步'"——"'因为'我们'习惯'了'反驳'"——"'不习惯'理解'"
- **我和你协同**: 柯维说"'我和你协同'是'第四步'"——"'共创第三选择'"。"'你们'一起'头脑风暴'"——"'寻找'一个'新的方案'"——"'这个方案'要'满足双方的'需求'"——"'比'各自的方案'都'好'"。"'协同'需要'创造力'"——"'需要'跳出'原有的'思维框架'"——"'需要'相信'有'更好的'方案'"。"柯维说'第三选择'是'存在的'"——"'只要'你们'愿意'协同'"——"'就'能'找到'"
- **史蒂芬·柯维的背景**: 史蒂芬·柯维（Stephen Covey, 1932-2012）——美国管理学家、作家——"《七个习惯》的作者"——"《第三选择》是他'晚年'的作品'"——"2011 年出版"。"柯维'是'全球'最'有影响力的'管理学家之一'"——"'他的'书'销量超过'2500 万册'"——"'被'翻译成'38 种语言'"。"《第三选择》是'他'对'冲突解决'的'思考'"——"'他'认为'世界上的'大部分问题'"——"'都可以'通过'第三选择'来'解决'"
