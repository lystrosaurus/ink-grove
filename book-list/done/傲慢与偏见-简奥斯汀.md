# 傲慢与偏见

状态：已读

作者：简·奥斯汀

一句话评价：简·奥斯汀（Jane Austen）"代表作'"——"'伊丽莎白与达西的双向成长（傲慢源于阶级优越/偏见源于自尊受伤）'"。"'柯林斯滑稽求婚讽刺功利婚姻'"。"'威克姆虚伪揭示外表魅力的危险'"。"'班纳特太太焦虑折射女性经济困境'"。"'奥斯汀用温柔讽刺揭示人性与阶级'"。"'英语文学最受欢迎小说'"

核心收获：
- **伊丽莎白的独立精神**: 本书核心是"'伊丽莎白的独立精神'"。"'在'19 世纪'英国'社会'"——"'女性'几乎'只能'通过'婚姻'获得'经济'保障'"。"'但'伊丽莎白'坚持'没有'爱情'就'不'结婚'"。"'她'拒绝'了'柯林斯'的'求婚'"——"'稳定'的'经济'保障'"。"'也'最初'拒绝'了'达西'的'求婚'"——"'巨大'的'财富'和'地位'"。"'这'在'当时'是'极其'前卫'的'态度'"
- **达西的转变与谦卑**: 本书"描述了达西的转变'"。"'达西'从'傲慢'的'贵族'绅士'"——"'到'愿意'放下'身段'"——"'真诚'追求'爱'的'人'"。"'他'的'第一次'求婚'是'居高临下'的'"。"'被'拒绝'后'他'写'了'长信'解释'"——"'然后'默默'帮助'伊丽莎白'的'家人'"。"'真正'的'爱情'让'人'学会'谦卑'"
- **傲慢与偏见的双向性**: 本书"分析了傲慢与偏见的双向性'"。"'达西'的'傲慢'源于'阶级'优越感'"。"'伊丽莎白'的'偏见'源于'自尊'被'伤害'"。"'两'人'都'在'误解'中'成长'"——"'达西'学会'了'放下'阶级'偏见'"——"'伊丽莎白'学会'了'不'凭'第一'印象'判断'人'"
- **威克姆的虚伪**: 本书"刻画了威克姆的形象'"。"'威克姆'是'全书'最'危险'的'反派'"。"'外表'迷人'"——"'谈吐'优雅'"——"'人人'喜欢'"。"'但'内心'自私'"——"'贪婪'"——"'不负责任'"。"'他'和'达西'形成'鲜明'对比'"——"'达西'外表'冷漠'但'内心'善良'"
- **柯林斯的滑稽求婚**: 本书"讽刺了柯林斯的求婚'"。"'柯林斯'是'全书'最'可笑'的'人物'"。"'趋炎附势'的'牧师'"——"'求婚'像'谈'生意'"。"'他'向'伊丽莎白'求婚'时'说'我'选择'你'因为'你'是'我家'教母'的'推荐'"。"'被'拒绝'后'立刻'转向'她'的'闺蜜'夏洛特'"
- **班纳特太太的困境**: 本书"描绘了班纳特太太的困境'"。"'一心'只'想'把'女儿'嫁'出去'的'母亲'"——"'可笑'又'可悲'"。"'但'奥斯汀'没有'把'她'写成'单纯'的'丑角'"。"'她'的'焦虑'是'真实'的'"。"'根据'当时'的'限定'继承权'法律'"——"'班纳特'先生'的'财产'只能'由'男性'继承'"
- **奥斯汀的讽刺艺术**: 本书"体现了奥斯汀的讽刺艺术'"。"'奥斯汀'用'幽默'和'反讽'揭示'社会'现实'"——"'但'不'说教'"。"'她'的'讽刺'是'温柔'的'"——"'她'嘲笑'柯林斯'但'不'恨'他'"。"'她'让'读者'自己'得出'结论'"
- **作者的背景**: 简·奥斯汀（Jane Austen, 1775-1817）——英国小说家——"《傲慢与偏见》的作者'"。"'她'出生'于'英国'乡村'牧师'家庭'"。"'《傲慢'与'偏见》'1813 年'出版'"。"'她'的'其他'著作'"："'《理智'与'情感》'——'《爱玛》'"。"'奥斯汀'的'主题'"："'婚恋'"——"'阶级'"——"'女性'处境'"。"'她'的'文学'地位'"："'英语'文学'中'最'受'欢迎'的'小说'之一'"
