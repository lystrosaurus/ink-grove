# 苏菲的世界

## 基本信息
- **书名**: 苏菲的世界
- **作者**: 乔斯坦 - 贾德
- **出版社**: 作家出版社
- **出版年份**: 2020
- **页数**: 560
- **分类**: 西方哲学/文学经典

## 选书理由
用小说形式讲述西方哲学史的入门经典。14 岁女孩苏菲收到神秘信件，开始学习从苏格拉底到萨特的哲学。"你是谁？世界从哪里来？"这些问题看似简单却直指哲学核心。虽然简化了某些理论，但对初学者极其友好。希望在奇幻叙事中轻松进入哲学世界，培养对终极问题的好奇心。
