# 了凡四训

状态：已读

作者：袁了凡

一句话评价：中国版"命运可自主"的启蒙书，用亲身经历讲透"改命"的底层逻辑——信、愿、行缺一不可。

核心收获：
- "命由我作，福自己求"——命运不是预设轨道，而是可修可改的因果系统
- 立命之学：知命而不认命，改过迁善是改变命运的唯一路径
- 积善之方：善有真假、端曲、阴阳、是非之分，行善需要分辨力
- 谦德之效：谦卑是承载福报的容器，骄者必败
- 改过之法：从事上改、从理上改、从心上改三层递进，心上改是根本
