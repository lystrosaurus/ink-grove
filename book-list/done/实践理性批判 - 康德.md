# 实践理性批判

状态：已读

作者：伊曼努尔·康德

一句话评价：康德道德哲学的核心著作（1788 年出版），是"三大批判"中的第二部（第一部《纯粹理性批判》1781 年讲认识论，第三部《判断力批判》1790 年讲美学）。康德在本书中提出了"绝对命令"（categorical imperative）的概念——"只按照你同时希望它成为普遍法则的准则去行动"，论证了道德法则的普遍性和人的自由意志。"人是目的而非手段"——这是现代人权观念的哲学基础。

核心收获：
- **实践理性 vs 纯粹理性**: 康德的"三大批判"分别对应人类理性的三个领域。《纯粹理性批判》回答"我能知道什么？"（认识论）——人类的认识能力有边界，我们只能认识"现象"，不能认识"物自体"。《实践理性批判》回答"我应该做什么？"（伦理学）——虽然我们不能"认识"自由、上帝、灵魂不朽，但在"道德实践"中我们必须"预设"它们存在。《判断力批判》回答"我可以希望什么？"（美学和目的论）——通过审美和自然目的来沟通"认识"和"道德"两个领域
- **绝对命令（Categorical Imperative）**: 康德道德哲学的核心概念。"命令"（imperative）是"你应该做什么"的指令。"假言命令"（hypothetical imperative）是"有条件的"——"如果你想成功，你就应该努力"（前提是"你想成功"）。"绝对命令"是"无条件的"——"你应该诚实"——不管你想不想、有没有好处，你都应该诚实。绝对命令的第一公式是"普遍法则公式"："只按照你同时希望它成为普遍法则的准则去行动"——换句话说，在你做一件事之前，问自己："如果所有人都这样做，世界还能运转吗？"如果不能，你就不应该做
- **普遍法则测试**: 康德用"普遍法则测试"来判断一个行为是否道德。比如"撒谎"——如果"所有人都撒谎"，那么"语言"就失去了交流的功能（因为没人相信任何人说的话），"撒谎"本身就变得不可能了。所以"撒谎"不能成为普遍法则——它是不道德的。比如"不帮助他人"——如果"所有人都不帮助他人"，那么当你需要帮助时也没人会帮你——这个法则"自我矛盾"。所以"帮助他人"是道德义务
- **人是目的而非手段**: 绝对命令的第二公式——"你要如此行动，即无论是你人格中的人性，还是其他任何一个人格中的人性，你在任何时候都同时当作目的，绝不仅仅当作手段来使用。"意思是：你不能把别人当成"实现你自己目标的工具"。比如你不能"欺骗"别人——因为你在利用他的"无知"来达到你的目的。你不能"奴役"别人——因为你在把他当成"工具"而不是"人"。这个原则是现代"人权"观念的哲学基础——每个人都有"内在的价值"（尊严），不能被"利用"或"物化"
- **自由意志**: 康德说道德的前提是"自由"——如果人的行为是"被决定的"（由基因、环境、本能决定），那么"道德责任"就没有意义了（你不能责怪一个"不得不这样做"的人）。所以康德说在"道德实践"中我们必须"预设"人有"自由意志"——虽然我们在"认识论"层面不能"证明"自由意志存在，但在"实践"层面我们必须"假设"它存在。这是一种"实践的必然性"
- **道德法则的普遍性**: 康德反对"道德相对主义"——他认为道德法则不是"文化的产物"或"社会的约定"，是"理性的产物"。理性是"普遍的"（所有理性存在者都有同样的理性），所以道德法则也是"普遍的"——不管你在什么文化、什么时代、什么社会，道德法则都是一样的。这和功利主义（"最大多数人的最大幸福"）形成鲜明对比——康德说道德不是"计算后果"，是"遵守义务"
- **义务 vs 倾向**: 康德区分了"出于义务"（from duty）和"符合义务"（in accordance with duty）。一个商人"童叟无欺"——如果他是"为了生意好"（符合义务），这不是道德行为；如果他是"因为诚实是义务"（出于义务），这才是道德行为。康德说真正的道德行为必须"出于义务"——不是因为你"喜欢"做善事，是因为你"知道这是对的"
- **至善（Summum Bonum）**: 康德说道德的最终目标是"至善"——"德性"和"幸福"的统一。但现实中"有德之人不一定幸福，幸福之人不一定有德"。为了解决这个矛盾，康德说我们必须"预设"两个东西：1）**灵魂不朽**——因为"完美的德性"需要无限的时间来修炼；2）**上帝存在**——因为只有上帝能保证"德性和幸福最终统一"。这不是"证明"上帝存在——是"道德实践的需要"
- **康德的背景**: 伊曼努尔·康德（1724-1804）是德国哲学家，出生于普鲁士的哥尼斯堡（今俄罗斯加里宁格勒）。他一辈子没有离开过哥尼斯堡，但他的思想影响了整个西方哲学。《实践理性批判》是他 1788 年的作品（64 岁），是他"批判哲学"体系的重要一环。康德的道德哲学对后世影响巨大——罗尔斯的"正义论"、哈贝马斯的"商谈伦理"、人权宣言中的"人人生而平等"——都直接或间接源自康德的思想
