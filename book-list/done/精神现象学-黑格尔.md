# 精神现象学

状态：已读

作者：黑格尔

一句话评价：西方哲学史上最具野心的著作之一，描述了意识从感性确定性到绝对知识的演进历程，"主奴辩证法"影响深远，但文本艰涩程度令人望而生畏。

核心收获：
- 理解了"辩证法"的基本模式：正题→反题→合题，每一个阶段都被下一个阶段"扬弃"（既否定又保留）
- "主奴辩证法"揭示了自我意识必须通过他者才能确立，权力关系中存在深刻的悖论
- 黑格尔的历史观：人类精神（Geist）通过矛盾和冲突不断自我认识、自我实现
