# 作为意志和表象的世界

状态：已读

作者：叔本华

一句话评价：叔本华的哲学体系总纲——世界有两面：表象（我们看到的）和意志（背后的盲目驱动力）。人生痛苦就是因为意志永不满足。

核心收获：
- 世界是表象：我们感知到的一切都是经过大脑加工的"表象"，不是事物本身
- 世界是意志：表象背后是盲目的、永不满足的"意志"——这是驱动一切生命活动的根本力量
- 意志永不满足→欲望不断产生→人生在痛苦（欲求不满）和无聊（欲望满足）之间摆动
- 摆脱痛苦有三条路：审美（暂时遗忘意志）、道德（同情他人超越自我意志）、禁欲（否定意志本身）
- 艺术的最高形式是音乐——音乐不模仿表象，直接表达意志本身
