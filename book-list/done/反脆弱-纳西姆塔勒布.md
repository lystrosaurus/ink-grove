# 反脆弱

状态：已读

作者：纳西姆·尼古拉斯·塔勒布

一句话评价：塔勒布（Nassim Nicholas Taleb）"不确定性三部曲之二'"——"'风会熄灭蜡烛却能使火越烧越旺比喻精妙'"。"'提出超越强韧的反脆弱概念'"。"'杠铃策略、冗余价值、皮肤在游戏中等洞见重塑风险认知'"。"'不确定性时代的生存哲学必读'"

核心收获：
- **反脆弱定义**: 本书核心是"'反脆弱定义'"。"'有些'东西'会'在'压力'下'变得'更强'"——"'肌肉'锻炼'"——"'免疫'系统'"——"'创业'公司'"。"'脆弱'是'害怕'波动'"——"'强韧'是'抵抗'波动'"——"'反脆弱'是'喜欢'波动'"
- **三元结构**: 本书"提出了三元结构'"。"'世界'上'的'事物'分'三种'"：1）**脆弱的**——"'怕'波动'"；2）**强韧的**——"'扛'波动'"；3）**反脆弱的**——"'爱'波动'"。"'大多数'人'只'看到'前'两种'"——"'忽略'了'第三'种'"
- **杠铃策略**: 本书"介绍了杠铃策略'"。"'极端'保守'加'极端'冒险'"——"'避开'中庸'"。"'比如'90% 资金'买'国债'"——"'10% 投'高风险'高'回报'的'项目'"。"'要么'稳赚'"——"'要么'大赚'"——"'不'会'大亏'"
- **过度干预的危害**: 本书"讨论了过度干预的危害'"。"'政府'"——"'医生'"——"'父母'最'常见'的'错误'"："'做'太'多'"。"'身体'的'自愈'能力'"——"'市场'的'自我'调节'"——"'经常'被'好心'的'干预'破坏'"
- **冗余的价值**: 本书"强调了冗余的价值'"。"'现代'人'追求'效率'最大化'"——"'但'冗余'是'反脆弱'的'核心'"。"'自然界'从来'不'追求'效率'"——"'你'有'两个'肾'"——"'两片'肺'"——"'这'就是'冗余'"
- **试错进化**: 本书"讨论了试错进化'"。"'进化'不'是'靠'计划'"——"'是'靠'试错'"。"'创业'"——"'投资'"——"'人生'决策'都'应该'用'小'成本'试错'"——"'而'不'是'All-in'"
- **皮肤在游戏中**: 本书"提出了皮肤在游戏的概念'"。"'做'决策'的'人'必须'承担'后果'"。"'银行家'拿'别人'的'钱'冒险'"——"'政客'发动'战争'自己'不'上'战场'"——"'这'是'现代'社会'最'危险'的'脆弱性'来源'"
- **作者的背景**: 纳西姆·尼古拉斯·塔勒布（Nassim Nicholas Taleb, 1960-）——黎巴嫩裔美国作家、数学家、交易员——"《反脆弱》的作者'"。"'他'是'不确定性'研究'专家'"。"'他'的'不确定性'三部曲'"："'《随机'漫步'的'傻瓜》'（2001）'"——"'《黑天鹅》'（2007）'"——"'《反脆弱》'（2012）'"。"'《反脆弱》'2012 年'出版'"。"'他'的'核心'思想'"："'世界'是'随机'的'"——"'人'类'低估'不确定性'"——"'反脆弱'是'应对'不确定性'的'方式'"
