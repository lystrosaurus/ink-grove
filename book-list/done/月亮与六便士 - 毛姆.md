# 月亮与六便士

状态：已读

作者：威廉·萨默塞特·毛姆

一句话评价：毛姆（W. Somerset Maugham）的代表作（1919 年出版）——以"法国画家高更"为原型——讲述了"中年证券经纪人斯特里克兰德'抛弃一切'去追求艺术理想"的故事。"满地都是六便士——他却抬头看见了月亮。""六便士"代表"世俗生活"——"月亮"代表"理想"。斯特里克兰德说"我必须画画——就像溺水的人必须挣扎。"这是"艺术追求最决绝的表达"。

核心收获：
- **斯特里克兰德的出走**: 斯特里克兰德是"一个'看似普通'的中年证券经纪人"——"有稳定的工作"——"有妻子和孩子"——"过着'体面的生活'"。但"有一天"——他"突然离开了"——"抛妻弃子"——"去了巴黎"——"去画画"。所有人都不理解——"你是不是有了外遇？"——他说"不是"——"我只是'必须画画'"。这种"决绝"让"所有人都觉得他疯了"——但"他知道'自己'在做什么"
- **艺术追求**: 斯特里克兰德对艺术的追求是"纯粹的"——不是为了"名利"——不是为了"被认可"——是"一种'内在的冲动'"。他说"我必须画画——就像溺水的人必须挣扎。"这种"创作冲动"是"无法抗拒的"——"即使'生活再苦'"——"即使'没有人欣赏'"——"即使'自己'画得不好'"——"他'还是要画'"。毛姆说"天才的'代价'是'巨大的'"——"斯特里克兰德'付出了'一切"
- **世俗与理想的冲突**: 全书的核心主题是"世俗与理想的冲突"。"六便士"代表"世俗"——"安稳的工作"——"体面的生活"——"家庭的责任"。"月亮"代表"理想"——"艺术的追求"——"精神的自由"——"灵魂的表达"。大多数人选择了"六便士"——因为"它'看得见摸得着'"。斯特里克兰德选择了"月亮"——因为"它'召唤'了他"。毛姆没有说"哪个选择'更好'"——他只是"呈现了'两种人生'"
- **塔希提岛**: 斯特里克兰德"最终去了塔希提岛"——南太平洋上的"一个原始小岛"。在塔希提——他"找到了'创作的自由'"——"没有人'评判他'"——"没有'世俗的规则'"——"只有'自然'和'色彩'"。他在塔希提"创作了'最伟大的作品'"——但也"染上了麻风病"——"失明"——"死在了小岛上"。塔希提代表了"艺术的'乌托邦'"——"纯粹的、不受世俗污染的创作环境"
- **天才与疯狂**: 毛姆探讨了"天才与疯狂的关系"。斯特里克兰德是"天才"——但"他的行为'如同疯子'"——"他不顾'家庭'"——"不顾'健康'"——"不顾'别人的看法'"。毛姆说"天才'往往是'疯狂的'"——因为"天才'看到的是'别人看不到'的东西'"——"他们的'思维方式'和'常人不同'"。但毛姆也质疑——"天才的'疯狂'是否'值得'？"——"斯特里克兰德的'家人'承受了'巨大的痛苦'"
- **人性的矛盾**: 毛姆最擅长的是"刻画人性的矛盾"。斯特里克兰德"对艺术'极度纯粹'"——但"对人'极度冷漠'"。他"能画出'最美的画'"——但"对'爱他的人'毫不留情"。毛姆没有"美化他"——也没有"丑化他"——他"如实呈现了'一个复杂的人'"——"伟大的艺术家"——同时是"糟糕的丈夫和父亲"
- **毛姆的背景**: 威廉·萨默塞特·毛姆（1874-1965）——英国作家——代表作《月亮与六便士》《人性的枷锁》《刀锋》。《月亮与六便士》（The Moon and Sixpence）以"保罗·高更"（Paul Gauguin）为原型——高更是"法国后印象派画家"——也是"中年'抛弃一切'去塔希提岛画画"——和斯特里克兰德的故事"惊人地相似"。毛姆说"这部小说'不是'高更的传记'"——是"以高更为'灵感'创作的'虚构故事'"
