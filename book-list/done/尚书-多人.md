# 尚书

状态：已读

作者：多人

一句话评价：中国最古老的政治文献汇编——记载了从尧舜禹到春秋时期的重要政治文献（训诰、誓命、典谟），是儒家"五经"之一。"尚"即"上"——"上古之书"。尚书中的名篇（《尧典》《舜典》《大禹谟》《汤誓》《牧誓》《金縢》）记载了上古圣王的治国理念和政治智慧，文字古奥艰涩——是中国最难读的经典之一，需要大量注释才能理解。

核心收获：
- **尚书的性质**: 《尚书》不是"历史书"——它是"政治文献汇编"。它收录的是"诏书""讲话""誓词""训诫"——类似于今天的"政府工作报告"和"领导人讲话"。它不是"客观记录"，是"官方文献"——每一篇都有"政治目的"（宣扬君主的德行、教训臣民、记录重大决策）。但正因为它是"官方文献"，它保存了中国最早的"政治话语"——中国人从几千年前就开始讨论"什么是好的统治"
- **尧典与舜典**: 《尧典》是《尚书》开篇——记载了尧帝的事迹和治国理念。尧的核心品质是"钦明文思安安"——敬畏、明达、文雅、深思、宽厚。他做的最重要的事是"禅让"——不把帝位传给自己的儿子（丹朱），而是传给了"德才兼备"的舜。《舜典》记载了舜帝的治国——他设立了各种官职（管农业的、管司法的、管教育的、管音乐的），这是中国最早的"政府机构设置"记录。舜也"禅让"给了禹
- **大禹谟**: 记载了大禹治水的功绩和他与舜帝的对话。大禹的核心品质是"克勤于邦，克俭于家"——对国家勤勉，对自己节俭。他治水 13 年，"三过家门而不入"——这是中国文化中"公而忘私"的典范。大禹谟中有一段著名的对话——舜问禹"什么是好的统治？"禹回答："安汝止"（安于你的位置，做好你的本分）和"惟几惟康"（时刻警惕，保持安康）。这是中国最早的"治国箴言"
- **汤誓**: 记载了商汤伐夏桀时的"战前动员令"。商汤说夏桀"有罪"——"夏王率遏众力，率割夏邑"（夏桀耗尽民力、残害百姓），所以他要"替天行道"。这是中国最早的"革命合法性"论述——推翻暴君是"正当的"。这种"天命"观念（天命不属于某个家族，属于"有德者"）是中国政治思想的核心
- **牧誓**: 记载了周武王伐商纣时的"战前动员令"。周武王列举了商纣的罪状——"惟妇言是用"（只听女人的话）、"昏弃厥肆祀弗答"（不祭祀祖先）、"昏弃厥遗王父母弟不迪"（不任用兄弟）——然后号召将士们"奋力作战"。牧誓是《尚书》中"最有气势"的篇章之一——短促有力、节奏感强
- **金縢**: 记载了周公的故事——周武王病重时，周公向祖先祈祷"让我代替武王去死"，然后把祈祷文封存在"金縢"（金属封套）中。后来武王还是去世了，周公辅佐年幼的成王。有人诬陷周公"想篡位"，成王怀疑了周公。后来成王打开"金縢"看到了周公的祈祷文，感动得哭了，迎回了周公。这个故事是中国"忠臣"典范
- **今文尚书 vs 古文尚书**: 《尚书》的版本问题是中国学术史上最复杂的"公案"之一。秦始皇"焚书坑儒"后《尚书》失传。汉初有一个叫伏生的老儒凭记忆背诵了 28 篇——这就是"今文尚书"（用汉代通行的隶书书写）。后来在孔子旧宅的墙壁中发现了用"古文"（先秦篆书）写的《尚书》——比今文多了 16 篇——这就是"古文尚书"。但"古文尚书"后来失传了。东晋时又出现了一部"古文尚书"——但到了清代，学者阎若璩考证这部"古文尚书"是"伪造的"（伪古文尚书）。今天通行的《尚书》是"今文 28 篇 + 伪古文 25 篇"的混合体
- **文字的难度**: 《尚书》是中国"最难读的经典"——比《论语》《孟子》难读得多。原因：1）时代太早——很多词汇和语法在后来消失了；2）经过多次传抄——错字、漏字、篡改很多；3）"诘屈聱牙"——韩愈说《尚书》"周诰殷盘，佶屈聱牙"（读起来拗口难懂）。读《尚书》需要借助注释——历代学者（从汉代的孔安国到清代的段玉裁）都做过注疏。现代人读《尚书》最好选择有"翻译和注释"的版本
- **尚书的地位**: 《尚书》是儒家"五经"之一（《诗》《书》《礼》《易》《春秋》），是中国古代"科举考试"的必读经典。它对中国政治思想的影响深远——"天命"观念、"禅让"理想、"民本"思想、"德治"理念——都源自《尚书》。从汉武帝到清朝末年，每一个中国皇帝都要学习《尚书》——因为它是"治国理政的教科书"
