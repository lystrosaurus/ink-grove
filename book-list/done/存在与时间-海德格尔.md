# 存在与时间

状态：已读

作者：海德格尔

一句话评价：20 世纪最重要的哲学著作之一，海德格尔对"存在"意义的追问，开创了存在主义和现象学的新传统。

核心收获：
- 理解了"存在"与"存在者"的区别——哲学史上一直在研究"存在者"（具体的事物），却忘了问"存在本身"是什么
- "此在"（Dasein）是海德格尔的核心概念——人作为一种特殊的存在，是唯一会追问自身存在意义的存在者
- "向死而生"：意识到死亡是不可避免的，才能真实地活着
