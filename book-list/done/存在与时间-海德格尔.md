# 存在与时间

状态：已读

作者：马丁·海德格尔

一句话评价：20 世纪最重要的哲学著作之一（1927 年出版），海德格尔对"存在"（Sein）意义的追问——哲学史上一直在研究"存在者"（具体的事物），却忘了问"存在本身"是什么。"此在"（Dasein）是海德格尔的核心概念——人作为一种特殊的存在，是唯一会追问自身存在意义的存在者。"向死而生"——意识到死亡是不可避免的，才能真实地活着。这本书开创了存在主义和现象学的新传统，影响了萨特、加缪、梅洛 - 庞蒂、德里达等无数哲学家。

核心收获：
- **存在的问题**: 海德格尔开篇就问了一个看似简单但从未被认真回答的问题——"存在的意义是什么？"哲学家们（从柏拉图到康德到黑格尔）一直在研究"存在者"（beings / entities）——桌子是什么？人是什么？上帝是什么？但没有人问过"存在本身"（Being / Sein）是什么。"存在"不是"一个东西"，是"让东西成为东西的那个东西"。海德格尔说"存在"被遗忘了——从柏拉图开始，哲学就走错了路
- **存在者 vs 存在**: 这是海德格尔最核心的区分。"存在者"（Seiendes）是具体的事物——桌子、椅子、人、石头、上帝。"存在"（Sein）是"让这些东西存在的那个过程"。比如一张桌子——你可以研究它的材质、形状、用途（这些都是"存在者"层面的问题），但你不会问"这张桌子为什么存在而不是不存在"（这是"存在"层面的问题）。海德格尔说哲学应该问"存在"的问题，而不是"存在者"的问题
- **此在（Dasein）**: 海德格尔用"此在"（Dasein，字面意思是"在这里存在"）来指代"人"。人和其他存在者不同——石头不会追问自己的存在，但人会。"此在"的特殊性在于它是"唯一会追问自身存在意义的存在者"。海德格尔说研究"存在"必须从"此在"入手——因为只有"此在"能理解"存在"
- **在世存在（In-der-Welt-sein）**: 海德格尔说"此在"不是"一个孤立的主体面对一个外在的世界"——这是笛卡尔以来的错误。"此在"是"在世界之中存在"（Being-in-the-world）——我们不是"先存在然后进入世界"，我们"一出生就在世界中"。世界不是"外在的客体"，是我们"活在其中"的环境。我们和世界是"一体的"
- **常人（Das Man）**: 海德格尔描述了"此在"的日常状态——我们大部分时间活在"常人"（Das Man，"人们"的意思）的状态中。"人们都这么做""人们都这么想"——我们按照"社会规范""大众期待"生活，不追问"这真的是我想做的吗"。这是一种"非本真"（inauthentic）的生存状态——不是"你自己在活"，是"别人在替你活"
- **向死而生（Being-towards-death）**: 海德格尔说"此在"只有面对"死亡"时才能从"常人"状态中觉醒。死亡是"最本己的可能性"——没有人能替你死，也没有人能逃避死。当你真正意识到"我终将一死"时，你会从"日常的麻木"中惊醒——"我为什么这样活？这真的是我想要的生活吗？"这种"向死而生"的态度是"本真"（authentic）生存的开始——不再"随波逐流"，而是"为自己的存在负责"
- **时间性**: 海德格尔说"存在"的理解必须通过"时间"。"此在"是"时间性的存在"——我们有"过去"（已经被抛入世界）、"现在"（正在活着）、"未来"（面向死亡）。"本真的时间"不是"钟表的时间"（一分钟一分钟地流逝），是"意义的时间"——你的过去（你的经历）塑造了你的现在，你的未来（你的目标）指引着你的现在。"向死而生"就是"把未来（死亡）纳入现在"——让死亡提醒你"怎样活才是本真的"
- **海德格尔的语言**: 《存在与时间》以"极其难读"著称。海德格尔创造了大量新词（此在、在世存在、常人、向死而生、沉沦、操心）——不是因为他"喜欢造词"，是因为"现有的语言无法表达他的思想"。他用德语的词源学（Sein、Seiendes、Dasein、Sorge）来构建哲学概念——这些词在德语中有特殊的"存在感"，翻译成其他语言后力量大打折扣
- **影响与争议**: 《存在与时间》影响了 20 世纪的几乎所有重要哲学流派——存在主义（萨特、加缪）、解释学（伽达默尔）、解构主义（德里达）、后结构主义（福柯）。但海德格尔本人有严重的政治污点——1933 年他加入纳粹党，担任弗莱堡大学校长，发表了支持希特勒的演讲。战后他拒绝道歉。这成为了 20 世纪哲学界最大的争议之一：一个伟大的哲学家怎么能是一个纳粹？他的哲学和他的政治有关系吗？这个问题至今没有令人满意的答案
- **海德格尔的背景**: 马丁·海德格尔（1889-1976）是德国哲学家，胡塞尔的学生。《存在与时间》是他 1927 年出版的代表作（当时 38 岁），但只完成了计划的一部分（第二部分从未发表）。他后来的思想发生了"转向"（Kehre）——从"此在的分析"转向"存在本身的历史"，关注语言、技术、艺术。他被认为是"20 世纪最有原创性的哲学家之一"
