# 写给大家看的设计书

状态：已读

作者：Robin-Williams

一句话评价：非设计师的设计入门圣经——四个原则（对比、重复、对齐、亲近性）就能让任何人的排版从"业余"变"专业"。

核心收获：
- 四大设计原则：对比（Contrast）、重复（Repetition）、对齐（Alignment）、亲近性（Proximity）——CRAP原则
- 对比：不要让元素"相似但不相同"。要么完全一样，要么明显不同
- 重复：整个文档统一使用相同的视觉元素（字体、颜色、布局），形成一致感
- 对齐：任何元素都不能随意放置，必须和页面上其他元素有视觉联系
- 亲近性：相关的东西放一起，不相关的分开。亲近性就是最好的分类
