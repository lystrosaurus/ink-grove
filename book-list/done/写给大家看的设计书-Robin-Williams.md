# 写给大家看的设计书

状态：已读

作者：Robin Williams

一句话评价：非设计师的设计入门圣经——四个原则（对比、重复、对齐、亲近性）就能让任何人的排版从"业余"变"专业"。薄薄一百多页，却改变了全球数百万人做 PPT、写文档、设计海报的方式。

核心收获：
- **CRAP 四大原则**: 对比（Contrast）、重复（Repetition）、对齐（Alignment）、亲近性（Proximity）。这是全书的核心，也是设计领域最简洁有效的入门框架。记住这四个词，你的排版就能超越 80% 的人
- **对比**: 不要让元素"相似但不相同"——这是最业余的做法。要么完全一样，要么明显不同。大标题和小正文要大到"一眼就能看出区别"，不要用"稍大一点"这种暧昧的处理
- **重复**: 整个文档统一使用相同的视觉元素——字体、颜色、布局、图标风格。重复创造一致感，一致感创造专业感。为什么大公司都有品牌手册？因为重复是品牌的根基
- **对齐**: 任何元素都不能随意放置，必须和页面上其他元素有视觉联系。左对齐最安全，居中对齐最常用但也最容易乱，右对齐最有设计感但最难驾驭。新手只用左对齐就够了
- **亲近性**: 相关的东西放一起，不相关的分开。亲近性就是最好的分类——标题和它对应的正文要靠近，图片和它的说明要靠近，不同区块之间要有足够的留白
- **留白的力量**: 设计新手最怕"空"，总想把每个角落都填满。但留白不是"没东西"，留白是设计的一部分。好的留白让读者知道该看哪里、不该看哪里
- **字体选择的常识**: 一份文档最多用两种字体——一种衬线体（正文），一种无衬线体（标题）。超过三种字体就乱了。Comic Sans 和 Papyrus 能不用就不用
- **设计的本质**: Robin Williams 反复强调：设计不是艺术，是沟通。好设计的目标不是"好看"，是"让信息更容易被理解"。这个认知转变是全书最重要的前提
