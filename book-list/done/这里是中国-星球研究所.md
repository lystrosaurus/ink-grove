# 这里是中国

状态：已读

作者：星球研究所

一句话评价：星球研究所的"地理科普佳作'"——"'用极致摄影和科普文字展现中国大地的地理之美'"。"'从青藏高原到沿海平原，从冰川到沙漠'"——"'每一页都是视觉盛宴'"。"'地貌决定文明——中国东西南北的地理差异如何塑造了不同的文化'"

核心收获：
- **地理视角**: 本书"提供了地理视角理解中国'"。"'什么'是'地理'视角'"——"'从'地形'——'气候'——'河流'看'中国'"。"'传统'视角'"——"'历史'——'文化'"。"'地理'视角'"——"'自然'环境'如何'塑造'文明'"。"'比如'"：1）**青藏高原**——"'隆起'影响'整个'亚洲'气候'"；2）**黄河**——"'塑造'了'北方'农业'"；3）**长江**——"'连接'了'南方'经济'"
- **中国的多样性**: 本书"展现了中国的多样性'"。"'中国'的'地理'差异'很'大'"：1）**东西**——"'西部'高原'——'东部'平原'"；2）**南北**——"'北方'干旱'——'南方'湿润'"；3）**地形**——"'山地'——'高原'——'平原'——'盆地'"。"'这种'多样性'"——"'塑造'了'不同'的'文化'"。"'比如'"：1）**北方**——"'小麦'——'面食'"；2）**南方**——"'水稻'——'米饭'"；3）**西北**——'**牧业'——'肉食'"
- **青藏高原**: 本书"详细介绍了青藏高原'"。"'青藏高原'是'什么'"——"'世界'屋脊'"——"'平均'海拔'4000 米'"。"'如何'形成'"——"'印度'板块'撞'欧亚'板块'"——"'隆起'"。"'影响'"：1）**气候**——"'阻挡'印度洋'水汽'"——"'西北'干旱'"；2）**河流**——"'长江'——'黄河'——'澜沧江'发源'"；3）**文明**——"'藏族'文化'"。"'青藏高原'是'中国'的'水塔'"
- **黄河与长江**: 本书"对比了黄河与长江'"。"'黄河'"：1）**特点**——"'含沙量'大'"——"'地上河'"；2）**文明**——"'中华'文明'发源地'"；3）**问题**——"'改道'——'泛滥'"。"'长江'"：1）**特点**——"'水量'大'"——"'通航'"；2）**经济**——"'南方'经济'大动脉'"；3）**工程**——"'三峡'大坝'"。"'两条'河'塑造'了'中国'"
- **科普与美学**: 本书"结合了科普与美学'"。"'科普'"——"'解释'地理'现象'"——"'用'通俗'语言'"。"'美学'"——"'精美'摄影'"——"'震撼'画面'"。"'比如'"：1）**冰川**——"'照片'展示'冰川'的'壮观'"；2）**沙漠**——"'照片'展示'沙漠'的'纹理'"；3）**城市**——"'照片'展示'城市'的'繁华'"。"'这种'结合'"——"'让'科普'不'枯燥'"——"'让'读者'既'学习'又'享受'"
- **人地关系**: 本书"探讨了人地关系'"。"'人'如何'适应'环境'"：1）**农业**——"'根据'气候'种'不同'作物'"；2）**建筑**——"'北方'四合院'"——"'南方'吊脚楼'"；3）**交通**——"'北方'马车'"——"'南方'船'"。"'人'如何'改造'环境'"：1）**梯田**——"'山坡'变'农田'"；2）**运河**——"'连接'水系'"；3）**城市**——"'改造'自然'"。"'人地'关系'"——"'和谐'才'能'可持续'"
- **星球研究所的背景**: 星球研究所——中国地理科普团队——"《这里是中国》的作者'"。"'他们'用'新媒体'方式'做'科普'"——"'公众号'——'视频'"。"'他们'的'特点'"：1）**高质量图片**——"'专业'摄影'"；2）**通俗文字**——"'不'用'术语'"；3）**宏大叙事**——"'从'大'视角'看'中国'"。"'《这里是中国》'2019 年'出版'"——"'畅销'百万册'"。"'他们'的'目的'"——"'让'更多'人'了解'中国'地理'"——"'热爱'中国'"
