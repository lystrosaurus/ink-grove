# 被讨厌的勇气

状态：已读

作者：岸见一郎

一句话评价：岸见一郎的"阿德勒心理学通俗解读'"——"'自由就是被讨厌的勇气'"——"'观点鲜明'"。"'课题分离'方法实用'"——"'自我接纳与人际关系改善的指南'"。"'世界很简单，人生也一样，不是世界复杂，而是你把世界复杂化了'"

核心收获：
- **自由就是被讨厌**: 本书提出了"'自由就是被别人讨厌'"的核心观点'"。"'为什么'"——"'因为'你'不'能'控制'别人'的'想法'"——"'别人'可能'讨厌'你'"——"'这'是'别人'的'课题'"。"'你'能'控制'的'"——"'只是'自己'的'行为'"。"'如果'你'在意'别人'的'看法'"——"'你'就'不'自由'"。"'所以'自由'"——"'就是'不'在意'别人'的'看法'"——"'即使'别人'讨厌'你'"——"'你'也'坚持'自己'"。"'这'不'是'鼓励'你'去'得罪'人'"——"'是'说'你'不'要'为'别人'的'评价'而'活'"
- **课题分离**: 本书介绍了"'课题分离'"的方法'"。"'什么'是'课题分离'"——"'区分'什么'是'你'的'课题'"——"'什么'是'别人'的'课题'"。"'你'的'课题'"：1）**你的行为**——"'你'做'什么'"；2）**你的想法**——"'你'想'什么'"；3）**你的情绪**——"'你'感受'什么'"。"'别人'的'课题'"：1）**别人的看法**——"'别人'怎么'看'你'"；2）**别人的情绪**——"'别人'的'感受'"；3）**别人的选择**——"'别人'做'什么'决定'"。"'人际'烦恼'"——"'都'是'因为'干涉'别人'的'课题'"——"'或'让'别人'干涉'你'的'课题'"。"'解决'方法'"——"'做好'自己'的'课题'"——"'不'干涉'别人'的'课题'"
- **目的论**: 本书"介绍了阿德勒的目的论'"。"'弗洛伊德'是'因果论'"——"'过去'决定'现在'"——"'童年'创伤'决定'成年'行为'"。"'阿德勒'是'目的论'"——"'现在'被'未来'目标'牵引'"——"'人'选择'行为'来'达成'目的'"。"'比如'社交恐惧'"——"'因果论'说'因为'童年'被'嘲笑'"——"'目的论'说'因为'不想'社交'的'目的'"——"'所以'制造'恐惧'情绪'"。"'目的论'的'意义'"——"'人'可以'改变'"——"'只要'改变'目的'"——"'行为'就'改变'"
- **共同体感觉**: 本书"探讨了共同体感觉'"。"'共同体'是'什么'"——"'家庭'——'学校'——'公司'——'社会'"——"'任何'人'的'集合'"。"'共同体'感觉'"——"'对'共同体'的'归属感'"——"'觉得'自己'是'其中'一员'"。"'如何'获得'共同体'感觉'"：1）**自我接纳**——"'接纳'自己'"；2）**他者信赖**——"'信任'别人'"；3）**他者贡献**——"'为'别人'做'贡献'"。"'幸福'"——"'就是'共同体'感觉'"——"'觉得'自己'有'价值'"——"'觉得'自己'被'需要'"
- **横向关系**: 本书"提倡横向关系'"。"'纵向'关系'"——"'有'上下'——'优劣'——'尊卑'"——"'父母'对'孩子'"——"'上司'对'下属'"。"'横向'关系'"——"'平等'的'关系'"——"'不'是'上下'"——"'是'伙伴'"。"'如何'建立'横向'关系'"：1）**不批评**——"'批评'是'纵向'关系'"——"'批评'意味着'你'比'对方'高'"；2）**不表扬**——"'表扬'也'是'纵向'关系'"——"'表扬'意味着'你'在'评价'对方'"；3）**鼓励**——"'鼓励'是'横向'关系'"——"'鼓励'是'感谢'——'尊重'"。"'横向'关系'的'核心'"——"'尊重'"——"'把'对方'当'平等'的'人'"
- **自我接纳**: 本书"强调了自我接纳的重要性'"。"'自我'接纳'"——"'接纳'自己'的'全部'"——"'包括'优点'和'缺点'"。"'不'是'自我'肯定'"——"'自我'肯定'是'告诉'自己'"我'很'棒'"——"'这'是'欺骗'"。"'自我'接纳'"——"'承认'自己'有'缺点'"——"'但'依然'接纳'"。"'如何'自我'接纳'"：1）**区分能改变和不能改变的**——"'能'改变'的'努力'改变'"——"'不能'改变'的'接纳'"；2）**关注当下**——"'不'纠结'过去'"——"'不'担忧'未来'"；3）**接纳不完美的自己**——"'完美'不'可能'"——"'不完美'也'可以'"
- **岸见一郎的背景**: 岸见一郎（1956-）——日本哲学家、阿德勒心理学研究者——"《被讨厌的勇气》的作者'"。"他'在'京都大学'学习'哲学'"——"'后来'研究'阿德勒'心理学'"。"他'是'日本'阿德勒'心理学'会'的'顾问'"。"《被讨厌的勇气》'2013 年'出版'"——"'采用'对话体'"——"'青年'与'哲学家'的'对话'"。"'这种'形式'让'复杂'的'理论'变得'易懂'"。"这本书'畅销'日本'和'中国'"——"'销量'超过'百万册'"。"岸见'的'其他'著作'"——"'《幸福的勇气》'——'《阿德勒心理学讲义》'"
