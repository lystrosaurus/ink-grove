# 金瓶梅

状态：已读

作者：兰陵笑笑生

一句话评价：兰陵笑笑生的"明代世情小说巅峰之作'"——"'借《水浒》中西门庆的故事展开'"。"'以冷峻笔法描绘市井社会的欲望图景'"——"'人物刻画之细腻堪称中国古典小说之最'"。"'第一部文人独立创作的白话长篇小说'"——"'西门庆一家的兴衰是明代中后期商业社会的缩影'"

核心收获：
- **文学史地位**: 《金瓶梅》有"重要的文学史地位'"。"'什么'地位'"：1）**第一部文人独立创作**——"'不'像'三国'——'水浒'是'集体'创作'"；2）**世情小说**——"'描写'市井'生活'"——"'不'是'英雄'——'神魔'"；3）**白话长篇小说**——"'用'白话'写作'"。"'《金瓶梅》'的'出现'"——"'标志'中国'小说'成熟'"。"'它'影响'了'后世'"——"'《红楼梦》'受'它'影响'"
- **西门庆的兴衰**: 本书"讲述了西门庆一家的兴衰'"。"'西门庆'是'谁'"——"'一个'商人'"——"'也'是'官员'"。"'他'如何'发家'"：1）**经商**——"'开'店铺'"——"'做'生意'"；2）**官商勾结**——"'贿赂'官员'"——"'获得'特权'"；3）**婚姻**——"'娶'富婆'"——"'获得'财产'"。"'他'如何'衰亡'"：1）**纵欲**——"'过度'享乐'"；2）**报应**——"'因果'报应'"。"'西门庆'的'兴衰'"——"'是'明代'商业'社会'的'缩影'"
- **三位女性形象**: 本书"刻画了三位主要女性'"。"'潘金莲'"：1）**美貌**——"'漂亮'"；2）**聪明**——"'有心机'"；3）**悲剧**——"'被'命运'左右'"。"'李瓶儿'"：1）**温柔**——"'善良'"；2）**富有**——"'有'财产'"；3）**早逝**——"'病死'"。"'庞春梅'"：1）**丫鬟**——"'出身'低'"；2）**上升**——"'后来'得势'"；3）**复杂**——"'善'与'恶'并存'"。"'这'三个'女性'"——"'不'是'简单'的'淫妇'标签'"——"'有'层次'"
- **市井社会的描绘**: 本书"描绘了明代市井社会'"。"'什么'是'市井'社会'"：1）**商业**——"'商人'地位'上升'"；2）**金钱**——"'金钱'至上'"；3）**欲望**——"'追求'享乐'"。"'书中'的'描绘'"：1）**官场**——"'腐败'"——"'官商'勾结'"；2）**家庭**——"'妻妾'争斗'"；3）**市井**——"'商人'——'妓女'——'帮闲'"。"'这种'描绘'"——"'真实'"——"'冷峻'"
- **欲望的主题**: 本书"的主题是欲望'"。"'什么'欲望'"：1）**财**——"'追求'财富'"；2）**色**——"'追求'美色'"；3）**权**——"'追求'权力'"。"'欲望'的'后果'"：1）**毁灭**——"'西门庆'纵欲'身亡'"；2）**报应**——"'因果'报应'"。"'作者'的'态度'"："'不'说教'"——"'冷峻'描绘'"——"'让'读者'自己'判断'"
- **艺术成就**: 本书"的艺术成就很高'"。"'人物'刻画'"："'细腻'"——"'立体'"——"'不'脸谱化'"。"'语言'"："'生动'"——"'市井'语言'"。"'结构'"："'严密'"——"'百回'长篇'"。"'细节'"："'生活'细节'丰富'"。"'《金瓶梅》'的'艺术'"——"'影响'了'《红楼梦》'"
- **作者的背景**: 兰陵笑笑生——明代小说家——"《金瓶梅》的作者'"。"'真名'不'详'"——"'兰陵'是'笔名'"。"'他'可能'是'文人'"——"'熟悉'市井'生活'"。"《金瓶梅》'成书'于'16 世纪'末'"——"'明代'万历'年间'"。"'这本书'长期'被'禁'"——"'因为'有'色情'描写'"。"'但'它'的'文学'价值'"——"'被'认可'"
