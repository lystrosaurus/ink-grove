# 蛤蟆先生去看心理医生

状态：已读

作者：罗伯特戴博德

一句话评价：用童话形式讲解心理咨询过程的心理学入门书，借《柳林风声》中的蛤蟆先生，展现了从抑郁到自我觉醒的心理治疗旅程。

核心收获：
- 理解了心理咨询的基本流程：从依赖型儿童自我状态到成人自我状态的转变
- "人生坐标"理论：四种基本人生态度（我好-你好、我好-你不好、我不好-你好、我不好-你不好）
- 心理治疗的核心不是给建议，而是帮助来访者自己找到答案
