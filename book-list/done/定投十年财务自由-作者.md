# 定投十年财务自由

状态：已读

作者：银行螺丝钉

一句话评价：面向中国投资者的指数基金定投指南，系统讲解了为什么定投、选什么基金、如何止盈，实操性强，是 A 股投资者的理财入门书。

核心收获：
- 理解了定投的核心逻辑：在波动市场中通过定期定额买入摊低成本，微笑曲线是定投收益的来源
- 学会了挑选指数基金的几个关键指标：费率、跟踪误差、规模、流动性
- 明白了"止盈不止损"的原则——定投不怕跌，怕的是在低点停止投入
