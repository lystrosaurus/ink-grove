# 股票大作手回忆录

状态：已读

作者：埃德温·勒菲弗

一句话评价：埃德温·勒菲弗（Edwin Lefèvre）的"投资经典'"——"'以传奇交易员杰西·利弗莫尔（Jesse Livermore）为原型'"——"'用小说体讲述了他从穷小子到华尔街之王的起伏人生'"。"'市场永远是对的'"——"'截断亏损，让利润奔跑'"——"'交易的核心是管理风险和情绪'"。"'投机像群山一样古老'"——"'人性不变，模式重复'"

核心收获：
- **市场永远是对的**: 利弗莫尔的"核心信条是'市场永远是对的'"。"'不要与趋势对抗'"——"'顺势而为'"。"'当你'认为'市场'错了'"——"'你'往往'是'错的'"。"'市场'有'它'自己的'意志'"——"'你'无法'战胜'市场'"——"'你'只能'跟随'市场'"。"'如何'判断'趋势'"——"'价格'是'唯一'的'真相'"——"'不'听'消息'"——"'不'听'分析'"——"'只'看'价格'"。"'当'价格'上涨'"——"'趋势'是'向上'的'"——"'买入'"。"'当'价格'下跌'"——"'趋势'是'向下'的'"——"'卖出'"。"'这'听起来'简单'"——"'但'执行'很难'"——"'因为'人性'总是'想'逆势'"
- **截断亏损，让利润奔跑**: 这是"利弗莫尔最著名的交易原则'"。"'截断亏损'意味着'"：1）**快速止损**——"'当'交易'亏损'时'"——"'立即'卖出'"——"'不'抱'幻想'"；2）**小亏**——"'每次'亏损'控制在'小'范围'"——"'不'让'亏损'扩大'"；3）**不补仓**——"'亏损'的'头寸'不'加仓'"——"'那'是'错误'的'"。"'让利润奔跑'意味着'"：1）**持有盈利**——"'当'交易'盈利'时'"——"'继续'持有'"——"'让'利润'增长'"；2）**不急于卖出**——"'不'因为'赚'了'一点'就'卖出'"；3）**移动止损**——"'随着'盈利'增加'"——"'提高'止损位'"。"'这个'原则'的'核心'"——"'亏'要'亏'得'小'"——"'赚'要'赚'得'大'"
- **管理风险和情绪**: 利弗莫尔认为"交易的核心不是预测市场，而是管理风险和情绪'"。"'风险管理'"：1）**仓位控制**——"'不'要'满仓'"——"'每次'交易'只'用'一小部分'资金'"；2）**止损**——"'每笔'交易'都'设'止损'"；3）**分散**——"'不'把'所有'资金'投入'一只'股票'"。"'情绪管理'"：1）**不贪婪**——"'赚'了'不'贪婪'"——"'知道'何时'退出'"；2）**不恐惧**——"'亏'了'不'恐惧'"——"'按'计划'执行'"；3）**不冲动**——"'不'因为'情绪'而'交易'"。"'利弗莫尔'说'最大的'敌人'是'自己'"——"'是'贪婪'和'恐惧'"
- **投机像群山一样古老**: 利弗莫尔有"一句名言：'投机像群山一样古老'"。"'为什么'投机'古老'"——"'因为'人性'不变'"。"'贪婪'——'恐惧'——'希望'——'绝望'"——"'这些'情绪'几百年来'没有'变化'"。"'所以'市场'模式'会'重复'"——"'历史'会'重演'"。"'但'重演'不是'简单'重复'"——"'是'相似'的'模式'"——"'细节'不同'"。"'理解'人性'"——"'理解'市场'"——"'理解'自己'"——"'才能'在'市场'中'生存'"
- **关键点交易**: 利弗莫尔"发展了'关键点'（Pivot Point）交易法'"。"'关键点'是'价格'的'转折'点'"。"'如何'识别'关键点'"：1）**新高**——"'价格'突破'前期'高点'"——"'是'买入'信号'"；2）**新低**——"'价格'跌破'前期'低点'"——"'是'卖出'信号'"；3）**成交量**——"'突破'时'成交量'放大'"——"'确认'信号'"。"'关键点'交易'的'优点'"——"'顺势'"——"'风险'小'"——"'收益'大'"。"'但'需要'耐心'"——"'等待'关键点'出现'"——"'不'提前'入场'"
- **利弗莫尔的起落**: 利弗莫尔的"人生充满戏剧性'"。"'他'15 岁'开始'交易'"——"'从'穷小子'起家'"。"'他'几次'大赚'"——"'赚'了'几百万'美元'"——"'成为'华尔街'传奇'"。"'他'也'几次'大亏'"——"'破产'了'三次'"——"'但'每次'都'东山再起'"。"'1929 年'大萧条'"——"'他'做空'美股'"——"'赚'了'1 亿美元'"——"'被称为'华尔街'大空头'"。"'但'晚年'他'再次'破产'"——"'1940 年'自杀'身亡'"。"'他'的'悲剧'在于'"——"'他'懂'市场'"——"'但'不'懂'自己'"——"'他'无法'控制'情绪'"
- **勒菲弗的背景**: 埃德温·勒菲弗（Edwin Lefèvre, 1871-1943）——美国作家、记者——"《股票大作手回忆录》的作者'"。"他'是'财经'记者'"——"'采访'了'很多'华尔街'人物'"。"《股票大作手回忆录》'1923 年'出版'"——"'以'杰西·利弗莫尔'为'原型'"——"'但'用'小说'形式'"。"'利弗莫尔'本人'参与'了'本书'的'创作'"——"'书中'的'很多'观点'来自'利弗莫尔'"。"这本书'被誉为'投资'经典'"——"'巴菲特'——'索罗斯'等'都'推荐'过'"
