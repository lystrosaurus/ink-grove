# 经济学的思维方式

状态：已读

作者：保罗·海恩

一句话评价：保罗·海恩（Paul Heyne）的"经济学入门经典'"——"'机会成本'——'边际思维'——'供需规律'等概念通俗讲解'"——"'经济学是一种思维方式，而不是一套结论'"——"'用经济学视角理解世界的指南'"。"'天下没有免费的午餐'"——"'每一个选择都有代价'"

核心收获：
- **经济学是思维方式**: 海恩说"'经济学是一种思维方式'"——"'不是'一套结论'"。"'经济学'不'告诉'你'应该'做什么'"——"'它'帮助'你'理解'人们'在做'什么'"——"'为什么'他们'这么做'"。"'经济学思维'的核心'"——"'从'个体'出发'"——"'理解'他们的'激励'"——"'预测'他们的'行为'"。"'这种思维'可以'应用到'任何'领域'"——"'政治'——'法律'——'家庭'——'教育'"
- **机会成本**: 海恩说"'机会成本'是'经济学最核心的概念'"。"'某种东西的成本'"——"'是为了得到它而放弃的东西'"。"'你'上大学的'成本'"——"'不是'学费'"——"'是'你'放弃的'工作收入'"。"'你'看'这场电影'的'成本'"——"'不是'票价'"——"'是'你'用'这段时间'能'做的'其他事'"。"'机会成本'提醒我们'"——"'资源是稀缺的'"——"'每一个'选择'都有'代价'"——"'天下没有免费的午餐'"
- **边际思维**: 海恩说"'理性人考虑边际量'"。"'边际'是'增量'"——"'下一个'单位'"。"'你'已经'吃了 3 个包子'"——"'第 4 个包子'的'满足感'"——"'就是'边际效用'"。"'企业'已经'生产了 100 件产品'"——"'第 101 件'的'成本'"——"'就是'边际成本'"。"'理性决策'是'在边际上'做出的'"——"'如果'边际收益>边际成本'"——"'就'继续'"。"'边际思维'帮助'我们'理解'"——"'为什么'水'很'有用'但'便宜'"——"'钻石'没'用'但'贵'"——"'因为'边际效用不同'"
- **供需规律**: 海恩说"'供给和需求'是'经济学的核心框架'"。"'需求'"——"'买者在不同价格下愿意购买的数量'"。"'供给'"——"'卖者在不同价格下愿意出售的数量'"。"'均衡价格'"——"'供给等于需求时的价格'"——"'在这个价格下'"——"'市场出清'"。"'如果'价格'高于均衡'"——"'供过于求'"——"'卖者降价'"。"'如果'价格'低于均衡'"——"'供不应求'"——"'买者加价'"。"'价格'是'信号'"——"'它'告诉'生产者'生产什么'"——"'告诉'消费者'消费什么'"
- **市场机制**: 海恩说"'市场是组织经济活动的好方法'"。"'市场'通过'价格'来'协调'千百万人的决策'"——"'不需要'中央计划'"。"'亚当·斯密的'看不见的手'"——"'每个人追求自身利益'"——"'但'市场'引导'他们'促进社会利益'"。"'市场'的'优点'"：1）**效率**——"'资源流向最有价值的用途'"；2）**自由**——"'个人可以自由选择'"；3）**创新**——"'竞争促使改进'"。"'市场'的'缺点'"：1）**外部性**——"'污染'"；2）**不平等**——"'收入分配'"；3）**公共物品**——"'国防'"
- **比较优势**: 海恩说"'比较优势是贸易的基础'"。"'即使'一个人'在所有方面'都'比另一个人强'"——"'他们'仍然'可以从贸易中受益'"。"'因为'每个人'在'某些方面'有'比较优势'"——"'机会成本较低'"。"'美国'生产'芯片'的机会成本'比'生产'衣服'低'"——"'中国'生产'衣服'的机会成本'比'生产'芯片'低'"——"'所以'美国'出口芯片'"——"'中国'出口衣服'"——"'双方'都'受益'"。"'贸易'不是'零和游戏'"——"'是'正和游戏'"
- **保罗·海恩的背景**: 保罗·海恩（Paul Heyne, 1931-1997）——美国经济学家——"华盛顿大学教授'"——"《经济学的思维方式》的作者'"。"这本书'1973 年'首次出版'"——"'已经'更新了'多次'"——"'是'美国'最受欢迎的'经济学入门书'"。"海恩'的'写作风格'通俗易懂'"——"'他'用'大量案例'解释'抽象概念'"——"'让'非专业读者'也能'理解经济学'"。"他'说'经济学的'目的'"——"'不是'培养'经济学家'"——"'是'培养'有'经济学思维'的'公民'"
