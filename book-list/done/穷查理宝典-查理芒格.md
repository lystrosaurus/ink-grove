# 穷查理宝典

状态：已读

作者：查理·芒格（彼得·考夫曼编）

一句话评价：查理·芒格（Charlie Munger）的"智慧合集"——"'多元思维模型'与'误判心理学'影响深远'"——"'手里拿着锤子的人，看什么都像钉子'"——"'如果你能比别人更聪明一点点，你就能比别人更富有一点点'"——"价值投资与人生智慧的集大成者"。"芒格'是'巴菲特的'搭档'"——"'伯克希尔·哈撒韦'的'副主席'"——"他的'智慧'不仅'体现在'投资上'"——"'也'体现在'人生的'方方面面'"

核心收获：
- **多元思维模型**: 芒格最核心的观点——"多元思维模型"。"'手里拿着锤子的人，看什么都像钉子'"——"'如果'你'只'有'一个'思维模型'"——"'你'就'会'用'它'来'解决'所有'问题'"——"'这'是'愚蠢的'"。"芒格说'你'必须'掌握'多个'学科'的'核心模型'"——"'物理学'的'临界点'"——"'生物学'的'进化论'"——"'心理学'的'认知偏差'"——"'经济学'的'机会成本'"——"'用'这些'模型'来'思考'问题'"——"'你'就'会'比'别人'更'聪明'"
- **误判心理学**: 芒格总结了"'25 种'人类误判心理学'"——"'人类'在'决策'时'会'犯'的'错误'"。"其中'最重要的'几种"：1）**奖励和惩罚超级反应倾向**——"'人们'会'为了'奖励'而'做'某事'"——"'会'为了'避免惩罚'而'不做'某事'"；2）**喜欢/热爱倾向**——"'人们'会'忽略'自己'喜欢的'人'或'事物'的'缺点'"；3）**避免不一致性倾向**——"'人们'会'坚持'自己'的'观点'"——"'即使'这个'观点'是'错的'"；4）**社会认同倾向**——"'人们'会'跟随'大多数人'"——"'即使'大多数人是'错的'"
- **逆向思考**: 芒格"提倡'逆向思考'"——"'反过来'想'问题'"。"'如果'你'想'成功'"——"'先'想'怎样'会'失败'"——"'然后'避免'这些'失败'"。"'如果'你'想'幸福'"——"'先'想'怎样'会'痛苦'"——"'然后'避免'这些'痛苦'"。"芒格说'我'的'成功'不是'因为'我'很'聪明'"——"'是'因为'我'避免了'很多'愚蠢'"——"'反过来'想'问题'"——"'让'我'看到了'别人'看不到'的'东西'"
- **能力圈**: 芒格提出了"'能力圈'的概念'"——"'你'必须'知道'自己'的'能力圈'在'哪里'"——"'在'能力圈内'行事'"——"'不要'超出'能力圈'"。"'能力圈'的'大小'不'重要'"——"'重要'的'是'你'知道'它的'边界'在'哪里'"。"巴菲特和芒格'的'成功'"——"'不是'因为'他们'什么都懂'"——"'是'因为'他们'知道'自己'不懂'什么'"——"'他们'只'在'自己'的'能力圈内'投资'"
- **Lollapalooza 效应**: 芒格提出了"'Lollapalooza 效应'"——"'多种'因素'共同作用'"——"'产生'一个'巨大的'结果'"。"'比如'拍卖'"——"'社会认同'（别人都在买）+'"——'**承诺和一致性**（我已经出过价了）+'——'**稀缺性**（只剩最后一个）+'"——"**奖励和惩罚**（不买就没了）"——"'这些'因素'叠加'"——"'让人'做出'非理性'的'决策'"。"理解'Lollapalooza 效应'"——"'可以'帮助'你'避免'被'操控'"
- **跨学科思维**: 芒格"强调'跨学科思维'"——"'你'不能'只'学'一个'学科'"——"'你'必须'学习'多个'学科'的'核心概念'"。"'他'自己'就'是'一个'跨学科的'学习者'"——"'他'学'了'物理学'——'生物学'——'心理学'——'经济学'——'历史学'"——"'把'这些'学科'的'模型'应用'到'投资'和'人生'"。"芒格说'我'的'成功'靠'的'不是'一个'秘密武器'"——"'是'一个'工具箱'"——"'这个'工具箱'里'有'很多'工具'"——"'每个'工具'都'来自'不同的'学科'"
- **查理·芒格的背景**: 查理·芒格（Charlie Munger, 1924-）——美国投资家——"伯克希尔·哈撒韦公司副主席"——"巴菲特的'黄金搭档'"。"他'和'巴菲特'一起'管理'伯克希尔'近 60 年'"——"'创造了'投资史上的'奇迹'"。"芒格'不仅'是'一个'投资家'"——"'也是'一个'思想家'"——"'他的'智慧'被'总结'在'《穷查理宝典》'中'"——"'这本书'不是'芒格'写的'"——"'是'彼得·考夫曼'编的'"——"'收录'了'芒格的'演讲、文章、格言'"
