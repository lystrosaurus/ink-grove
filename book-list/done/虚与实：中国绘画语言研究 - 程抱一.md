# 虚与实：中国绘画语言研究

状态：已读

作者：程抱一

一句话评价：程抱一（François Cheng）的"中国艺术研究经典'"——"'从中西比较视角解读中国绘画的语言体系'"。"'虚与实是中国绘画的核心语言'"——"'留白不是空白，而是画面的一部分'"。"'中国绘画与西方绘画的根本区别：西方追求再现，中国追求传神'"

核心收获：
- **虚与实的核心**: 程抱一提出"'虚与实'是中国绘画的核心语言'"。"'实'是'什么'"——"'实'是'画面'上'的'物象'"——"'山'——'水'——'树'——'石'"。"'虚'是'什么'"——"'虚'是'留白'"——"'是'空白'"——"'是'不'画'的'地方'"。"'但'虚'不'是'没有'"——"'虚'是'画面'的'一部分'"——"'虚'与'实'相互'依存'"。"'没有'虚'"——"'实'就'太'满'"——"'没有'呼吸'的'空间'"。"'没有'实'"——"'虚'就'太'空'"——"'没有'内容'"。"'虚'与'实'的'关系'"——"'虚实相生'"——"'虚'中有'实'"——"'实'中有'虚'"
- **留白的意义**: 程抱一"详细分析了留白的意义'"。"'留白'不'是'空白'"——"'留白'有'多重'意义'"：1）**想象空间**——"'让'观者'想象'"——"'空白'可以'是'云'"——"'可以'是'水'"——"'可以'是'天'"；2）**呼吸空间**——"'让'画面'有'节奏'"——"'不'太'拥挤'"；3）**精神空间**——"'虚'是'道'的'体现'"——"'道'是'无'"——"'无'中生'有'"。"'中国'画家'说'计白当黑'"——"'把'空白'当'黑色'一样'重要'"——"'精心'设计'空白'"
- **中西绘画比较**: 程抱一"从中西比较视角分析中国绘画'"。"'西方绘画'"：1）**追求再现**——"'力求'真实'"——"'透视法'"——"'解剖学'"；2）**满构图**——"'画面'画'满'"——"'不留'空白'"；3）**焦点透视**——"'从'一个'视点'看'"。"'中国绘画'"：1）**追求传神**——"'不'求'形似'"——"'求'神似'"；2）**留白构图**——"'留'大量'空白'"；3）**散点透视**——"'从'多个'视点'看'"。"'这种'区别'源于'不同的'世界观'"——"'西方'主客二分'"——"'中国'天人合一'"
- **诗书画印的统一**: 程抱一"探讨了中国绘画的独特传统'"。"'中国'有'诗书画印'统一'的'传统'"。"'诗'"——"'画'上'题诗'"——"'诗'补充'画'的'意境'"。"'书'"——"'题字'用'书法'"——"'书法'本身'有'美感'"。"'画'"——"'主体'是'画'"。"'印'"——"'印章'"——"'是'作者'的'标记'"——"'也'是'构图'的'一部分'"。"'这'四者'统一'在'一幅'画'上'"——"'形成'综合'艺术'"。"'西方'没有'这种'传统'"——"'西方'的'画'上'不'题诗'"——"'不'写字'"
- **气的概念**: 程抱一"分析了中国绘画中'气'的概念'"。"'气'是'什么'"——"'气'是'生命力'"——"'是'能量'"——"'是'流动'的'东西'"。"'一幅'好画'"——"'有'气韵生动'"——"'画面'有'生命'"——"'在'流动'"。"'如何'表现'气'"：1）**笔墨**——"'笔墨'的'流动'"——"'表现'气'"；2）**构图**——"'构图'的'节奏'"——"'表现'气'"；3）**虚实**——"'虚实'的'变化'"——"'表现'气'"。"'气'是'中国'美学'的'核心'概念'"——"'不'只是'绘画'"——"'书法'——'诗歌'——'音乐'都'讲'气'"
- **道的体现**: 程抱一"探讨了中国绘画与道的关系'"。"'道'是'中国'哲学'的'核心'"——"'道'是'无'"——"'道'是'虚'"。"'中国'绘画'追求'道'"——"'不'是'再现'物象'"——"'是'表现'道'"。"'如何'表现'道'"：1）**虚**——"'留白'体现'道'的'无'"；2）**简**——"'简'洁'体现'道'的'朴素'"；3）**自然**——"'自然'体现'道'的'无为'"。"'所以'中国'绘画'不'是'艺术'"——"'是'修行'"——"'是'体道'的'方式'"
- **程抱一的背景**: 程抱一（François Cheng, 1929-）——法籍华裔学者、诗人、翻译家——"《虚与实》的作者'"。"他'1929 年'出生'于'中国'"——"'1948 年'赴'法国'"。"他'在'法国'学习'文学'和'艺术'"——"'后'来'在'巴黎'第七大学'任教'"。"他'的'写作'用'法语'"——"'但'主题'是'中国'文化'"。"《虚与实》'1979 年'出版'"——"'在'法国'引起'轰动'"。"他'还'翻译'了'中国'诗歌'"——"'《唐诗》'——'《宋词》'"。"2002 年'"——"'他'当选'法兰西学术院'院士'"——"'是'第'一'位'亚裔'院士'"
