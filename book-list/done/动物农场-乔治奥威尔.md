# 动物农场

状态：已读

作者：乔治·奥威尔

一句话评价：乔治·奥威尔（George Orwell）的"政治寓言经典'"——"'144 页讲透革命异化全过程'"。"'所有动物一律平等，但有些动物比其他动物更相等'"——"'成千古名句'"。"'拳击手的悲剧最令人心碎'"。"'20 世纪最具影响力的政治讽刺小说'"——"'与《1984》并列为反极权主义双璧'"

核心收获：
- **动物起义**: 本书核心是"'动物起义'"的故事'"。"'老少校'（影射'马克思/列宁）'的'革命'理想'"——"'所有'动物'一律'平等'"。"'但'理想'很快'被'现实'扭曲'"
- **七诫的篡改**: 本书"描述了七诫的篡改'"。"'从'凡'动物'皆'不'可'杀人'"——"'到'凡'动物'皆'不'可'无'理由'杀人'"。"'这'是'权力'的'语言'游戏'"
- **拿破仑夺权**: 本书"讲述了拿破仑夺权'"。"'用'暴力'（狗）'和'宣传'（斯奎拉）'建立'独裁'"。"'革命'领袖'变成'新'暴君'"
- **雪球被逐**: 本书"描述了雪球被逐'"。"'理想主义者'被'实用主义者'淘汰'"——"'革命'中'的'路线'斗争'"
- **拳击手的悲剧**: 本书"刻画了拳击手的悲剧'"。"'最'忠诚'"——"'最'勤劳'的'马'"——"'最后'被'卖给'屠马场'"。"'我会'更加'努力'工作'"——"'的'悲剧'"
- **猪变人**: 本书"结局是猪变人'"。"'革命'的'最终'结局'"："'猪'和'人'坐'在'同一张'桌子'上'"。"'革命者'变成'了'他们'曾经'反对'的'人'"
- **奥威尔的洞察**: 本书"揭示了奥威尔的洞察'"。"'权力'腐蚀'的'不'是'好人'"——"'是'所有'掌握'权力'的'人'"。"'制度'比'个人'更'重要'"
- **作者的背景**: 乔治·奥威尔（George Orwell, 1903-1950）——英国作家、记者——"《动物农场》的作者'"。"'他'经历'了'西班牙'内战'"——"'目睹'了'左翼'内部'的'清洗'"。"'《动物'农场》'1945 年'出版'"——"'影射'苏联'革命'的'异化'"。"'他'的'其他'著作'"："'《1984》'"。"'奥威尔'的'主题'"："'极权主义'"——"'权力'腐蚀'"——"'语言'操控'"
