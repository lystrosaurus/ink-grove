# 许三观卖血记

状态：已读

作者：余华

一句话评价：余华的"代表作之一'"——"'许三观卖血度日的故事催人泪下'"。"'一盘炒猪肝，二两黄酒'成经典意象'"——"'中国底层百姓的生存史诗'"——"'苦难中的人性光辉'"

核心收获：
- **卖血的生存方式**: 本书的核心是"'卖血'"这一生存方式'"。"'许三观'是'谁'"——"'一个'普通'的'丝厂'工人'"。"'他'生活'在'什么'时代'"——"'大跃进'——'文革'——'困难'时期'"。"'为了'生存'"——"'他'选择'卖血'"。"'卖血'在'当时'很'常见'"——"'穷人'没有'其他'赚钱'方式'"——"'只能'卖血'"。"'许三观'一生'卖血'12 次'"——"'每次'卖血'都'有'一个'原因'"——"'娶亲'——'还债'——'治病'——'渡荒'"。"'卖血'是'许三观'的'生存'策略'"——"'也是'他'的'尊严'"
- **家庭责任**: 本书"展现了许三观的家庭责任'"。"'许三观'对'家庭'的'责任'"：1）**妻子**——"'他'爱'许玉兰'"——"'即使'她'犯'了'错'"——"'他'也'原谅'她'"；2）**孩子**——"'三个'儿子'"——"'其中'一个'不是'他'亲生'"——"'但'他'依然'爱'他们'"；3）**家庭**——"'为了'家庭'"——"'他'可以'卖血'"——"'可以'牺牲'自己'"。"'最后一次'卖血'"——"'他'已经'老'了'"——"'血头'不'要'他'的'血'"——"'他'哭'了'"——"'因为'他'觉得'自己'没有'用'了'"。"'这'种'责任'感'"——"'是'中国'父亲'的'典型'"
- **饥荒年代**: 本书"描写了饥荒年代的苦难'"。"'大跃进'后'"——"'全国'大'饥荒'"。"'许三观'一家'"——"'每天'喝'玉米粥'"——"'还是'不够'"。"'最'困难'的'时候'"——"'他们'用'嘴'炒菜'"——"'想象'着'吃'肉'"。"'许三观'去'卖血'"——"'用'卖血'的'钱'买'食物'"——"'让'家人'活'下去'"。"'这'段'描写'真实'"——"'余华'不'回避'苦难'"——"'他'直面'苦难'"。"'但'苦难'中'有'温暖'"——"'一家人'相互'扶持'"——"'一起'渡过'难关'"
- **人性的善良**: 本书"展现了人性的善良'"。"'虽然'生活'艰难'"——"'但'人性'依然'善良'"。"'许三观'的'善良'"：1）**宽容**——"'对'许玉兰'的'原谅'"；2）**无私**——"'为'一乐'治病'"——"'一路'卖血'到'上海'"；3）**坚韧**——"'再'苦'再'累'"——"'也'不'放弃'"。"'其他人'的'善良'"：1）**血头**——"'虽然'剥削'"——"'但'也'给'许三观'机会'"；2）**路人**——"'在'许三观'卖血'路上'"——"'帮助'他'"。"'这'种'善良'"——"'是'苦难'中'的'光'"——"'让'人'看到'希望'"
- **经典意象**: 本书"有很多经典意象'"。"'一盘炒猪肝，二两黄酒'"——"'这是'许三观'卖血'后'的'犒劳'"。"'黄酒'要'温'一'温'"——"'这'是'仪式感'"。"'这个'意象'反复'出现'"——"'每次'卖血'后'"——"'许三观'都'要'吃'炒猪肝'——'喝'黄酒'"。"'这'代表'了'什么'"——"'代表'了'许三观'对'生活'的'热爱'"——"'即使'再'苦'"——"'他'也'要'享受'一'下'"。"'还有'嘴'炒菜'"——"'用'嘴'想象'炒菜'"——"'既'心酸'又'温暖'"
- **余华的写作风格**: 本书"展现了余华的写作风格'"。"'余华'的'风格'"：1）**简洁**——"'语言'简单'"——"'不'华丽'"；2）**冷静**——"'叙述'冷静'"——"'不'煽情'"；3）**重复**——"'反复'出现'的'句子'"——"'形成'节奏'"。"'比如'许三观'每次'卖血'后'"——"'都'说'一盘'炒猪肝，二两黄酒'"——"'反复'说'"——"'形成'了'韵律'"。"'这种'风格'"——"'让'苦难'不'那么'沉重'"——"'让'故事'有'了'幽默感'"。"'余华'说'他'想'用'喜剧'的'方式'写'悲剧'"
- **余华的背景**: 余华（1960-）——中国当代作家——"《许三观卖血记》的作者'"。"他'是'先锋'文学'代表'"——"'早期'作品'暴力'——'血腥'"。"'90 年代'后'"——"'他'转向'写实'"——"'《活着》'——'《许三观卖血记》'"。"《许三观卖血记》'1995 年'出版'"——"'获得'好评'"。"'它'被'翻译成'多'种'语言'"——"'在'世界'范围'内'有'影响'"。"余华'的'其他'作品'"——"'《在细雨中呼喊》'——'《兄弟》'——'《第七天》'"——"'都'关注'底层'"——"'关注'苦难'"
