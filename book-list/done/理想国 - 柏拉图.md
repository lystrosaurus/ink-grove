# 理想国

状态：已读

作者：柏拉图

一句话评价：柏拉图的"哲学对话录"——"西方哲学的'源头'"——"'洞穴比喻'与'哲人王'思想'影响两千年'"。"'未经审视的人生不值得度过'"——"'除非哲学家成为国王'——'否则人类将永无宁日'"。《理想国》"探讨了'正义'——'知识'——'灵魂'——'政治'——'教育'——'艺术'"——"几乎'所有'哲学问题"——"都是'从这里开始的'"。

核心收获：
- **洞穴比喻**: 柏拉图最著名的"洞穴比喻"——"一群人'从小'被'锁在'一个洞穴里"——"他们'只能'看到'墙上'的影子'"——"以为'影子'就是'真实的世界'"。"有一天'一个人'被'解放了'"——"他'走出了洞穴'"——"看到了'真正的'阳光和世界"——"他'回到了洞穴'"——"想'告诉'其他人'真相'"——"但'其他人'不相信他"——"甚至'想杀了他'"。柏拉图说"哲学家'就是'那个'走出洞穴的人'"——"他'看到了'真理'"——"他的'责任'是'回到洞穴'"——"'引导'其他人'也'看到'真理'"
- **哲人王**: 柏拉图提出了"哲人王"（philosopher-king）的理想——"'除非哲学家成为国王'——'或者国王成为哲学家'"——"'否则人类将永无宁日'"。"因为'政治家'只'关心权力'"——"'商人'只'关心利益'"——"'只有哲学家'关心'真理和正义'"。"哲人王'不是'一个'独裁者'"——"他'是一个'有智慧的人'"——"他'用'理性'来'治理国家'"——"'不'被'欲望'和'情绪'左右"
- **理念论**: 柏拉图提出了"理念论"（Theory of Forms）——"'真实的世界'不是'我们看到的'世界'"——"'是'理念的世界'"。"我们看到的'桌子'"——"'只是一个'不完美的'桌子的'复制品'"——"'真正的'桌子'是'理念世界中的'桌子的'理念'"。"同样的"——"我们看到的'正义'、'美'、'善'"——"'都是'不完美的'"——"'真正的'正义、美、善'存在于'理念世界'"。"哲学的'任务'"——"'就是'去'认识'理念世界"
- **正义论**: 《理想国》的'核心问题'是"'什么是正义'"。"柏拉图'通过'苏格拉底'与'其他人的'对话'"——"'探讨了'几种'对正义的定义'"：1）**正义是'欠债还钱'**——"苏格拉底说'这'太简单了'"；2）**正义是'强者的利益'**——"苏格拉底说'那'暴君的'行为'也是正义的吗"；3）**正义是'每个人做自己该做的事'**——"这是柏拉图'最终的答案'"。"在一个'理想的国家'中"——"统治者'负责'智慧'"——"战士'负责'勇敢'"——"生产者'负责'节制'"——"当'每个人都'做自己该做的事'"——"这个国家'就是'正义的"
- **灵魂三分**: 柏拉图"把人的灵魂分为三部分"：1）**理性**（reason）——"追求'真理和智慧'"——"应该在'灵魂中'占据'主导地位'"；2）**激情**（spirit）——"追求'荣誉和勇气'"——"应该'支持'理性'"；3）**欲望**（appetite）——"追求'食物、性、财富'"——"应该'被'理性'和'激情'控制'"。"当'理性'控制'灵魂'"——"这个人'就是'正义的'"——"当'欲望'控制'灵魂'"——"这个人'就是'不正义的'"
- **乌托邦构想**: 柏拉图"构想了一个'理想的国家'"——"在这个国家中"：1）**统治者'是'哲学家'"——"'因为他们'有'最高的智慧'"；2）**战士'负责'保卫国家'"——"'他们'被'训练'得'勇敢而忠诚'"；3）**生产者'负责'生产和贸易'"——"'他们'的'欲望'被'节制'"；4）**所有人'都'接受'教育'"——"'教育'是'国家最'重要的'职能'"；5）**艺术'被'严格审查'"——"'因为'艺术'会'影响人的'灵魂'"。柏拉图的"乌托邦"被"后人批评为'极权主义'"——"但'他的初衷'是'建立一个'正义的国家'"
- **柏拉图的背景**: 柏拉图（Plato, 约公元前 428-前 348 年）——古希腊哲学家——"苏格拉底的学生"——"亚里士多德的老师"。"西方哲学史'可以'分为'两部分'"——"'柏拉图之前'和'柏拉图之后'"——"因为'几乎所有'哲学问题'都'是从'柏拉图'开始的'"。《理想国》是"柏拉图'最著名'的作品"——"也是'西方哲学史上'最'有影响力的'著作之一"
