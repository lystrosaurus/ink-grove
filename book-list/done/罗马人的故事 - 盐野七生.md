# 罗马人的故事

状态：已读

作者：盐野七生

一句话评价：盐野七生的"15 卷巨著'"——"'讲述罗马从建国到灭亡的千年历史'"——"'以小说般的叙事手法让历史变得生动可读'"。"'罗马成功的关键'"——"'包容性（接纳被征服者）'——'法治精神（法律面前人人平等）'——'实用主义（吸收各方长处）'"。"'罗马的衰落'不是'因为外敌入侵'"——"'是内部制度的腐化'"——"'从共和到帝制的转变是转折点'"

核心收获：
- **罗马的包容性**: 盐野七生说"'罗马成功的关键是包容性'"。"'罗马'不'像'其他'帝国'"——"'它'不'屠杀'被征服者'"——"'它'接纳'他们'"。"'被征服者'可以'成为'罗马公民'"——"'可以'参军'"——"'可以'当官'"。"'凯撒'是'罗马人'"——"'图拉真'是'西班牙人'"——"'哈德良'是'西班牙人'"——"'塞维鲁'是'非洲人'"。"'这种'包容性'"——"'让'罗马'能够'吸收'各方'人才'"——"'让'被征服者'愿意'成为'罗马人'"
- **法治精神**: 盐野七生说"'罗马的另一个关键是法治精神'"。"'罗马法'是'西方'法律体系的'基础'"。"'罗马法'的'原则'"：1）**法律面前人人平等**——"'公民'都'受'法律'保护'"；2）**契约精神**——"'承诺'必须'遵守'"；3）**私有财产神圣不可侵犯**——"'财产'受'法律保护'"；4）**程序正义**——"'审判'必须'按'程序'进行'"。"'罗马法'的'影响'"——"'至今'仍'在'欧洲'和'美洲'使用'"
- **实用主义**: 盐野七生说"'罗马人是实用主义者'"。"'他们'不'拘泥于'传统'"——"'他们'吸收'各方'长处'"。"'罗马'的'建筑'技术'来自'希腊'"——"'罗马'的'军事'战术'来自'迦太基'"——"'罗马'的'法律'来自'希腊'和'东方'"。"'罗马人'的'座右铭'"——"'如果'它'有用'"——"'就'采用'它'"。"'这种'实用主义'"——"'让'罗马'能够'不断改进'"——"'保持'竞争力'"
- **从共和到帝制**: 盐野七生"详细讲述了罗马从共和到帝制的转变'"。"'共和制'的'罗马'"——"'元老院'是'最高权力机构'"——"'执政官'由'选举'产生'"。"'但'随着'罗马'扩张'"——"'共和制'的'问题'暴露'"——"'军阀'崛起'"——"'苏拉'——'凯撒'——'庞培'"。"'凯撒'被'刺杀'"——"'屋大维'建立'帝国'"——"'共和制'结束'"。"'盐野七生'说'这是'罗马的'转折点'"——"'帝制'让'罗马'稳定'"——"'但'也'埋下了'衰落的'种子'"
- **罗马的衰落**: 盐野七生"分析了罗马衰落的原因'"。"'罗马'不是'被'外敌'灭亡'的'"——"'是'内部'腐化'导致'的'"。"'衰落'的'原因'"：1）**军阀割据**——"'军队'效忠'将领'不'效忠'国家'"；2）**经济崩溃**——"'通货膨胀'——'税收'过重'"；3）**道德沦丧**——"'奢侈'——'腐败'"；4）**蛮族入侵**——"'日耳曼人'——'匈人'"。"'公元 476 年'"——"'西罗马帝国'灭亡'"——"'但'东罗马帝国'（拜占庭）'又'延续'了'1000 年'"
- **历史是人的故事**: 盐野七生"把历史写成了人的故事'"。"'她'不'写'枯燥的'事件'"——"'她'写'人'"。"'凯撒'不是'一个'名字'"——"'是'一个'有血有肉'的'人'"——"'他'有'野心'"——"'有'魅力'"——"'有'弱点'"。"'屋大维'不是'一个'符号'"——"'是'一个'精明的'政治家'"——"'他'建立'了'帝国'"——"'但'也'失去'了'朋友'"。"盐野七生说'历史'不是'事件'的'记录'"——"'是'人的'故事'"
- **盐野七生的背景**: 盐野七生（1937-）——日本历史学家、作家——"《罗马人的故事》的作者'"。"她'26 岁'时'去'意大利'"——"'在'那里'生活'了'几十年'"——"'研究'罗马历史'"。"《罗马人的故事》'共 15 卷'"——"'写了'15 年'"——"'1992 年'开始'出版'"——"'2006 年'完成'"。"这套书'在日本'和'中国'都'非常畅销'"——"'它'让'很多人'对'罗马历史'产生'兴趣'"
