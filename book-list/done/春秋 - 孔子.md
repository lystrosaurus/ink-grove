# 春秋

状态：已读

作者：孔子（修订）

一句话评价：中国"第一部编年体史书"——记载了"鲁国从隐公元年（前 722 年）到哀公十四年（前 481 年）共 242 年的历史"。孔子"据鲁史修订"——开创了"春秋笔法"的史学传统——"用极简的文字隐含褒贬"——"微言大义"——"一个字的选用就有道德评判"。春秋时期是"礼崩乐坏的时代"——"周天子权威衰落"——"诸侯争霸"——孔子用《春秋》来"维护道义秩序"。

核心收获：
- **春秋笔法**: 孔子修《春秋》最核心的贡献是"春秋笔法"——"用极简的文字隐含褒贬"。"微言大义"——"一句话中'暗含'深刻的'道德评判'"。比如"杀"和"弑"——"杀"是"中性"的——"弑"是"贬义"的——"臣杀君叫'弑'"——"用'弑'字就是'谴责'这种行为"。比如"伐"和"侵"——"伐"是"有正当理由的攻打"——"侵"是"无正当理由的入侵"——"用一个字就表达了'孔子的是非判断'"
- **礼崩乐坏的时代**: 春秋时期是"中国历史上'最动荡'的时代之一"。"周天子的权威'衰落'"——"诸侯'不听'天子的"——"大国'吞并'小国"——"臣子'杀死'国君"——"儿子'杀死'父亲"——"传统的'礼乐秩序'全面崩溃"。孔子"痛心疾首"——他说"八佾舞于庭——是可忍也——孰不可忍也"（季氏用了天子规格的舞蹈——"这种事都能忍——还有什么不能忍"）。《春秋》就是"孔子对'礼崩乐坏'的'回应'"
- **孔子的道德史观**: 孔子修《春秋》的目的不是"记录历史"——是"维护道义"。他"用'历史'来'教育'后人"——"让'乱臣贼子惧'"（孟子语）。孔子的"道德史观"是"历史不仅仅是'发生了什么'"——是"什么'是对的'——什么'是错的'"。他"通过'文字的选用'来表达'是非判断'"——让"后人在'读历史'的同时'学到'做人的道理"
- **春秋三传**: 《春秋》本身"极其简略"——"全文只有约 16000 字"——"平均每年'只写了'60 多字"——"一句话记录'一年的大事'"。所以后人写了"三传"来"解释《春秋》"：1）**《左传》**（左丘明）——"最详细"——"用'丰富的故事'来'补充'《春秋》的'简略记录'"——是"中国最早的'叙事体'史书"；2）**《公羊传》**——"侧重'义理'"——"解释《春秋》中的'微言大义'"；3）**《谷梁传》**——"也侧重'义理'"——"与《公羊传》'观点有时不同'"
- **重要事件**: 《春秋》记载了"春秋时期'最重要的历史事件'"：1）**郑伯克段于鄢**——"郑庄公'击败'叛乱的弟弟"——孔子用"克"字（"兄弟相残——如两国相'克'"）来"谴责"；2）**齐桓公称霸**——"齐桓公在管仲辅佐下'成为霸主'"——孔子"肯定"他"尊王攘夷"的功绩；3）**晋楚城濮之战**——"晋文公'击败'楚军"——奠定了"晋国的霸主地位"；4）**弭兵之会**——"晋楚两国'停战'"——"小国'暂时获得'了和平"
- **《春秋》的地位**: 《春秋》是"儒家五经之一"——是"中国古代'史学的'源头'"。它开创了"三个传统"：1）**编年体**——"按'时间顺序'记录历史"——后来的"《资治通鉴》"继承了这一传统；2）**春秋笔法**——"用文字表达'是非判断'"——成为中国史学的"核心方法"；3）**以史为鉴**——"历史不只是'记录'"——是"'教训'和'镜子'"——"以史为鉴可以知兴替"
- **孔子的背景**: 孔子（公元前 551-前 479 年）——名丘——字仲尼——鲁国人。他"修订《春秋》"是在"晚年"——"回到鲁国后"——"根据鲁国的官方史书'整理'而成"。孟子说"孔子成《春秋》——而乱臣贼子惧"——"《春秋》的'威慑力'"来自于"它'代表了'道德评判'的标准"。孔子修《春秋》的"最终目的"不是"写历史"——是"'匡正天下'"——"用'道德的力量'来'纠正'礼崩乐坏的局面"
