# 孝经

状态：已读

作者：曾子

一句话评价：儒家十三经中篇幅最短的一部（仅一千八百余字），却系统构建了从个人修身到治国平天下的"孝道"伦理体系。"孝"在中国不仅是家庭伦理，更是政治伦理和社会秩序的基石。

核心收获：
- **孝的三重境界**: "身体发肤，受之父母，不敢毁伤，孝之始也。立身行道，扬名于后世，以显父母，孝之终也。"孝的起点是爱护自己的身体（不让父母担心），中间是侍奉父母、忠于职守，终点是"立身行道"——成就一番事业，让父母的名声因你而荣耀
- **"孝"与"治"的同构**: 《孝经》最核心的政治哲学是"以孝治天下"。"夫孝，始于事亲，中于事君，终于立身。"家庭中的"孝"（对父母）直接对应政治中的"忠"（对君主）。汉代"举孝廉"制度就是这一思想的制度化——选官先看孝不孝
- **五等之孝**: 《孝经》把"孝"分为五个等级：天子之孝（爱敬天下人的父母）、诸侯之孝（保其社稷）、卿大夫之孝（守其宗庙）、士之孝（保其禄位）、庶人之孝（谨身节用）。不同身份的人，孝的内涵不同。天子之孝是"博爱广敬"，庶人之孝是"勤俭养亲"
- **"谏诤"的智慧**: 《孝经》不提倡"愚孝"。"父有争子，则身不陷于不义。"父母有过错，子女必须劝谏（谏诤）。如果不劝谏，就是陷父母于不义，这是更大的不孝。这和后世"天下无不是的父母"的极端化完全不同
- **孝与礼的关系**: 孝不仅是情感，更是制度。"生，事之以礼；死，葬之以礼，祭之以礼。"父母在世时按礼节侍奉，去世后按礼节约束哀伤（"三日而食""三年之丧"）。礼是孝的外在表现，孝是礼的内在动力
- **《孝经》的成书**: 传统认为是曾子记录孔子的言论，但现代学者多认为是战国至汉代儒家学者的托名之作。它的成书过程反映了"孝"观念从孔子时代的"家庭伦理"到汉代的"国家意识形态"的演变
- **孝道的现代反思**: 《孝经》中的"孝"有时代局限——它建立在等级制和父权制基础上，强调"顺从"多于"平等"。但它的核心精神——感恩、责任、代际关怀——在现代社会依然有价值。关键在于剥离等级制的外壳，保留"互爱互敬"的内核
- **《孝经》与《论语》的关系**: 《论语》讲孝是零散的（"今之孝者是谓能养""色难"），《孝经》把孝系统化、理论化、政治化。可以说《论语》是孝的"散论"，《孝经》是孝的"专论"
