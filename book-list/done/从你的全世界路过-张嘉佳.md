# 从你的全世界路过

状态：已读

作者：张嘉佳

一句话评价：张嘉佳"睡前故事集'"——"'段子手的幽默加诗人的深情'"。"'茅十八和荔枝的爱情展现理工男的极致浪漫'"。"'猪头的痴情引发反思'"。"'这个人也许永远不回来了，也许明天回来是中国现代文学最有名的结尾'"。"'网络文学走向主流文学的典型案例'"。"'笑中带泪的风格打动数百万读者'"

核心收获：
- **茅十八和荔枝的爱情**: 本书核心是"'茅十八和荔枝的爱情'"。"'理工男'的'极致'浪漫'"——"'把'导航'语音'换成'荔枝'"——"'我'好'喜欢'你'"。"'让'荔枝'每次'开车'都'被'表白'"。"'科技'与'爱情'的'结合'"——"'把'冷冰冰'的'机器'变成'了'温暖'的'信使'"。"'这个'故事'告诉'我们'"："'浪漫'不'是'花'多少'钱'"——"'是'花'了'多少'心思'"
- **陈末和幺鸡的救赎**: 本书"讲述了陈末和幺鸡的故事'"。"'电台'DJ 陈末'和'听众'幺鸡'的'互相'救赎'"。"'两'个'孤独'的'人'在'城市'中'相遇'"——"'彼此'温暖'"。"'陈末'用'声音'陪伴'幺鸡'度过'最'黑暗'的'时光'"。"'幺鸡'用'真诚'唤醒'了'陈末'对'生活'的'热情'"。"'这个'故事'的'核心'是'陪伴'"
- **猪头的痴情**: 本书"描绘了猪头的痴情'"。"'燕子'"——"'没有'你'我'怎么'活'"。"'猪头'追着'燕子'离开'的'出租车'狂奔'"——"'是'全书'最'催泪'的'场景'"。"'猪头'的'爱情'是'付出型'的'"。"'他'把'自己'的'一切'都'给'了'燕子'"。"'但'燕子'还是'走'了'"。"'这个'故事'不'是'歌颂'痴情'"——"'是'反思'痴情'"
- **路过的意义**: 本书"探讨了路过的意义'"。"'书名'从'你的'全世界'路过'的'含义'"："'每个'人'都会'从'你'的'全世界'路过'"。"'有'的'人'停留'"——"'有'的'人'离开'"。"'重要'的'是'那些'停留'的'瞬间'"。"'路过'不'是'遗憾'"——"'是'经历'"
- **张嘉佳的叙事风格**: 本书"体现了张嘉佳的叙事风格'"。"'段子手'的'幽默'加'诗人'的'深情'"。"'张嘉佳'的'写法'是'先'让'你'笑'"——"'再'让'你'哭'"。"'他'用'幽默'的'段子'开头'"——"'用'深情'的'结尾'收尾'"
- **城市孤独感**: 本书"反映了城市孤独感'"。"'每个'故事'都'在'讲'都市人'的'孤独'与'渴望'连接'"。"'南京'"——"'北京'"——"'上海'"。"'这些'大'城市'里'有'无数'孤独'的'人'"。"'张嘉佳'抓住'了'这个'时代'最'普遍'的'情感'"——"'孤独'"
- **网络时代的文学现象**: 本书"是网络时代的文学现象'"。"'这'本书'最初'是'张嘉佳'在'微博'上'写'的'睡前'故事'"。"'每天'一个'"——"'积累'了'33 个'后'结集'出版'"。"'它'是'网络'文学'走向'主流'文学'的'典型'案例'"
- **作者的背景**: 张嘉佳（1980-）——中国作家、编剧——"《从你的全世界路过》的作者'"。"'他'出生'于'江苏'南通'"。"'《从'你的'全世界'路过》'2013 年'出版'"。"'他'的'其他'著作'"："'《让我'留在'你'身边》'"。"'张嘉佳'的'风格'"："'幽默'"——"'深情'"——"'笑中带泪'"
