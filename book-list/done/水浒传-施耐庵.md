# 水浒传

状态：已读

作者：施耐庵

一句话评价：施耐庵的"古典名著"——中国"四大名著"之一——"108 位好汉聚义梁山"——"逼上梁山"成为了"中国文化符号"。"替天行道"——"官逼民反"——"兄弟义气"——"英雄群像"——"招安悲剧"。《水浒传》是"一部'悲剧史诗'"——"一群'好人'被'逼上'了'绝路'"——最终"在'招安'中'走向灭亡'"。

核心收获：
- **108 将聚义**: 《水浒传》最"引人入胜"的部分是"108 位好汉'一个一个地上梁山'"。每一个好汉都有"自己的故事"——"林冲"（被高衙内陷害——"逼上梁山"）、"武松"（为兄报仇——"杀了西门庆和潘金莲"）、"鲁智深"（为救金翠莲——"三拳打死镇关西"）、"宋江"（杀了阎婆惜——"被迫逃亡"）。每个人都是"被逼无奈"——"没有一个'天生'想当'强盗'"
- **逼上梁山**: "逼上梁山"是"全书的核心主题"——也是"中国文化中最'深入人心'的成语之一"。"官逼民反"——"不是'百姓想反'"——是"官府'逼得'他们'无路可走'"。林冲"本来是一个'安分守己'的教头"——"被高俅'一步步逼'到了'梁山'"。杨志"本来是'名门之后'"——"被'贪官污吏'逼得'卖刀杀人'"。"逼上梁山"的"悲剧性"在于"好人'没有'好出路"
- **替天行道**: 梁山的"口号"是"替天行道"——"替'上天'执行'正义'"——因为"人间的'法律'已经'失效了'"。但"替天行道"本身就是一个"悖论"——"一群'强盗'在'执行正义'"——这"本身就'不合理'"。施耐庵通过这个口号"表达了对'社会不公'的'愤怒'"——"当'合法途径'走不通时"——"'暴力'是否'正当'"
- **兄弟义气**: "兄弟义气"是"水浒英雄'最看重的'品质"。"为兄弟'两肋插刀'"——"为兄弟'赴汤蹈火'"——"为兄弟'死也甘心'"。但"义气"也是"梁山'最终灭亡'的原因之一"——"宋江为了'义气'"——"接受了'招安'"——"带着兄弟们'去征方腊'"——"结果'死伤大半'"。"义气"在"个人层面"是"美德"——在"政治层面"可能是"灾难"
- **招安悲剧**: 《水浒传》最"令人心痛"的部分是"招安"。宋江"一直想'被朝廷招安'"——他认为"当'强盗'不是'长久之计'"——"只有'被招安'"——"才能'名正言顺'"。但"招安的结果"是"悲剧的"——"朝廷'利用'梁山好汉'去'征方腊'"——"108 将'死了'大半"——"活下来的'也没有'好下场'"。施耐庵用"招安的悲剧"表达了"对'封建统治'的'深刻批判'"
- **英雄群像**: 《水浒传》最"了不起"的地方是"塑造了'108 个'鲜活的人物'"——"每一个都有'鲜明的个性'"。"鲁智深的'豪爽'"——"林冲的'隐忍'"——"武松的'刚烈'"——"李逵的'鲁莽'"——"吴用的'机智'"——"花荣的'忠义'"。施耐庵用"寥寥数笔"就能"让一个人物'立起来'"——这是"中国古典小说中'最出色'的人物群像"
- **施耐庵的背景**: 施耐庵（约 1296-约 1370）——元末明初小说家——《水浒传》的作者（与罗贯中合著的说法也存在）。《水浒传》的故事"源于'北宋末年'的'宋江起义'"——在"民间流传了'几百年'"——施耐庵将其"整理、加工、创作"成了"一百二十回的长篇小说"。《水浒传》是"中国文学史上'第一部'以'农民起义'为题材的'长篇小说'"
