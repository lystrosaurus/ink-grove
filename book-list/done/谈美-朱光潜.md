# 谈美

状态：已读

作者：朱光潜

一句话评价：朱光潜的"中国美学最佳入门书'"——"'以书信体娓娓道来'"。"'将西方美学与中国传统艺术融会贯通'"——"'文字亲切通透，读来如与一位智者对谈'"。"'美感态度需要与现实功利保持距离'"——"'无所为而为地欣赏'"——"'人生的艺术化是终极关怀'"

核心收获：
- **美感态度**: 朱光潜提出"'美感态度'"的核心概念'"。"'什么'是'美感态度'"——"'审美'需要'与'现实'功利'保持'距离'"。"'你'看'一棵'松树'"——"'如果'想'它'能'卖'多少'钱'"——"'这'不'是'美感'"。"'如果'你'只'欣赏'它'的'形态'"——"'它'的'姿态'"——"'这'才'是'美感'"。"'朱光潜'说'无所为而为'"——"'不'为'实用'目的'"——"'只'为'欣赏'本身'"。"'这种'态度'"——"'是'审美'的'前提'"——"'没有'这种'态度'"——"'就'没有'美感'"
- **移情作用**: 朱光潜"解释了移情作用'"。"'什么'是'移情'"——"'我们'把'自己'的'情感'投射'到'对象'上'"。"'比如'我们'看'柳树'"——"'觉得'柳树'在'悲伤'"——"'其实'柳树'没有'情感'"——"'是'我们'在'悲伤'"。"'移情'的'作用'"：1）**产生共鸣**——"'我们'与'对象'产生'情感'连接'"；2）**美化对象**——"'对象'变'得'有'生命'"。"'移情'是'审美'的'关键'"——"'没有'移情'"——"'对象'是'冷冰冰'的'"
- **人生的艺术化**: 朱光潜的"终极关怀是人生的艺术化'"。"'什么'是'人生的艺术化'"——"'把'人生'当'成'艺术品'"——"'用'审美'的'态度'过'生活'"。"'不'是'要'你'成为'艺术家'"——"'是'要'你'有'艺术家'的'态度'"。"'如何'做到'"：1）**超脱功利**——"'不'被'名利'束缚'"；2）**欣赏当下**——"'欣赏'生活'的'每一刻'"；3）**创造美好**——"'让'自己'的'生活'变'得'美'"。"'朱光潜'说'"——"'人生'的'艺术化'"——"'是'人格'的'完成'"
- **距离说**: 朱光潜"介绍了布洛的距离说'"。"'距离'是'什么'"——"'审美'需要'适当'的'距离'"。"'太'近'"——"'你'被'功利'束缚'"——"'没有'美感'"。"'太'远'"——"'你'不'理解'对象'"——"'也'没有'美感'"。"'适当'的'距离'"——"'既'超脱'功利'"——"'又'能'理解'对象'"。"'比如'看'戏'"——"'你'知道'是'假'的'"——"'但'你'投入'情感'"——"'这'是'适当'距离'"
- **艺术与道德**: 朱光潜"讨论了艺术与道德的关系'"。"'艺术'与'道德'是'两回事'"。"'艺术'的'标准'"——"'美'不'美'"。"'道德'的'标准'"——"'善'不'善'"。"'有'的'作品'美'但'不'善'"——"'有'的'作品'善'但'不'美'"。"'但'朱光潜'认为'"——"'最高'的'艺术'"——"'美'与'善'统一'"。"'艺术家'的'人格'"——"'影响'作品'的'境界'"
- **书信体风格**: 本书"采用书信体风格'"。"'朱光潜'写'给'一个'青年'"——"'用'朋友'的'口吻'"。"'这种'风格'的'好处'"：1）**亲切**——"'像'对谈'"——"'不'说教'"；2）**通俗**——"'用'简单'的'语言'"——"'不'用'术语'"；3）**系统**——"'13 封信'"——"'涵盖'美学'的'主要'问题'"。"'这种'风格'"——"'让'普通'读者'也'能'读'"
- **朱光潜的背景**: 朱光潜（1897-1986）——中国美学家、文艺理论家——"《谈美》的作者'"。"他'在'英国'学习'美学'"——"'回来'后'在'北大'——'武大'任教'"。"《谈美》'1932 年'出版'"——"'是'他'为'青年'写'的'美学'入门'"。"'他'的'其他'著作'"——"'《西方美学史》'——'《文艺心理学》'"。"'朱光潜'的'贡献'"——"'把'西方'美学'介绍'给'中国'"——"'用'中国'的'例子'解释'"
