# 花朵的秘密生命

状态：已读

作者：沙曼·阿普特萝赛

一句话评价：沙曼·阿普特萝赛（Sharman Apt Russell）的"植物学科普佳作'"——"'以花为主题，揭示花朵在色彩、气味、形态背后的进化策略和生态关系'"。"'花朵的一切特征都是与传粉者长期协同进化的结果'"——"'花不是为人类的美而存在'"——"'植物的智慧远超人类想象'"

核心收获：
- **协同进化**: 本书的核心是"'协同进化'（Co-evolution）"的概念'"。"'花朵'和'传粉者'"——"'昆虫'——'鸟类'——'蝙蝠'"——"'长期'共同'进化'"。"'花朵'的'颜色'"——"'吸引'特定'传粉者'"——"'蜜蜂'喜欢'蓝色'和'黄色'"——"'蜂鸟'喜欢'红色'"。"'花朵'的'气味'"——"'吸引'特定'传粉者'"——"'香甜'吸引'蜜蜂'"——"'腐臭'吸引'苍蝇'"。"'花朵'的'形状'"——"'匹配'特定'传粉者'"——"'长'花冠'匹配'长'喙'的'蛾'"。"'这种'协同进化'持续'了'数千万年'"——"'形成'了'精妙'的'适应'"
- **花的颜色**: 阿普特萝赛"详细讲解了花朵颜色的进化'"。"'为什么'花'有'颜色'"——"'不'是'为了'人类'的'美'"——"'是'为了'吸引'传粉者'"。"'不同'传粉者'看到'的'颜色'不同'"：1）**蜜蜂**——"'能'看到'紫外线'"——"'花'有'紫外线'图案'"——"'引导'蜜蜂'找到'花蜜'"；2）**鸟类**——"'能'看到'红色'"——"'红色'花'主要'由'鸟类'传粉'"；3）**蝙蝠**——"'夜间'活动'"——"'白色'花'在'月光'下'显眼'"。"'所以'花的'颜色'是'一种'信号'"——"'是'花'与'传粉者'的'语言'"
- **花的气味**: 阿普特萝赛"分析了花朵气味的策略'"。"'气味'是'化学'信号'"——"'在'夜间'特别'重要'"。"'香甜'气味'"——"'吸引'蜜蜂'——'蝴蝶'"。"'腐臭'气味'"——"'吸引'苍蝇'"——"'苍蝇'以为'是'腐肉'"——"'来'产卵'"——"'顺便'传粉'"。"'夜间'开花'的'花'"——"'气味'特别'浓烈'"——"'因为'夜间'视觉'不'重要'"——"'气味'是'主要'信号'"。"'有些'花'能'模拟'昆虫'的'性激素'"——"'吸引'雄虫'"——"'雄虫'以为'是'雌虫'"——"'来'交配'"——"'顺便'传粉'"
- **花的形状**: 阿普特萝赛"描述了花朵形状的适应性'"。"'花的'形状'是'为'特定'传粉者'设计'的'"。"'管状'花'"——"'适合'长'喙'的'蜂鸟'——'蛾'"——"'其他'昆虫'够'不到'花蜜'"。"'唇形'花'"——"'适合'蜜蜂'降落'"——"'蜜蜂'的'重量'打开'花'的'机关'"——"'花粉'弹'到'蜜蜂'身上'"。"'辐射对称'花'"——"'任何'方向'都'能'访问'"。"'两侧对称'花'"——"'只有'特定'方向'能'访问'"——"'更'精确'地'控制'传粉'"
- **植物的智慧**: 阿普特萝赛"展示了植物的智慧'"。"'植物'能'感知'"：1）**光线**——"'知道'什么'时候'开花'"——"'知道'光'的'方向'"；2）**温度**——"'知道'季节'变化'"；3）**化学信号**——"'能'闻到'邻近'植物'的'气味'"；4）**触觉**——"'含羞草'被'碰'会'闭合'"。"'植物'能'交流'"——"'被'虫害'的'植物'释放'化学'信号'"——"'邻近'植物'收到'信号'"——"'提前'产生'防御'物质'"。"'这种'智慧'不'需要'大脑'"——"'是'亿万年'进化'的'结果'"
- **花与人类**: 阿普特萝赛"也讨论了花与人类的关系'"。"'人类'被'花'吸引'"——"'可能'是'进化'的'结果'"——"'花'意味着'水源'"——"'意味着'果实'"。"'花'在'人类'文化'中'的'地位'"：1）**爱情**——"'玫瑰'象征'爱情'"；2）**宗教**——"'莲花'在'佛教'中'神圣'"；3）**艺术**——"'花'是'绘画'——'诗歌'的'主题'"。"'但'人类'的'欣赏'不'影响'花'的'进化'"——"'花'不'为'人类'而'美'"——"'花'为'传粉者'而'美'"
- **阿普特萝赛的背景**: 沙曼·阿普特萝赛（Sharman Apt Russell）——美国作家、自然科学家——"《花朵的秘密生命》的作者'"。"她'在'新墨西哥州'生活'"——"'那里'的'沙漠'生态'影响'了'她'的'写作'"。"《花朵的秘密生命》'2005 年'出版'"——"'获得'好评'"。"她'的'写作风格'"——"'科学'与'文学'结合'"——"'既'准确'又'优美'"。"她'的'其他'著作'"——"'《饥饿》'——'《边界》'"——"'都'是'自然'主题'"
