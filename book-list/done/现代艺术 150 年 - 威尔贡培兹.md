# 现代艺术 150 年

状态：已读

作者：威尔·贡培兹

一句话评价：威尔·贡培兹（Will Gompertz）的"最好的现代艺术入门书之一"——"以'泰特美术馆前高管'的身份"——"用'幽默通俗的语言'梳理了'从印象派到当代艺术的'150 年发展历程"——"让'看不懂的'现代艺术变得'可以理解'"。"现代艺术的'核心转向'"——"从'画得像'到'表达什么'"——"'形式'让位于'观念'"。"杜尚的'现成品'艺术（小便池）"——"彻底'重新定义了'什么是艺术"。

核心收获：
- **现代艺术的发展脉络**: 贡培兹"用'一条清晰的线索'"串起了"现代艺术的 150 年"：1）**印象派**（莫奈、雷诺阿）——"关注'光和色'"——"不再追求'画得像'"；2）**后印象派**（梵高、塞尚、高更）——"关注'情感和形式'"；3）**野兽派**（马蒂斯）——"用'大胆的色彩'"；4）**立体主义**（毕加索、布拉克）——"从'多个角度'同时'描绘一个物体'"；5）**达达主义**（杜尚）——"挑战'什么是艺术'"；6）**超现实主义**（达利、马格利特）——"描绘'梦境和潜意识'"；7）**抽象表现主义**（波洛克、罗斯科）——"纯粹的'色彩和形式'"；8）**波普艺术**（沃霍尔）——"把'大众文化'变成'艺术'"；9）**观念艺术**——"'想法'比'作品'更重要"
- **从"画得像"到"表达什么"**: 贡培兹说"现代艺术的'核心转向'"——"是从'画得像'到'表达什么'"。"在'印象派之前'"——"艺术的标准是'像不像'"——"画得越像'就'越好'"。"但'照相机'发明后"——"画得'再像'也'比不过'照相机'"——"所以'艺术家们'开始'寻找'新的方向'"。"他们'不再追求'像'"——"而是追求'表达'"——"表达自己的'情感'"、"'观念'"、"'对世界的'理解"
- **杜尚的现成品**: 贡培兹说"杜尚的'现成品'艺术（小便池）'彻底重新定义了'什么是艺术"。1917 年——"杜尚把'一个'普通的小便池'"——"'签上'自己的名字"——"'命名为'《泉》（Fountain）"——"'送到'艺术展'"。"所有人'都觉得'这是'一个玩笑'"——"但杜尚说'这就是艺术'"——"因为'他选择了它'"——"他'赋予了它'新的'语境'"。"杜尚的贡献是'让'所有人'意识到'"——"'艺术不是'关于'物品本身'"——"是'关于'语境和观念'"
- **看不懂的艺术**: 贡培兹"专门解释了'为什么现代艺术'让人'看不懂'"。"因为'现代艺术'不再'描绘'你'能认出的'东西'"——"它'在表达'一种'观念'或'情感'"。"比如'波洛克的'滴画'"——"看起来'就是'一堆'乱七八糟的'颜料'"——"但'波洛克'想表达的是'"——"'创作过程中的'自由'和'随机性'"。"比如'罗斯科的'色块画'"——"看起来'就是'几个'大色块'"——"但'罗斯科'想表达的是'"——"'人类最基本的'情感"——"悲剧、狂喜、毁灭"
- **泰特美术馆的视角**: 贡培兹"作为'泰特美术馆的前高管'"——"他的'视角'很独特"。"他'不是'一个'纯粹的艺术史学家'"——"他'是一个'艺术的'推广者'"——"他的'使命'是'让'普通人'也能'理解'现代艺术'"。"所以他'用'幽默的语言'"——"'用'生活中的'例子'"——"'用'故事'而不是'理论'"——"来'讲解'艺术"——"这让'这本书'非常'好读'"
- **艺术与时代的关系**: 贡培兹"反复强调'艺术与时代的关系'"。"每一次'艺术运动'的'兴起'"——"都'与'当时的'社会变革'有关'"。"印象派'兴起于'工业革命后'"——"'城市生活'变得'丰富多彩'"——"艺术家们'想捕捉'这种'变化'"。"达达主义'兴起于'一战期间'"——"'人们对'理性'和'文明'感到'绝望'"——"所以'用'荒诞'来'抗议'"。"波普艺术'兴起于'消费社会'"——"'大众文化'成为了'新的'艺术素材"
- **威尔·贡培兹的背景**: 威尔·贡培兹（Will Gompertz）——英国艺术评论家——"泰特美术馆前传播总监"——"BBC 艺术编辑"。他"擅长'用'通俗易懂的语言'讲解'艺术'"——"他的'Talks'系列视频'在'YouTube 上'有'数百万次'播放"——"他'让'很多'不看艺术的人'开始'对艺术'感兴趣"。《现代艺术 150 年》是他"最畅销的作品"——"被'翻译成'多种语言"——"是'现代艺术入门'的'最佳读物之一'"
