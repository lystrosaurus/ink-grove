# 利维坦

状态：已读

作者：霍布斯

一句话评价：西方政治哲学的奠基之作，霍布斯用"自然状态是人类对人类的战争"这一冷酷假设，论证了强大的主权者（利维坦）是维持社会秩序的必要条件。没有政府，人生是"孤独、贫困、卑污、残忍而短寿"的。

核心收获：
- **自然状态（State of Nature）**: 霍布斯的思想实验——如果没有政府、没有法律、没有公共权力，人类会处于什么状态？答案是"所有人对所有人的战争"（bellum omnium contra omnes）。不是因为人类"天生邪恶"，是因为资源有限、人人平等（最弱的人也能杀死最强的人）、没有信任。在自然状态下，没有"你的"和"我的"之分——谁强谁就有
- **人生是"孤独、贫困、卑污、残忍而短寿"**: 这是霍布斯对自然状态下人类生活的著名描述。没有工业、没有农业、没有艺术、没有文学、没有社会——因为所有精力都用在"保护自己"上。这不是"历史事实"，是"逻辑推演"——霍布斯用这个推演证明为什么需要政府
- **社会契约论**: 为了逃离自然状态，人们"自愿"让渡部分权利给一个主权者（个人或者集体），换取安全和秩序。这个契约不是"人民和政府之间的"，是"人民彼此之间的"——我们互相约定服从同一个主权者。主权者不受契约约束——因为主权者不是签约方，他是契约的执行者
- **利维坦的隐喻**: 书名"利维坦"是《圣经》中的海中巨兽，象征绝对权力。霍布斯说主权者就是"人造的神"——他的权力不是来自上帝，是来自人民的授权。但这个授权是不可撤销的——一旦你让渡了权利，就不能收回。这是霍布斯和洛克、卢梭的根本分歧
- **绝对主权的必要性**: 霍布斯认为主权者的权力必须是"绝对的"——不能被分割、不能被限制、不能被挑战。因为任何对主权的削弱都会导致社会回到自然状态。他反对分权（立法、行政、司法分立），认为分权就是"多头政治"，必然导致内战
- **霍布斯的人性观**: 霍布斯是"性恶论"者——他认为人类行为的根本动机是"自我保存"和"权力欲望"。"人对人是狼"（homo homini lupus）。这不是道德批判，是"科学观察"——霍布斯试图用机械论的方式解释人类行为，就像解释物理现象一样
- **霍布斯与洛克、卢梭的对比**: 三人都用"自然状态 + 社会契约"的框架，但结论完全不同。霍布斯：自然状态是战争→需要绝对主权。洛克：自然状态是自由的但有不便→需要有限政府（保护财产权）。卢梭：自然状态是美好的但被文明腐蚀→需要直接民主。三种方案代表了政治哲学的三种基本立场
- **《利维坦》的当代意义**: 霍布斯的问题在今天依然尖锐——安全和自由怎么平衡？当恐怖主义、疫情、网络攻击威胁社会秩序时，我们愿意让渡多少自由来换取安全？霍布斯的回答是"尽可能多"——因为"没有安全，一切自由都是空的"。这个回答不让人舒服，但很难反驳
