# 博弈论

状态：已读

作者：迪克西特

一句话评价：博弈论最佳入门书，用大量日常案例讲解囚徒困境、纳什均衡、承诺与威胁等核心概念，不需要数学基础也能理解策略性思维的精髓。

核心收获：
- 理解了"策略性思维"的本质：在做决定时不仅要考虑自己的行动，还要考虑对方会如何回应
- 囚徒困境揭示了个人理性与集体理性的冲突——每个人都做出对自己最有利的选择，结果却对所有人都不利
- 承诺、威胁和可信度的区别：一个策略只有在对方相信你会上台时才有效
