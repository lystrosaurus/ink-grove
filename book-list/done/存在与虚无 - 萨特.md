# 存在与虚无

状态：已读

作者：让 - 保罗·萨特

一句话评价：萨特存在主义哲学的集大成之作（1943 年出版），近 800 页的巨著系统阐述了"存在先于本质""自由与责任""他者即地狱""自欺（mauvaise foi）"等核心命题。通过"自在存在"（en-soi）和"自为存在"（pour-soi）的区分，萨特论证了人的绝对自由——"人被判定为自由"，必须为自己的选择承担全部责任。这是 20 世纪最重要的哲学著作之一。

核心收获：
- **自在存在 vs 自为存在**: 萨特把世界上的存在分为两种。"自在存在"（being-in-itself / en-soi）是"物"的存在——石头、桌子、树。物的存在是"固定的、完成的、没有可能性的"——一块石头就是那块石头，它不会变成别的东西。"自为存在"（being-for-itself / pour-soi）是"人"的存在——人的存在是"开放的、未完成的、充满可能性的"。人永远"不是"他"现在是的样子"——因为你下一秒就可以改变。人的本质是"虚无"——不是"没有"，是"有无限的可能性"
- **存在先于本质**: 这是萨特最著名的命题。"本质"是"一个事物的定义"——刀的本质是"用来切割的工具"，在刀被制造出来之前，它的本质就已经被定义好了。但人不同——人"首先存在"，然后通过自己的选择和行动"定义自己是谁"。你不是"天生的好人或坏人"——你是通过一次次选择"成为了好人或坏人"。没有"人性"这回事——人性是你自己创造的
- **自由与责任**: 萨特说"人被判定为自由"（condemned to be free）。"判定"（condemned）这个词很重要——自由不是"礼物"，是"判决"。因为你没有"上帝"、"命运"、"人性"可以依靠——你必须自己做出每一个选择，而且你必须为每一个选择承担全部责任。你不能说"我这样做是因为我的性格""是因为社会环境""是因为上帝让我这么做"——这些借口都是"自欺"（mauvaise foi）
- **自欺（mauvaise foi / bad faith）**: "自欺"是萨特最深刻的心理学洞察——人经常"骗自己"说自己"没有选择"。比如一个服务员过度热情——他不是在"真诚地服务"，是在"扮演服务员的角色"，好像他的行为是"角色要求的"而不是"他自己选择的"。萨特说这就是"自欺"——你明明有选择，但你假装"不得不这样做"。自欺是一种"逃避自由"的方式——因为自由太沉重了，承担全部责任太可怕了
- **他者即地狱**: 萨特在戏剧《禁闭》中说"他人即地狱"（L'enfer, c'est les Autres）。这不是说"别人都是坏人"——是说"人与人之间的关系本质上是冲突的"。原因是：每个人都想"定义别人"（把别人当成"客体"），但每个人都不想"被别人定义"（不想成为"客体"）。当两个人相遇时，他们在争夺"主体性"——"我是看的人，不是被看的人"。这种"争夺"是人际关系的核心动力——爱情、嫉妒、羞耻、骄傲都源于此
- **焦虑（anguish）**: 萨特说当你意识到"你是完全自由的"时，你会感到"焦虑"。不是因为"你害怕做不好"，是因为"你意识到没有任何东西可以替你决定"。萨特举了一个士兵的例子——他在战场上害怕的不是"死亡"，是"他的选择决定了他的生死"。这种焦虑不是"病理性的"——它是"自由的必然伴随物"。你越自由，你越焦虑
- **虚无的力量**: 萨特说人的意识本质上是"虚无"——意识不是"一个东西"，是"对东西的意识"。你看一棵树——你的意识不是"树"，是"对树的意识"。意识永远"指向"某物，但本身"什么都不是"。这种"虚无"恰恰是"自由"的基础——因为意识"什么都不是"，所以它"可以成为任何东西"
- **萨特与海德格尔**: 萨特深受海德格尔影响（《存在与时间》1927 年），但两人的思想有重要区别。海德格尔关注"存在"的问题（存在 vs 存在者），萨特关注"人的自由"的问题。海德格尔后来与萨特划清界限（认为萨特误解了他的思想），但萨特的存在主义比海德格尔更"入世"——他积极参与政治（反战、反殖民、1968 年学生运动），是 20 世纪最有公共影响力的哲学家之一
- **萨特的背景**: 让 - 保罗·萨特（1905-1980）是法国哲学家、作家、剧作家。《存在与虚无》（1943）是他最重要的哲学著作，他因此拒绝了 1964 年的诺贝尔文学奖（他说"作家不应该被制度化"）。他与西蒙娜·德·波伏娃的"开放式关系"是 20 世纪知识分子生活的传奇。萨特的存在主义在二战后风靡全球——因为它给了人们一个"在没有上帝的世界中如何活下去"的答案
