# 贫穷的本质

状态：已读

作者：阿比吉特·班纳吉、埃丝特·迪弗洛

一句话评价：阿比吉特·班纳吉（Abhijit Banerjee）和埃丝特·迪弗洛（Esther Duflo）的"诺贝尔经济学奖获奖研究'"——"'通过在发展中国家的实地实验'"。"'揭示了穷人为什么穷、以及什么方法能真正帮助他们脱贫'"。"'贫穷陷阱：穷人不是因为懒或笨而穷，而是因为贫穷本身制造了更多贫穷'"

核心收获：
- **贫穷陷阱**: 本书核心是"'贫穷陷阱'"的概念'"。"'什么'是'贫穷'陷阱'"——"'穷人'不'是'因为'懒'或'笨'而'穷'"——"'因为'贫穷'本身'制造'了'更多'贫穷'"。"'比如'"：1）**健康**——"'穷人'没钱'看病'"——"'生病'"——"'不'能'工作'"——"'更'穷'"；2）**教育**——"'穷人'没钱'上学'"——"'没'技能'"——"'只能'做'低'收入'工作'"——"'更'穷'"；3）**信贷**——"'穷人'借不到'钱'"——"'不'能'投资'"——"'没'法'脱贫'"。"'贫穷'陷阱'的'启示'"——"'帮助'穷人'"——"'要'打破'陷阱'"
- **实地实验方法**: 本书"采用了实地实验方法'"。"'什么'是'实地'实验'"——"'在'真实'世界'中'做'实验'"。"'比如'想'知道'免费'校餐'是否'有效'"——"'随机'选择'一些'学校'"——"'提供'免费'校餐'"——"'对比'没有'提供'的'学校'"。"'这种'方法'的'好处'"：1）**因果推断**——"'能'确定'是'干预'导致'结果'"；2）**真实**——"'在'真实'环境'中'"；3）**可复制**——"'有效'的'话'可以'推广'"。"'作者'的'贡献'"——"'把'随机'对照'实验'引入'发展'经济学'"
- **小额信贷的效果**: 本书"分析了小额信贷的效果'"。"'小额'信贷'是'什么'"——"'给'穷人'小额'贷款'"——"'让'他们'创业'"。"'理论'上'应该'有效'"——"'但'实验'结果'呢'"。"'作者'发现'"：1）**有些有效**——"'有'创业'能力'的'人'受益'"；2）**有些无效**——"'很多'人'用'贷款'消费'"——"'不'是'投资'"；3）**不'能'解决'根本'问题'"——"'小额'信贷'不'是'万能'的'"。"'结论'"——"'小额'信贷'有'用'"——"'但'不'是'脱贫'的'唯一'方法'"
- **教育补贴**: 本书"研究了教育补贴的效果'"。"'穷人'不'送'孩子'上学'"——"'不'是'不'重视'教育'"——"'是'因为'成本'太高'"。"'作者'实验'"——"'提供'教育'补贴'"——"'免'学费'"——"'提供'校服'"——"'提供'课本'"。"'结果'"：1）**入学率提高**——"'更多'孩子'上学'"；2）**学习效果**——"'但'学习'效果'不'一定'提高'"——"'学校'质量'差'"。"'结论'"——"'教育'补贴'有效'"——"'但'要'同时'提高'学校'质量'"
- **健康干预**: 本书"讨论了健康干预的效果'"。"'穷人'的'健康'问题'"：1）**疟疾**——"'用'蚊帐'"——"'有效'"；2）**寄生虫**——"'驱虫'药'"——"'有效'"；3）**疫苗**——"'免费'疫苗'"——"'提高'接种率'"。"'为什么'穷人'不'用'这些'服务'"——"'不'是'不'知道'"——"'是'因为'麻烦'"——"'因为'短期'成本'"。"'如何'提高'使用率'"：1）**简化**——"'让'服务'更'方便'"；2）**激励**——"'提供'小'奖励'"；3）**默认**——"'让'好'的'选择'成为'默认'"
- **穷人的决策逻辑**: 本书"分析了穷人的决策逻辑'"。"'穷人'的'决策'不'理性'"——"'但'有'原因'"。"'为什么'穷人'买'电视'"——"'不'买'食物'"——"'因为'生活'太'苦'"——"'需要'娱乐'"。"'为什么'穷人'借'高利贷'"——"'因为'急需用钱'"——"'没有'其他'选择'"。"'理解'穷人的'决策'"——"'不'能'简单'评判'"——"'要'理解'他们'的'约束'"。"'政策'设计'"——"'要'考虑'穷人'的'决策'逻辑'"
- **作者的背景**: 阿比吉特·班纳吉（Abhijit Banerjee, 1961-）——印度裔美国经济学家、MIT 教授。埃丝特·迪弗洛（Esther Duflo, 1972-）——法国经济学家、MIT 教授。"'两人'是'夫妻'"——"'一起'研究'发展'经济学'"。"'2019 年'获得'诺贝尔经济学奖'"——"'表彰'他们'的'实地'实验'方法'"。"《贫穷的本质》'2011 年'出版'"——"'畅销'全球'"。"'他们'的'贡献'"——"'把'发展'经济学'从'理论'转向'实证'"
