# 观看之道

状态：已读

作者：约翰·伯格

一句话评价：约翰·伯格（John Berger）的"视觉文化批评开山之作'"——"'用马克思主义和女性主义视角解构艺术品背后的权力关系、性别政治和消费主义'"。"'我们观看的方式受我们所知所信的影响'"——"'观看不是被动接收，而是主动建构'"。"'欧洲油画中的女性裸体是男性凝视的产物'"

核心收获：
- **观看的建构性**: 伯格提出"'我们观看的方式受我们所知所信的影响'"。"'观看'不'是'被动'的'接收'"——"'观看'是'主动'的'建构'"。"'我们'看到'的'不'是'事物'本身'"——"'是'我们'对'事物'的'理解'"。"'比如'同样'一幅'画'"——"'不同'的'人'看到'不同'的'东西'"——"'因为'他们'的'知识'和'信仰'不同'"。"'所以'没有'纯粹'的'观看'"——"'所有'观看'都'被'文化'——'历史'——'意识形态'所'塑造'"
- **男性凝视**: 伯格"用女性主义视角分析欧洲油画'"。"'欧洲'油画'中'的'女性'裸体'"——"'不'是'艺术'"——"'是'男性'凝视'的'产物'"。"'这些'画'是'画'给'男性'看'的'"——"'女性'是'被'看'的'对象'"——"'不'是'主体'"。"'画'中'的'女性'往往'看着'观众'"——"'她'知道'自己'被'看'"——"'她'在'邀请'观众'看'"。"'这种'传统'一直'延续'到'今天'"——"'广告'——'媒体'——'电影'中'的'女性'形象'"——"'依然'是'男性'凝视'的'产物'"
- **原真性**: 伯格"讨论了摄影和复制技术对艺术的影响'"。"'本雅明'提出'了'原真性'（Aura）'的'概念'"。"'原真性'是'什么'"——"'原作'的'独特'存在'"——"'它'在'特定'时间'和'空间'的'存在'"。"'摄影'和'复制'技术'"——"'摧毁'了'原真性'"——"'因为'复制品'可以'在'任何'地方'出现'"。"'但'伯格'说'这'不'是'坏事'"——"'复制'让'艺术'民主化'"——"'让'更多人'能'看到'艺术'"。"'问题'是'图像'被'滥用'"——"'被'广告'——'被'权力'利用'"
- **油画与资本**: 伯格"用马克思主义视角分析油画'"。"'油画'的'兴起'与'资本主义'的'兴起'同步'"。"'油画'表现'了'什么'"——"'表现'了'财富'"——"'表现'了'财产'"。"'油画'中'的'静物'"——"'水果'——'花卉'——'器皿'"——"'都'是'财产'的'展示'"。"'油画'中'的'肖像'"——"'穿着'华丽'衣服'"——"'显示'地位'"。"'所以'油画'不'是'纯粹'的'艺术'"——"'油画'是'资本'的'表达'"——"'是'权力'的'展示'"
- **广告与艺术**: 伯格"分析了广告与艺术的关系'"。"'广告'借用'了'艺术'的'语言'"。"'广告'中'的'图像'"——"'像'油画'一样'"——"'展示'了'理想'的'生活'"。"'但'广告'的'目的'"——"'是'让'你'不满足'"——"'是'让'你'买'东西'"。"'广告'说'"——"'买'了'这个'产品'"——"'你'就'会'像'图像'中'的'人'一样'幸福'"。"'但'这'是'谎言'"——"'因为'幸福'不'能'用'钱'买'"。"'广告'利用'了'人'的'嫉妒'"——"'人'嫉妒'图像'中'的'生活'"——"'所以'去'买'"
- **图像的力量**: 伯格"探讨了图像的力量'"。"'图像'不'是'中立'的'"——"'图像'有'意识形态'"。"'图像'可以'塑造'人'的'观念'"——"'可以'强化'权力'关系'"。"'比如'媒体'中'的'图像'"——"'塑造'了'人'对'美'——'成功'——'幸福'的'理解'"。"'如何'抵抗'"：1）**批判性观看**——"'不'被动'接受'图像'"——"'要'质疑'图像'"；2）**理解背景**——"'理解'图像'的'生产'背景'"——"'谁'制作'的'"——"'为什么'制作'"；3）**创造'自己'的'图像**——"'用'图像'表达'自己'的'观点'"
- **伯格的背景**: 约翰·伯格（John Berger, 1926-2017）——英国艺术评论家、作家、画家——"《观看之道》的作者'"。"他'在'BBC'主持'了'同名'电视'节目'"——"'1972 年'播出'"。"《观看之道》'基于'这个'节目'"——"'书'和'节目'都'引起'轰动'"。"伯格'的'写作'风格'"——"'通俗'——'犀利'"——"'用'简单'的'语言'讲'深刻'的'道理'"。"他'的'其他'作品'"——"'《G.》'——'《第七人》'"——"'都'关注'底层'——'关注'权力'关系'"
