## 深入理解

### 锚定效应的运作
**机制**:
- 第一个信息成为锚点
- 后续判断围绕锚点调整
- 调整通常不足

**《经济学人》实验解析**:
- 诱饵选项 (仅印刷版$125) 无实际价值
- 但使电子 + 印刷版显得超值
- 84% 的人选择最贵选项

**启示**: 消费者偏好可以被设计。

### 免费的心理学
**为什么免费如此强大？**
- 零价格触发情感反应
- 规避风险的直觉
- 社会交换vs市场交换

**代价**:
- 忽视质量差异
- 购买不需要的东西
- 付出更高隐性成本

### 社会规范与市场规范
**关键区别**:
| 社会规范 | 市场规范 |
|---------|---------|
| 友好请求 | 明码标价 |
| 基于互惠 | 基于等价 |
| 长期关系 | 即时交易 |
| 情感联系 | 冷冰冰 |

**混合的代价**:
- 托儿所罚款实验：迟到增加
- 家庭付钱做家务：关系受损
- 公司过度激励：内在动机下降

### 所有权与禀赋效应
**现象**: 拥有后估值显著提高

**原因**:
- 损失厌恶 (失去的痛苦>获得的快乐)
- 情感依附
- 现状偏见

**应用**:
- 试用期 (建立虚拟所有权)
- 无理由退货 (降低购买门槛)
- 个性化定制 (增强归属感)

### 期望的力量
**期望→体验的机制**:
1. 期望形成 (品牌、价格、他人评价)
2. 选择性注意 (确认期望的证据)
3. 解释偏差 (模糊信息符合期望)
4. 真实体验被塑造

**实验**: 告知加醋的啤酒评价差，事后告知评价好。

- 稀缺原理←→免费与稀缺
