# 文明现代化价值投资与中国

状态：已读

作者：李录

一句话评价：李录的演讲和文章合集，分为文明现代化和价值投资两部分。前半部分从人类文明演进的角度解释为什么现代化是必然趋势，后半部分系统阐述了价值投资的理念和方法。

核心收获：
- 理解了"3.0 文明"概念：科学革命和市场经济的结合创造了可持续增长的人类文明新形态
- 价值投资的核心是买股票就是买公司——以企业所有者的眼光做投资
- 中国作为 3.0 文明的后来者，有后发优势但也有独特的制度和文化挑战
