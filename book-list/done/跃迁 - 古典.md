# 跃迁

状态：已读

作者：古典

一句话评价：古典的个人成长方法论——在互联时代实现非线性成长，不是靠拼命努力，而是靠正确的选择和策略。"跃迁"是量子力学的概念：电子从低能级跳到高能级，不需要经过中间状态。人的成长也可以这样——找到杠杆点，一跃而过。

核心收获：
- **"跃迁"的本质**: 非线性成长不是"慢慢积累"，是"找到杠杆点后一跃而过"。古典用物理学的"跃迁"概念类比人的成长——电子从低能级跳到高能级，不需要经过中间状态。关键是找到那个"杠杆点"——正确的领域、正确的平台、正确的方法
- **"头部效应"**: 在一个小领域做到头部（前 20%），比在大领域做到中游（50%）更有价值。头部的人获得更多的资源、更多的机会、更多的正反馈，然后进入"头部→资源→更头部"的正循环。所以策略不是"什么都做"，是"找到一个小领域，做到第一"
- **"借力"思维**: 善用平台、趋势和他人经验，比自己蛮干效率高得多。"站在巨人的肩膀上"不是谦虚，是策略。借力包括：借平台的力（选对平台）、借趋势的力（顺势而为）、借高手的力（向已经成功的人学习）
- **"系统思考"**: 个人成长不是孤立的事件，是一个系统。你的能力、人脉、平台、趋势互相影响。系统思考要求你看清整个系统的全貌，找到"关键节点"——改变一个节点，整个系统都会变。比如：换一个城市，可能改变你的整个人脉圈和机会
- **"高价值区"的选择**: 古典说成功的关键是"在高价值区做正确的事"。高价值区是"有复利效应"的领域——今天做的事，明天还在产生价值。写文章、做产品、建人脉都有复利；送外卖、做重复性工作没有复利
- **"联机学习" vs "单机学习"**: 单机学习是自己看书、自己琢磨；联机学习是和高手交流、加入社群、输出倒逼输入。在互联时代，联机学习的效率远高于单机学习。"教是最好的学"——输出一次，比读十遍记得牢
- **"反直觉"的成长策略**: 1）不要急于努力，先想清楚方向——方向错了，越努力越惨；2）不要什么都学，聚焦一个领域做到头部；3）不要闭门造车，多和高手交流；4）不要追求完美，先完成再完美
- **古典的背景**: 古典是"新精英生涯"创始人，之前写过《拆掉思维里的墙》。他本身就是"跃迁"的实践者——从新东方老师到职业规划师到知识 IP，每一次都是找到新的杠杆点实现跃迁。他的方法论不是纸上谈谈，是自己验证过的
