# 聪明的投资者

状态：已读

作者：本杰明·格雷厄姆

一句话评价：本杰明·格雷厄姆（Benjamin Graham）的"价值投资奠基之作'"——"'安全边际'与'市场先生'概念影响深远'"——"'巴菲特导师经典'"——"'理性投资的圣经'"。"'投资操作是基于透彻的分析，确保本金的安全并获得满意的回报'"——"'防御型投资者与分析型投资者的区分'"——"'分散投资与长期持有的智慧'"

核心收获：
- **价值投资的核心原则**: 格雷厄姆"定义了什么是价值投资'"。"'价值投资'的'核心'"：1）**透彻分析**——"'基于'事实'和'数据'"——"'不是'投机'"——"'不是'跟风'"；2）**本金安全**——"'首要'目标是'保护'本金'"——"'不是'追求'高收益'"；3）**满意回报**——"'合理'的'回报'就'足够'"——"'不'追求'暴利'"。"'格雷厄姆'说'投资'与'投机'的'区别'在于'是否'有'安全边际'"——"'投机'是'赌博'"——"'投资'是'理性'的'分析'"
- **安全边际**: 格雷厄姆提出了"'安全边际'（Margin of Safety）"的概念'"。"'安全边际'意味着'"：1）**价格低于价值**——"'用'50 美分'买'1 美元'的'东西'"；2）**留有缓冲**——"'即使'你'算错'了'"——"'也'不会'亏'太多'"；3）**降低风险**——"'安全边际'越'大'"——"'风险'越'小'"。"'如何'获得'安全边际'"——"'买'被'低估'的'股票'"——"'买'股价'低于'净资产'的'公司'"。"'安全边际'是'价值投资'的'核心'"——"'没有'安全边际'"——"'就'不是'价值投资'"
- **市场先生**: 格雷厄姆提出了"'市场先生'（Mr. Market）"的'比喻'"。"'市场先生'是'你的'仆人'"——"'不是'你的'老师'"。"'市场先生'每天'来'找你'"——"'他'给出'一个'价格'"——"'想'买'你的'股票'"——"'或'想'卖'股票'给'你'"。"'市场先生'的'情绪'不稳定'"——"'有时'乐观'"——"'有时'悲观'"——"'有时'理性'"——"'有时'疯狂'"。"'你'应该'利用'市场先生'"——"'他'悲观'时'买入'"——"'他'乐观'时'卖出'"——"'但'不要'被'他'左右'"。"'市场'是'为你服务'的'"——"'不是'指导你'的'"
- **防御型投资者**: 格雷厄姆"区分了两种投资者'"。"'防御型投资者'是'被动'的'"。"'防御型投资者'的'策略'"：1）**分散投资**——"'买'一篮子'股票'"——"'不'把'鸡蛋'放在'一个'篮子里'"；2）**长期持有**——"'买'了'就'持有'"——"'不'频繁'交易'"；3）**指数基金**——"'格雷厄姆'推荐'指数基金'"——"'低成本'"——"'分散风险'"；4）**股债平衡**——"'股票'和'债券'各'50%'"——"'定期'再平衡'"。"'防御型投资者'不'需要'花'很多'时间'"——"'适合'普通人'"
- **分析型投资者**: "'分析型投资者'是'主动'的'"。"'分析型投资者'的'策略'"：1）**深入研究**——"'分析'公司'财报'"——"'分析'行业'前景'"；2）**寻找低估**——"'找'被'市场'低估'的'股票'"；3）**集中投资**——"'集中'在'少数'优质'公司'"；4）**积极管理**——"'定期'审视'持仓'"——"'适时'调整'"。"'分析型投资者'需要'专业知识'"——"'需要'花'大量'时间'"——"'但'可能'获得'超额收益'"。"'巴菲特'是'分析型投资者'的'代表'"
- **分散投资与长期持有**: 格雷厄姆"强调分散投资和长期持有'"。"'分散投资'的'好处'"：1）**降低风险**——"'单只'股票'暴跌'"——"'不'影响'整体'"；2）**平滑波动**——"'不同'股票'涨跌'互抵'"；3）**无需择时**——"'不'需要'预测'市场'"。"'长期持有'的'好处'"：1）**复利效应**——"'时间'越'长'"——"'收益'越'大'"；2）**避免交易成本**——"'频繁'交易'费用'高'"；3）**减少错误**——"'交易'越'少'"——"'错误'越'少'"。"'格雷厄姆'说'长期'是'投资者'的'朋友'"
- **格雷厄姆的背景**: 本杰明·格雷厄姆（Benjamin Graham, 1894-1976）——美国投资家、经济学家——"价值投资之父'"。"他'在'哥伦比亚大学'教书'"——"'他的'学生'包括'巴菲特'"——"'芒格'等'投资大师'"。"《聪明的投资者》'1949 年'出版'"——"'被誉为'投资圣经'"。"格雷厄姆'的'其他'著作'"——"'《证券分析》'——'更'专业'"——"'《聪明的投资者》'——'更'通俗'"。"格雷厄姆'的'投资理念'影响'了'几代'投资者'"——"'巴菲特'说'格雷厄姆'的'理论'是'他'投资'的'基石'"
