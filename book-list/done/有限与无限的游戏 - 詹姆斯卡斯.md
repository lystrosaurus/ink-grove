# 有限与无限的游戏

状态：已读

作者：詹姆斯·卡斯

一句话评价：詹姆斯·卡斯（James P. Carse）的"薄薄的小书"（1986 年出版）——提出了一个"极具解释力的世界观框架"——"世上只有两种游戏"——"有限游戏'以赢为目的'"——"无限游戏'以延续游戏为目的'"。这个框架"影响了从商业到人生的诸多领域"。"无限游戏玩家'不追求赢'"——"追求'让游戏继续'"——这是一种"完全不同的思维方式"。

核心收获：
- **两种游戏的根本区别**: 卡斯说"世上只有两种游戏"。**有限游戏**——"有明确的规则、边界和终点"——"目的是'赢'"——"赢的人'获得头衔'"——"游戏'就结束了'"。比如"一场足球赛"——"有规则"——"有边界（球场）"——"有终点（90 分钟）"——"赢的人'获得冠军'"。**无限游戏**——"规则'可以改变'"——"边界'可以扩展'"——"没有终点"——"目的是'让游戏继续'"。比如"文化"——"没有'终点'"——"规则'在不断变化'"——"目的是'让文化'延续下去"
- **商业的重新理解**: 卡斯用"有限 vs 无限"重新理解了"商业"。"有限游戏式商业"——"目标是'打败竞争对手'"——"获得'市场份额'"——"成为'行业第一'"——"然后就'结束了'"。"无限游戏式商业"——"目标是'让商业生态'延续下去'"——"和竞争对手'共存'"——"创造'更大的价值'"——"让'整个行业'繁荣"。卡斯说"伟大的企业'都是'无限游戏玩家'"——他们"不追求'一时的胜利'"——追求"长期的'可持续发展'"
- **婚姻的重新理解**: 卡斯说"婚姻"也可以是"有限游戏"或"无限游戏"。"有限游戏式婚姻"——"目标是'赢得'对方'"——"让对方'服从自己'"——"控制'家庭的主导权'"。"无限游戏式婚姻"——"目标是'让关系'延续下去'"——"不断'调整和成长'"——"让'两个人'都在关系中得到发展"。有限游戏式的婚姻"最终会'走向破裂'"——因为"一方的'胜利'意味着另一方的'失败'"。无限游戏式的婚姻"会'越来越深厚'"——因为"双方都在'为关系'付出"
- **文化的本质**: 卡斯说"文化是'最典型的'无限游戏"。文化"没有'终点'"——"没有'赢家'"——"它的目的'是'延续'"。文化"在不断'变化'"——"吸收新的元素"——"淘汰旧的元素"——"但'核心'一直'存在'"。卡斯说"文明'衰落'的原因"不是"'被打败了'"——是"'停止了变化'"——"当一个文化'拒绝改变'"——"它就'不再是'无限游戏了"——"它变成了'有限游戏'"——"最终'消亡'"
- **权力的两种形式**: 卡斯区分了"两种权力"。"有限游戏的权力"——"是'对别人的控制'"——"让别人'做你想做的事'"。"无限游戏的权力"——"是'让别人'成为'他自己'"——"赋能"——"让每个人'都能参与'游戏"。卡斯说"真正的'伟大领袖'"不是"控制别人的人"——是"让别人'变得更强'的人"
- **死亡的视角**: 卡斯在书的结尾讨论了"死亡"。他说"有限游戏玩家'害怕死亡'"——因为"死亡意味着'游戏结束'"——"他们'输了'"。"无限游戏玩家'不怕死亡'"——因为"他们的目的'不是'永远活着'"——是"在'活着的时候'让'游戏继续'"——"通过'传承'——通过'影响他人'"——"游戏'不会'因为一个人的死亡'而结束'"
- **詹姆斯·卡斯的背景**: 詹姆斯·卡斯（James P. Carse, 1932-）——美国宗教学家和哲学家——纽约大学宗教史教授。《有限与无限的游戏》（Finite and Infinite Games）1986 年出版——虽然"很薄"（不到 200 页）——但"影响深远"。这本书"影响了硅谷的企业家"（如 Stewart Brand、Kevin Kelly）——"影响了商业思想家"（如 Simon Sinek）——"成为了'创新思维'的经典框架"
