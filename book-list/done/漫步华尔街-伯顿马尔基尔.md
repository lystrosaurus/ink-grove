# 漫步华尔街

状态：已读

作者：伯顿·马尔基尔

一句话评价：伯顿·马尔基尔（Burton Malkiel）的"投资经典中的经典"——"系统论证了'有效市场假说'和'指数投资的优越性'"——"用'历史数据'戳穿了'技术分析和基本分析的幻觉'"。"'随机漫步'理论"——"'股价短期变动'本质上是'随机的'"——"'无法被可靠预测'"。"指数基金'长期跑赢'绝大多数'主动管理基金'"——"这是'被几十年数据反复验证'的事实"。"投资的核心策略"——"'分散化'——'低成本'——'长期持有'——'定期再平衡'"。

核心收获：
- **随机漫步理论**: 马尔基尔说"股价的短期变动'本质上是'随机的"——"'无法被可靠预测'"。"如果'股价变动'有'规律'"——"那'所有人'都会'利用这个规律'"——"规律就'消失了'"。"这就是'有效市场假说'"——"市场价格'已经反映了'所有已知信息'"——"所以你'无法'通过分析'获得'超额收益'"。"股价'就像'一个'醉汉走路'"——"你'不知道'他下一步'会往哪边走'"——"这就是'随机漫步'"
- **指数投资的优越性**: 马尔基尔用"几十年的历史数据"证明了"指数基金'长期跑赢'绝大多数'主动管理基金'"。"主动管理基金的'平均收益率'"——"'低于'指数基金"——"因为'管理费'和'交易成本'"。"指数基金'不试图'跑赢市场'"——"它'就是'市场'"——"所以'它的收益率'就是'市场的收益率'"——"减去'极低的管理费'"。"长期来看"——"'90% 以上的'主动管理基金'跑不赢'指数基金"
- **技术分析的幻觉**: 马尔基尔"用数据和逻辑'戳穿了'技术分析的'幻觉'"。"技术分析'认为'股价有'趋势'和'形态'"——"但'马尔基尔'用'大量数据'证明"——"'那些形态'在统计学上'没有预测能力'"。"技术分析'之所以'让人觉得'有用'"——"是因为'人们'在'随机数据中'也能'看到模式'"——"这是'人类的'认知偏差"
- **基本分析的局限**: 马尔基尔也"分析了'基本分析'的局限性"。"基本分析'认为'通过分析'公司财务'可以'找到'被低估的股票'"——"但'问题是'"——"'所有投资者'都在'做基本分析'"——"所以'价格已经反映了'这些信息'"。"而且'基本分析'依赖'对未来盈利的预测'"——"但'预测'往往是'错的'"
- **泡沫的历史**: 马尔基尔"回顾了'历史上'的'几次大泡沫'"：1）**荷兰郁金香泡沫**（1637）——"一株郁金香'能换'一栋房子"；2）**南海泡沫**（1720）——"牛顿'也亏了钱'"——"他说'我能计算天体的运行'——'但计算不了'人类的疯狂'"；3）**互联网泡沫**（2000）——"没有盈利的公司'估值上百亿'"。马尔基尔说"泡沫'总是'会破'"——"但'每次'人们都'觉得'这次不一样'"
- **生命周期投资指南**: 马尔基尔"给出了'不同年龄段的'投资建议"。"年轻人'应该'多配置'股票'"——"因为'你有时间'来'平滑波动'"。"中年人'应该'逐步'增加'债券比例'"——"降低'风险'"。"老年人'应该'以'债券和现金'为主"——"保护'本金'"。"他还建议'定期再平衡'"——"每'半年或一年'"——"把'资产配置'调整回'目标比例'"
- **伯顿·马尔基尔的背景**: 伯顿·马尔基尔（Burton Malkiel, 1932-）——普林斯顿大学经济学教授——"《漫步华尔街》的作者"。这本书"1973 年'首次出版'"——"已经'更新了'12 版'"——"是'投资领域'最畅销的'经典之一'"。马尔基尔"自己'就是'指数投资的'践行者'"——"他'管理'的'基金'主要'投资'指数基金"——"他'用自己的'投资实践'证明了'指数投资的'优越性"
