# 刻意练习

状态：已读

作者：安德斯艾利克森

一句话评价：颠覆"天赋决定论"的心理学经典，提出杰出表现不是天生的，而是通过特定方式的练习——刻意练习——获得的，为普通人提升技能提供了科学路径。

核心收获：
- 理解了"刻意练习"的四个要素：明确目标、专注投入、即时反馈、走出舒适区
- "天真的练习"（只是重复做）与"刻意练习"（有目标、有反馈地改进）有本质区别
- 心理表征是刻意练习的核心成果——高手和新手的区别在于脑中建立了精细的心理模型
