# 今日简史

状态：已读

作者：尤瓦尔·赫拉利

一句话评价：赫拉利（Yuval Noah Harari）"简史三部曲终章'"——"'从历史转向当下诊断 21 世纪核心危机（科技颠覆/生态崩溃/假新闻/自由主义叙事衰退）'"。"'承认无知比已知知识更重要'"。"'推荐冥想对抗算法操控'"。"'在不确定时代保持清醒思考的指引之作'"

核心收获：
- **科技颠覆**: 本书核心是"'科技颠覆'"的分析'"。"'AI 和'生物'技术'的'结合'正在'改变'人'的'定义'"。"'当'算法'比'你'更'了解'你'自己'"——"'当'基因'编辑'可以'定制'婴儿'"——"'当'脑机'接口'可以'上传'意识'"。"'人类'这个'概念'的'边界'在'哪里'"。"'赫拉利'说'21 世纪'最大'的'问题'"："'什么'是'人'"
- **生态崩溃**: 本书"讨论了生态崩溃'"。"'气候'变化'不'是'未来'的'问题'"——"'是'现在'的'问题'"。"'赫拉利'说'"："'生态'崩溃'和'科技'颠覆'是'21 世纪'的'两大'存在性'威胁'"。"'但'人类'对'这'两个'问题'的'反应'太'慢'"——"'因为'它们'是'全球性'的'"——"'而'政治'是'地方性'的'"
- **假新闻与后真相**: 本书"分析了假新闻问题'"。"'假新闻'不'是'新'问题'"——"'人类'几千年'来'一直'生活'在'谎言'中'"。"'但'社交媒体'让'假新闻'的'传播'速度'比'真相'快'6 倍'"。"'赫拉利'说'后'真相'时代'的'本质'"："'不'是'人们'相信'谎言'"——"'是'人们'不'再'相信'真相'"。"'当'所有'信息'都'不'可信'时'"——"'人们'退回'到'部落主义'"
- **自由主义叙事的危机**: 本书"讨论了自由主义的危机'"。"'赫拉利'说'20 世纪'是'自由主义'的'胜利'"——"'但'21 世纪'自由主义'正在'失去'吸引力'"。"'自由主义'的'三个'核心'承诺'"：1）**政治自由**——"'被'民粹主义'冲击'"；2）**经济自由**——"'被'保护主义'冲击'"；3）**个人自由**——"'被'算法'操控'冲击'"
- **无知的重要性**: 本书"强调了无知的重要性'"。"'在'信息'不'充足'的'情况'下'"——"'要'保持'怀疑'"。"'赫拉利'说'现代'教育'的'问题'"："'教'学生'知道'什么'"——"'而'不'是'教'学生'承认'无知'"。"'在'快速'变化'的'时代'"——"'已知的'知识'很快'过时'"。"'承认'无知'的'能力'才'是'最'重要'的'"
- **世俗与宗教的纠缠**: 本书"分析了宗教的角色'"。"'赫拉利'说'宗教'不'是'迷信'"——"'是'社会'合作'的'工具'"。"'宗教'让'人类'和'陌生人'合作'"。"'但'宗教'也'有'黑暗'面'"——"'当'宗教'和'民族主义'结合'"——"'就'会'产生'宗教'民族主义'"
- **冥想与自我认知**: 本书"最后推荐了冥想'"。"'赫拉利'在'最后'一章'推荐'了'冥想'"——"'他'本人'已经'冥想'20 多年'"。"'他'说'在'算法'即将'比'你'自己'更'了解'你'的'时代'"——"'认识'自己'变得'前所未有'的'重要'"。"'冥想'不'是'宗教'仪式'"——"'是'观察'自己'内心'的'工具'"
- **作者的背景**: 尤瓦尔·赫拉利（Yuval Noah Harari, 1976-）——以色列历史学家、作家——"《今日简史》的作者'"。"'《今日'简史》'2018 年'出版'"——"'是'简史'三部曲'的'终章'"。"'前'两部'"："'《人类'简史》'（2011）'"——"'《未来'简史》'（2015）'"。"'赫拉利'的'态度'"："'不'是'悲观主义者'"——"'也'不'是'乐观主义者'"——"'是'清醒'的'现实主义者'"。"'他'说'未来'不'是'注定'的'"——"'取决于'我们'今天'的'选择'"
