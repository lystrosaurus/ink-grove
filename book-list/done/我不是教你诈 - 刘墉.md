# 我不是教你诈

状态：已读

作者：刘墉

一句话评价：刘墉的处世智慧系列——用"短小精悍的故事和案例"教你"识别人性陷阱"——在"社会中保护自己而不伤害他人"。书名"我不是教你诈"——意思是"我不是在教你'骗人'"——是"教你'不被骗'"。"诈"的本质不是"恶意伤害"——是"人性中的自私和利益驱动导致的行为偏差"。刘墉的写作风格"亲切易懂"——每个案例都"来自真实生活"——有很强的"实用价值"。

核心收获：
- **"诈"的本质**: 刘墉说"诈"不是"坏人的专利"——是"人性中的自私"在特定条件下的"自然反应"。"每个人都可能在'利益驱动'下做出'不诚实'的行为"——不是"因为他们是坏人"——是"因为'利益'太大了"或"风险太小了"。理解这一点不是为了"变得愤世嫉俗"——是为了"在'不伤害别人的前提下'保护自己"
- **职场陷阱**: 刘墉分析了职场中常见的"人性陷阱"：1）**被利用**——同事让你"帮忙"——其实是"把他的工作推给你"；2）**被背叛**——你信任的人"在关键时刻'背后捅刀'"；3）**被误解**——你的"善意"被别人理解为"别有用心"；4）**背黑锅**——出了问题"所有人都说是你的责任"。刘墉的建议是"不轻易承诺"、"留下书面记录"、"不要让'所有人都依赖你'"——因为"依赖一旦变成'理所当然'"——你就"没有退路了"
- **生活中的陷阱**: 生活中的"诈"更加隐蔽——因为"发生在'亲密关系'中"：1）**亲戚借钱**——"借了不还"——你"不好意思要"；2）**朋友合伙做生意**——"赚了钱'分配不均'"——"亏了钱'互相推诿'"；3）**婚姻中的'经济控制'"——一方"掌握所有财务"——另一方"完全没有话语权"。刘墉说"在亲密关系中——更要'把话说清楚'"——因为"越是亲密——越容易被'不好意思'伤害"
- **化解而非对抗**: 刘墉的核心理念——面对"诈"——"不要用'对抗'来回应"——用"智慧来化解"。"对抗"会让"矛盾升级"——"智慧"会让"问题自然消失"。比如同事"推活给你"——不是"直接拒绝"（会得罪人）——也不是"默默接受"（会被一直利用）——是"说'我很乐意帮忙——但我现在手头有 X 事——您看哪个优先？'"——让对方"自己做选择"
- **识人的智慧**: 刘墉说"识人"不是"猜心思"——是"观察行为模式"：1）**看他如何对待'不如他的人'"**——对服务员态度差的人——不值得信任；2）**看他"在利益面前的表现'"**——能"公平分配利益"的人——值得合作；3）**看他"'说了'和'做了'是否一致'"**——言行一致的人——值得深交。刘墉说"不要听'一个人说了什么'"——要看"他做了什么"
- **自我保护的底线**: 刘墉强调"保护自己"不等于"伤害别人"——他的原则是"不主动害人——但也不当'冤大头'"。"善良"不是"没有底线"——"善良"是"有选择地付出"。刘墉说"你要做一个'好人'"——但"不是一个'好欺负的好人'"
- **刘墉的背景**: 刘墉（1949-）——台湾作家、画家——他的作品以"处世智慧"和"人生哲理"著称。"我不是教你诈"系列是他"最受欢迎的作品"之一——在台湾和大陆的销量超过百万册。刘墉的写作风格"亲切、幽默、接地气"——他不用"大道理"——用"小故事"——让读者"在'笑'中'悟'"
