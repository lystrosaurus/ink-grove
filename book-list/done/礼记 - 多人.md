# 礼记

状态：已读

作者：多人（西汉戴圣编撰）

一句话评价：儒家经典之一——"系统记载了'先秦时期的'礼仪制度、社会规范和伦理思想'"——"《大学》《中庸》《礼运》等名篇'影响深远'"——"是'理解中国传统社会秩序的'钥匙'"。"'礼'不仅是'仪式和规矩'"——"'更是'一套'维系社会秩序、规范人际关系的'完整体系'"。"'格物致知诚意正心修身齐家治国平天下'"——"是'儒家的'人生进阶蓝图'"。

核心收获：
- **礼的本质**: 《礼记》中"'礼'的'本质'"——"'不'只是'仪式和规矩'"——"'是'一套'维系社会秩序的'完整体系'"。"'礼'规范了'人与人之间的'关系'"——"'君臣'——'父子'——'夫妇'——'兄弟'——'朋友'"——"'五伦'"——"'每一种'关系'都有'相应的'礼仪'"。"'礼'的'目的'"——"'不是'束缚人'"——"'是'让'社会'有序'"——"'让'人'知道'自己的'位置'"——"'知道'自己'应该'怎么做'"
- **《大学》的八条目**: 《大学》提出了"'八条目'"——"'格物、致知、诚意、正心、修身、齐家、治国、平天下'"。"这是'儒家的'人生进阶蓝图'"：1）**格物**——"'探究'事物的'原理'"；2）**致知**——"'获得'知识'"；3）**诚意**——"'使'意念'真诚'"；4）**正心**——"'端正'内心'"；5）**修身**——"'修养'自身'"；6）**齐家**——"'管理'好'家庭'"；7）**治国**——"'治理'好'国家'"；8）**平天下**——"'使'天下'太平'"。"这八个步骤'从'内到外'"——"'从'个人'到'天下'"——"'层层递进'"
- **《中庸》的中道**: 《中庸》的核心是"'中道'"——"'不偏不倚'"——"'无过无不及'"。"'中庸'不是'平庸'"——"'是'一种'恰到好处的'智慧'"。"'喜怒哀乐之未发，谓之中；发而皆中节，谓之和'"——"'情绪'没有'发作'时'"——"'内心'是'平静的'"——"'这叫'中'"——"'情绪'发作'时'"——"'能'控制'在'适当的'程度'"——"'这叫'和'"。"'致中和，天地位焉，万物育焉'"——"'达到'中和'的'境界'"——"'天地'就'各就其位'"——"'万物'就'生长繁衍'"
- **《礼运》的大同理想**: 《礼运》提出了"'大同'理想'"——"'大道之行也，天下为公'"。"'大同社会'的'特征'"：1）**天下为公**——"'天下'是'大家的'"——"'不是'一家的'"；2）**选贤与能**——"'选拔'有'德行'和'才能'的'人'来'治理'"；3）**讲信修睦**——"'人人'都'讲信用'"——"'人人'都'和睦相处'"；4）**老有所终，壮有所用，幼有所长**——"'每个人'都'能'得到'照顾'"。"'大同'理想'是'中国最早的'乌托邦构想'"——"'影响至今'"
- **丧祭之礼**: 《礼记》中"'详细记载'了'丧礼和祭礼'"。"'丧礼'的'核心'"——"'孝'"——"'父母'去世'"——"'子女'要'守丧三年'"——"'表达'哀思'"。"'祭礼'的'核心'"——"'敬'"——"'祭祀'祖先'"——"'表达'敬意'"。"'儒家'认为'丧祭之礼'的'目的'"——"'不'是'迷信'"——"'是'让'人'不忘本'"——"'让'人'知道'自己'从'哪里来'"——"'让'人'懂得'感恩'"
- **教育思想**: 《礼记》中"'有很多'关于'教育'的'思想'"。"'教学相长'"——"'教'和'学'是'互相促进'的'"——"'教'别人'的'过程'"——"'也是'自己'学习'的'过程'"。"'玉不琢，不成器；人不学，不知道'"——"'人'必须'学习'"——"'才能'明白'道理'"。"'温故而知新'"——"'复习'旧的'知识'"——"'能'获得'新的'理解'"。"这些'教育思想'"——"'影响了'中国'几千年的'教育传统'"
- **礼记的背景**: 《礼记》是"儒家经典之一"——"'西汉'戴圣'编撰'"——"'收录'了'先秦'到'汉初'的'礼仪文献'"。"'《礼记》'与'《周礼》'——'《仪礼》'合称'三礼'"——"'是'儒家'最重要的'经典之一'"。"'《礼记》'中的'很多'篇章'"——"'如'《大学》'——'《中庸》'"——"'后来'被'朱熹'选入'四书'"——"'成为了'科举考试的'必读书'"——"'影响了'中国'读书人'的'思想'"
