# 像艺术家一样思考

状态：已读

作者：贝蒂-艾德华

一句话评价：把"画画"从天赋变成可训练的技能——核心不是手，是眼睛和大脑。学会"看"比学会"画"重要一百倍。

核心收获：
- 画画不是手的技能，是"看"的技能。大多数人画不好不是因为手笨，是因为"看"的方式错了
- L模式（语言/逻辑/分析）和R模式（视觉/空间/直觉）的切换是学会画画的关键。要画画必须关闭L模式、打开R模式
- 五种基本技能：感知边缘、感知空间、感知关系、感知光影、感知整体（格式塔）
- "倒着画"练习的妙处：把对象倒过来，L模式认不出它是什么，R模式被迫接管——结果画得更好
- 每个人都能学会画画，这不是天赋，是感知方式的转换
