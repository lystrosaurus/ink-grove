# 人工智能的边界

状态：已读

作者：相关作者

一句话评价：理清了AI"能做什么"和"不能做什么"的边界，破除 hype 和恐慌两个极端，建立理性的技术认知。

核心收获：
- AI的能力边界：模式识别、数据处理、规则优化是强项，但创造力、共情、价值判断是短板
- 当前AI的本质是统计模型，不是"理解"，是"拟合"
- AI伦理问题：算法偏见、数据隐私、就业冲击、自主武器
- 人机协作的未来不是"AI取代人类"，而是"善用AI的人取代不用AI的人"
