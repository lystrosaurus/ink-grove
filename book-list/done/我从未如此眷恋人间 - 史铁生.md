# 我从未如此眷恋人间

状态：已读

作者：史铁生

一句话评价：史铁生的散文精选集——以"地坛"为核心意象——记录了他在"双腿瘫痪后"对"生命、死亡、亲情和人间百态"的深刻思考。《我与地坛》是中国当代散文的"巅峰之作"——一个残疾青年与一座废弃古园的"对话"——在"最绝望的处境中"找到"眷恋世界的理由"。史铁生对母亲的描写"催人泪下"——"我从来没有想过——母亲为我做过的那些事——会在我心里留下这么深的印子。"文字"沉静而有力量"——是"中国当代文学中最打动人心的作品"之一。

核心收获：
- **在最绝望中眷恋人间**: 史铁生 21 岁时"双腿瘫痪"——从此"坐上了轮椅"——这是他"人生最大的打击"。他"一度想要自杀"——但他"没有"——因为他"在地坛中找到了'活下去的理由'"。"地坛"是"一座废弃的古园"——"荒凉、安静、没有人打扰"——史铁生"每天摇着轮椅去那里"——"一坐就是一整天"。在地坛中——他"想通了"——"死是一件不必急于求成的事"——"死是一个必然会降临的节日"——所以"不必急着去"——"不如先'好好活着'"
- **《我与地坛》**: 这是全书的"核心篇章"——也是中国当代散文的"经典之作"。史铁生在地坛中"观察"——"观察蚂蚁"、"观察落叶"、"观察四季的变化"——他"从自然中领悟了'生命的意义'"。他说"园子荒芜但并不衰败"——"荒芜"是"外表"——"不衰败"是"内在的生命力"。这就是他的"人生态度"——"身体可以'荒芜'"——但"精神不能'衰败'"
- **对母亲的描写**: 这是全书"最催泪"的部分。史铁生瘫痪后——他的母亲"承受了比他更大的痛苦"——但"她从不抱怨"。她"每天悄悄跟在他后面"——怕他"想不开"——又"不敢让他发现"。史铁生写道："多年来我头一次意识到——这园中不单是处处都有过我的车辙——有过我的车辙的地方也都有过母亲的脚印。"母亲在他"还没有完全理解'母爱'的时候"就"去世了"——这成为了他"一生的遗憾"。他说"我从来没有想过——母亲为我做过的那些事——会在我心里留下这么深的印子"
- **对死亡的思考**: 史铁生说"死是一件不必急于求成的事"——"死是一个必然会降临的节日"——"所以不必急着去"。他把"死亡"看作"生命的'终点站'"——不是"生命的'对立面'"。因为有了"终点"——"生命才有了'意义'"——如果"永远不死"——"活着就'没有紧迫感'"——"也就'没有了意义'"。这种"向死而生"的态度——和"海德格尔"的哲学有相通之处——但史铁生的表达更加"朴素"和"个人化"
- **对生命的热爱**: 史铁生在"最不幸的处境"中——写下了"最热爱生命的文字"。他说"我从未如此眷恋人间"——不是"因为人间'完美'"——是因为"人间'值得'"。"值得"不是因为"没有痛苦"——是因为"有'爱'、有'美'、有'希望'"。史铁生说"即使'命运对你不公'"——你"仍然可以'选择'如何去'面对'"——你可以"选择'怨恨'"——也可以"选择'眷恋'"。他选择了"眷恋"
- **残疾与尊严**: 史铁生从不"回避'残疾'"——也不"美化'残疾'"。他说"残疾不是'一种荣耀'"——是"一种'现实'"。但"残疾不意味着'失去尊严'"——"尊严来自'你如何面对'残疾"。他说"有些残疾人'把自己当成了'弱者'"——"处处要求'特殊照顾'"——"这反而'失去了尊严'"。真正的"尊严"是"不把自己当'弱者'"——是"在'有限'的条件下'做到最好'"
- **史铁生的背景**: 史铁生（1951-2010）——中国当代作家——北京人——1969 年去延安插队——1972 年因"腰疾"双腿瘫痪——此后"一生与疾病为伴"——后来又患"肾病"——最终在 2010 年去世——享年 59 岁。史铁生的"写作生涯"几乎全部在"轮椅上"完成——他的代表作包括《我与地坛》《务虚笔记》《病隙碎笔》《命若琴弦》。他被誉为"中国当代最'有思想深度'的散文家"——他的文字"沉静、深邃、充满力量"——影响了无数"在困境中挣扎的人"
