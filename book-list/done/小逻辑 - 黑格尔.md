# 小逻辑

状态：已读

作者：格奥尔格·威廉·弗里德里希·黑格尔

一句话评价：黑格尔哲学的浓缩精华版（1817 年第一版，1830 年第三版为定本），系统阐述了"存在论"（Sein）、"本质论"（Wesen）和"概念论"（Begriff）三大逻辑范畴。作为《哲学科学全书纲要》的第一部分，是理解黑格尔辩证法的核心著作——"正题、反题、合题"的三段式思维框架，每一个概念都与整个体系紧密相连。比《大逻辑》更精炼但也更抽象，是哲学史上最难读但也最有深度的著作之一。

核心收获：
- **小逻辑 vs 大逻辑**: 黑格尔写了两本"逻辑学"——《大逻辑》（Wissenschaft der Logik, 1812-1816）是三卷本巨著，《小逻辑》（Encyclopaedia Logic, 1817）是《哲学科学全书纲要》的"逻辑学部分"，是《大逻辑》的"浓缩版"。《小逻辑》更"精炼"（篇幅短得多），但也更"抽象"（省略了大量论证过程）。初学者通常从《小逻辑》入手，然后再读《大逻辑》
- **存在论（Being）**: 小逻辑的第一部分。黑格尔从"存在"（Sein / Being）这个最抽象、最空泛的概念开始——"存在"是什么？什么都没有说——它就是"存在"。但"纯粹的存在"和"纯粹的无"（Nothing）是"同一个东西"——因为两者都是"空的"。从"存在"和"无"的矛盾中产生了"变易"（Becoming）——"存在"变成了"某物"（Something）。这就是辩证法的第一步——从"最抽象的概念"通过"内在矛盾"推演出"更具体的概念"
- **本质论（Essence）**: 小逻辑的第二部分。"存在"发展到一定程度就会追问"它是什么"——这就是"本质"。本质论探讨的是"表象与本质""原因与结果""必然与偶然""可能与现实"等范畴。黑格尔说"本质必须显现"——本质不是"隐藏在现象背后的东西"，是"通过现象展现出来的东西"。没有脱离现象的本质，也没有脱离本质的现象——两者是"统一的"
- **概念论（Concept）**: 小逻辑的第三部分，也是"最高级"的部分。"概念"不是"头脑中的想法"——在黑格尔那里，"概念"是"存在和本质的统一"——是"事物本身的理性结构"。概念论分为"主观概念"（概念本身）、"客观概念"（机械性、化学性、目的性）和"理念"（生命、认识、绝对理念）。"绝对理念"是黑格尔哲学的顶点——它是"整个逻辑学发展的最终结果"，是"思维与存在的完全统一"
- **辩证法的三段式**: 黑格尔的辩证法经常被概括为"正题 - 反题 - 合题"（thesis-antithesis-synthesis）——但实际上黑格尔本人很少用这三个词。他更常用的表述是"自在 - 自为 - 自在自为"（in-itself, for-itself, in-and-for-itself）。辩证法的核心是"矛盾推动发展"——一个概念（或事物）内部包含着"自身的对立面"，这种"矛盾"推动了它向"更高层次的统一"发展。比如"存在"（正题）包含了"无"（反题），两者的统一是"变易"（合题）
- **具体的普遍**: 黑格尔反对"抽象的普遍"（"所有人都是一样的"），主张"具体的普遍"（"普遍性中包含了差异性"）。比如"人"这个概念——如果只说"人是有理性的动物"，这是"抽象的普遍"。但"具体的人"有性别、种族、阶级、时代、性格等差异——这些差异不是"普遍的例外"，是"普遍本身的组成部分"。没有差异的普遍是"空洞的"
- **黑格尔的影响**: 黑格尔的逻辑学影响了几乎所有 19 世纪和 20 世纪的哲学流派——马克思（辩证唯物主义）、存在主义（通过克尔凯郭尔的反黑格尔立场）、分析哲学（罗素和摩尔早期是黑格尔主义者后来反叛了）、法兰克福学派、结构主义。理解黑格尔是理解"现代哲学史"的前提——不管你喜欢他还是讨厌他，你都绕不开他
- **为什么难读**: 黑格尔被公认为"最难读的哲学家"之一。原因：1）他的概念体系"自成一格"——他不使用日常语言的含义，而是重新定义了几乎所有哲学概念；2）他的论证是"螺旋式"的——不是"线性推理"，是"不断回到前面的概念但在更高层次上重新讨论"；3）他的德文本身就"极其晦涩"——翻译成中文后难度翻倍。读黑格尔需要"耐心"和"反复"——第一遍看不懂是正常的
- **黑格尔的背景**: 格奥尔格·威廉·弗里德里希·黑格尔（1770-1831）是德国哲学家，与康德、费希特、谢林并称"德国观念论四大哲学家"。他出生于斯图加特，在图宾根大学与谢林、荷尔林是同学。他的主要著作包括《精神现象学》（1807）、《逻辑学》（大逻辑，1812-1816）、《哲学科学全书纲要》（含小逻辑，1817）、《法哲学原理》（1820）。他 1818 年到柏林大学任教，1830 年成为校长，1831 年死于霍乱
