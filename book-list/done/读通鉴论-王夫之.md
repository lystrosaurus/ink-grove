# 读通鉴论

状态：已读

作者：王夫之

一句话评价：王夫之的"史论集巅峰之作'"——"'读《资治通鉴》的史论'"。"'以史为鉴评点历代政治得失'"。"'历史不是简单的事件记录，而是"理"与"势"的交织'"。"'对中国古代史论的深远影响'"

核心收获：
- **理与势的史观**: 王夫之提出了"'理"与"势'"的史观'"。"'什么'是'理'"——"'理'是'历史'的'规律'"——"'是'不变'的'道理'"。"'什么'是'势'"——"'势'是'历史'的'趋势'"——"'是'时代'的'潮流'"。"'历史'就'是'理'与'势'的'交织'"。"'理'不'能'脱离'势'"——"'势'不'能'违背'理'"。"'比如'封建制'变'郡县制'"——"'这'是'势'"——"'但'背后'有'理'"——"'中央集权'更'有利于'统一'"。"'这种'史观'"——"'比'简单'的'道德'评判'更'深刻'"
- **对封建制的批判**: 王夫之"批判了封建制'"。"'封建制'是'什么'"——"'分封'诸侯'"——"'各自'为政'"。"'王夫之'认为'"——"'封建制'导致'分裂'"——"'导致'战争'"。"'周朝'封建'"——"'后来'春秋战国'战乱'"。"'秦'废'封建'"——"'行'郡县'"——"'这'是'进步'"。"'但'王夫之'也'指出'"——"'郡县制'有'问题'"——"'中央'权力'太'大'"——"'皇帝'昏庸'就'亡国'"。"'他'的'态度'"——"'不'简单'肯定'或'否定'"——"'分析'利弊'"
- **对科举制的分析**: 王夫之"分析了科举制'"。"'科举制'的'优点'"：1）**公平**——"'不论'出身'"——"'都'可以'考'"；2）**选才**——"'选拔'有'才学'的'人'"；3）**稳定**——"'读书人'有'出路'"——"'不'造反'"。"'科举制'的'缺点'"：1）**僵化**——"'考'八股'"——"'束缚'思想'"；2）**无用**——"'考'的'东西'不'实用'"；3）**腐败**——"'有'舞弊'"。"'王夫之'的'态度'"——"'科举'比'九品中正'好'"——"'但'也'有'问题'"
- **以史为鉴**: 王夫之"强调以史为鉴'"。"'读史'的'目的'"——"'不'是'记住'事件'"——"'是'理解'规律'"。"'如何'以史为鉴'"：1）**理解理势**——"'理解'历史'的'理'和'势'"；2）**吸取教训**——"'从'失败'中'学习'"；3）**指导现实**——"'用'历史'智慧'指导'现在'"。"'王夫之'说'"——"'读史'可以'明智'"——"'明智'不'是'知道'很多'事'"——"'是'理解'历史'规律'"
- **对历代帝王的评价**: 王夫之"评价了历代帝王'"。"'他'对'秦始皇'"——"'功'在'统一'"——"'过'在'暴政'"。"'他'对'汉武帝'"——"'功'在'开疆'"——"'过'在'劳民'"。"'他'对'唐太宗'"——"'贞观之治'值得'称赞'"——"'但'晚年'也'有'过失'"。"'他'的'评价'"——"'不'一味'歌颂'"——"'不'一味'批评'"——"'客观'分析'"
- **王夫之的史论方法**: 王夫之"的史论方法独特'"。"'他'的'方法'"：1）**就事论理**——"'从'具体'事件'中'提炼'道理'"；2）**古今对照**——"'用'古代'对照'现在'"；3）**辩证分析**——"'不'简单'肯定'或'否定'"。"'他'的'文字'"——"'犀利'"——"'深刻'"。"'他'敢'说'真话'"——"'批判'专制'"——"'批判'暴政'"
- **王夫之的背景**: 王夫之（1619-1692）——明末清初思想家、史学家——"《读通鉴论》的作者'"。"他'是'明朝'遗民'"——"'清兵'入关'后'"——"'他'隐居'不'出'"。"'他'著书'立说'"——"'批判'专制'"——"'反思'明朝'灭亡'"。"《读通鉴论》'是'他'晚年'作品'"——"'他'读'《资治通鉴》'"——"'写'下'心得'"。"'王夫之'的'思想'"——"'对'后世'影响'很大'"——"'梁启超'——'谭嗣同'都'受'他'影响'"
