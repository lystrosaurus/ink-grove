# 谁在看中国画

状态：已读

作者：柯律格

一句话评价：柯律格（Craig Clunas）的"艺术史佳作'"——"'跳出"名作赏析"的传统框架'"。"'从观看者的社会身份出发'"——"'揭示中国绘画在不同时代、不同人群中的功能和意义'"。"'谁在看'比'看什么'更重要'"——"'同一幅画，帝王、文人、商人、工匠看到的东西完全不同'"

核心收获：
- **观看者的身份**: 本书核心是"'谁在看'"的问题'"。"'传统'艺术史'"——"'关注'作品'本身'"——"'这是'名作'"——"'那是'杰作'"。"'柯律格'说'不对'"——"'要'关注'谁'在'看'"。"'帝王'看画'"——"'看'权力'"——"'看'统治'的'合法性'"。"'文人'看画'"——"'看'品味'"——"'看'身份'的'象征'"。"'商人'看画'"——"'看'投资价值'"——"'看'社交'资本'"。"'工匠'看画'"——"'看'技术'"——"'看'如何'模仿'"。"'所以'同一幅画'"——"'不同'人'看到'不同'的'东西'"
- **社会身份的展演**: 柯律格提出"'鉴赏是社会身份的展演'"。"'什么'意思'"——"'看画'不'是'纯'审美'活动'"——"'是'表演'自己的'身份'"。"'文人'雅集'"——"'一起'看画'"——"'品评'"——"'题跋'"——"'这'是'在'展示'自己'的'文化'资本'"。"'商人'收藏'"——"'买'名画'"——"'请'文人'题跋'"——"'这'是'在'提升'自己'的'地位'"。"'所以'艺术'不'是'超脱'的'"——"'艺术'是'社会'的'"
- **文化资本的较量**: 柯律格"用布迪厄的文化资本理论分析中国画'"。"'什么'是'文化'资本'"——"'知识'——'品味'——'教养'"。"'文化'资本'可以'转化'为'社会'资本'"——"'有'文化'的'人'"——"'地位'高'"。"'中国'画'的'鉴赏'"——"'需要'知识'"——"'需要'品味'"。"'有'知识'的'人'"——"'能'看懂'画'"——"'能'题诗'"——"'能'品评'"。"'这'些'人'"——"'有'文化'资本'"——"'地位'高'"。"'没有'知识'的'人'"——"'看'不'懂'"——"'地位'低'"
- **艺术市场的形成**: 柯律格"分析了中国艺术市场的形成'"。"'明代'以后'"——"'商品经济'发展'"——"'艺术'市场'形成'"。"'画作'成为'商品'"——"'可以'买卖'"。"'价格'如何'确定'"——"'不'是'艺术'价值'"——"'是'作者'名声'"——"'是'稀缺性'"。"'名'画家'的'画'"——"'价格'高'"。"'假货'泛滥'"——"'因为'有'利润'"。"'艺术'市场'的'影响'"：1）**促进创作**——"'画家'可以'卖画'为生'"；2）**导致庸俗**——"'画家'为'市场'创作'"——"'迎合'买家'"
- **收藏与流通**: 柯律格"讨论了收藏与流通'"。"'收藏'的'目的'"：1）**欣赏**——"'自己'看'"；2）**投资**——"'等'升值'"；3）**社交**——"'送'礼'"——"'炫耀'"。"'流通'的'方式'"：1）**买卖**——"'市场'交易'"；2）**馈赠**——"'朋友'之间'赠送'"；3）**继承**——'**传给**后代'"。"'收藏'的'记录'"——"'著录'"——"'让'后人'知道'哪些'画'流传'过'"
- **复制与传播**: 柯律格"分析了复制与传播'"。"'古代'没有'印刷术'"——"'画作'如何'传播'"：1）**摹本**——"'临摹'"——"'复制'"；2）**版画**——"'刻'成'版画'"——"'大量'复制'"；3）**著录**——"'文字'描述'"。"'复制'的'好处'"——"'让'更多'人'看到'"。"'复制'的'坏处'"——"'真假'难辨'"——"'原作'的'价值'被'稀释'"
- **柯律格的背景**: 柯律格（Craig Clunas, 1954-）——英国艺术史家、汉学家——"《谁在看中国画》的作者'"。"他'在'牛津大学'任教'"——"'研究'中国'艺术'史'"。"他'的'研究'方法'"——"'社会'史'方法'"——"'不'是'传统'的'风格'分析'"。"《谁在看中国画》'2010 年代'出版'"——"'获得'好评'"。"'他'的'其他'著作'"——"'《长物志》研究'"——"'《中国'绘画'社会'史》'"——"'都'从'社会'角度'研究'艺术'"
