# 第一哲学沉思集

状态：已读

作者：笛卡尔

一句话评价：近代哲学开山之作，笛卡尔用"普遍怀疑"方法找到了不可怀疑的基点——"我思故我在"，确立了理性主义哲学传统。

核心收获：
- 理解了"普遍怀疑"方法：怀疑一切可以怀疑的东西，直到找到不可怀疑的基点
- "我思故我在"（Cogito ergo sum）——即使我在怀疑一切，"我在怀疑"这件事本身是不可怀疑的
- 笛卡尔用"我思"确立了主体的存在，开启了近代哲学从"世界是什么"到"我能知道什么"的转向
