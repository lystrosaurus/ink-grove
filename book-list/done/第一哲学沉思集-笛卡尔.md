# 第一哲学沉思集

状态：已读

作者：勒内·笛卡尔

一句话评价：勒内·笛卡尔（René Descartes）的"近代哲学开山之作"——"用'普遍怀疑'方法找到了'不可怀疑的'基点"——"'我思故我在'（Cogito ergo sum）"——"'确立了'理性主义哲学传统'"。"'怀疑一切'可以怀疑的东西'"——"'直到'找到'不可怀疑的'基点'"——"'即使'我'在'怀疑一切'"——"'我'在'怀疑'这件事本身'是'不可怀疑的'"——"笛卡尔用'我思'确立了'主体的存在'"——"'开启了'近代哲学'从'世界是什么'到'我能知道什么'的'转向'"

核心收获：
- **普遍怀疑方法**: 笛卡尔提出了"'普遍怀疑'方法'"——"'怀疑一切'可以怀疑的东西'"。"'感官'可能'欺骗我们'"——"'我们'看到'的'东西'可能'是'幻觉'"——"'我们'听到'的'东西'可能'是'假的'"。"'数学'可能'也'不可靠'"——"'可能'有'一个'邪恶的'恶魔'在'欺骗我们'"——"'让'我们'相信'1+1=2'"——"'但'其实'不是'"。"笛卡尔说'我'要'怀疑'一切'"——"'直到'找到'一个'不可怀疑的'东西'"
- **我思故我在**: 笛卡尔找到了"'不可怀疑的'基点'"——"'我思故我在'（Cogito ergo sum）"。"'即使'我'在'怀疑一切'"——"'我'在'怀疑'这件事本身'是'不可怀疑的'"——"'因为'如果'我'不'存在'"——"'我'就'不能'怀疑'"。"'我思'证明了'我'的存在'"——"'我'是'一个'思考的东西'"——"'我'是'一个'心灵'"——"'我'的'本质'是'思考'"。"笛卡尔说'这是'哲学'的'第一原理'"——"'从'这个'原理'出发'"——"'可以'建立'整个'知识体系'"
- **身心二元论**: 笛卡尔提出了"'身心二元论'"——"'心灵'和'身体'是'两个'不同的'实体'"。"'心灵'的'本质'是'思考'"——"'身体'的'本质'是'广延'（占据空间）'"。"'心灵'不'依赖'身体'而'存在'"——"'身体'也不'依赖'心灵'而'存在'"。"'身心二元论'的'问题'"——"'心灵'和'身体'如何'相互作用'"——"'如果'它们'是'两个'不同的'实体'"——"'它们'怎么'能够'互相影响'"——"'这个'问题'困扰'了'后世'哲学家'几百年'"
- **上帝的存在证明**: 笛卡尔"用'我思'证明了'上帝的存在'"。"'我'有'一个'上帝的'观念'"——"'这个'观念'包含'无限'——'完美'——'全能'等'属性'"。"'我'是'有限的'——'不完美的'"——"'所以'这个'观念'不'可能'来自'我'"。"'这个'观念'必须'来自'一个'无限'——'完美的'存在'"——"'这个'存在'就是'上帝'"。"笛卡尔说'上帝'是'完美的'"——"'所以'上帝'不会'欺骗我们'"——"'所以'我们'的'感官'在'一定程度'上'是'可靠的'"
- **清楚明白的观念**: 笛卡尔提出了"'清楚明白的观念'标准'"。"'什么'是'真理'"——"'清楚明白'的'观念'就'是'真理'"。"'我思故我在'是'清楚明白'的'"——"'所以'它'是'真理'"。"'上帝'的'观念'是'清楚明白'的'"——"'所以'上帝'存在'"。"笛卡尔说'清楚明白'的'标准'来自'上帝'"——"'因为'上帝'是'完美的'"——"'所以'他'不会'让'我们'被'清楚明白'的'观念'欺骗'"
- **近代哲学的转向**: 笛卡尔"开启了'近代哲学的'转向'"。"'在'笛卡尔'之前'"——"'哲学'问'的'是'世界是什么'"——"'世界的'本质'是什么'"。"'在'笛卡尔'之后'"——"'哲学'问'的'是'我能知道什么'"——"'知识'的'基础'是什么'"。"笛卡尔'把'哲学'的'焦点'从'本体论'转向'了'认识论'"——"'从'世界'转向'了'主体'"——"'从'客体'转向'了'思维'"——"'这个'转向'影响'了'整个'近代哲学'"
- **勒内·笛卡尔的背景**: 勒内·笛卡尔（René Descartes, 1596-1650）——法国哲学家、数学家、科学家——"近代哲学的奠基人"——"《第一哲学沉思集》是他'最'重要的'哲学著作'"——"1641 年出版"——"他'还'是'解析几何'的'创始人'"——"'笛卡尔坐标系'就是'他'发明的'"。"笛卡尔'的'名言'我思故我在'"——"'成为'了'哲学史上'最'著名的'命题之一'"
