# 数学文化透视

状态：已读

作者：汪晓勤

一句话评价：从文化视角解读数学发展史，揭示数学与哲学、艺术、宗教、社会的深层联系，是一本跨学科的数学人文读物。

核心收获：
- 理解了数学不仅是工具和文化——它本身就是一种文化现象，反映了一个时代的思维方式
- 数学史上的重大转折（如无理数的发现、非欧几何的诞生）不仅是数学进步，更是人类认知范式的转变
- 数学与艺术的联系：黄金分割、分形几何、对称群——数学美是艺术美的重要组成部分
