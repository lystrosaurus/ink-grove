# 百年孤独

## 基本信息
- **书名**: 百年孤独
- **作者**: 加西亚 - 马尔克斯
- **出版社**: 南海出版公司
- **出版年份**: 2020
- **页数**: 360
- **分类**: 文学经典

## 选书理由
魔幻现实主义巅峰之作，讲述布恩迪亚家族七代人的兴衰。"多年以后，面对行刑队..."这个开头被无数人模仿。马尔克斯用神话般的笔法写出了一个大陆的孤独命运——不是缺乏爱，而是缺乏爱的能力。希望在拉美文学的魔幻叙事中，理解孤独的普遍性与人类命运的循环。
