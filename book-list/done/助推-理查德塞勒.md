# 助推

状态：已读

作者：理查德·塞勒

一句话评价：行为经济学经典之作，提出"自由家长主义"——在不限制选择自由的前提下，通过改变"选择架构"来引导人们做出更好的决策。从养老金自动加入到器官捐献默认选项，助推正在改变政策设计的底层逻辑。

核心收获：
- **助推（Nudge）的定义**: 助推是"在选择架构中做出的任何改变，它以可预见的方式改变人们的行为，而不禁止任何选项，也不显著改变经济激励"。简单说：不强迫、不禁止、不惩罚，只是"轻轻推一下"，让人们更容易做出好的选择
- **自由家长主义（Libertarian Paternalism）**: 这是全书的核心哲学。"自由"意味着人们有选择的自由；"家长主义"意味着设计者希望引导人们做出"更好"的选择。两者看似矛盾，但助推实现了统一——你可以选择"不助推"，但大多数人会顺着设计者的引导走
- **选择架构（Choice Architecture）**: 每一个选择都有"架构"——选项的排列顺序、默认设置、信息呈现方式都会影响人的决定。没有"中性"的选择架构——不管你怎麼设计，都会影响人的行为。所以问题不是"要不要设计"，是"怎么设计才对人们有利"
- **默认选项的力量**: 默认选项是最强大的助推工具。比如养老金计划——如果默认是"不参加"，大部分人不会参加；如果默认是"参加"，大部分人不会退出。器官捐献也是——"默认捐献"国家的捐献率远高于"默认不捐献"国家。人们倾向于"不做改变"，所以默认设置决定了大多数人的选择
- **人类并非理性经济人**: 传统经济学假设人是理性的，会做出最优选择。行为经济学证明人不是——人有认知偏差、短视、从众、损失厌恶。助推不是"欺骗"，是"帮助不完美的人做出更好的选择"。就像在悬崖边设栏杆——不是限制你的自由，是防止你不小心掉下去
- **助推的实际应用**: 1）健康——食堂把健康食品放在显眼位置，垃圾食品放在角落；2）财务——自动加入养老金计划，但允许随时退出；3）环保——电费账单上显示"你的用电量比邻居多 20%"；4）教育——发送短信提醒家长孩子的作业截止日期
- **助推的伦理争议**: 批评者说助推是"温柔的操纵"——虽然不禁止选择，但利用了人的认知弱点。塞勒的回应是：1）选择架构总是存在的，不设计也是一种设计；2）助推是透明的，可以被公开讨论；3）助推的目标是帮助人们实现他们"真正想要的"，而不是设计者想要的
- **塞勒的背景**: 理查德·塞勒是 2017 年诺贝尔经济学奖得主，是行为经济学的奠基人之一。他和卡斯·桑斯坦（哈佛法学院教授）合著了这本书。塞勒的研究证明了"人不是理性的"，这本书证明了"不理性的人也可以被善意地引导"
