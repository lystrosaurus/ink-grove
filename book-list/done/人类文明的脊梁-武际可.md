# 人类文明的脊梁

状态：已读

作者：武际可

一句话评价：用力学视角串联人类文明史，从拱桥到摩天大楼，从水利工程到航天器，每一座"脊梁"背后都是人类与自然对抗又和解的故事。

核心收获：
- 力学是理解人类文明的一把钥匙——建筑、交通、水利、航天，核心都是"力的平衡"
- 中国古代的工程智慧：赵州桥、都江堰、应县木塔，在没有现代力学理论的情况下凭经验做到了极致
- 工程失败的价值：塔科马海峡大桥倒塌、挑战者号爆炸，每次灾难都推动了力学的进步
- 结构之美和力学之真是统一的——好的工程作品同时是艺术品
- 现代工程面临的挑战：超高层建筑、深海探测、太空站，每一个都在推演力学的边界
