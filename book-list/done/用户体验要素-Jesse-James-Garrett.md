# 用户体验要素

状态：已读

作者：Jesse-James-Garrett

一句话评价：交互设计领域的方法论经典，提出了用户体验的五层模型（战略层、范围层、结构层、框架层、表现层），是产品设计和开发的必备指南。

核心收获：
- 理解了用户体验的五层模型：从抽象到具体，每一层都建立在前一层之上
- 战略层（用户需求和产品目标）是最基础的一层——没有清晰的战略，后面的设计都是空中楼阁
- 每一层的决策都应该基于上一层——不能跳过战略层直接做视觉设计
