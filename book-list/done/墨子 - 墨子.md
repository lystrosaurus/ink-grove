# 墨子

状态：已读

作者：墨子

一句话评价：先秦诸子中最具实用主义、科学精神和平民色彩的一家。"兼爱非攻"是超越时代的和平主义宣言，"墨经"中的逻辑学和自然科学成就比西方早了两百多年。墨家在秦汉后几乎消失，但它的精神在民间始终存在。

核心收获：
- **"兼爱" vs "仁爱"**: 墨子的"兼爱"是"无差别的爱"——爱别人的父亲像爱自己的父亲，爱别人的国家像爱自己的国家。这直接挑战了儒家的"差序格局"（爱有等差，从亲人推到外人）。墨子认为天下大乱的根源就是"别"（区分你我），解决办法是"兼"（视人如己）
- **"非攻"的正义观**: "非攻"不是反对所有战争，是反对"不义之战"（侵略战争）。墨子不反对"诛"（讨伐暴君的正义战争）。他不仅理论上反战，还实践反战——听说楚国要打宋国，走了十天十夜去劝阻，并用守城技术威慑楚国。这是中国古代最彻底的和平主义实践
- **墨家的科学成就**: 《墨经》（《墨子》中的《经上》《经下》《经说上》《经说下》）包含了中国古代最系统的自然科学知识：光学（小孔成像、光的直线传播、反射）、力学（杠杆原理、力的定义"刑之所以奋也"）、几何学（圆的定义"一中同长也"）、逻辑学（"故""理""类"三物逻辑）。这些成就比古希腊晚一点，但在中国古代是绝无仅有的
- **"尚贤"与"尚同"**: "尚贤"是选贤任能——不管出身贵贱，有才能就提拔。这是对世袭贵族制的直接挑战。"尚同"是自下而上地统一意见——里长统一里的意见，乡长统一乡的意见，国君统一国的意见，天子统一天下的意见。最后"上同于天"。这有集权倾向，但初衷是消除混乱
- **"节用"与"节葬"**: 墨子极度反对奢侈浪费。"节用"要求衣食住行够用就好，不要追求华丽。"节葬"反对儒家的"三年之丧"和厚葬——人死了花那么多钱埋起来，活人却吃不饱，这是本末倒置。墨子的节俭观代表了底层劳动者的价值观
- **"天志"与"明鬼"**: 墨子相信天有意志（"天志"），相信鬼神存在（"明鬼"）。这不是迷信，是道德约束——他怕人做坏事没人管，所以用"天"和"鬼神"来威慑。"天欲义而恶不义"——天的意志就是兼爱、非攻、尚贤。这是墨家的"宗教"维度
- **墨家的衰落**: 墨家在战国时和儒家并称"显学"（"世之显学，儒墨也"），但秦汉后几乎消失。原因：1）墨家的组织太严密（类似宗教团体），不利于大一统帝国；2）"兼爱"太理想化，不符合人性；3）儒家被独尊后，墨家被边缘化；4）墨家的科学精神在中国古代没有制度性传承
- **墨家的现代意义**: 墨家的"兼爱"对应现代的"博爱"，"非攻"对应"反战"，"尚贤"对应" meritocracy"，"节用"对应"环保"，"墨经"的科学精神对应"理性"。胡适说墨家是"中国最早的科学家和哲学家"。如果墨家没有衰落，中国的科学传统可能会完全不同
