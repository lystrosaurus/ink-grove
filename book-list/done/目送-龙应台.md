# 目送

状态：已读

作者：龙应台

一句话评价：龙应台的"亲情散文集"——"以'父亲的逝、母亲的老、儿子的远'为主线'"——"写尽了'人到中年'面对'亲情离别'的'无奈与深情'"。"'所谓父女母子一场，只不过意味着，你和他的缘分就是今生今世不断地在目送他的背影渐行渐远。'"——"这是'全书'最'打动人心'的句子"。"亲情中最痛的'部分'不是'争吵'"——"是'沉默'"——"父母老了"——"孩子远了"——"你只能'目送'"。

核心收获：
- **目送的意象**: "目送"是"全书的核心意象"。"龙应台'写'儿子的'背影'"——"'他'背着'书包'走进'校门'"——"'没有'回头'"——"'她'站在'原地'"——"'看着'他的'背影'消失'"。"她'写'父亲的'背影'"——"'他'被'推进'手术室'"——"'没有'回头'"——"'她'站在'门外'"——"'看着'门'关上'"。"目送'是'一种'无奈'"——"'你'知道'他'要'离开'"——"'你'知道'你'留不住'"——"'你'只能'看着'"
- **父亲的逝**: 书中"第一部分"是"关于父亲"。"父亲'得了'阿兹海默症'"——"'他'渐渐'忘记'了'一切'"——"'他'忘记'了'自己的'名字'"——"'他'忘记'了'女儿'"——"'最后'"——"'他'忘记'了'怎么'说话'"——"'怎么'吃饭'"——"'怎么'走路'"。"龙应台'看着'父亲'一天天'衰弱'"——"'她'想'抓住'什么'"——"'但'什么'也'抓不住'"——"'她'只能'目送'父亲'走向'终点'"
- **母亲的老**: 书中"第二部分"是"关于母亲"。"母亲'老了'"——"'她'的'眼睛'花了'"——"'她'的'耳朵'聋了'"——"'她'的'腿'脚'不灵便了'"。"龙应台'带'母亲'去'看病'"——"'母亲'像个'孩子'一样'依赖'她'"——"'她'突然'意识到'"——"'角色'互换了'"——"'小时候'是'母亲'照顾'她'"——"'现在'是'她'照顾'母亲'"。"这种'角色的'转换'"——"让'她'既'心疼'又'无奈'"
- **儿子的远**: 书中"第三部分"是"关于儿子"。"儿子'长大了'"——"'他'要'去'国外'读书'"——"'他'要'离开'家'"。"龙应台'送'儿子'去'机场'"——"'儿子'头也'不回'地'走了'"——"'她'站在'原地'"——"'看着'儿子的'背影'消失'在'安检口'"。"她'说'所谓'母子一场'"——"'就是'你'看着'他的'背影'渐行渐远'"——"'而'他'用'背影'告诉你'"——"'不必'追'"
- **沉默的痛**: 龙应台说"亲情中最痛的'部分'不是'争吵'"——"是'沉默'"。"父亲'病重'时'"——"'他们'之间'没有'太多'话'"——"'因为'该说的'都说了'"——"'不该说的'说不了'"。"儿子'要离开'时'"——"'他们'之间'也'没有'太多'话'"——"'因为'说了'也'留不住'"。"这种'沉默'"——"'比'任何'语言'都'更'沉重'"
- **中年的感悟**: 龙应台"在书中'写了'她'人到中年的'感悟'"。"中年'是'一个'特殊的'阶段'"——"'你'上有'老'"——"'下'有'小'"——"'你'是'夹心层'"。"你'看着'父母'老去'"——"'你'看着'孩子'长大'"——"'你'在'中间'"——"'你'承担着'一切'"——"'但'你'也'在'失去'一切'"。"龙应台说'中年'就是'学会'接受'失去'"
- **龙应台的背景**: 龙应台（1952-）——台湾作家、文化学者——"《目送》是她'最著名'的'亲情散文集'"——"2009 年出版"——"在'华语世界'引起了'巨大的'共鸣"。她"还写过'《野火集》'"——"'《孩子你慢慢来》'"——"'《亲爱的安德烈》'"——"但'《目送》'是'她'最'打动人心'的作品'"——"因为'它'写的是'每个人'都'会经历的'亲情离别'"
