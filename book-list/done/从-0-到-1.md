## 一句话评价
PayPal 创始人 Peter Thiel 的创业哲学。"0 到 1 是创造，1 到 n 是复制"是核心区分，"竞争是失败的，垄断才是目标"是反直觉洞见。

## 核心收获
1. 0 到 1vs1 到 n：创造新事物 vs 复制已有，垂直进步 vs 水平进步
2. 未来四种态度：明确/模糊 × 乐观/悲观，现在是模糊的乐观
3. 竞争是失败：竞争消灭利润，垄断才能积累资本
4. 垄断四特征：专利技术（10 倍好）、网络效应、规模经济、品牌优势
5. 垄断者谎言：隐藏垄断说面临竞争，非垄断者夸大独特性
6. 后发优势：last mover advantage 比 first mover 重要，让先行者犯错
7. 秘密问题：什么重要真相很少人同意你？答案就是创业机会
8. 基础定律：公司早期错误无法修复，必须一开始就做对
9. 销售重要：产品不会自动卖，销售是 skill 被严重低估
10. 清洁技术教训：没有 0 到 1 创新， incremental 改进必败
11. 创始人悖论：极端特质是双刃剑，但要接受利用而非"正常化"
12. 垄断不是违法：是创造无可替代产品的结果

## 实践改变
- 追求 0 到 1：在工作中寻找创造机会，不只是复制
- 重新思考竞争：不以竞争为目标，追求独特价值
- 寻找秘密：问"什么真相很少人同意我"，独立思考
- 重视基础：做决定考虑长期，不急于行动，建立正确基础
- 学习销售：不忽视销售，学习表达，产品需要被销售
