# 万历十五年

状态：已读

作者：黄仁宇

一句话评价：用"大历史观"解剖明朝——万历十五年（1587年）看似平平无奇，但帝国的衰亡种子在这一年已经种下。

核心收获：
- 大历史观：不是看"这一年发生了什么大事"，而是看"制度层面的问题如何在这一年集中暴露"
- 万历皇帝：28年不上朝，不是懒，是"消极抵抗"——他发现自己做不了主，索性什么都不做
- 张居正：改革派的悲剧。活着时权倾天下，死后被抄家清算。改革触动了所有人的利益
- 海瑞：道德楷模但政治上毫无建树。道德是个人修养，不能替代制度和治理
- 戚继光：军事天才但在文官体系中处处受制。再好的将领也打不赢体制的仗
- 文官集团：明朝真正的统治者不是皇帝，是文官集团。他们用道德绑架一切，用程序拖延一切
