# 人生的智慧

状态：已读

作者：叔本华

一句话评价：悲观主义哲学家里最实用的一本——告诉你幸福的关键不是获得更多，而是减少痛苦。

核心收获：
- 幸福取决于三个因素：人是什么（人格）、人有什么（财产）、人在他人眼中是什么（声誉）。其中"人是什么"最重要
- 痛苦的根源是欲望。欲望不满足就痛苦，满足了就无聊。人生在痛苦和无聊之间摆动
- 独处是最高级的能力。无法独处的人注定被外界支配
- 健康比财富重要一万倍。宁要健康的乞丐，不要生病的国王
- 对他人期待越低，人际关系越和谐
