# 绘画中的世界观

状态：已读

作者：迈耶·夏皮罗

一句话评价：迈耶·夏皮罗（Meyer Schapiro）的"艺术评论集'"——"'从绘画、雕塑到建筑，探讨了艺术形式与思想观念之间的深层联系'"。"'艺术形式不是单纯的审美选择'"——"'是'世界观的表达'"——"'每一种风格背后都有一种看待世界的方式'"。"夏皮罗的分析方法'"——"'把艺术作品放在历史、社会、哲学的语境中解读'"——"'不是孤立的审美评判'"

核心收获：
- **艺术形式是世界观的表达**: 夏皮罗说"'艺术形式不是单纯的审美选择'"——"'是'世界观的表达'"。"'每一种风格'"——"'背后'都有'一种'看待世界的方式'"。"'中世纪艺术'的'平面化'"——"'因为'中世纪人'认为'精神世界'高于'物质世界'"——"'他们'不追求'透视'"——"'他们'追求'象征意义'"。"'文艺复兴艺术'的'透视法'"——"'因为'文艺复兴人'相信'人是万物的尺度'"——"'他们'从'人的视角'看世界'"。"'现代艺术'的'抽象化'"——"'因为'现代人'怀疑'传统'的'再现方式'"——"'他们'追求'形式本身'"
- **历史语境的重要性**: 夏皮罗的"分析方法'强调历史语境'"。"'不能'孤立地'看'一件作品'"——"'要'把它'放在'历史中'"——"'理解'它'产生的'时代背景'"。"'为什么'梵高'用'强烈的色彩'"——"'因为'他'生活'在'工业革命后'"——"'他'看到'了'传统'的'崩溃'"——"'他'用'色彩'表达'内心的'痛苦'"。"'为什么'毕加索'画'立体主义'"——"'因为'他'生活'在'一战前'"——"'他'看到'了'传统'视角'的'局限'"——"'他'用'多个视角'同时'呈现'物体'"
- **社会与艺术的关系**: 夏皮罗"分析了社会与艺术的关系'"。"'艺术'不是'脱离社会'的'"——"'它'反映'社会'的'价值观'"——"'它'也'影响'社会'的'观念'"。"'中世纪'的'宗教艺术'"——"'服务于'教会'"——"'强化'宗教'信仰'"。"'文艺复兴'的'世俗艺术'"——"'反映'了'人文主义'的'兴起'"。"'现代艺术'的'反叛性'"——"'反映'了'现代社会'的'异化'"。"夏皮罗说'艺术家'不是'孤立的'天才'"——"'他'是'社会的一部分'"——"'他的'作品'是'社会'的'产物'"
- **符号学分析**: 夏皮罗"使用符号学方法分析艺术'"。"'艺术作品'中的'每一个元素'"——"'都是'一个'符号'"——"'它'有'特定的'含义'"。"'比如'基督教艺术'中的'羔羊'"——"'象征'耶稣'"。"'比如'中国山水画'中的'松'"——"'象征'长寿'"。"'符号'的'含义'不是'固定的'"——"'它'随'时代'变化'"。"'理解'符号'的'含义'"——"'需要'理解'那个时代'的'文化'"
- **现代艺术的革命**: 夏皮罗"分析了现代艺术的革命'"。"'现代艺术'的'革命'"——"'不是'形式上的'突破'"——"'是'对'艺术是什么'的'重新思考'"。"'印象派'"——"'不再'追求'再现'现实'"——"'追求'光'和'色'的'感受'"。"'立体主义'"——"'不再'追求'单一视角'"——"'追求'多角度'同时'呈现'"。"'抽象表现主义'"——"'不再'追求'描绘'物体'"——"'追求'情感'的'直接'表达'"。"夏皮罗说'现代艺术'的'核心问题'"——"'艺术'的'界限'在哪里'"——"'什么'可以'是'艺术'"
- **艺术与哲学的联系**: 夏皮罗"探讨了艺术与哲学的联系'"。"'艺术'和'哲学'都是'对世界的'理解方式'"——"'艺术'用'形象'"——"'哲学'用'概念'"。"'存在主义'的'焦虑'"——"'在'表现主义艺术'中'得到'表达'"。"'结构主义'的'符号分析'"——"'在'波普艺术'中'得到'体现'"。"'后现代主义'的'解构'"——"'在'观念艺术'中'得到'实践'"。"夏皮罗说'理解艺术'"——"'需要'理解'背后的'哲学思想'"
- **迈耶·夏皮罗的背景**: 迈耶·夏皮罗（Meyer Schapiro, 1904-1996）——美国艺术史学家——"哥伦比亚大学教授'"。"他是'20 世纪'最'有影响力的'艺术史学家之一'"。"他的'研究'涵盖'中世纪艺术'——'现代艺术'"——"'他'的'分析方法'融合了'形式分析'——'社会史'——'符号学'"。"《绘画中的世界观》'是他'的'艺术评论集'"——"'收录'了'他'关于'艺术'与'思想'的'文章'"
