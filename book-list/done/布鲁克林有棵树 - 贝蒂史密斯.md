# 布鲁克林有棵树

状态：已读

作者：贝蒂·史密斯

一句话评价：20 世纪最温暖的成长小说之一（1943 年出版），讲述 20 世纪初纽约布鲁克林贫民窟女孩弗兰西在贫困中通过阅读和教育改变命运的故事。弗兰西一家住在破旧的公寓里，父亲是个有才华但酗弱的歌手，母亲凯蒂做着清洁工的工作养活一家人。但弗兰西有一样东西——图书馆。她一本一本地读书（从 A 到 Z 按字母顺序读完布鲁克林公共图书馆的所有书），用阅读对抗贫困，用教育打开未来。"愿你是那棵在水泥地里也能长成的树。"

核心收获：
- **阅读的力量**: 弗兰西改变命运的工具是"阅读"。她每周去布鲁克林公共图书馆借一本书——按字母顺序从 A 开始读（A 作者的所有书读完再读 B）。她不是"选好书读"——是"所有书都读"。这种"饥渴的阅读"让她看到了"布鲁克林贫民窟之外的世界"。她读莎士比亚、读狄更斯、读圣经——这些书给了她"想象力"和"希望"。史密斯说"一个读书的人永远不会真正贫穷"——因为"书是你最便宜的旅行票"
- **凯蒂的力量**: 弗兰西的母亲凯蒂是全书最强大的角色。她不识字、做清洁工、生活困苦，但她有"钢铁般的意志"和"清醒的头脑"。她知道"教育是唯一能让孩子摆脱贫困的路"——所以她拼命存钱（把零钱藏在床垫下），确保孩子们能上学。她教弗兰西一个重要的道理："每天读一页圣经，即使你不理解。"不是因为宗教，是因为"习惯的力量"——每天做一件事，久而久之它就成为了你的一部分
- **父亲的悲剧**: 弗兰西的父亲约翰尼是一个"有才华但失败的人"——他是一个好歌手、好讲故事的人、好父亲，但他酗酒、没有固定工作、最终因肺炎（酒精削弱了免疫力）在 34 岁时去世。约翰尼代表了"有天赋但没有毅力"的人——他的才华没有转化为"现实的成就"。史密斯对约翰尼的态度是"同情但不美化"——他爱孩子，但他的软弱给家庭带来了巨大的痛苦
- **贫困中的尊严**: 《布鲁克林有棵树》最动人的主题是"贫困不是堕落的借口"。弗兰西一家很穷——吃不起肉、穿不起新衣服、住在漏雨的公寓里——但他们保持着"尊严"。母亲凯蒂从不"乞讨"或"抱怨"——她"工作、存钱、计划未来"。弗兰西在学校被富家孩子嘲笑时，她不"自卑"——她知道"贫穷不是我的错，也不定义我是谁"。史密斯说"尊严可以在任何环境中保持"——它不取决于你有多少钱，取决于你"如何看待自己"
- **教育的意义**: 史密斯通过弗兰西的故事回答了"教育到底有什么意义"。教育不是"为了找一份好工作"（虽然这也是结果之一）——教育是"让你看到更大的世界"。弗兰西通过阅读知道了"布鲁克林之外还有曼哈顿、曼哈顿之外还有欧洲、欧洲之外还有整个世界"。教育给了她"野心"——不是"我要赚很多钱"，是"我要过一种更丰富的生活"。最终弗兰西考上了大学——成为了家族中第一个大学生
- **布鲁克林的天堂树**: 书名中的"树"指的是"天堂树"（Tree of Heaven / Ailanthus）——一种生命力极其顽强的树，它能在水泥地的裂缝中生长，不需要阳光和水，只需要"一点点土壤"就能存活。史密斯用这棵树隐喻弗兰西——一个在"贫民窟的水泥地"中长大的女孩，靠着"顽强的生命力"和"对知识的渴望"，最终"长成了一棵大树"
- **贝蒂·史密斯的背景**: 贝蒂·史密斯（Betty Smith, 1896-1972）原名伊丽莎白·韦纳，出生于纽约布鲁克林的贫民窟——她的成长经历和弗兰西高度相似。她也来自一个贫困的德裔移民家庭，也通过阅读和自学改变了自己的命运。《布鲁克林有棵树》（A Tree Grows in Brooklyn）1943 年出版后立刻成为了畅销书——1945 年被改编为同名电影。这是史密斯最著名的作品——她后来还写了续集《明天会更好》（Maggie-Now），但影响力远不及处女作
- **当代意义**: 在今天"阶层固化"和"教育焦虑"的语境下，《布鲁克林有棵树》的意义更加凸显——它提醒我们"教育仍然是底层孩子突破阶层的最有效工具"。弗兰西没有"天才智商"、没有"富有的父母"、没有"特殊的机会"——她只有"一本书接一本书地读"的毅力。这种"最朴素的方法"恰恰是"最有效的方法"
