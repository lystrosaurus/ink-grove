# OKR工作法

状态：已读

作者：克里斯蒂娜-沃特克

一句话评价：用故事讲OKR的落地方法——不是KPI的换皮，而是"聚焦+对齐+追踪"的目标管理系统，硅谷公司从Google到Stripe都在用。

核心收获：
- OKR vs KPI：KPI是考核工具，OKR是沟通工具。KPI告诉你"做到了没"，OKR告诉你"该往哪走"
- Objectives（目标）要鼓舞人心，Key Results（关键结果）要可量化。O是方向，KR是里程碑
- 每周一的OKR check-in是关键：本周要推进哪几个KR？信心指数是多少？有什么阻碍？
- 取消绩效挂钩：OKR一旦和奖金挂钩，员工就会定低目标，失去激励意义
- 透明化：所有人的OKR对全公司可见，这是对齐的基础
