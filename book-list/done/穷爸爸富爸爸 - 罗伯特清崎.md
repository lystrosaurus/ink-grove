# 穷爸爸富爸爸

状态：已读

作者：罗伯特·清崎

一句话评价：罗伯特·清崎（Robert Kiyosaki）的"个人理财启蒙经典"——"用'两个爸爸'（亲爸和朋友的爸）的'不同金钱观'对比'"——"'揭示了'富人和穷人'在财务思维上的'根本差异'"。"'资产'和'负债'的区别"——"'资产'把钱放进你的口袋'"——"'负债'把钱从你的口袋拿走'"。"'穷人和中产阶级为钱工作'"——"'富人让钱为自己工作'"——"'财务自由的关键'不是高薪'"——"'是'建立资产组合'"——"'让被动收入覆盖生活支出'"

核心收获：
- **资产与负债的区别**: 清崎说"'资产'和'负债'的区别"是"财务知识的基础"。"'资产'是'把钱放进你口袋的东西'"——"'如'出租房'——'股票'——'债券'——'版税'"。"'负债'是'把钱从你口袋拿走的东西'"——"'如'自住房'——'汽车'——'信用卡债务'"。"清崎说'穷人'和'中产阶级'的'问题'"——"'他们'把'负债'当成'资产'"——"'他们'买'大房子'"——"'买'好车'"——"'这些'东西'每月'都在'从'他们口袋'里'拿钱'"
- **为钱工作 vs 让钱工作**: 清崎说"'穷人和中产阶级为钱工作'"——"'富人让钱为自己工作'"。"'穷爸爸'（清崎的亲爸）'说'"——"'好好'学习'"——"'找个'好工作'"——"'为'公司'工作'"——"'为'钱'工作'"。"'富爸爸'（朋友的爸）'说'"——"'不要'为'钱'工作'"——"'要'让'钱'为'你'工作'"——"'建立'你的'资产组合'"——"'让'资产'产生'被动收入'"。"'当'你的'被动收入'超过'你的'生活支出'时'"——"'你'就'财务自由'了'"
- **财务自由的关键**: 清崎说"'财务自由的关键'不是高薪'"——"'是'建立资产组合'"。"'高薪'的人'可能'依然'不自由'"——"'因为'他们'的'支出'也'很高'"——"'他们'有'大房子'"——"'有好车'"——"'有'高额贷款'"。"'财务自由'的'定义'"——"'被动收入'超过'生活支出'"。"'要'实现'财务自由'"——"'你'必须'建立'资产组合'"——"'让'资产'产生'现金流'"——"'用'现金流'覆盖'支出'"
- **四个象限**: 清崎提出了"'四个象限'的概念'"：1）**E（Employee）**——"'雇员'"——"'为'别人'工作'"——"'拿'工资'"；2）**S（Self-employed）**——"'自雇人士'"——"'为'自己'工作'"——"'但'依然'在'卖时间'"；3）**B（Business owner）**——"'企业主'"——"'有'一个'系统'"——"'别人'为'他'工作'"；4）**I（Investor）**——"'投资者'"——"'钱'为'他'工作'"。"清崎说'要'实现'财务自由'"——"'你'必须'从'E 或 S'象限'转移到'B 或 I'象限'"
- **财商教育的重要性**: 清崎"强调'财商教育的重要性'"。"'学校'不'教'财务知识'"——"'所以'大多数人'不'懂'钱'的'规律'"。"'财商'包括'四个方面'"：1）**会计**——"'读懂'财务报表'"；2）**投资**——"'让'钱'生钱'"；3）**市场**——"'供给与需求'"；4）**法律**——"'利用'税法'保护'自己'"。"清崎说'财商'是'可以'学习的'"——"'你'必须'主动'学习'"——"'不能'依赖'学校'"
- **富人不为钱工作**: 清崎说"'富人不为钱工作'"——"'他们'为'学习'而'工作'"。"'富爸爸'让'清崎'在'他的'店里'工作'"——"'但'不'给'他'钱'"——"'清崎'很'生气'"——"'富爸爸'说'"——"'我'在'教'你'一个'道理'"——"'不要'为'钱'工作'"——"'要'为'学习'而'工作'"——"'学习'如何'让'钱'为你'工作'"。"清崎说'这个'道理'改变'了'他'的'一生'"
- **罗伯特·清崎的背景**: 罗伯特·清崎（Robert Kiyosaki）——美国企业家、投资家、作家——"《穷爸爸富爸爸》的作者"——"1997 年出版"——"全球销量超过'4000 万册'"——"被'翻译成'50 多种语言"。"清崎'说'他'写'这本书'的'目的'"——"'是'帮助'普通人'理解'财务知识'"——"'实现'财务自由'"。"他'还'写了'很多'书'"——"'如'《富爸爸穷孩子》'——'《富爸爸财务自由之路》'"——"'形成了'一个'庞大的'富爸爸'系列'"
