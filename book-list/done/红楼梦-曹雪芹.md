# 红楼梦

状态：已读

作者：曹雪芹

一句话评价：曹雪芹的"中国古典文学巅峰之作'"——"'贾宝玉与林黛玉的爱情悲剧'"——"'贾府兴衰折射封建社会没落'"——"'满纸荒唐言，一把辛酸泪'"——"'假作真时真亦假，无为有处有还无'"——"中国封建社会的百科全书'"——"古典小说最高成就'"

核心收获：
- **宝黛爱情悲剧**: 宝玉和黛玉的爱情是"全书的核心线索'"。"他们是'木石前盟'"——"'前世'是'神瑛侍者'和'绛珠仙草'"——"'今生'来'还泪'"。"但'他们的爱情'注定'悲剧'"——"'因为'不符合'封建礼教'"——"'黛玉'身体不好'"——"'家世衰落'"——"'不是'合适的'宝玉媳妇'"。"最终'黛玉'在'宝玉大婚之夜'含恨而终'"——"'焚稿断痴情'"——"'质本洁来还洁去'"
- **金陵十二钗**: 《红楼梦》塑造了"'金陵十二钗'"——"'十二个'命运各异的'女子'"。"'林黛玉'"——"'才华横溢'但'多愁多病'"；"'薛宝钗'"——"'端庄稳重'但'世故圆滑'"；"'王熙凤'"——"'精明强干'但'心狠手辣'"；"'史湘云'"——"'豪爽开朗'但'命运坎坷'"；"'贾元春'"——"'贵为妃子'但'深宫寂寞'"；"'贾探春'"——"'有才干'但'庶出身份'"；"'贾迎春'"——"'懦弱'被'虐待致死'"；"'贾惜春'"——"'看破红尘'出家为尼'"；"'李纨'"——"'守寡'一生'清心寡欲'"；"'秦可卿'"——"'早逝'身世成谜'"；"'巧姐'"——"'被卖'后被'救'"；"'妙玉'"——"'洁癖'最终'被劫'"
- **贾府兴衰**: 贾府的兴衰是"全书的另一条主线'"。"'贾府'曾是'赫赫扬扬'的'百年望族'"——"'宁国府'和'荣国府'两府'相连'"——"'仆从如云'"——"'富贵无双'"。"但'贾府'的'子孙'一代不如一代'"——"'贾赦'荒淫'"——"'贾珍'乱伦'"——"'贾琏'好色'"——"'贾宝玉'不愿读书'"。"最终'贾府'被'抄家'"——"'树倒猢狲散'"——"'落了个'白茫茫大地真干净'"
- **大观园**: 大观园是"《红楼梦》中最美的空间'"。"它是'为元妃省亲'而建的'皇家园林'"——"'后来'成为'宝玉和姐妹们'的'居住地'"。"大观园'是'一个'理想世界'"——"'少男少女'在这里'吟诗作对'"——"'无忧无虑'"。"但'大观园'最终'被现实摧毁'"——"'抄检大观园'"——"'晴雯被逐'"——"'芳官出家'"——"'姐妹们'一个一个'离开'"——"'大观园'从'乐园'变成'废墟'"
- **封建礼教的批判**: 曹雪芹"通过《红楼梦》批判了封建礼教'"。"'封建礼教'要求'女子'三从四德'"——"'但'黛玉'追求'爱情自由'"——"'晴雯'追求'人格尊严'"——"'鸳鸯'拒绝'做妾'"。"'封建礼教'要求'男子'读书做官'"——"'但'宝玉'厌恶'功名'"——"'他说'女儿是水做的骨肉'"——"'男子是泥做的骨肉'"。"曹雪芹'通过'宝玉的'叛逆'"——"'表达了对'封建礼教的'不满'"
- **人生如梦的主题**: 《红楼梦》的主题是"'人生如梦'"。"'开篇'的'甄士隐'经历'了'人生起伏'"——"'最终'看破红尘'"——"'随'跛足道人'出家'"。"'太虚幻境'的'对联'"——"'假作真时真亦假，无为有处有还无'"——"'点明了'全书的'主题'"。"'结尾'的'宝玉出家'"——"'他'经历了'荣华富贵'"——"'经历了'爱恨情仇'"——"'最终'发现'一切'都是'空'"——"'他'选择了'出家'"——"'完成了'从'入世'到'出世'的'转变'"
- **曹雪芹的背景**: 曹雪芹（约 1715-约 1763）——清代小说家——"《红楼梦》的作者'"。"他'出身'于'江宁织造'曹家'"——"'曹家'曾是'康熙皇帝的'宠臣'"——"'但'在'雍正年间'被'抄家'"——"'曹雪芹'从'富贵'跌入'贫困'"。"他'晚年'在北京'西山'写'《红楼梦》'"——"'披阅十载，增删五次'"——"'书未成，泪已尽而逝'"。"《红楼梦》'前 80 回'是'曹雪芹'写的'"——"'后 40 回'是'高鹗'续写的'"
