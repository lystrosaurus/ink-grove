# 人类群星闪耀时

状态：已读

作者：茨威格

一句话评价：茨威格用 14 个历史特写镜头，捕捉了人类历史上那些决定性的瞬间——一个念头、一次犹豫、一秒之差，改变了整个世界走向。

核心收获：
- 理解了"历史的关键时刻"往往不是长期趋势的结果，而是瞬间的决定——滑铁卢的一秒之差、拜占庭的一扇小门
- 历史的戏剧性在于：伟大人物在关键时刻的表现决定了无数人的命运
- 茨威格的叙事技巧：用文学的手法写历史，让历史人物像小说角色一样鲜活
