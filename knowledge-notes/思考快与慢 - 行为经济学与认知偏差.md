## 核心要点

### 要点 1：双系统理论

| 系统 | 特点 | 例子 |
|------|------|------|
| 系统 1 | 快速、直觉、自动 | 2+2=?,认人脸 |
| 系统 2 | 缓慢、理性、费力 | 17×24=?,填税表 |

**关键发现**: 我们多用系统 1，系统 2 很懒。

---

### 要点 2：常见偏差

| 偏差 | 说明 |
|------|------|
| 锚定效应 | 先入为主，第一个数字影响 |
| 可得性启发 | 容易想起的更重要 |
| 损失厌恶 | 损失比收益更痛（2 倍） |
| 过度自信 | 高估自己能力 |
| 框架效应 | 表述方式影响选择 |
| 确认偏差 | 寻找支持证据 |

---

### 要点 3：前景理论

| 概念 | 说明 |
|------|------|
| 参照点 | 感受取决于参照点 |
| 价值函数 | S 形，损失更陡 |
| 确定性效应 | 确定的更有价值 |
| 风险偏好 | 得时规避，失时寻求 |

---

### 要点 4：实践应用

| 领域 | 应用 |
|------|------|
| 投资 | 避免追涨杀跌 |
| 消费 | 识别营销陷阱 |
| 政策 | 助推设计 |
| 生活 | 重要决策慢思考 |

## 实践启发

### 个人决策
- 认识偏差
- 慢下来思考
- 重要事用系统 2
- 寻求外部意见

### 投资行为
- 避免损失厌恶
- 不听消息炒股
- 长期视角
- 理性分析

### 消费选择
- 识别锚定
- 比较真实价值
- 不被框架影响
- 延迟满足

### 经典名句
