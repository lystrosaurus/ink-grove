# 沧浪诗话 - 阅读笔记

核心主题：中国诗论与审美哲学

## 关键知识点
1. **妙悟说**：诗歌最高境界不是技巧学问而是直觉感悟——类似禅宗顿悟
2. **诗有别材**：诗歌需要才情和感受力——不是学问和道理
3. **尊唐抑宋**：对唐诗推崇对宋诗批评奠定后世诗学传统
4. **兴趣说**：好诗应该有"言有尽而意无穷"的韵味——可感不可触
5. **以禅喻诗**：用禅宗概念来解释诗歌创作和鉴赏

## 深度理解
《沧浪诗话》是宋代最重要的诗论著作。严羽以禅喻诗，提出"妙悟"说和"兴趣"说。"妙悟"是全书的核心概念：诗歌的最高境界不是技巧和学问，而是直觉和感悟。"诗有别材，非关书也；诗有别趣，非关理也"——诗歌需要的是才情和感受力，不是学问和道理。

## 实践启发
- 写诗和读诗都需要"妙悟"——不是分析出来的而是感受出来的
- 好诗的标准不是辞藻华丽而是"言有尽而意无穷"
