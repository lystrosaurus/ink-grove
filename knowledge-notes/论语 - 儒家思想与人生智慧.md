## 核心要点

### 要点 1：核心概念

| 概念 | 说明 | 名句 |
|------|------|------|
| 仁 | 爱人，核心道德 | 己所不欲，勿施于人 |
| 义 | 道义，正当行为 | 君子喻于义 |
| 礼 | 规范，社会秩序 | 克己复礼为仁 |
| 智 | 智慧，明辨是非 | 知之为知之 |
| 信 | 诚信，立身之本 | 人而无信不知其可 |

**五常**: 仁义礼智信，儒家核心价值。

---

### 要点 2：学习之道

| 原则 | 说明 |
|------|------|
| 学而时习之 | 学习并实践，是快乐的 |
| 温故而知新 | 复习旧知识，获得新理解 |
| 学思结合 | 学而不思则罔，思而不学则殆 |
| 三人行必有我师 | 虚心向他人学习 |
| 知之为知之 | 诚实面对自己的无知 |
