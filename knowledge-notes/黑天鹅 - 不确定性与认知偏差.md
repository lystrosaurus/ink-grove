## 核心要点

### 要点 1：黑天鹅定义

| 特征 | 说明 |
|------|------|
| 罕见 | 发生概率极低 |
| 影响巨大 | 改变历史或个人命运 |
| 事后可预测 | 事后找原因解释 |

**例子**: 911、金融危机、疫情、互联网

---

### 要点 2：认知偏差

| 偏差 | 说明 |
|------|------|
| 叙事谬误 | 用故事解释随机事件 |
| 确认偏差 | 只看支持自己观点的证据 |
| 沉默证据 | 看不见的更重要 |
| 游戏谬误 | 把现实当有规则的游戏 |

---

### 要点 3：应对策略

| 策略 | 说明 |
|------|------|
| 承认无知 | 不假装知道 |
| 不预测 | 预测不可靠 |
| 杠铃策略 | 保守端 + 激进端 |
| 反脆弱 | 从波动受益 |

---

### 要点 4：生活应用

| 领域 | 应用 |
|------|------|
| 投资 | 90% 保守 +10% 激进 |
| 职业 | 稳定工作 + 副业探索 |
| 学习 | 广泛阅读 + 深度钻研 |
| 健康 | 适度压力训练 |

---

## 实践启发

### 决策思维
- 承认不确定性
- 不依赖预测
- 考虑极端情况
- 建立保护

### 投资策略
- 杠铃配置
- 避免中间风险
- 利用正面黑天鹅
- 规避负面黑天鹅

### 职业发展
- 保持稳定收入
- 探索冒险机会
- 开放可能性
- 从波动受益
