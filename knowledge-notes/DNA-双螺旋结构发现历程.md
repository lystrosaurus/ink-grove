# DNA 双螺旋结构发现历程

## 背景

1950 年代初，科学界已知基因是遗传物质，但其化学结构未知。DNA（脱氧核糖核酸）的化学组成已知，但空间排列未知。

## 核心发现过程

1. **富兰克林的 X 射线衍射**：
  -罗莎琳德·富兰克林在伦敦国王学院通过 X 射线晶体学拍摄了高质量的 DNA 衍射照片
  -著名的"Photo 51"展示了清晰的 X 形图案，这是螺旋结构的典型特征

2. **沃森与克里克的模型构建**：
  -在剑桥大学卡文迪许实验室，沃森和克里克尝试搭建物理模型
  -他们意识到磷酸骨架在外部，碱基在内部
  -通过尝试不同配对方式，最终发现 A-T 和 G-C 的完美匹配

3. **查伽夫规则**：
  -埃尔文·查伽夫发现 DNA 中嘌呤和嘧啶的比例相等
  -这为碱基配对提供了化学基础

## 关键结论

- **双螺旋结构**：两条反向平行的多核苷酸链围绕同一轴心盘旋
- **碱基配对**：A 与 T 通过两个氢键配对，G 与 C 通过三个氢键配对
- **复制机制**：互补配对暗示了 DNA 半保留复制的可能机制

## 历史意义

1. 1953 年发表于《自然》杂志
2. 1962 年沃森、克里克、威尔金斯获诺贝尔生理学或医学奖
3. 开启了分子生物学时代
4. 为后来的基因工程、人类基因组计划奠定基础

## 争议与反思

- 富兰克林未获诺奖（她于 1958 年去世，诺奖不追授）
- 科学发现中的伦理问题：未经同意使用他人数据
- 女性在科学史上的贡献常被低估
