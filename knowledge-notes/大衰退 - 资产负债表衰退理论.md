## 核心理论框架

### 资产负债表衰退模型

```
┌─────────────────────────────────────────────────────────────────┐
│                  资产负债表衰退模型                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  正常经济周期：                                                  │
│  扩张 → 资产↑ → 投资↑ → 利润↑ → 继续扩张                           │
│                                                                 │
│  资产负债表衰退：                                                │
│  泡沫破裂 → 资产价格暴跌 → 资不抵债                                │
│              ↓                                                  │
│      企业目标：利润最大化 → 债务最小化                            │
│              ↓                                                  │
│      行为：借钱投资 → 存钱还债                                   │
│              ↓                                                  │
│      宏观：总需求持续萎缩 → 长期衰退                              │
│                                                                 │
│  政策应对：                                                      │
│  货币政策 → 失效（借贷方消失）                                    │
│  财政政策 → 必需（维持总需求）                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 核心命题

1. **目标函数转变命题**：企业从利润最大化转向债务最小化
2. **货币政策失效命题**：问题不是资金成本，是借贷方消失
3. **财政必需命题**：政府支出是防止崩溃的唯一工具
4. **长期性命题**：修复资产负债表需要时间

---

## 两种衰退对比

| 维度 | 正常衰退 | 资产负债表衰退 |
|------|---------|---------------|
| 触发原因 | 需求不足 | 资产泡沫破裂 |
| 企业状态 | 盈利但谨慎 | 资不抵债 |
| 企业目标 | 利润最大化 | 债务最小化 |
| 借贷行为 | 不愿借 | 不能借 |
| 货币政策 | 有效（降息刺激） | 失效（零利率无用） |
| 财政政策 | 辅助 | 必需 |
| 持续时间 | 6-18 个月 | 5-15 年 |

## 历史案例

### 日本 1990-2005

| 指标 | 1989 峰值 | 2005 谷底 | 变化 |
|------|---------|---------|------|
| 日经指数 | 38915 | 16000 | -59% |
| 商业地价 | 峰值 | 谷底 | -87% |
| 企业贷款 | 增长 | 负增长 | 持续还债 |
| 利率 | 6% | 0% | 降至零 |
| 财政刺激 | - | 占 GDP40% | 累计 |

**关键教训**：
- 财政刺激防止了大萧条
- 但被误解为"浪费"
- 恢复缓慢但避免了崩溃

### 1929 大萧条

| 指标 | 峰值 | 谷底 | 变化 |
|------|------|------|------|
| 股市 | 381 | 42 | -89% |
| GDP | 峰值 | 谷底 | -27% |
| 政策 | 紧缩 | 紧缩 | 错误应对 |
| 结果 | - | 大萧条 | 灾难 |

**对比教训**：
- 大萧条：财政紧缩→深度衰退
- 日本：财政扩张→避免崩溃

### 2008 年全球金融危机

**美国应对**：
- 快速财政刺激
- 大规模量化宽松
- 恢复相对较快

**与日本对比**：
- 行动更快
- 规模更大
- 承认问题性质

## 政策框架

### 识别指标

| 指标 | 正常衰退 | 资产负债表衰退 |
|------|---------|---------------|
| 资产价格 | 小幅下跌 | 大幅暴跌（>50%） |
| 企业贷款 | 下降 | 持续负增长 |
| 利率敏感度 | 高 | 低 |
| 私人储蓄 | 下降 | 上升（还债） |
| 银行准备金 | 正常 | 超额堆积 |

### 应对策略

**货币政策**：
- 认识局限性
- 配合财政政策
- 防止金融崩溃

**财政政策**：
- 规模足够大（占 GDP 5%+）
- 持续足够长（直到修复完成）
- 重点是维持需求，不是效率

**退出时机**：
- 私人部门借贷恢复
- 资产价格企稳
- 企业目标函数回归

### 与皮凯蒂整合

| 维度 | 皮凯蒂 | 辜朝明 |
|------|-------|-------|
| 资产价格 | 持续上升 (r>g) | 可能破裂 |
| 不平等 | 财富集中 | 衰退冲击底层 |

**整合**：
- 资产价格上升加剧不平等
- 资产价格破裂引发衰退
