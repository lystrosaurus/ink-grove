## 核心要点

### 要点 1：核心问题

| 问题 | 说明 |
|------|------|
| 为什么 | 欧洲人征服美洲 |
| 而非 | 美洲人征服欧洲 |
| 原因 | 环境差异，非种族 |

---

### 要点 2：欧亚优势链

| 环节 | 优势 | 结果 |
|------|------|------|
| 地理轴线 | 东西向 | 传播容易 |
| 可驯化物种 | 植物动物多 | 农业发展 |
| 食物生产 | 剩余粮食 | 人口增长 |
| 病菌免疫 | 与家畜同居 | 征服优势 |
| 技术积累 | 人口多交流多 | 枪炮钢铁 |

---

### 要点 3：三大支柱

| 支柱 | 说明 |
|------|------|
| 枪炮 | 军事技术优势 |
| 病菌 | 免疫优势，杀死 95% 美洲人 |
| 钢铁 | 工业技术优势 |

---

### 要点 4：地理决定

| 因素 | 影响 |
|------|------|
| 轴线方向 | 东西向传播快 |
| 物种分布 | 可驯化物种不均 |
| 大陆面积 | 人口规模 |
| 隔离程度 | 技术积累 |
