# 战国策 - 阅读笔记

核心主题：战国纵横术与外交智慧

## 关键知识点
1. **合纵连横**：弱国联合抗强（合纵）vs 强国拉拢弱国各个击破（连横）
2. **代表人物**：苏秦（合纵）、张仪（连横）、范雎（远交近攻）、邹忌、冯谖
3. **游说技巧**：善用比喻、寓言、数据分析，逻辑严密地打动君主
4. **经典寓言**：画蛇添足、狐假虎威、南辕北辙、鹬蚌相争等出自此书
5. **文献性质**：刘向整理汇编，有史实但也有文学加工和夸大

## 深度理解
《战国策》是中国历史上最精彩的"外交案例集"。战国时代七雄并立，没有一家能独大，这就创造了一个极其复杂的国际博弈环境。在这个环境中，"士"（策士、纵横家）成为了最重要的角色——他们没有军队、没有领土，只凭一张嘴和一颗大脑就能改变国家命运。苏秦一人佩六国相印，张仪以连横破合纵，这些都是中国历史上最具戏剧性的政治故事。《战国策》中的游说辞令堪称说服艺术的教科书：先分析形势让君主意识到危机，再指出利害关系让他感到紧迫，最后给出方案让他看到希望。这种"形势-利害-方案"的三段式结构，至今仍然是商业提案和政治演讲的标准模板。需要注意的是，《战国策》不完全是信史——策士们的故事在流传中被不断美化和夸大，但它所展现的智慧和语言艺术是真实的。

## 实践启发
- 说服他人的关键是先让对方意识到"这跟你有关"——从对方的利益出发
- 国际关系中的联盟和博弈逻辑从古至今没有变，只是工具和规模不同
- 好的语言需要好的结构——分析形势→指出利害→给出方案
