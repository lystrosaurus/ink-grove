## 核心理论框架

### 开发 - 测量 - 认知循环

```
假设 → 开发 (MVP) → 测量 (数据) → 认知 (学习) → 调整
  ↑                                              │
  └──────────────────────────────────────────────┘
```

### 关键概念

| 概念 | 说明 |
|------|------|
| MVP | 最小可行产品，快速验证假设 |
| 验证式学习 | 用实验验证商业假设 |
| 创新核算 | 衡量创新进展的指标 |
| 转型/坚持 | 根据数据调整方向 |

## 与传统创业对比

| 维度 | 传统创业 | 精益创业 |
|------|---------|---------|
| 规划 | 详细计划 | 假设驱动 |
| 开发 | 完美产品 | MVP |
| 发布 | 大规模 | 小测试 |
| 指标 | 收入利润 | 学习进展 |
| 决策 | 直觉 | 数据 |
