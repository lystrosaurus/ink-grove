## 核心理论框架

### r > g 不等式模型

```
┌─────────────────────────────────────────────────────────────────┐
│                      r > g 不平等模型                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  r = 资本收益率 (通常 4-5%)                                      │
│  g = 经济增长率 (通常 1-2%)                                      │
│                                                                 │
│  当 r > g 时：                                                   │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │  财富增长 > 收入增长                                       │    │
│  │  继承财富 > 劳动积累                                       │    │
│  │  财富集中度上升                                            │    │
│  │  世袭资本主义回归                                          │    │
│  └─────────────────────────────────────────────────────────┘    │
│                                                                 │
│  历史证据：                                                      │
│  1800-1914: r>g → 高度不平等                                    │
│  1950-1980: g>r → 短暂平等 (战争 + 高增长)                        │
│  1980-2020: r>g → 不平等回升                                    │
│                                                                 │
│  解决方案：全球累进资本税                                         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 核心命题

1. **r > g 常态命题**：资本收益率高于经济增长率是历史常态
2. **财富集中命题**：r > g 导致财富自动集中
3. **战争平等命题**：20 世纪平等是战争摧毁资本的结果
4. **世袭回归命题**：继承财富重要性正在回归 19 世纪水平

## 不平等的三种形式

### 1. 劳动收入不平等

**表现**：
- 顶层 1% 收入份额上升
- 高管薪酬暴涨（1980 年代后）
- 技术变革偏向高技能

**驱动因素**：
- 超级经理人薪酬
- 技术偏向性变革
- 工会力量下降

### 2. 资本收入不平等

**表现**：
- 资本所有权高度集中
- 顶层 10% 占有 60-90% 资本
- 资本的资本（基金、保险）

**驱动因素**：
- r > g 逻辑
- 高储蓄率差异
- 资本回报再投资

### 3. 继承财富不平等

**表现**：
- 继承财富占比回升
- 富二代优势扩大
- 代际流动下降

**驱动因素**：
- 财富积累效应
- 人口老龄化
- 房产增值

## 20 世纪平等化的真相

### 传统叙事 vs 皮凯蒂发现

| 维度 | 库兹涅茨曲线 | 皮凯蒂发现 |
|------|------------|-----------|
| 趋势 | 先升后降 | U 型曲线 |
| 原因 | 经济发展 | 战争冲击 |
| 持续性 | 自动收敛 | 需要干预 |
| 政策含义 | 等待增长 | 主动调节 |

### 战争冲击效应

```
一战 (1914-1918)
  ↓
资本摧毁 (欧洲资本存量↓50%)
  ↓
大萧条 (1929-1939)
  ↓
资产价格暴跌
  ↓
二战 (1939-1945)
  ↓
进一步摧毁 + 高额累进税
  ↓
1950s-1970s: 短暂平等时代
```

### 高税率的作用

| 时期 | 美国顶层税率 | 欧洲顶层税率 |
|------|------------|------------|
| 1930 年代前 | 20-30% | 20-30% |
| 1950s-1970s | 80-90% | 70-80% |
| 1980s 后 | 30-40% | 40-50% |

## 解决方案：全球资本税

### 设计方案

```
全球累进资本税：

资产规模          税率
────────────────────────
<100 万欧元         0%
100 万 -500 万       1%
500 万 -1000 万      2%
1000 万 -1 亿        5%
>1 亿               10%
```

### 预期效果

| 目标 | 机制 |
|------|------|
| 遏制财富集中 | 降低 r 净回报 |
| 减少避税激励 | 全球协调 |
| 增加政府收入 | 用于公共服务 |
| 促进机会平等 | 投资教育 |

### 实施困难

1. **国际合作**
   - 需要主要经济体协调
   - 避税天堂问题
   - 执行机制缺失

2. **政治阻力**
   - 富人游说
   - 意识形态反对
   - 民族主义兴起

3. **技术挑战**
   - 财富评估
   - 信息共享
   - 执行成本

## 与已读知识的对话

### 与林毅夫整合

| 维度 | 林毅夫 | 皮凯蒂 |
|------|-------|-------|
| 核心 | 增长 | 分配 |
| 不平等 | 会收敛 | 会上升 |
| 政府 | 促进增长 | 调节分配 |
| 时间 | 40 年中国 | 200 年多国 |

**整合**：增长创造财富，分配决定公平。两者需要平衡。

### 与温铁军整合

| 维度 | 温铁军 | 皮凯蒂 |
|------|-------|-------|
| 核心 | 危机转嫁 | 资本积累 |
| 层次 | 中国国内 | 全球比较 |
| 机制 | 制度性 | 经济性 |

**整合**：
```
不平等 = 危机转嫁 (温) + 资本积累 (皮)
国内维度            全球维度
```

### 与《八次危机》对话

- 温铁军：危机时成本向弱势转嫁
- 皮凯蒂：平时财富向资本集中
- 结果：不平等持续加剧
