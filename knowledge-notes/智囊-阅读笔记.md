# 智囊-阅读笔记

核心主题：中国智慧与谋略文化

## 关键知识点
1. **分类体系**：上智、明智、察智、胆智、术智——每种智慧有不同应用场景
2. **智谋文化**：从政治斗争到军事策略到商业谈判到日常生活
3. **案例式智慧**：一千多个故事——每个故事几百字配点评
4. **冯梦龙**：不仅是文学家（三言二拍）也是智慧故事搜集者
5. **历史真实性**：部分是史实部分是传说

## 深度理解
《智囊》是明代冯梦龙编纂的智慧故事集。收录了从先秦到明代的一千多个智谋故事。"智囊"的分类体系：上智（大智慧）、明智（明辨是非）、察智（洞察人心）、胆智（勇敢决断）、术智（策略技巧）。

## 实践启发
- 智慧不是抽象的——它体现在具体的问题解决中
- 从历史案例中学习智谋——前人的经验是最好的教材
