## 核心要点

### 要点 1：群体特征

| 特征 | 说明 |
|------|------|
| 冲动易变 | 情绪化不稳定 |
| 易受暗示 | 轻信谣言 |
| 情绪夸张 | 非黑即白 |
| 缺乏理性 | 不懂逻辑 |
| 需要领袖 | 盲从权威 |

---

### 要点 2：智力下降

| 现象 | 说明 |
|------|------|
| 个人变愚蠢 | 智力下降 |
| 情绪主导 | 理性丧失 |
| 从众心理 | 失去判断 |
| 责任分散 | 行为极端 |

---

### 要点 3：说服技巧

| 技巧 | 说明 |
|------|------|
| 断言 | 简洁有力，不讲推理 |
| 重复 | 不断重复，深入人心 |
| 传染 | 情绪传播，快速扩散 |
| 名望 | 成功带来信任 |

---

### 要点 4：现代启示

| 启示 | 说明 |
|------|------|
| 社交媒体 | 群体更易形成 |
| 谣言传播 | 更快更广 |
| 民粹主义 | 需要警惕 |
| 独立思考 | 更加重要 |
