# 三国志-史实与演义

核心主题：三国历史的原始记录、史笔与文学的区别、人物评价

## 关键知识点
1. **三书结构**：《魏书》《蜀书》《吴书》三部分，以魏为正统。陈寿是蜀人，蜀亡后入晋写史
2. **文风简练**：陈寿惜墨如金，一个人的一生可能只有几百字，但每句都有分量
3. **曹操的复杂性**：军事天才、文学爱好者、用人高手、多疑残忍。"治世之能臣，乱世之奸雄"
4. **诸葛亮的真实形象**：杰出政治家和外交家，军事才能一般。演义中的"神机妙算"多为虚构
5. **裴松之注解**：补充大量三国志正文没有的史料，价值不亚于正文本身
6. **史与演义的区别**：草船借箭、空城计、借东风、桃园结义——全是小说虚构

## 深度理解
三国志最大的价值是"第一手材料"。它是了解三国历史最接近真相的来源。但读三国志需要和裴注一起读，也需要和《三国演义》对照读——对照之后才能分辨什么是历史、什么是文学。

和《三国演义》《后汉书》《廿二史札记》对照阅读：三国志是原始材料，演义是文学加工，后汉书补充东汉背景，廿二史札记提供史学批评。

## 实践启发
- 读历史时区分"史实"和"演义"——很多"常识"其实是小说编的
- 评价历史人物时考虑史料来源的立场：陈寿以魏为正统，会影响叙事角度
- 裴注的注解方法值得学习：好的注释不是解释正文，是补充正文
