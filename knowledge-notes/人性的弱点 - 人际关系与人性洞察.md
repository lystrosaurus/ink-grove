## 核心要点

### 要点 1：人性核心

**核心洞察**: 人性的弱点是渴望被重视。

| 渴望 | 表现 |
|------|------|
| 被认可 | 希望被赞美 |
| 被理解 | 希望被倾听 |
| 被重视 | 希望名字被记住 |
| 被需要 | 希望有价值感 |

**卡耐基的观点**: 满足这个渴望，就能影响他人。

---

### 要点 2：待人技巧

| 技巧 | 说明 |
|------|------|
| 真诚赞美 | 发现优点，真诚表达 |
| 记住名字 | 名字是一个人最重要的声音 |
| 微笑 | 最简单的善意 |
| 谈论对方兴趣 | 建立连接的捷径 |
| 倾听 | 让对方多说 |

**关键**: 真诚是基础，不是技巧表演。

---

### 要点 3：说服他人

| 原则 | 说明 |
|------|------|
| 避免争论 | 争论没有赢家 |
| 尊重意见 | 不说"你错了" |
| 让对方说是 | 从小请求开始 |
| 站在对方角度 | 换位思考 |
| 激发高尚动机 |  appeal to noble motives |

**核心**: 不强迫，是引导。

---

### 要点 4：领导他人

| 方法 | 说明 |
|------|------|
| 赞美开始批评 | 三明治法 |
| 先说自己错误 | 降低防御 |
| 保全面子 | 尊重对方 |
| 让对方觉得主意是他的 | 激发 ownership |
| 激发成就感 | 让对方乐于做 |

**领导本质**: 不是命令，是影响。

---

## 实践启发

### 日常交往
- 每天真诚赞美至少一人
- 用心记对方名字
- 多倾听，少说
- 谈论对方感兴趣的事

### 工作沟通
- 避免争论
- 先找共同点
- 从对方角度看问题
- 保全面子

### 领导管理
- 用赞美开始批评
- 先说自己的错误
- 让下属觉得主意是他们的
- 激发成就感和价值感

### 核心心态
- 真诚关心他人
- 换位思考
- 满足被重视的渴望
- 不是操纵，是双赢
