## 核心要点

### 要点 1：核心框架

| 要素 | 说明 | 应用 |
|------|------|------|
| 观察 | 客观事实 | 不评判 |
| 感受 | 内心体验 | 真诚表达 |
| 需要 | 深层渴望 | 理解根源 |
| 请求 | 具体行动 | 建设性 |

---

### 要点 2：沟通原则

| 原则 | 说明 |
|------|------|
| 不评判 | 客观观察 |
| 不指责 | 理解需要 |
| 不比较 | 尊重差异 |
| 不命令 | 提出请求 |

---

### 要点 3：倾听技巧

| 技巧 | 说明 |
|------|------|
| 专注 | 全身心关注 |
| 同理 | 理解感受 |
| 不打断 | 耐心等待 |
| 不评判 | 开放接纳 |

---

### 要点 4：情绪管理

| 情绪 | 处理方式 |
|------|---------|
| 愤怒 | 识别需要 |
| 感激 | 具体表达 |
| 悲伤 | 接纳理解 |
| 恐惧 | 探寻根源 |
