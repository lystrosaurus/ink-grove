# 财务自由第一课-阅读笔记

核心主题：理财入门与资产配置

## 关键知识点
1. **财务自由定义**：被动收入 >= 生活支出——不需要为钱工作
2. **复利思维**：时间越长，复利效果越惊人——年化 10%，7 年翻倍
3. **指数基金定投**：最适合普通人的投资方式——不需要选股，长期获得市场平均回报
4. **资产配置**：分散到股票、债券、现金等不同资产，降低风险
5. **先存后花**：每月先存下一笔钱再消费，而不是花剩下的才存

## 深度理解
《财务自由第一课》是面向普通人的投资理财入门指南。帅张的核心贡献是把复杂的投资概念用通俗语言讲清楚。"财务自由"的定义极其简单：被动收入 >= 生活支出。这意味着你不需要为钱工作——你的资产产生的收益足够覆盖你的生活开销。复利思维是投资的第一原则——年化 10% 的收益，7 年后资产翻倍；年化 15%，5 年翻倍。时间越长，复利的效果越惊人。指数基金定投是最适合普通人的投资方式——你不需要懂选股，不需要看财报，只需要定期定额买入指数基金，长期坚持就能获得市场平均回报。资产配置比选股更重要——把资金分散到不同类型的资产中（股票、债券、现金），可以在不降低预期收益的情况下显著降低风险。"先存后花"是最简单的理财习惯——每月先存下一笔钱再消费，而不是花剩下的才存。这个简单的习惯改变了很多人的财务状况。

## 实践启发
- 每月发工资后先存一笔钱再消费——这个习惯比任何投资技巧都重要
- 不要试图选股——指数基金定投长期来看跑赢大部分专业基金经理
- 资产配置的核心是分散——不要把所有鸡蛋放在一个篮子里
