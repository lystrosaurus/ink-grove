## 核心框架整合

### 创新类型

```
                    创新
                     │
       ┌─────────────┴─────────────┐
       │                           │
  维持性创新                  颠覆性创新
  (Sustaining)              (Disruptive)
       │                           │
  渐进改进                    范式转变
  服务主流客户               服务边缘/新客户
  大企业优势                 创业公司机会
```

### 创业方法论

| 方法 | 核心 | 适用 |
|------|------|------|
| 精益创业 | MVP + 验证学习 | 不确定性高 |
| 四步创业法 | 客户开发四步 | B2B/B2C |
| 商业模式画布 | 九宫格分析 | 商业模式设计 |

### 创新者困境解决

**大企业困境**：
- 资源依赖主流客户
- 无法投资小市场
- 能力变成障碍

**解决方案**：
- 独立组织
- 不同成本结构
- 独立决策权

**创业公司优势**：
- 灵活快速
- 接受小市场
- 低成本结构

### 创业公司

2. 客户开发优先

4. 灵活调整方向
