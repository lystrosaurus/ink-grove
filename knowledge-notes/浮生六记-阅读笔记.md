# 浮生六记-阅读笔记

核心主题：文人生活与夫妻情深

## 关键知识点
1. **闲情价值**：在困顿生活中寻找美和乐趣——文人的生活智慧
2. **夫妻深情**：沈复与芸的爱情是中国文学史上最动人的夫妻故事之一
3. **四记现存**：六记原本失传两记——闺房记乐、闲情记趣、坎坷记愁、浪游记快
4. **平淡深情**：没有华丽辞藻但每一句都打动人心
5. **生活美学**：插花、园艺、品茶、赏月——日常中的美

## 深度理解
《浮生六记》是清代沈复的自传体散文。"闺房记乐"是全书最温暖的部分——沈复与芸的爱情是中国文学史上最动人的夫妻故事之一。"闲情记趣"讲的是在困顿生活中寻找美和乐趣——插花、园艺、品茶、赏月。"坎坷记愁"是最伤感的部分——芸的病逝、家庭的变故、生活的困顿。

## 实践启发
- 在平凡生活中寻找美——不需要昂贵的消费，只需要一颗感受美的心
- 珍惜身边的人——最珍贵的不是远方的风景而是身边的陪伴
