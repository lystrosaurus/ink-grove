## 核心主题

中国古典文学 3000 年的演进可以简化为一条从"集体之歌"到"个体之声"的线索: **集体吟唱(诗经) → 个人咏怀(唐诗) → 文人抒情(宋词) → 世俗叙事(明清小说)**。每个时代都有其独特的文体和情感表达方式，但共同的核心是"文以载道"——文学不仅仅是艺术，还是道德的承载者。

## 四大文体的演进

### 一、诗经：集体之声 (公元前 1000-600 年)

**《诗经》305 篇的核心**:
- **风**: 民间歌谣(最鲜活、最感人的部分)
- **雅**: 朝廷正乐(政治/祭祀)
- **颂**: 宗庙颂歌(王权/祖先)

**核心贡献**: 建立了中国文学的"比兴"传统——用自然景物来暗喻人的情感。
- "关关雎鸠，在河之洲" = 男女相恋的渴望
- "昔我往矣，杨柳依依" = 离别的不舍

**中国文学的源头**:
- 赋(描写) + 比(比喻) + 兴(联想) → 唐诗宋词的基础
- 现实主义的起点(十五国风 = 人民的声�音)

### 二、唐诗：个人之声 (618-907 年)

**《唐诗三百首》的三大传统**:

| 诗人 | 诗风 | 核心主题 | 代表作 |
|------|------|---------|--------|
| 李白 | 浪漫、飘逸、酒神 | 自由、幻想、不服规则 | 将进酒 |
| 杜甫 | 现实、沉郁、哀民生 | 人间疾苦、历史记录 | 春望 |
| 王维 | 空灵、禅意、画中有诗 | 自然、退隐、内心平静 | 山居秋暝 |

**唐诗的"黄金时代"为何出现在唐朝？**
- 科举以诗取士(学诗是入仕的门槛)
- 文化交流(丝绸之路带来了新的意象)
- 国力强盛(诗人有机会游历天下)

### 三、宋词：细腻之声 (960-1279 年)

**宋词的贡献**:
- 诗的"公共场合" → 词的"私人场合"(可以写私人情感)
- 长短句 → 更自由、更细腻、更接近"音乐"
- 苏轼开创了词的"士大夫化"(词可以写政治、人生、哲学)

**柳永 vs 苏轼**:
| 柳永 | 苏轼 |
|------|------|
| 婉约派代表 | 豪放派开创者 |
| 写给歌伎、写爱情 | 写人生、写历史、写哲学 |
| "今宵酒醒何处？杨柳岸晓风残月" | "大江东去，浪淘尽，千古风流人物" |

### 四、四大名著：世俗之声 (14-18 世纪)

**四部小说的不同关怀**:

| 小说 | 核心追问 | 文体 |
|------|---------|------|
| 三国演义 | 什么是"义"？(忠义/兄弟/天下) | 历史演义 |
| 水浒传 | 什么是"忠"？(对谁忠？) | 英雄传奇 |
| 西游记 | 什么是"心"？(心猿=杂念、修行=降心) | 神魔小说 |
| 红楼梦 | 什么是"情"？(爱、执著、觉悟) | 人情小说 |

**四大名著的共同点**:
- 都从一个"理想状态"(桃园/梁山/取经/大观园)开始，都以幻灭告终
- 都探索了"人如何在这个社会中安身立命"
- 都与《史记》的"纪传体"传统有所继承(写人、写命运)

### 《史记》的位置

《史记》是中国文学的"元文本"——不仅是一部历史书，还是文学的最高级。鲁迅称《史记》为"史家之绝唱，无韵之离骚"。司马迁的笔触从帝王将相到底层平民——为所有人立传。

## 核心脉络

```
诗经(集体之声)
    ↓ 风雅颂、赋比兴
唐诗(个人之声)  →  李白(浪漫) + 杜甫(现实) + 王维(空灵)
    ↓
宋词(细腻之声)  →  柳永(婉约) + 苏轼(豪放)
    ↓
史记(纪传之声)  →  从帝王到平民的"传记"
    ↓
四大名著(世俗之声)  →  三国(义) → 水浒(忠) → 西游(心) → 红楼(情)
```

## 与西方文学的对比

| 中国 | 西方 |
|------|------|
| 诗为主流文体(至宋) | 叙事/戏剧为主流(荷马→莎士比亚) |
| "诗言志"(诗=情感表达) | "诗=模仿"(亚里士多德诗学) |
| 含蓄、留白 | 直接、冲突 |
| 文学=道德载体(文以载道) | 文学=美/净化(亚里士多德净化论) |
