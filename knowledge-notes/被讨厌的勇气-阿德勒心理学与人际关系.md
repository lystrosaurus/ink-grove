## 核心要点

### 要点 1：阿德勒核心概念

| 概念 | 说明 |
|------|------|
| 目的论 | 不是过去决定现在，而是你选择现在 |
| 课题分离 | 分清什么是你的事，什么是别人的事 |
| 横向关系 | 平等合作，而非上下竞争 |
| 自我接纳 | 接受不能改变的，改变能改变的 |

---

### 要点 2：人际关系烦恼

| 烦恼来源 | 说明 |
|----------|------|
| 认可欲求 | 想要别人认可，活在期待里 |
| 比较竞争 | 纵向关系，自卑或优越 |
| 干涉他人 | 不分的课题界限 |
| 害怕被讨厌 | 失去自由 |

---

### 要点 3：幸福三要素

| 要素 | 说明 |
|------|------|
| 自我接纳 | 接纳不完美的自己 |
| 他者信赖 | 无条件相信他人 |
| 他者贡献 | 为他人做贡献，获得价值感 |

---

### 要点 4：自由与当下

| 概念 | 说明 |
|------|------|
| 自由 | 就是被别人讨厌 |
| 当下 | 人生是连续的刹那 |
| 意义 | 自己赋予的，不是普遍存在的 |

## 实践启发

### 人际关系
- 课题分离：不干涉别人
- 横向关系：平等对待
- 不寻求认可：活出自我
- 不怕被讨厌：获得自由

### 自我成长
- 自我接纳：接受不完美
- 目的论：主动选择
- 改变思维：从原因到目的
- 持续练习：改变习惯

### 工作态度
- 他者贡献：为他人创造价值
- 过程导向：认真做每一件事
- 不求认可：内在动机
- 合作思维：横向而非纵向

### 生活哲学
- 活在当下：每一刻都充实
- 没有目的地：过程就是全部
- 自己赋予意义：主动创造
- 认真生活：此时此地

### 经典名句
- "目的论：不是过去决定现在，而是你选择现在"
- "一切烦恼都来自人际关系"
- "课题分离：分清什么是你的事，什么是别人的事"
- "自由就是被别人讨厌"
- "认可欲求是枷锁"
- "自我接纳：接受不能改变的，改变能改变的"
- "人生是连续的刹那，活在当下"
