## 核心要点

### 要点 1：核心概念

| 概念 | 说明 |
|------|------|
| 乡土性 | 中国社会本质，农业文明基础 |
| 差序格局 | 以自己为中心向外推展 |
| 熟人社会 | 彼此熟悉，无需法律 |
| 礼治 | 传统规范治理 |
| 家族 | 事业组织，经济单位 |
| 无讼 | 不打官司，调解为主 |

---

### 要点 2：中西社会对比

| 方面 | 中国乡土 | 西方现代 |
|------|----------|----------|
| 社会结构 | 差序格局 | 团体格局 |
| 治理方式 | 礼治 | 法治 |
| 人际关系 | 熟人社会 | 陌生人社会 |
| 权力基础 | 长老统治 | 契约授权 |
| 信任基础 | 熟悉 | 制度 |

### 要点 3：社会特征

| 特征 | 说明 |
|------|------|
| 不流动 | 土地束缚，安土重迁 |
| 世代定居 | 生于斯死于斯 |
| 血缘纽带 | 家族为核心 |
| 礼俗治理 | 传统规范内化 |

### 要点 4：权力类型

| 权力 | 说明 |
