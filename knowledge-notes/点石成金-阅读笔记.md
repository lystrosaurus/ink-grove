# 点石成金 - 阅读笔记

核心主题：Web 可用性与用户体验

## 关键知识点
1. **别让我思考**：好的界面让用户一眼就知道怎么用——不需要说明书
2. **5 用户测试**：找 5 个用户测试比找 50 个用户问卷更有效
3. **用户非设计师**：不要用设计师的逻辑要求用户理解界面
4. **导航清晰**：用户不应该在任何页面迷路
5. **幽默风格**：用漫画和插图让枯燥设计原则变得生动

## 深度理解
《点石成金》是 Web 可用性领域的经典之作。史蒂夫·克鲁格的核心原则"别让我思考"是可用性设计的第一定律。用户不是设计师——不要用设计师的逻辑来要求用户理解界面。

## 实践启发
- 设计产品时先让一个完全不懂的人试用——如果他需要思考怎么用，就说明设计失败了
- 可用性测试不需要大样本——5 个用户就能发现 80% 的问题
