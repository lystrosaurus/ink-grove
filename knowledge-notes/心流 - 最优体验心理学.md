## 核心要点

### 要点 1：心流定义

| 特征 | 说明 |
|------|------|
| 完全沉浸 | 全神贯注于活动 |
| 忘记时间 | 时间感扭曲 |
| 忘记自我 | 自我意识消失 |
| 内在奖励 | 活动本身即奖励 |

---

### 要点 2：心流条件

| 条件 | 说明 |
|------|------|
| 清晰目标 | 知道要做什么 |
| 即时反馈 | 知道做得如何 |
| 挑战技能平衡 | 匹配当前能力 |
| 控制感 | 能影响结果 |

---

### 要点 3：心流通道

| 状态 | 原因 |
|------|------|
| 心流 | 挑战=技能 |
| 焦虑 | 挑战>技能 |
| 无聊 | 挑战<技能 |
| 放松 | 低挑战高技能 |

---

### 要点 4：幸福观

| 观点 | 说明 |
|------|------|
| 过程非状态 | 幸福是体验过程 |
| 主动非被动 | 需要主动创造 |
| 内在非外在 | 不依赖外部条件 |
| 秩序非混乱 | 内在秩序建立 |
