## 核心要点

### 要点 1：核心概念

| 概念 | 说明 |
|------|------|
| 稀缺 | 资源有限，必须选择 |
| 机会成本 | 放弃的最佳替代选择 |
| 边际分析 | 增量决策，边际收益=边际成本 |
| 激励 | 人会对激励做出反应 |
| 价格 | 信息载体，协调行动 |
| 市场 | 自发秩序，资源配置 |
| 比较优势 | 相对效率，贸易基础 |

---

### 要点 2：经济学思维

| 思维方式 | 说明 |
|------|------|
| 理性选择 | 成本收益分析 |
| 激励分析 | 预测行为反应 |
| 系统思考 | 整体与长期后果 |
|  unintended consequences | 非意图结果 |

---

### 要点 3：市场机制

| 机制 | 说明 |
|------|------|
| 供求定律 | 价格由供求决定 |
| 价格功能 | 传递信息，激励行为 |
| 资源配置 | 市场自发协调 |
| 政府干预 | 可能扭曲市场 |

---

### 要点 4：贸易理论

| 理论 | 说明 |
|------|------|
| 比较优势 | 相对效率差异 |
| 分工合作 | 专业化生产 |
| 互利共赢 | 贸易创造财富 |
| 开放繁荣 | 保护损害长期 |
