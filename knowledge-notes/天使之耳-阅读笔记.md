# 天使之耳 - 阅读笔记

核心主题：交通事故推理与短篇悬疑

## 关键知识点
1. **小切口风格**：从交通事故、日常纠纷中挖掘人性
2. **证词推理**：真相不在物证里，而在人的叙述中
3. **盲女证人**："天使之耳"篇中超常听力成为破案关键
4. **短篇节奏**：反转集中、不需要长篇铺垫

## 深度理解
《天使之耳》是东野圭吾短篇推理中较有特色的一部。它以交通事故为共同主题，把日常中容易被忽视的碰撞、剐蹭变成了悬疑的起点。"天使之耳"是全书最出色的篇章：一个盲女凭借超乎寻常的听力，记住了事故发生时每一辆车的声音细节。东野在这个故事中玩了一个经典的推理游戏——证人的证词是否可靠？看似完美的证词背后可能隐藏着怎样的真相？这种"不可靠叙述者"的技巧在推理文学中很常见，但东野用得比较自然。其他几个短篇质量不一，有些反转过于明显，但整体上保持了东野短篇的一贯水平——推理不是最精妙的，但人情味是足够的。

## 实践启发
- 好的短篇不需要宏大的设定，一个精准的切入点就够了
- 证词和叙述天然带有主观性——理解这一点有助于在冲突中保持客观
- 交通事故看似简单，但真相往往藏在细节中
