# 反时间管理 - 阅读笔记

核心主题：能量管理与注意力保护

## 关键知识点
1. **注意力 > 时间**：问题不在时间不够，而在注意力被分散
2. **精简任务**：减少不重要的事比增加技巧更有效
3. **能量管理**：按精力曲线安排工作，高峰时段做最重要的事
4. **"不"的力量**：拒绝是保护注意力的第一道防线
5. **深度工作**：设定不受打扰的专注时段，比碎片化高效更有价值

## 深度理解
《反时间管理》的核心贡献是它挑战了时间管理行业的基本假设。传统时间管理告诉你：把时间切块、列待办清单、用番茄钟、 multitasking。诺顿说这些都错了——问题不是你不会管理时间，而是你在用时间管理来合理化一个已经超载的生活。真正的解决方案不是做更多，而是做更少但更好。"能量管理"是全书最实用的概念：每个人一天中的精力不是均匀分布的，有人早上清醒，有人下午高效。把最重要的工作安排在精力高峰时段，比试图"管理"每一分钟有效得多。书中对"忙碌文化"的批判也很犀利：在现代社会，忙碌成了一种身份标志，但忙碌不等于有产出。很多人用忙碌来逃避真正重要但困难的事——因为回复 50 封邮件比写一份战略报告容易得多。

## 实践启发
- 每天早上问自己：今天哪一件事做完后会让其他事变得不那么重要
- 保护自己的注意力比保护时间更重要——关掉通知、减少会议
- 学会说"不"不是消极，而是对重要事情的积极承诺
