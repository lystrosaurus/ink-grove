# 七堂极简物理课-物理之美

核心主题：现代物理学七大主题、科学之美、宇宙与人的关系

## 关键知识点
1. **广义相对论**：空间和时间不是舞台背景，是会弯曲、扭曲的"演员"。物质告诉空间怎么弯曲，空间告诉物质怎么运动
2. **量子力学**：世界不连续，能量是一份一份的（量子）。微观世界只能用概率描述
3. **宇宙结构**：宇宙在膨胀，始于138亿年前的大爆炸。宇宙有年龄、有历史、有终点
4. **基本粒子**：标准模型中的夸克、电子、中微子、希格斯玻色子是万物的积木
5. **量子引力**：广义相对论（大尺度）和量子力学（小尺度）矛盾，是物理学最大未解之谜
6. **时间的本质**：时间可能不是基本的物理量，而是从更基本的规律中"涌现"出来的
7. **人与宇宙**：我们由星尘构成，大脑是宇宙理解自己的方式

## 深度理解
这本书最有价值的是"诗意"。罗韦利把物理学从公式和计算中解放出来，还原为人类对世界的好奇和惊叹。他证明了：好的科普不需要降低难度，只需要换一种语言。

和《大设计》《时间简史》对比：霍金偏重"知识传递"，罗韦利偏重"美的感受"。一个让你"知道"，一个让你"感受"。

## 实践启发
- 理解物理学的核心不是"做题"，是"感受世界的美"
- 在焦虑时想想"宇宙视角"——我们的烦恼在138亿年的宇宙尺度下微不足道
- 对科学保持敬畏：人类对宇宙的理解还极其有限
