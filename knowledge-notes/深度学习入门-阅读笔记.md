# 深度学习入门 - 阅读笔记

核心主题：深度学习原理与 Python 实现

## 关键知识点
1. **感知机**：最简单的神经网络单元，通过权重和偏置实现逻辑门
2. **神经网络**：多层感知机的堆叠，通过激活函数引入非线性
3. **反向传播**：用计算图理解链式法则，高效计算梯度
4. **CNN**：卷积层提取局部特征，池化层降维，适合图像任务
5. **RNN**：处理序列数据的网络结构，具有记忆功能
6. **过拟合对策**：权重衰减、Dropout、数据增强

## 深度理解
斋藤康毅的这本书是深度学习入门的标杆之作。它的核心理念是"理解一个东西最好的方式就是自己实现它"。通过从零开始用 Python 和 NumPy 编写每一层、每一个算法，读者不再是调用 API 的"调参侠"，而是真正理解了神经网络内部发生了什么。反向传播一章尤其精彩——传统的数学推导让人望而生畏，但用计算图的方式，每一步的前向传播和反向传播都变成了可视化的数据流。这种教学方式值得所有技术写作借鉴。不足之处是书中没有涉及 Transformer 等更新近的架构，但作为入门书，它完成的任务已经足够。

## 实践启发
- 学习新技术最好的方式是动手实现，而不是只看理论
- 理解底层原理后，使用高级框架时能做出更明智的选择
- 图解和可视化是理解复杂概念的最有效工具
