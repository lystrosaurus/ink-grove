# 用户体验要素-阅读笔记

核心主题：UX设计与产品架构

## 关键知识点
1. **五层模型**：战略层、范围层、结构层、框架层、表现层——从抽象到具体
2. **战略层**：用户需求和产品目标——没有清晰战略后面都是空中楼阁
3. **范围层**：把战略转化为具体功能——功能规格和内容需求
4. **结构层**：交互设计和信息架构——定义用户如何与产品互动
5. **框架层**：界面设计、导航设计、信息设计——把结构转化为页面布局

## 深度理解
《用户体验要素》是交互设计领域的方法论经典。Jesse James Garrett 提出的五层模型是 UX 领域最经典的框架。从抽象到具体，每一层都建立在前一层之上。战略层（用户需求和产品目标）是最基础的一层——没有清晰的战略，后面的设计都是空中楼阁。范围层（功能规格和内容需求）把战略转化为具体的功能。结构层（交互设计和信息架构）定义用户如何与产品互动。框架层（界面设计、导航设计、信息设计）把结构转化为具体的页面布局。表现层（视觉设计）是最具体的一层——颜色、字体、排版。每一层的决策都应该基于上一层——不能跳过战略层直接做视觉设计。

## 实践启发
- 做产品时先问战略层：用户需要什么？我们的目标是什么——不要跳过这一步
- 每一层都画出来——可视化能帮团队对齐认知
- 好的用户体验是系统工程的产物——不是靠灵光一闪，而是靠层层递进的设计
