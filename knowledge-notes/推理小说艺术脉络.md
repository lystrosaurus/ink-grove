## 推理小说艺术脉络 (福尔摩斯→阿加莎→东野圭吾)

**爱伦·坡 (1841)**: 创造"侦探小说"文体。莫格街凶杀案 = 世界第一部侦探小说。核心公式:谜题+逻辑推理+意外的结论。

**柯南·道尔 (1887-1927)**: 《福尔摩斯探案集》。将"科学推理"植入文学——法医学、化学、心理学。福尔摩斯+华生 = 侦探+叙述者——成为范式。

**阿加莎·克莉斯蒂 (1920-1976)**: 本格推理的女王。"谁干的？"(whodunit)为驱动。东方快车谋杀案(12共同犯罪)、罗杰疑案(叙事者=凶手)颠覆了范式。马普尔小姐+波洛 = 两种截然不同的"正义感"。

**松本清张 (1950s)**: 社会派推理。不再问"谁干的？"而是问"为什么会这样？"动机深入日本社会深层(阶级/腐败/战争创伤)。推理 = 社会批判的工具。

**东野圭吾 (1985-至今)**: 融合本格+社会派。嫌疑人X的献身(已知凶手，悬念在"如何掩盖")、白夜行(19年的罪与爱)。推理不再只是谜题，更是人性的深度剖析。数学/物理学的"极端逻辑"常常对比"人性的脆弱"。

### 发展脉络
```
爱伦·坡(谜题) → 柯南·道尔(科学) → 阿加莎(本格⇄颠覆) 
    → 松本清张(社会派) → 东野圭吾(人性)
```

## 阅读方法论整合 (如何阅读/深阅读/好好学习)

**艾德勒《如何阅读一本书》(1940)**: 四个层次——基础→检视→分析→主题阅读。核心:大部分书只需检视(15min全书概览)，少数需要分析(深度对话)，更少数需要主题阅读(跨书整合)。

**斋藤孝《深阅读》**: 日本版的"如何阅读"。注重"阅读的格斗"(与文本的正面搏斗)、"与作者的对话"(不是被动接受)、经典的价值(可读一百遍)。

**成甲《好好学习》**: 知识管理角度的阅读。临界知识(跨领域的底层规律)、反思日记(输出倒逼输入)、知识的复利(持续+关联+应用)、以教为学(费曼技巧)。

### 三层阅读体系
```
第一层: 检视阅读(15min概览) → 这本书值不值得读？
第二层: 分析阅读(深度笔记) → 这本书的核心是什么？
第三层: 主题阅读(跨书整合) → 多本书的共同主题是什么？
```

## 进化与人类学视角

**道金斯《自私的基因》**: 基因是进化的真正单位，个体是基因的"生存机器"。利他=基因的自利。

**赫拉利《人类简史》**: 三大革命(认知/农业/科学)。虚构故事创造了大规模人类合作。

**戴蒙德《枪炮、病菌与钢铁》**: 地理+技术+病菌决定了征服的方向。为什么是欧洲征服美洲而非反之？

**斯塔夫里阿诺斯《全球通史》**: 全球史观——文明在同一时间的平行发展和互动。

### 三层因果链
```
地理/气候(戴蒙德)
    → 技术/制度(斯塔夫里阿诺斯)
        → 文化/故事(赫拉利)
            → 基因/行为(道金斯)
```
