## 核心总结

《资本论》是马克思对资本主义生产方式的最系统的批判。核心概念"剩余价值"揭示：资本家的利润来自于对工人的"无偿劳动"的剥削。商品的价值由"社会必要劳动时间"决定，但工人拿到的只够"再生产劳动力"（工资），其余的劳动时间全部用于为资本家创造"剩余价值"。资本主义是一个"吞噬人"的机器——但也内含自我毁灭的种子（利润率下降趋势）。

---

## 深入理解

### 核心概念

**商品拜物教 (Commodity Fetishism)**
- 人们看到的是商品（物），看不到的是背后的社会关系（谁为这个东西付出了什么）
- 你看到的 iPhone 是"价值的魔幻结晶"，但你看不到的是中国工人的长时间高强度劳动
- "拜物教"：把人造的东西（商品）视为自然/天赐的——忘记了它来自人的劳动

**剩余价值 (Surplus Value)**
```
商品价值 = 不变资本(原材料/机器) + 可变资本(劳动力) + 剩余价值(利润)

工人工作 8 小时
├─ 前 4 小时 = 工资(足够的补偿)
└─ 后 4 小时 = 剩余价值(被资本家无偿拿走)
```

**资本主义悖论**：
- 资本家追求利润 → 投资机器替代工人 → 降低可变资本 → 降低剩余价值 → 降低利润率
- 这是资本主义"自我消灭"的内部逻辑

### 与亚当·斯密的对话

| 斯密《国富论》 | 马克思《资本论》 |
|-------------|----------------|
| 市场 = 自然和谐 (看不见的手) | 市场 = 阶级冲突 (剥削的机器) |
| 价值 = 自然价格 (劳动+资本+地租) | 价值 = 劳动 (只有劳动创造价值) |
| 分工提高效率 = 好消息 | 分工异化劳动者 = 坏消息 |
| 资本主义 = 最终的好事情 | 资本主义 = 终将被扬弃的阶段 |

### 影响与批评

**影响**: 马克思主义改变了 20 世纪的政治版图(苏联/中国/古巴等)

**批评 1**: 劳动价值论——为什么"只有劳动创造价值"？机器/技术/创意不也创造价值吗？

**批评 2**: 预测失败——资本主义没有崩溃，反而通过福利国家/技术创新/全球化"自我修复"

**批评 3**: "剩余价值"的计算困难——"社会必要劳动时间"难以量化
