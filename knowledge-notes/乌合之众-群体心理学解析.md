## 核心要点

### 要点 1：群体特征

| 特征 | 说明 |
|------|------|
| 冲动易变 | 情绪主导，反复无常 |
| 易受暗示 | 轻信传言，缺乏判断 |
| 情绪夸张 | 极端表达，非黑即白 |
| 偏执专横 | 不容异见，强制服从 |
| 道德两极 | 要么崇高要么卑劣 |

---

### 要点 2：个体在群体中变化

| 变化 | 说明 |
|------|------|
| 理性丧失 | 智力下降，情绪主导 |
| 责任分散 | 法不责众心理 |
| 从众行为 | 放弃独立思考 |
| 行为极端 | 做出个体不会做的事 |

---

### 要点 3：领袖操纵手法

| 手法 | 说明 |
|------|------|
| 断言 | 简单直接，不加论证 |
| 重复 | 不断重复，强化印象 |
| 传染 | 情绪扩散，快速传播 |
| 名望 | 权威背书，盲目信任 |

---

### 要点 4：群体信仰形式

| 特点 | 说明 |
|------|------|
| 宗教形式 | 盲目崇拜，不容质疑 |
| 非理性 | 情感驱动，逻辑无效 |
| 偶像崇拜 | 领袖神化，绝对服从 |
| 排他性 | 异端清除，内部统一 |
