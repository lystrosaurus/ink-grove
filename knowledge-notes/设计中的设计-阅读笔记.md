# 设计中的设计-阅读笔记

核心主题：东方美学与设计哲学

## 关键知识点
1. **还原而非添加**：设计不是添加，而是把事物还原到最本质的状态
2. **白的哲学**：不是空无一物，而是包含了所有可能性的"空"
3. **无印良品哲学**："无品牌"——没有 Logo、没有花哨包装，只有产品本身
4. **本来就是这样**：最高境界的设计让人感觉"理所当然"
5. **东方美学**：虚无、留白、侘寂——与西方"形式追随功能"不同的设计传统

## 深度理解
《设计中的设计》是原研哉的设计哲学随笔集。作为无印良品（MUJI）的艺术总监，原研哉的设计理念与无印良品的品牌哲学高度一致：设计不是添加，而是还原。把事物还原到最本质的状态，去掉一切不必要的装饰，留下的就是最纯粹的"设计"。"白"是全书最核心的概念。在西方语境中，"白"通常被理解为空无一物。但在东方美学中，"白"是包含了所有可能性的"空"——就像中国画中的留白，不是空白，而是给想象留下空间。无印良品的设计哲学就是"无品牌"——没有 Logo、没有花哨的包装，只有产品本身。但这种"无"不是"没有"，而是"不张扬"。设计的最高境界是让使用者感受到"本来就是这样"——不张扬、不抢眼，但恰到好处。原研哉的文字像散文诗——不是技术手册，而是设计哲学。这本书适合慢慢品味，不适合一口气读完。

## 实践启发
- 设计时先做减法——去掉不必要的元素，剩下的才是真正需要的
- 留白不是浪费空间，而是给使用者留下想象和呼吸的空间
- 最好的设计是让人感觉"理所当然"的设计——不显眼但不可或缺
