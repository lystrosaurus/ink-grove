## 核心要点

### 要点 1：多元思维模型

**核心**: 手里拿着锤子，看什么都是钉子。

| 学科 | 模型举例 |
|------|----------|
| 物理 | 临界点、平衡 |
| 化学 | 自催化 |
| 生物 | 进化、适应 |
| 心理 | 25 种倾向 |
| 经济 | 机会成本、激励 |

---

### 要点 2：人类误判心理学

**25 种倾向**（部分）:

| 倾向 | 说明 |
|------|------|
| 奖励惩罚 | 激励决定行为 |
| 避免不一致 | 不愿改变观念 |
| 社会认同 | 从众心理 |
| 禀赋效应 | 高估自己拥有的 |
| 对比错误 | 相对而非绝对 |
