## 核心要点

### 要点 1：乡土性

| 特点 | 说明 |
|------|------|
| 不流动 | 生于斯死于斯 |
| 熟人社会 | 彼此了解 |
| 礼治秩序 | 传统约束 |
| 无需诉讼 | 舆论约束 |

---

### 要点 2：差序格局

| 特征 | 说明 |
|------|------|
| 自我中心 | 以自己为中心 |
| 一圈圈推 | 愈推愈远 |
| 公私有别 | 家内家外不同 |
| 特殊主义 | 对不同人不同标准 |

**对比西方**:
| 中国 | 西方 |
|------|------|
| 差序格局 | 团体格局 |
| 特殊主义 | 普遍主义 |
| 关系本位 | 个人本位 |

### 要点 3：礼治秩序

|------|------|
